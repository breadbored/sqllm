How can I apply a path mode to only a portion of a longer path pattern?

--- AGENT ---

Enclose the portion of the path where the mode should apply in a subpath pattern and place the path mode before that subpath. The mode applies only within the subpath, and edges outside it are not constrained by the mode.

```googlesql
GRAPH FinGraph
MATCH
  (TRAIL (a1:Account)-[t1:Transfers]->{3}(a4:Account))
  -[t4:Transfers]->(a5:Account)
RETURN COUNT(1) as num_paths
```
