How do I calculate the difference in minutes between two TIME values?

--- AGENT ---

TIME_DIFF() returns the number of unit boundaries between two TIME values at a specified granularity. This example calculates the difference in MINUTE granularity, showing that 15:30:00 is 55 minutes after 14:35:00.

```googlesql
SELECT
  TIME "15:30:00" as first_time,
  TIME "14:35:00" as second_time,
  TIME_DIFF(TIME "15:30:00", TIME "14:35:00", MINUTE) as difference;
```
