--- USER ---

Show me a BigQuery query that extracts user question-answer interactions from a documentation event stream, normalizes the question text, and creates behavioral tags for both question-answer pairs and standalone questions.

--- AGENT ---

This query extracts documentation question events, normalizes the question text by removing punctuation and converting to lowercase, and produces two types of behavioral tags per interaction.

The `question_query` CTE processes events where `name = 'docs-questions'` by:
- Parsing the `action_meta` JSON field
- Using UNNEST to expand the answers object (which contains answer keys with 'true'/'false' values) into separate rows
- Filtering to only answers marked as 'true'
- Cleaning the question text by splitting on ' (', removing non-alphanumeric characters, replacing remaining special characters with hyphens, and lowercasing the result

The main query then produces two branches via UNION ALL:
1. **First branch**: Creates a tag combining the normalized question and answer key as `question[answer]`, tagged as 'behavior', with one row per answer
2. **Second branch**: Creates a tag using only the normalized question, also tagged as 'behavior', deduplicated with DISTINCT

Both branches preserve sub_session_id, url_path, app, and created timestamp from the source events.

```sql
WITH
question_query AS (
    SELECT
        e.id,
        e.session_id,
        e.sub_session_id,
        PARSE_JSON(e.action_meta) as meta,
            JSON_EXTRACT_SCALAR(
            e.action_meta,
            '$.question'
            ) as whole_question,
        LOWER(
            REPLACE(
            REGEXP_REPLACE(
                REGEXP_REPLACE(
                SPLIT(
                    JSON_EXTRACT_SCALAR(
                    e.action_meta,
                    '$.question'
                    ),
                    ' ('
                )[0], 
                r'[^a-zA-Z0-9\s]', 
                ''
                ),
                r'([^a-zA-Z0-9])',
                '-'
            ),
            '--',
            '-'
            )
        ) as question,
        a as answer,
        PARSE_JSON(JSON_EXTRACT(e.action_meta, '$.answers'))[a] as debug_answered,
        e.action_meta as debug,
        e.url_path,
        e.app,
        e.created
    FROM
        ${ref('base_buckets_events_docs_speedy')} as e, UNNEST(JSON_KEYS(PARSE_JSON(JSON_EXTRACT(e.action_meta, '$.answers')))) as a
    WHERE
        e.name = 'docs-questions'
    AND
        JSON_EXTRACT_SCALAR(PARSE_JSON(JSON_EXTRACT(e.action_meta, '$.answers'))[a], '$.') = 'true'
)

SELECT
    qq.sub_session_id as sub_session_id,
    CONCAT(qq.question, '[', qq.answer, ']') as tag,
    'behavior' as class,
    qq.url_path as url_path,
    qq.app as app,
    qq.created as dt
FROM
    question_query as qq

UNION ALL

SELECT DISTINCT
    qq.sub_session_id as sub_session_id,
    qq.question as tag,
    'behavior' as class,
    qq.url_path as url_path,
    qq.app as app,
    qq.created as dt
FROM
    question_query as qq
```
