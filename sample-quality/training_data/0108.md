--- USER ---

How does APPROX_TOP_SUM treat NULL values in the expression being aggregated?

--- AGENT ---

APPROX_TOP_SUM includes NULL as a regular value in the result set, ranking it alongside non-NULL values based on the sum of its associated weights.

```googlesql
SELECT APPROX_TOP_SUM(x, weight, 2) AS approx_top_sum FROM
UNNEST([STRUCT("apple" AS x, 0 AS weight), (NULL, 2)]);
```
