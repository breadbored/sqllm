How do I zip arrays with a transformation lambda without the named argument?

--- AGENT ---

Use `ARRAY_ZIP` with a lambda expression as the final argument without the `transformation =>` keyword. The function applies the lambda to corresponding elements from the input arrays and returns the computed results.

```googlesql
SELECT ARRAY_ZIP([1, 2], [3, 4], (e1, e2) -> (e1 + e2))
```
