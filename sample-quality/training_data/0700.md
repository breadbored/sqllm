--- USER ---

How do I filter, sort, and limit the results of a subquery before returning them?

--- AGENT ---

A `CALL` subquery can include `WHERE`, `ORDER BY`, and `LIMIT` clauses to filter, sort, and restrict its output before returning to the outer query. This example finds the three largest transfers over $50 for each person.

```googlesql
GRAPH FinGraph
MATCH (p:Person)
CALL (p) {
  MATCH (p)-[:Owns]->(a:Account)-[t:Transfers]->()
  WHERE t.amount > 50
  RETURN a.Id AS account_id, t.amount
  ORDER BY t.amount DESC
  LIMIT 3
}
RETURN p.name, account_id, amount
ORDER BY p.name, amount DESC;
```
