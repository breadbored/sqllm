--- USER ---

How do I create a function that extracts the last element from an array using array indexing?

--- AGENT ---

This query creates a reusable function with ANY TYPE parameter that uses ARRAY_LENGTH to determine the last position and ORDINAL to access it. The function works with arrays of any element type, demonstrating how templated parameters allow a single function definition to handle multiple data types.

```googlesql
CREATE TEMP FUNCTION LastArrayElement(arr ANY TYPE)
AS (
  arr[ORDINAL(ARRAY_LENGTH(arr))]
);

SELECT
  names[OFFSET(0)] AS first_name,
  LastArrayElement(names) AS last_name
FROM
  (
    SELECT ['Fred', 'McFeely', 'Rogers'] AS names UNION ALL
    SELECT ['Marie', 'Skłodowska', 'Curie']
  );
```
