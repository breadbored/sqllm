How do I apply explicit per-group contribution bounds with a privacy unit column?

--- AGENT ---

This query demonstrates explicit clamping with `DIFFERENTIAL_PRIVACY` using per-group contribution bounds, which are applied per grouping key. The bounds are uniformly applied to all aggregations and should reflect public information.

```googlesql
SELECT WITH DIFFERENTIAL_PRIVACY
  OPTIONS (
    epsilon = 1e20,
    delta = .01,
    privacy_unit_column=id
  )
  item,
  AVG(quantity, contribution_bounds_per_group=>(0,100)) AS average_quantity
FROM view_on_professors
GROUP BY item;
```
