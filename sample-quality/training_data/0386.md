What is the default type for a simple integer array literal without explicit type declaration?

--- AGENT ---

When you create an array without an explicit type declaration, GoogleSQL infers the type from the element values. For integer literals, the default type is ARRAY<INT64>.

```googlesql
SELECT [1, 2, 3] AS numbers;
```
