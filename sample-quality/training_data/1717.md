--- USER ---

Show me how to extract checkbox interaction events from an events table and tag them at different levels of granularity—basic event type, by action, and by action plus metadata answer name.

--- AGENT ---

This query extracts checkbox interaction events from the `base_buckets_events_docs_speedy` table and produces three different tagging/classification schemes in a single output using UNION ALL:

**Branch 1** filters for checkbox events and groups by sub_session_id and created timestamp, using MIN_BY to select the earliest app and url_path values for each group. All events in this branch are tagged simply as 'checkbox'.

**Branch 2** filters for checkbox events and groups by sub_session_id, action type, url_path, and created timestamp. It tags each event as 'checkbox_' concatenated with the action value, with app hardcoded to 'docs'. This creates action-specific tags.

**Branch 3** filters for checkbox events that contain an answer_name field in the action_meta JSON column, grouping by sub_session_id, action, url_path, and timestamp. It creates more granular tags as 'checkbox_' + action + '_' + answer_name, also with app hardcoded to 'docs'.

All three branches return rows with the same schema: sub_session_id, tag, class (always 'behavior'), app, url_path, and created timestamp (dt).

```sql
SELECT
    e.sub_session_id as sub_session_id,
    'checkbox' as tag,
    'behavior' as class,
    MIN_BY(e.app, e.created) as app,
    MIN_BY(e.url_path, e.created) as url_path,
    e.created as dt

FROM
    ${ref('base_buckets_events_docs_speedy')} as e 

WHERE
    e.name != 'page_view'
AND
    e.name != 'open_link'
AND
    e.name = 'checkbox'

GROUP BY
    sub_session_id,
    tag,
    dt

UNION ALL 

SELECT
  e.sub_session_id as sub_session_id,
  CONCAT('checkbox_', e.action) as tag,
  'behavior' as class,
  'docs' as app,
  e.url_path as url_path,
  e.created as dt
FROM
  ${ref('base_buckets_events_docs_speedy')} as e
WHERE
  e.name = 'checkbox'
GROUP BY
  sub_session_id,
  tag,
  url_path,
  dt

UNION ALL 

SELECT
  e.sub_session_id as sub_session_id,
  CONCAT('checkbox_', e.action, '_', JSON_EXTRACT_SCALAR(e.action_meta, '$.answer_name')) as tag,
  'behavior' as class,
  'docs' as app,
  e.url_path as url_path,
  e.created as dt
FROM
  ${ref('base_buckets_events_docs_speedy')} as e
WHERE
  e.name = 'checkbox'
AND
  JSON_EXTRACT_SCALAR(e.action_meta, '$.answer_name') IS NOT NULL
GROUP BY
  sub_session_id,
  tag,
  url_path,
  dt
```
