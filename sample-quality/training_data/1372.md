How do I decode a base64-encoded string?

--- AGENT ---

The FROM_BASE64 function converts a base64-encoded string into BYTES format using the standard RFC 4648 alphabet [A-Za-z0-9+/=].

```googlesql
SELECT FROM_BASE64('/+A=') AS byte_data;
```
