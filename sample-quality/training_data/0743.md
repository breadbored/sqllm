How do I sort results within the RETURN statement?

--- AGENT ---

The `ORDER BY` clause in the `RETURN` statement sorts results and is evaluated after aggregation. This example returns distinct paths sorted by hop count and then destination account ID.

```googlesql
GRAPH FinGraph
MATCH (src:Account {id: 7})-[e:Transfers]->{1, 3}(dst:Account)
RETURN DISTINCT ARRAY_LENGTH(e) AS hops, dst.id AS destination_account_id
ORDER BY hops, destination_account_id
```
