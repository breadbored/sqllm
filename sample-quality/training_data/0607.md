How can I inspect the complete type signature and all properties of a matched graph element node?

--- AGENT ---

This query matches a single `Person` node from the `FinGraph` property graph and uses the `TYPEOF` function to inspect the complete type definition of the matched graph element. The result reveals the node's label, all available property names, and their data types.

```googlesql
GRAPH FinGraph
MATCH (n:Person)
RETURN TYPEOF(n) AS t
LIMIT 1

/*----------------------------------------------+
 | t                                            |
 +----------------------------------------------+
 | GRAPH_NODE(FinGraph)<Id INT64, ..., DYNAMIC> |
 +---------------------------------------------*/
```
