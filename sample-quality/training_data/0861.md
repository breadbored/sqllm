How do I create a JSON object by grouping rows and aggregating key-value pairs?

--- AGENT ---

Use `JSON_OBJECT` with `ARRAY_AGG` to group rows and build a JSON object from aggregated keys and values.

```googlesql
WITH
  Fruits AS (
    SELECT 0 AS id, 'color' AS json_key, 'red' AS json_value UNION ALL
    SELECT 0, 'fruit', 'apple' UNION ALL
    SELECT 1, 'fruit', 'banana' UNION ALL
    SELECT 1, 'ripe', 'true'
  )
SELECT JSON_OBJECT(ARRAY_AGG(json_key), ARRAY_AGG(json_value)) AS json_data
FROM Fruits
GROUP BY id
```
