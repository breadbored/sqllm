Insert new concert rows by copying data from existing concert records where a specific singer performed.

--- AGENT ---

This query inserts new rows into the Concerts table by selecting data from existing concert records, allowing you to copy concert information while changing specific values like the concert date.

```googlesql
INSERT INTO Concerts
  (VenueId, SingerId, ConcertDate, BeginTime, EndTime, TicketPrices)
SELECT c.VenueId, c.SingerId, DATE "2015-06-01", c.BeginTime, c.EndTime, c.TicketPrices
  FROM Concerts c WHERE c.SingerId = 1;
```
