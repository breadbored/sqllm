--- USER ---

How do I return all columns from the working table and then select a subset in the next statement?

--- AGENT ---

The `RETURN *` clause returns all columns from the working table, and subsequent statements can select specific columns from those results. This example shows returning all columns including variables, then computing a new column from them.

```googlesql
GRAPH FinGraph
MATCH (p:Person)
LET a = 1, b = 2, c = 3
RETURN *

NEXT

RETURN p.name, (a + b + c) AS d
```
