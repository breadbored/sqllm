How do I convert a SQL value to JSON safely, handling unsupported types?

--- AGENT ---

The `SAFE_TO_JSON` function converts SQL values to JSON, producing a JSON null for unsupported field types instead of raising an error.

```googlesql
SELECT SAFE_TO_JSON(CAST(b'' AS UNSUPPORTED_TYPE)) as result;
```
