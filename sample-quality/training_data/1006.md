--- USER ---

How can I compute percentile values while treating NULL as a valid value that affects the interpolation?

--- AGENT ---

PERCENTILE_CONT with RESPECT NULLS includes NULL in the calculation; when interpolating between NULL and a non-NULL value, it returns the non-NULL value, and between two NULLs it returns NULL. This produces different results than ignoring nulls, with the minimum percentile returning NULL if the dataset contains nulls.

```googlesql
SELECT
  PERCENTILE_CONT(x, 0 RESPECT NULLS) OVER() AS min,
  PERCENTILE_CONT(x, 0.01 RESPECT NULLS) OVER() AS percentile1,
  PERCENTILE_CONT(x, 0.5 RESPECT NULLS) OVER() AS median,
  PERCENTILE_CONT(x, 0.9 RESPECT NULLS) OVER() AS percentile90,
  PERCENTILE_CONT(x, 1 RESPECT NULLS) OVER() AS max
FROM UNNEST([0, 3, NULL, 1, 2]) AS x LIMIT 1;
```
