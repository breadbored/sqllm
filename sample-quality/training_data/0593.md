How do I find a point at a specific fraction along a linestring geometry?

--- AGENT ---

`ST_LINEINTERPOLATEPOINT` returns a point at a specified fraction (0 to 1) along a linestring geography, where 0 is the start and 1 is the end.

```googlesql
WITH fractions AS (
    SELECT 0 AS fraction UNION ALL
    SELECT 0.5 UNION ALL
    SELECT 1 UNION ALL
    SELECT NULL
  )
SELECT
  fraction,
  ST_LINEINTERPOLATEPOINT(ST_GEOGFROMTEXT('LINESTRING(1 1, 5 5)'), fraction)
    AS point
FROM fractions
```
