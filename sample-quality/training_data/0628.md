How can I count the number of distinct transfer amounts along multiple paths in a graph?

--- AGENT ---

You can use horizontal aggregation with group variables to count and filter based on aggregate properties of a matched path pattern. The `COUNT(DISTINCT)` function counts unique values in the array of matched elements.

```googlesql
GRAPH FinGraph
MATCH (src:Account)-[e:Transfers]->{1, 3}(dst:Account)
WHERE src != dst
LET num_transfers = COUNT(e)
LET unique_amount_transfers = COUNT(DISTINCT e.amount)
FILTER unique_amount_transfers != num_transfers
RETURN src.id as src_id, num_transfers, unique_amount_transfers, dst.id AS destination_account_id
```
