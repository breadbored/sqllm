How do I use `CASE` to match a value against multiple alternatives?

--- AGENT ---

Use `CASE` with a search expression to evaluate the expression once, then test it for equality against each `WHEN` value until a match is found. The first matching branch executes.

```googlesql
DECLARE product_id INT64 DEFAULT 1;
CASE product_id
  WHEN 1 THEN
    SELECT CONCAT('Product one');
  WHEN 2 THEN
    SELECT CONCAT('Product two');
  ELSE
    SELECT CONCAT('Invalid product');
END CASE;
```
