--- USER ---

How do I chain multiple linear query statements and filter results from previous statements?

--- AGENT ---

The `NEXT` statement chains linear queries where each statement can filter or transform results from the previous one. This example chains three statements: first aggregating transfers per account, then finding owners, and finally filtering by transfer count.

```googlesql
GRAPH FinGraph
MATCH (:Account)-[:Transfers]->(account:Account)
RETURN account, COUNT(*) AS num_incoming_transfers
GROUP BY account

NEXT

MATCH (account:Account)<-[:Owns]-(owner:Person)
RETURN
  account.id AS account_id, owner.name AS owner_name,
  num_incoming_transfers

NEXT

FILTER num_incoming_transfers < 2
RETURN account_id, owner_name
```
