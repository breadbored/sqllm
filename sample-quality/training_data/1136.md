How do I see the schema and metadata of an intermediate query result?

--- AGENT ---

The `DESCRIBE` pipe operator returns textual metadata about the table at that point in the query, including column names, types, table aliases, and CTEs. It does not execute the input query—only analyzes it.

```googlesql
FROM
  (SELECT 1 AS col1, 2 AS col2) AS table1,
  (SELECT 'abc' AS col3, 1.0, 123 AS col5) AS table2
|> EXTEND col1 + col5 AS computed, 'anonymous column'
|> DESCRIBE;
```
