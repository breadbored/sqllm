How does EXTRACT handle intervals that have a negative sign applied to the hours, minutes, and seconds?

--- AGENT ---

When an interval has a negative sign preceding the time part, the negative sign distributes across the hours, minutes, and seconds components. EXTRACT returns the signed value for each component separately.

```googlesql
SELECT
  EXTRACT(HOUR FROM i) AS hour,
  EXTRACT(MINUTE FROM i) AS minute
FROM
  UNNEST([INTERVAL '10 -12:30' DAY TO MINUTE]) AS i
```
