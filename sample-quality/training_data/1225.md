How do I modify a protocol buffer map by adding new keys, updating existing ones, or removing entries?

--- AGENT ---

This query demonstrates PROTO_MODIFY_MAP operations: passing a NULL value for key 'A' removes it, passing a new value for key 'B' replaces it, and adding key 'C' inserts a new entry into the map.

```googlesql
SELECT
  PROTO_MODIFY_MAP(m.purchased, 'A', NULL, 'B', 4, 'C', 6) AS result_map
FROM
  (SELECT AS VALUE CAST("purchased { key: 'A' value: 2 } purchased { key: 'B' value: 3}" AS Item)) AS m;
```
