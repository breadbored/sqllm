--- USER ---

What error occurs when I assert that a value must exist in a dataset but that value is not actually present?

--- AGENT ---

The ASSERT statement evaluates the EXISTS condition, which is FALSE because the value 7919 does not appear in the dataset. When the assertion fails, GoogleSQL generates an error and displays the custom error message provided in the AS clause. This example demonstrates the failure case where the expected value is missing from the dataset.

```googlesql
ASSERT
  EXISTS(
    (SELECT X FROM UNNEST([7877, 7879, 7883, 7901, 7907]) AS X WHERE X = 7919))
AS 'Column X must contain the value 7919';
```
