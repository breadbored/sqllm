How do I convert a JSON array of mixed types to a string array?

--- AGENT ---

The `LAX_STRING_ARRAY` function converts a JSON array to `ARRAY<STRING>`, flexibly converting each element using LAX_STRING rules.

```googlesql
SELECT LAX_STRING_ARRAY(JSON '["purple", "10"]') AS result;
```
