Generate a BigQuery SQL query that extracts cookie banner interaction events and creates behavioral event tags, both aggregated and granular by interaction type.

--- AGENT ---

This query extracts cookie banner behavioral events from an events table and creates two categories of event tags per sub-session.

The **first branch** aggregates all cookie banner events (regardless of type) into a single "cookie_banner" tag per sub_session_id. It captures the earliest app and URL path from any cookie banner event in that session, along with the timestamp of the first such event.

The **second branch** creates type-specific tags (e.g., "cookie_banner_accept", "cookie_banner_required_only") for the same events, allowing finer-grained behavioral tracking. Each combination of sub_session_id and interaction type gets its own row, again capturing the earliest app, URL path, and timestamp from events of that specific type.

Both branches filter to cookie_banner events with type values of either 'accept' or 'required_only', and both use `MIN_BY()` to retrieve the app and url_path values from whichever record occurred first (by created timestamp). The union of these two branches produces a behavioral event tag table suitable for cohort analysis or behavioral feature engineering.

```sql
SELECT
    es.sub_session_id as sub_session_id,
    'cookie_banner' as tag,
    'behavior' as class,
    MIN_BY(es.app, es.created) as app,
    MIN_BY(es.url_path, es.created) as url_path,
    MIN(es.created) as dt

FROM
    ${ref("base_buckets_events_speedy")} as es
WHERE
    es.name = 'cookie_banner'
AND
    es.type IN (
        'accept',
        'required_only'
    )

GROUP BY
    sub_session_id,
    tag

UNION ALL

SELECT
    es.sub_session_id as sub_session_id,
    CONCAT('cookie_banner_', es.type) as tag,
    'behavior' as class,
    MIN_BY(es.app, es.created) as app,
    MIN_BY(es.url_path, es.created) as url_path,
    MIN(es.created) as dt

FROM
    ${ref("base_buckets_events_speedy")} as es
WHERE
    es.name = 'cookie_banner'
AND
    es.type IN (
        'accept',
        'required_only'
    )

GROUP BY
    sub_session_id,
    tag
```
