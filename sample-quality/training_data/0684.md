--- USER ---

How can I apply a path mode to one path pattern while ignoring it for another in the same MATCH clause?

--- AGENT ---

Path modes apply only to the specific path pattern they are attached to. When multiple path patterns are separated by commas, each one can have a different path mode, and the mode does not affect the other patterns.

```googlesql
GRAPH FinGraph
MATCH TRAIL (a1)-[t1]-(a2), (a2)-[t1]-(a3)
RETURN COUNT(1) as num_paths
```
