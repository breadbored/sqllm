How do I verify if a graph path is simple, meaning it has no repeated nodes except possibly at the endpoints?

--- AGENT ---

The `IS_SIMPLE` function returns `TRUE` if a graph path contains no repeated nodes, or only repeats its starting and ending nodes (forming a closed loop). This is stricter than checking for acyclicity and is useful for validating certain path structures.

```googlesql
GRAPH FinGraph
MATCH p=(a1:Account)-[t1:Transfers where t1.amount > 200]->
        (a2:Account)-[t2:Transfers where t2.amount > 200]->
        (a3:Account)-[t3:Transfers where t3.amount > 100]->(a4:Account)
RETURN
  IS_SIMPLE(p) AS is_simple_path,
  a1.id as a1_id, a2.id as a2_id, a3.id as a3_id, a4.id as a4_id
```
