How do I get a slice when the start offset is negative and the end is in the middle of the array?

--- AGENT ---

Use `ARRAY_SLICE` with a negative start offset (which counts from the end of the array) and a positive end offset. If the negative start position comes after the positive end position, the result is an empty array.

```googlesql
SELECT ARRAY_SLICE(['a', 'b', 'c', 'd', 'e'], -1, 3) AS result
```
