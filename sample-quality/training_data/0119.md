How do I verify that all values in one array are present in another array?

--- AGENT ---

Use `ARRAY_INCLUDES_ALL` to check whether every element from the search values array appears in the target array. The function returns TRUE only if all search values are found; otherwise it returns FALSE or NULL.

```googlesql
SELECT
  ARRAY_INCLUDES_ALL([1,2,3,4,5], [3,4,5]) AS a1,
  ARRAY_INCLUDES_ALL([1,2,3,4,5], [4,5,6]) AS a2;
```
