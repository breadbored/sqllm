How do I use implicit bounds with differential privacy based on a privacy unit column?

--- AGENT ---

This query demonstrates implicit clamping with `DIFFERENTIAL_PRIVACY`, where bounds are derived from the data itself rather than explicitly specified. The `privacy_unit_column` parameter is used to define the privacy unit.

```googlesql
SELECT WITH DIFFERENTIAL_PRIVACY
  OPTIONS (
    epsilon = 1e20,
    delta = .01,
    privacy_unit_column=id
  )
  item,
  AVG(quantity) average_quantity
FROM view_on_professors
GROUP BY item;
```
