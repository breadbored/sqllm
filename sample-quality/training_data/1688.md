--- USER ---

Provide me with a BigQuery query that tags user sub-sessions with a positive feedback behavior indicator, including the app and path context when that feedback was given.

--- AGENT ---

This query identifies user sub-sessions where positive feedback was recorded and creates a behavioral tag for those sessions. It joins events to feedback records on `user_id`, filters for positive sentiment responses (sentiment = 1), and validates temporal alignment—ensuring the feedback timestamp falls within the time window of that session's events via the HAVING clause. The output tags each matching sub-session with a "gave_positive_feedback" behavior classification, including the app and URL path context from the events table.

```sql
SELECT
  es.sub_session_id AS sub_session_id,
  'gave_positive_feedback' AS tag,
  'behavior' as class,
  es.app AS app,
  es.url_path AS url_path,
  f.created AS dt
FROM
  ${ref("base_buckets_events_app_speedy")} AS es
  JOIN ${ref("stg_feedback")} AS f ON es.user_id = f.user_id
WHERE
  f.response_sentiment = 1
GROUP BY
  es.sub_session_id,
  es.app,
  es.url_path,
  f.created
HAVING
  f.created BETWEEN MIN(es.created) AND MAX(es.created)
```
