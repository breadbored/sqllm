How can I assign a unique sequential number to each row in an ordered result?

--- AGENT ---

ROW_NUMBER() assigns a unique 1-based sequential number to each row in the window order, regardless of ties. This query numbers the rows sequentially, so even though there are duplicate values (like two 2's and three 10's), each row gets a unique integer from 1 to 7.

```googlesql
WITH Numbers AS
 (SELECT 1 as x
  UNION ALL SELECT 2
  UNION ALL SELECT 2
  UNION ALL SELECT 5
  UNION ALL SELECT 8
  UNION ALL SELECT 10
  UNION ALL SELECT 10
)
SELECT x,
  ROW_NUMBER() OVER (ORDER BY x) AS row_num
FROM Numbers
```
