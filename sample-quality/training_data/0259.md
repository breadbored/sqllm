How do I cast a string with timezone-aware timestamps to a RANGE<TIMESTAMP>?

--- AGENT ---

CAST to RANGE<TIMESTAMP> converts a string containing two timezone-aware timestamp values into a RANGE type with TIMESTAMP boundaries that preserve the timezone information.

```googlesql
SELECT CAST(
  '[2014-09-27 12:30:00+08, 2016-10-17 11:15:00+08)'
  AS RANGE<TIMESTAMP>) AS string_to_range
```
