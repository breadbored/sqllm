How do I extract just the time portion from a DATETIME value?

--- AGENT ---

TIME() can convert a DATETIME to a TIME value by extracting only the time component, discarding the date portion. The result is a TIME object containing only the hour, minute, and second.

```googlesql
SELECT TIME(DATETIME "2008-12-25 15:30:00.000000") AS time_dt;
```
