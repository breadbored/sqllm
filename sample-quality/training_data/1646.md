--- USER ---

Show me all user answers submitted in course assessments alongside the correct answers for each question, including metadata about the user, course, and when they answered.

--- AGENT ---

This query denormalizes user course assessment answer records by joining them with their supporting metadata. For each answer submission, it retrieves:
- The answering user, course, and assessment identifiers
- The question text and the answer option the user selected
- The correct answer option for that question
- The submission timestamp

The core table is `stg_user_course_assessment_answers`, which records each individual answer submission. The query left-joins to `stg_questions` to get question text, to `stg_question_options` to label the user's selected answer, and to `stg_courses` to provide course context.

A subquery aliased `con` reconstructs the correct answer by joining `stg_question_sets` (which holds the correct option ID per question) with `stg_question_options` to retrieve the correct option's name. The main query then joins this to retrieve the correct answer label.

Finally, the query left-joins to `stg_users` and filters with `WHERE u.test_user = 0`, excluding records from test accounts.

The output is a flat rowset suitable for analytics: one row per user answer, showing what was submitted and what should have been submitted.

```sql
SELECT
  ucaa.user_id,
  ucaa.course_id,
  c.container_id,
  c.name       AS course_name,
  ucaa.question_id,
  q.name       AS question,
  ucaa.answer_option_id,
  qo.name      AS user_answer,
  qs.correct_option_id,
  con.name     AS correct_answer,
  ucaa.created AS answer_date,
  ucaa.assessment_id
FROM
  ${ref("stg_user_course_assessment_answers")} AS ucaa
LEFT JOIN
  ${ref("stg_questions")} AS q
  ON
    ucaa.question_id = q.id
LEFT JOIN
  ${ref("stg_question_options")} AS qo
  ON
    ucaa.answer_option_id = qo.id
LEFT JOIN
  ${ref("stg_courses")} AS c
  ON
    ucaa.course_id = c.id
LEFT JOIN
  ${ref("stg_question_sets")} AS qs
  ON
    ucaa.question_id = qs.question_id
LEFT JOIN (
  SELECT
    qo.name,
    qs.correct_option_id
  FROM
    ${ref("stg_question_sets")} AS qs
  LEFT JOIN
    ${ref("stg_question_options")} AS qo
    ON
      qo.id = qs.correct_option_id
) AS con
  ON
    qs.correct_option_id = con.correct_option_id
LEFT JOIN
  ${ref("stg_users")} AS u
  ON
    ucaa.user_id = u.id
WHERE
  u.test_user = 0
```
