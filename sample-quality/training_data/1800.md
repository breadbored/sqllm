This query defines a BigQuery table-valued function (TVF) that accepts a client ID and container ID as parameters and returns a count of courses that have reached 95% or higher completion progress.

The function queries the `reporting_user_courses` table and uses a CASE statement to count only those rows where the `course_progress_percentage` column is greater than or equal to 0.95. The WHERE clause filters results by matching the input `clientId` parameter against values in the `client_id` field, and constrains results to a specific `containerId`.

The output is a single-column result set with the column name `courses_at_100`, containing the count of courses meeting the progress threshold.

```sql
CREATE OR REPLACE TABLE
  FUNCTION ${self()} (clientId INT64,
    containerId INT64) AS (
  SELECT
    COUNT(CASE
        WHEN course_progress_percentage >= 0.95 THEN 1
    END
      ) AS courses_at_100
  FROM
    ${ref("reporting_user_courses")} uc
  WHERE
    clientId IN (
    SELECT
      *
    FROM
      uc.client_id)
    AND container_id = containerId)
```
