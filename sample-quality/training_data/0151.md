How do I apply a transformation function to each element of an array?

--- AGENT ---

Use `ARRAY_TRANSFORM` with a lambda expression that takes an element and returns a transformed value. The output array has the same length as the input, with each element replaced by the result of the transformation.

```googlesql
SELECT
  ARRAY_TRANSFORM([1, 4, 3], e -> e + 1) AS a1,
  ARRAY_TRANSFORM([1, 4, 3], (e, i) -> e + i) AS a2;
```
