How can I find how many animals have a population within 1 unit of each animal's population?

--- AGENT ---

This query uses a RANGE-based window frame that creates a logical range around the order column. For each row, it counts animals whose population falls within 1 unit above or below the current row's population value, ordered by population.

```googlesql
SELECT animal, population, category, COUNT(*)
  OVER (
    ORDER BY population
    RANGE BETWEEN 1 PRECEDING AND 1 FOLLOWING
  ) AS similar_population
FROM Farm;
```
