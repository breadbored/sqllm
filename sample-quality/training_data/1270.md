How do I skip rows at the beginning of a result set?

--- AGENT ---

`OFFSET` skips a specified number of rows before applying the LIMIT, useful for pagination when combined with ORDER BY.

```googlesql
SELECT *
FROM UNNEST(ARRAY<STRING>['a', 'b', 'c', 'd', 'e']) AS letter
ORDER BY letter ASC LIMIT 3 OFFSET 1;
```
