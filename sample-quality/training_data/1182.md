Can I construct a protocol buffer by passing an array of key-value pairs directly to the NEW operator?

--- AGENT ---

This query constructs an `Item` protocol buffer using the `NEW` operator, passing an array of typeless structs that represent key-value pairs for the map field. Each tuple in the array becomes one entry in the `purchased` map field.

```googlesql
SELECT NEW Item([('A', 32), ('B', 9)] AS purchased)
```
