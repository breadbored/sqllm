How do I check if an array with one NULL value is distinct?

--- AGENT ---

Use `ARRAY_IS_DISTINCT` with an array containing NULL. Since a single NULL value is not repeated, the function returns TRUE for arrays containing one NULL and other unique elements.

```googlesql
SELECT ARRAY_IS_DISTINCT([1, 2, NULL]) AS is_distinct
```
