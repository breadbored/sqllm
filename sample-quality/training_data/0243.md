How do I use IF to return one value when a comparison is false and another when it is true?

--- AGENT ---

The IF function checks the boolean condition in its first argument and returns the corresponding result: the second argument if TRUE, the third argument if FALSE. Short-circuiting prevents evaluation of the unused branch.

```googlesql
SELECT
  30 AS A,
  20 AS B,
  IF(30 < 20, 'true', 'false') AS result
```
