How do I construct an array from multiple columns in a subquery?

--- AGENT ---

You can wrap column references in square brackets to create an array from multiple columns. The columns must have compatible types or will be coerced to a common supertype.

```googlesql
SELECT [a, b, c]
FROM
  (SELECT 5 AS a,
          37 AS b,
          406 AS c);
```
