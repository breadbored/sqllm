How do I find the smallest value in an array?

--- AGENT ---

Use `ARRAY_MIN` to retrieve the minimum non-NULL value from an array. The function returns NULL if the array is empty, contains only NULLs, or if the input array itself is NULL.

```googlesql
SELECT ARRAY_MIN([8, 37, NULL, 4, 55]) as min
```
