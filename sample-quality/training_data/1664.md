## Question
Show me a BigQuery SQL example that extracts behavioral markers from event data at the sub-session level.

--- AGENT ---

This query extracts behavioral events tagged as 'checkbox' actions at the sub-session level from an events table. It filters for events with the name `'expand_box'`, groups by sub-session ID, and captures structural information about the first occurrence of that event within each sub-session — specifically the app and url_path values associated with the earliest timestamp, along with that earliest timestamp itself.

The query uses `MIN_BY()` to retrieve the app and url_path values that correspond to the minimum created timestamp within each group, effectively identifying the originating context of the behavioral event. The hardcoded literals 'checkbox' and 'behavior' serve as classification tags in the output.

Here is the original SQL:

```sql
SELECT
  es.sub_session_id as sub_session_id,
  'checkbox' as tag,
  'behavior' as class,
  MIN_BY(es.app, es.created) as app,
  MIN_BY(es.url_path, es.created) as url_path,
  MIN(es.created) as dt

FROM
  ${ref("base_buckets_events_speedy")} as es

WHERE
  es.name = 'expand_box'

GROUP BY 
  sub_session_id,
  tag
```
