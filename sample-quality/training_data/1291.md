How do I verify that a specific date does not fall within a date range?

--- AGENT ---

RANGE_CONTAINS with a range and a scalar value returns FALSE if that date is not contained in the range. This query shows that April 1, 2023 is not contained in the range [2022-01-01, 2023-01-01) because 2023-04-01 is after the upper boundary.

```googlesql
SELECT RANGE_CONTAINS(
  RANGE<DATE> '[2022-01-01, 2023-01-01)',
  DATE '2023-04-01') AS results;
```
