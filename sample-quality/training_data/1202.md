How do I unnest a standard repeated field in a protocol buffer using a correlated join?

--- AGENT ---

This query uses a correlated INNER JOIN to unnest a standard repeated field without requiring an explicit UNNEST operator. The join `INNER JOIN T.proto_field.repeated_value AS value` creates one row for each element in the repeated field, allowing you to work with individual array elements.

```googlesql
WITH
  ExampleData AS (
    SELECT
      CAST("""
        record_key: 1
        repeated_value: 1
        repeated_value: 2
        repeated_value: 3
        [some.package.Extension.repeated_extension_value]: 4
        [some.package.Extension.repeated_extension_value]: 5
        [some.package.Extension.repeated_extension_value]: 6"""
        AS some.package.Example) AS proto_field
  )
SELECT
  T.proto_field.record_key,
  value
FROM
  ExampleData AS T
INNER JOIN
  T.proto_field.repeated_value AS value;
```
