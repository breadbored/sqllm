--- USER ---

How can I get a list of all usernames assigned to the red team as a single array value?

--- AGENT ---

An ARRAY subquery collects multiple rows from a subquery result into an array. If the subquery returns zero rows, it returns an empty array rather than NULL.

```googlesql
SELECT
  ARRAY(SELECT username FROM NPCs WHERE team = 'red') AS red;
```
