## Question
Show me a BigQuery query that retrieves all learning paths with their details and enriches each one with the list of clients associated with it.

--- AGENT ---

This query retrieves all learning paths along with their core metadata, enriching each record with an array of client IDs.

The work happens in two stages:

**Stage 1 (CTE)**: The `learning_path_clients` CTE aggregates the `clients_learning_paths` table, collecting all distinct non-null client IDs for each learning path into an array.

**Stage 2 (Main Query)**: The main SELECT pulls all columns from the `learning_paths` table. Several text columns (headline, description, accredible_group_id, credly_template_id, color, seo) are normalized to lowercase. The learning paths table is left-joined to the CTE on `learning_path_id`, so each path gets its associated client array. If a learning path has no row in the CTE (meaning no clients were found), the COALESCE function defaults the `client_ids` column to the array `[1]`.

The result is a denormalized, client-enriched view of all learning paths, with a fallback behavior that assigns client ID 1 to any path without explicit client associations.

```sql
WITH
learning_path_clients AS (
  SELECT
    clp.learning_path_id as learning_path_id,
    ARRAY_AGG(DISTINCT clp.client_id IGNORE NULLS) as client_ids
  FROM
    ${ref("clients_learning_paths")} as clp
  GROUP BY
    clp.learning_path_id
)
SELECT
  lp.id,
  lp.name,
  lp.created,
  lp.updated,
  lp.thumbnail,
  LOWER(lp.headline) AS headline,
  LOWER(lp.description) AS description,
  lp.published,
  lp.archived,
  lp.created_admin_id,
  lp.updated_admin_id,
  lp.has_been_published,
  lp.badge_image,
  LOWER(lp.accredible_group_id) AS accredible_group_id,
  lp.is_public,
  lp.lock_order,
  LOWER(lp.credly_template_id) AS credly_template_id,
  LOWER(lp.color) AS color,
  lp.discover_page,
  LOWER(lp.seo) AS seo,
  COALESCE(lpc.client_ids, [1]) AS client_ids
FROM
  ${ref("learning_paths")} as lp
LEFT JOIN
  learning_path_clients as lpc
ON
  lpc.learning_path_id = lp.id
```
