How do I calculate the sample standard deviation of a set of values?

--- AGENT ---

The `STDDEV_SAMP` function computes the sample (unbiased) standard deviation. NULL values are ignored, and the function requires at least two non-NULL values; otherwise it returns NULL.

```googlesql
SELECT STDDEV_SAMP(x) AS results FROM UNNEST([10, 14, 18]) AS x
```
