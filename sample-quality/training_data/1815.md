## Question
Show me a BigQuery SQL query that calculates the total time spent on courses for a specific client.

--- AGENT ---

This query creates a reusable table function that accepts a client ID as a parameter and returns a single aggregated metric. It queries the reporting_user_courses table, filtering for rows where the input client ID appears within the client_id array field (using UNNEST for array membership checking), and sums all time_on_course_minutes values across those matching rows to produce the total time on course.

```sql
CREATE OR REPLACE TABLE
  FUNCTION ${self()} (clientId INT64) AS (
  SELECT
    SUM(time_on_course_minutes) AS total_time_on_course_minutes
  FROM
    ${ref("reporting_user_courses")}
  WHERE
    clientId IN UNNEST(client_id))
```
