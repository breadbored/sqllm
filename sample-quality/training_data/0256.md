How do I parse multiple different interval string formats at once?

--- AGENT ---

CAST to INTERVAL accepts multiple string formats including the standard ISO 8601 duration format and the interval literal format 'Y-M D H:M:S.F', allowing you to convert various interval representations to a standardized INTERVAL type.

```googlesql
SELECT input, CAST(input AS INTERVAL) AS output
FROM UNNEST([
  '1-2 3 10:20:30.456',
  '1-2',
  '10:20:30',
  'P1Y2M3D',
  'PT10H20M30,456S'
]) input
```
