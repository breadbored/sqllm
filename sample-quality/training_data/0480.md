How do I parse a string in natural language date format into a DATETIME value?

--- AGENT ---

PARSE_DATETIME converts a string to a DATETIME value using a format string. The format string %A, %B %e, %Y matches a natural language format with full weekday name, full month name, day of month, and four-digit year.

```googlesql
SELECT PARSE_DATETIME('%A, %B %e, %Y','Wednesday, December 19, 2018')
  AS datetime;
```
