How do I find the first non-null value in a list of expressions that contains a null?

--- AGENT ---

COALESCE evaluates its arguments left to right and returns the first non-NULL value it encounters, skipping any NULL expressions. If the first argument is NULL, it continues to the next until a non-NULL value is found.

```googlesql
SELECT COALESCE(NULL, 'B', 'C') as result
```
