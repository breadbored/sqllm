How do I find common rows while matching columns by name instead of position with INTERSECT?

--- AGENT ---

The `BY NAME` modifier matches columns by name across queries instead of by position. This ensures that columns with the same name are correctly aligned even if they appear in different orders.

```googlesql
WITH
  NumbersTable AS (
    SELECT 1 AS one_digit, 10 AS two_digit
    UNION ALL
    SELECT 2, 20
    UNION ALL
    SELECT 3, 30
  )
SELECT one_digit, two_digit FROM NumbersTable
|> INTERSECT ALL BY NAME
    (SELECT 10 AS two_digit, 1 AS one_digit);
```
