How do I flatten an array field after accessing a specific element within a nested array?

--- AGENT ---

Combine array indexing with `FLATTEN()` and dot notation to first select a specific element from a nested array, then flatten that result. Use `SAFE_OFFSET()` to return null for out-of-range indexes rather than raising an error.

```googlesql
SELECT FLATTEN(items.sales.prices[SAFE_OFFSET(1)]) AS second_prices
FROM ItemsTable

/*------------------------+
 | second_prices          |
 +------------------------+
 | [50, NULL, NULL, NULL] |
 +------------------------*/
```
