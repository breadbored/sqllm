Update all rows in a table to reset a column to its default value.

--- AGENT ---

This query sets the SingerInfo column to its default value (NULL) for every row in the Singers table by using WHERE true to match all rows.

```googlesql
UPDATE Singers s
SET s.SingerInfo = DEFAULT
WHERE true;
```
