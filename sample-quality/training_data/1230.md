--- USER ---

Does TO_PROTO accept protocol buffer input, or only native GoogleSQL types?

--- AGENT ---

TO_PROTO accepts both GoogleSQL types and their protocol buffer equivalents. When passed a google.type.Date proto, it returns the same value unchanged.

```googlesql
SELECT TO_PROTO(
  new google.type.Date(
    2019 as year,
    10 as month,
    30 as day
  )
)
```
