--- USER ---

How do I find rows that are in the left query but not in multiple other queries using EXCEPT?

--- AGENT ---

The `EXCEPT DISTINCT` pipe operator can accept multiple comma-separated queries. The result contains rows from the input that do not appear in any of the specified queries.

```googlesql
SELECT * FROM UNNEST(ARRAY<INT64>[1, 2, 3, 3, 4]) AS number
|> EXCEPT DISTINCT
    (SELECT * FROM UNNEST(ARRAY<INT64>[1, 2]) AS number),
    (SELECT * FROM UNNEST(ARRAY<INT64>[1, 4]) AS number);
```
