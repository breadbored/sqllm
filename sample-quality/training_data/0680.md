--- USER ---

How can I retrieve multiple cheapest paths instead of just one?

--- AGENT ---

Use the CHEAPEST k prefix where k is the number of cheapest paths you want to return from each pair of endpoints. All qualifying paths will be returned in order of increasing cost.

```googlesql
GRAPH FinGraph
MATCH CHEAPEST 2 (a:Account)-[t:Transfers COST t.amount]->{1,3}(b:Account)
LET total_cost = sum(t.amount)
RETURN a.id AS a_id, b.id AS b_id, total_cost
```
