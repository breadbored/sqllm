How do I convert binary IP addresses back to their text representation?

--- AGENT ---

This query uses NET.IP_TO_STRING to convert binary addresses in network byte order back to human-readable STRING format, automatically detecting IPv4 (4 bytes) or IPv6 (16 bytes) based on input length. The FORMAT("%T", ...) displays the binary input in escaped notation for clarity.

```googlesql
SELECT FORMAT("%T", x) AS addr_bin, NET.IP_TO_STRING(x) AS ip_to_string
FROM UNNEST([
  b"0123",
  b"\x00\x00\x00\x00\x00\x00\x00\x00\x00\x00\x00\x00\x00\x00\x00\x01",
  b"0123456789@ABCDE",
  b"\x00\x00\x00\x00\x00\x00\x00\x00\x00\x00\xff\xff\xc0\x00\x02\x80"
]) AS x;
```
