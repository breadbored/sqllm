--- USER ---

How can I sort strings so that numeric substrings are treated as numbers rather than individual digits?

--- AGENT ---

The 'kn-true' numeric ordering extension enables numeric-aware string collation, where digits within strings are compared as numeric values. This query sorts strings with numeric content using the 'en-us-u-kn-true' collation, placing 'a1b' before 'a12b' because 1 is numerically less than 12.

```googlesql
SELECT *
FROM UNNEST([
  'a12b',
  'a1b'
]) AS ids
ORDER BY ids COLLATE 'en-us-u-kn-true'
```
