How do I convert an array of strings into a single delimited string?

--- AGENT ---

Use `ARRAY_TO_STRING(array, delimiter)` to concatenate array elements into a single string with the specified delimiter between elements. The result is a single string value.

```googlesql
WITH Words AS
  (SELECT ["Hello", "World"] AS greeting)
SELECT ARRAY_TO_STRING(greeting, " ") AS greetings
FROM Words;

/*-------------+
 | greetings   |
 +-------------+
 | Hello World |
 +-------------*/
```
