How do I aggregate results by grouping on a node in the RETURN statement?

--- AGENT ---

The `RETURN` statement supports `GROUP BY` to aggregate results per group. This example counts incoming transfers for each account by grouping on the account node.

```googlesql
GRAPH FinGraph
MATCH (:Account)-[:Transfers]->(account:Account)
RETURN account, COUNT(*) AS num_incoming_transfers
GROUP BY account

NEXT

MATCH (account:Account)<-[:Owns]-(owner:Person)
RETURN owner.name AS owner_name, num_incoming_transfers
ORDER BY num_incoming_transfers DESC
```
