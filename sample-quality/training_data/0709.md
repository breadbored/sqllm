What happens when a FOR statement unnests an empty array?

--- AGENT ---

When the `FOR` statement processes an empty array, it produces zero output rows from that input row. This example shows that matching three persons but then unnesting an empty array results in no output rows.

```googlesql
-- No rows produced
GRAPH FinGraph
MATCH (p:Person)
FOR element in [] WITH OFFSET AS off
RETURN p.name, element, off
```
