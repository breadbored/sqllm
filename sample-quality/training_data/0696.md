--- USER ---

How do I use an inline subquery to count the total number of accounts owned by each person?

--- AGENT ---

The `CALL` statement with an aggregation function in the subquery's `RETURN` statement allows you to compute aggregate values like counts per matched row. The subquery can include aggregation functions without requiring an explicit `GROUP BY` clause.

```googlesql
GRAPH FinGraph
MATCH (p:Person)
CALL (p) {
  MATCH (p)-[:Owns]->(a:Account)
  RETURN count(a) AS num_accounts
}
RETURN p.name, num_accounts
ORDER BY num_accounts DESC, p.name;
```
