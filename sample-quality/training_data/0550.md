How do I format a large negative number to display in scientific notation?

--- AGENT ---

The CAST function with the FORMAT clause converts a number to a formatted string. The EEEE format element outputs the number in scientific notation with an exponent, showing the mantissa and the power of 10.

```googlesql
SELECT CAST(-123456 AS STRING FORMAT '9.999EEEE') as a;
```
