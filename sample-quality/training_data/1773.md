This is a BigQuery table function that accepts a client ID and container ID as parameters. It computes a single metric: average time per user on a course, calculated as the sum of all time_on_course_minutes divided by the count of distinct users. The results are filtered to a specific client and container via the WHERE clause. The function queries the reporting_user_courses table and returns a single row with one column (avg_time_per_user).

Note: The WHERE clause contains an unusual pattern `clientId IN UNNEST(client_id)` that compares the input parameter against an unnested column reference—this pattern is uncommon and may indicate a schema detail not fully visible in the query text.

```sql
CREATE OR REPLACE TABLE
  FUNCTION ${self()} (clientId INT64,
    containerId INT64) AS (
  SELECT
    (SUM(time_on_course_minutes)/COUNT(DISTINCT user_id)) AS avg_time_per_user
  FROM
    ${ref("reporting_user_courses")}
  WHERE
    clientId IN UNNEST(client_id)
    AND container_id = containerId )
```
