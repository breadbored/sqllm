How do I calculate the sum of only distinct values?

--- AGENT ---

`SUM` with the `DISTINCT` keyword sums only unique values. This example adds distinct integers, counting each unique value once:

```googlesql
SELECT SUM(DISTINCT x) AS sum
FROM UNNEST([1, 2, 3, 4, 5, 4, 3, 2, 1]) AS x;
```
