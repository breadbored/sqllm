How do I specify a default protocol buffer value for a column?

--- AGENT ---

In a CREATE TABLE statement, you can use the DEFAULT clause with a protocol buffer constructor to set the default value.

```googlesql
CREATE TABLE TableName(
  proto_column googlesql.examples.music.Chart DEFAULT (
    { rank: 1 chart_name: '2' }))
```
