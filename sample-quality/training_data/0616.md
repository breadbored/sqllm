How can I extract all nodes from a matched graph path as an array?

--- AGENT ---

The `NODES` function returns an array of all nodes in a graph path while preserving their original order. You can then access individual nodes by index or extract properties from the entire array.

```googlesql
GRAPH FinGraph
MATCH p=(src:Account)-[t1:Transfers]->(mid:Account)-[t2:Transfers]->(dst:Account)
LET ns = NODES(p)
RETURN
  JSON_QUERY(TO_JSON(ns)[0], '$.labels') AS labels,
  JSON_QUERY(TO_JSON(ns)[0], '$.properties.nick_name') AS nick_name;
```
