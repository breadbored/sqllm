How do I parse a string in ISO date-time format into a DATETIME value?

--- AGENT ---

PARSE_DATETIME converts a string to a DATETIME value using a format string that matches the structure of the input string. This example parses an ISO 8601 formatted string with year-month-day and hour-minute-second components.

```googlesql
SELECT PARSE_DATETIME('%Y-%m-%d %H:%M:%S', '1998-10-18 13:45:55') AS datetime;
```
