How do I create an index on array elements along with their position offsets?

--- AGENT ---

The `WITH OFFSET` clause with `UNNEST` returns both the element value and its position in the array as separate index entries.

```googlesql
CREATE index i1 on Books UNNEST(BookList) WITH OFFSET (BookList, offset);
```
