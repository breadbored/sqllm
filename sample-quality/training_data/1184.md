How do I remove a key-value pair from a protocol buffer map field?

--- AGENT ---

This query removes a key-value pair from a map field by calling `PROTO_MODIFY_MAP` with the key and `NULL` as the value. When you pass `NULL` as the value for an existing key, that key-value pair is deleted from the map.

```googlesql
SELECT
  PROTO_MODIFY_MAP(m.purchased, 'A', NULL) AS result_map
FROM
  (SELECT AS VALUE CAST("purchased { key: 'A' value: 22 } purchased { key: 'B' value: 11 }" AS Item)) AS m;
```
