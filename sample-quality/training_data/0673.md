--- USER ---

How can I match nodes using an OR label expression to find nodes with any of multiple labels?

--- AGENT ---

Combine label names using the pipe character in a label expression. This matches nodes that have at least one of the specified labels.

```googlesql
GRAPH FinGraph
MATCH (n:Person|Account)
RETURN n.id
```
