How do I convert a JSON array of booleans to a SQL array?

--- AGENT ---

The `BOOL_ARRAY` function converts a JSON array of boolean values to a SQL `ARRAY<BOOL>` value. The function raises an error if the JSON value is not an array of booleans.

```googlesql
SELECT BOOL_ARRAY(JSON '[true, false]') AS vacancies;
```
