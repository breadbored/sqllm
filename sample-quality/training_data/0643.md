--- USER ---

How do you check whether an optionally-matched node has a specific label, handling the case where no match is found?

--- AGENT ---

The IS LABELED predicate can be used in the RETURN clause to test a node's label. When the node is NULL from an OPTIONAL MATCH that finds no result, the IS LABELED predicate returns NULL rather than FALSE.

```googlesql
GRAPH FinGraph
MATCH (a:Account {Id: 7})
OPTIONAL MATCH (a)-[:OWNS]->(b)
RETURN a.Id AS a_id, b.Id AS b_id, b IS LABELED Account AS b_is_account
```
