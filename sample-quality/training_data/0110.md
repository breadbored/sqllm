How do I create an array from individual values using a subquery?

--- AGENT ---

The `ARRAY` function constructs an array with one element per row returned by a subquery. The subquery must return exactly one column, and each value in that column becomes an element of the output array.

```googlesql
SELECT ARRAY
  (SELECT 1 UNION ALL
   SELECT 2 UNION ALL
   SELECT 3) AS new_array;
```
