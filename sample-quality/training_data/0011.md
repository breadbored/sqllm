How do I count non-null items per group in a view using differential privacy?

--- AGENT ---

This query counts non-NULL values in the item column across a view with differential privacy, applying contribution bounds per group before aggregation. Noise is added based on the epsilon parameter.

```googlesql
SELECT
  WITH DIFFERENTIAL_PRIVACY
    OPTIONS(epsilon=10, delta=.01, max_groups_contributed=1)
    item,
    COUNT(item, contribution_bounds_per_group=>(0, 100)) times_requested
FROM {{USERNAME}}.view_on_professors
GROUP BY item;
```
