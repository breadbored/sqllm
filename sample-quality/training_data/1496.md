--- USER ---

I want to select all players from the Players table using a subquery in the FROM clause.

--- AGENT ---

A table subquery in the FROM clause treats the subquery result as a table for the outer query. The subquery must be aliased so it can be referenced by the outer query.

```googlesql
SELECT results.username
FROM (SELECT * FROM Players) AS results;
```
