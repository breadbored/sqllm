Show me a BigQuery query that extracts the first occurrence of a specific event type within a user sub-session and captures its attributes.

--- AGENT ---

This query filters for 'open_course' events from the source table, groups them by sub-session, and for each sub-session extracts the earliest event's timestamp along with the app and URL path from that first event. The MIN_BY function retrieves the app and url_path values from the row with the minimum (earliest) created timestamp, while MIN captures that earliest timestamp directly. The result is labeled with a 'open_course' tag and 'behavior' class for categorization.

```sql
SELECT
    es.sub_session_id as sub_session_id,
    'open_course' as tag,
    'behavior' as class,
    MIN_BY(es.app, es.created) as app,
    MIN_BY(es.url_path, es.created) as url_path,
    MIN(es.created) as dt

FROM
    ${ref('base_buckets_events_docs_speedy')} as es 

WHERE
    es.name != 'page_view'
AND
    es.name != 'open_link'
AND
    es.name = 'open_course'

GROUP BY
    sub_session_id,
    tag
```
