--- USER ---

What happens when I assert that a dataset must contain more than 10 rows when it only has 6?

--- AGENT ---

The ASSERT statement evaluates the condition COUNT(*) > 10, which is FALSE because the dataset only contains 6 rows. When an assertion fails, GoogleSQL generates an error and displays the custom error message provided in the AS clause. This example demonstrates the failure case where the row count does not meet the required threshold.

```googlesql
ASSERT (
  (SELECT COUNT(*) > 10 FROM UNNEST([1, 2, 3, 4, 5, 6]))
) AS 'Table must contain more than 10 rows.';
```
