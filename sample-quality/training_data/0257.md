How do I cast a string representing a date range to a RANGE<DATE> type?

--- AGENT ---

CAST to RANGE<DATE> converts a string in range literal format to a RANGE type with DATE boundaries, where the brackets indicate whether boundaries are included ([) or excluded ()).

```googlesql
SELECT CAST(
  '[2020-01-01, 2020-01-02)'
  AS RANGE<DATE>) AS string_to_range
```
