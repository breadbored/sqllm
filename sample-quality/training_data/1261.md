How do I sort by a column using a specific language's collation rules?

--- AGENT ---

The `COLLATE` clause in ORDER BY specifies language-specific sorting rules for string columns. You can specify a language tag and optional collation attributes like case-insensitivity.

```googlesql
SELECT Place
FROM Locations
ORDER BY Place COLLATE "en_CA"
```
