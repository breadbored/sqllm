How do I extract the fractional part of a second with 2 digits of precision from a time value?

--- AGENT ---

The CAST function with the FORMAT clause converts a time to a string. The FF2 format element outputs 2 digits of the fractional part of the second, rounded as needed to fit the specified precision.

```googlesql
SELECT CAST(TIME '21:30:25.16' AS STRING FORMAT 'FF2') AS date_time_to_string;
```
