This is a BigQuery table function that accepts a `clientId` parameter and returns aggregate assessment statistics for courses belonging to that client.

The query uses a CTE to establish a baseline: `course_users` counts the distinct users enrolled in courses (where course_type = "course") for the given client from a reporting table. This count serves as the denominator for percentage calculations.

The main query then joins assessment records (`stg_user_course_assessments`) with supporting tables for users, courses, and client-course mappings. It aggregates across all assessments for the client to produce:
- **Participation metrics**: distinct users who started assessments, total assessment records started
- **Pass metrics**: count of individual assessments marked as passed, distinct users who passed at least one
- **Percentage rates**: start rate and pass rate expressed as percentages of the total enrolled `course_users`
- **Baseline**: the total `course_users` count from the CTE

The WHERE clause filters to only the assessments belonging to the specified `clientId`. The result is a single row of summary statistics per invocation.

```sql
CREATE OR REPLACE TABLE
  FUNCTION ${self()} (clientId INT64) AS (
  WITH
    course_users AS (
    SELECT
      COUNT(DISTINCT user_id) AS course_users
    FROM
      ${ref("reporting_user_courses")} uc
    WHERE
      clientId IN UNNEST(client_id)
      AND course_type = "course")
  SELECT
    COUNT(DISTINCT uca.user_id) AS assessment_users,
    ROUND(COUNT(DISTINCT uca.user_id) / cu.course_users * 100) start_percentage,
    COUNT(uca.id) AS assessments_started,
    COUNT(CASE
        WHEN uca.passed = 1 THEN 1
    END
      ) AS assessments_passed,
    COUNT(DISTINCT
      CASE
        WHEN uca.passed = 1 THEN uca.user_id
    END
      ) AS users_passed,
    ROUND(COUNT(DISTINCT
        CASE
          WHEN uca.passed = 1 THEN uca.user_id
      END
        ) / cu.course_users * 100) pass_percentage,
    cu.course_users
  FROM
    ${ref("stg_user_course_assessments")} uca
  JOIN
    ${ref("stg_users")} u
  ON
    u.id = uca.user_id
  JOIN
    ${ref("stg_courses")} c
  ON
    c.id = uca.course_id
  JOIN
    ${ref("stg_clients_course_containers")} ccc
  ON
    ccc.course_container_id = c.container_id
  CROSS JOIN
    course_users cu
  WHERE
    ccc.client_id = clientId
  GROUP BY
    cu.course_users)
```
