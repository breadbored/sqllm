How do I conditionally apply different transformations based on a constant condition?

--- AGENT ---

The `IF` pipe operator runs different subpipelines based on a constant boolean condition. If the condition is true, the first subpipeline runs; otherwise, `ELSEIF` or `ELSE` subpipelines run. Unselected cases are not validated.

```googlesql
FROM Produce
|> IF true THEN
  (
    |> AGGREGATE COUNT(*) AS total
  ) ELSE
  (
    |> SELECT *  -- Never evaluated
  )
|> AS Inventory;
```
