How do I use STRING_AGG as a window function to see cumulative concatenations?

--- AGENT ---

`STRING_AGG` with an `OVER` clause concatenates strings within a window frame for each row. This example shows cumulative concatenations as rows are processed in order of fruit length:

```googlesql
SELECT
  fruit,
  STRING_AGG(fruit, " & ") OVER (ORDER BY LENGTH(fruit)) AS string_agg
FROM UNNEST(["apple", NULL, "pear", "banana", "pear"]) AS fruit;
```
