How do I use the FOR statement followed by WITH if I need to fully qualify the offset clause?

--- AGENT ---

To use `WITH` after a `FOR` statement, you must complete the `WITH OFFSET` clause first; otherwise, `WITH` is interpreted as the start of an offset clause. This example shows the corrected syntax with `WITH OFFSET` followed by another `WITH` statement.

```googlesql
GRAPH FinGraph
FOR element in [1,2,3] WITH OFFSET
WITH element as col
RETURN col
ORDER BY col
```
