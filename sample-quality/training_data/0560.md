How do I get an array of S2 cell IDs that completely cover a geographic region, with control over cell size?

--- AGENT ---

`S2_COVERINGCELLIDS` returns an array of S2 cell IDs that cover the input geography, allowing you to constrain the minimum cell level to control precision.

```googlesql
WITH data AS (
  SELECT 1 AS id, ST_GEOGPOINT(-122, 47) AS geo
  UNION ALL
  SELECT 2 AS id, ST_GEOGFROMTEXT('POINT EMPTY') AS geo
  UNION ALL
  SELECT 3 AS id, ST_GEOGFROMTEXT('LINESTRING(-122.12 47.67, -122.19 47.69)') AS geo
)
SELECT id, S2_COVERINGCELLIDS(geo, min_level => 12) cells
FROM data;
```
