How do I create a Lua function that uses conditional logic to return different messages based on an integer input?

--- AGENT ---

This query creates a Lua UDF with an if-else conditional that returns different greetings based on whether the input is less than 12. The raw string literal preserves the Lua syntax and control flow statements without escaping issues.

```googlesql
CREATE TEMP FUNCTION CustomGreeting(i INT32)
RETURNS STRING
LANGUAGE lua
AS r'''
  if i < 12 then
    return 'Good Morning!'
  else
    return 'Good Evening!'
  end
''';

SELECT CustomGreeting(13) AS message;
```
