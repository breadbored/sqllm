--- USER ---

How can I compute the 1st percentile, median, 90th percentile, and the min/max values across a numeric column while ignoring NULL values?

--- AGENT ---

PERCENTILE_CONT computes percentile values using linear interpolation and by default ignores NULL values. Specifying percentile values of 0, 0.01, 0.5, 0.9, and 1 gives the minimum, first percentile, median, 90th percentile, and maximum respectively.

```googlesql
SELECT
  PERCENTILE_CONT(x, 0) OVER() AS min,
  PERCENTILE_CONT(x, 0.01) OVER() AS percentile1,
  PERCENTILE_CONT(x, 0.5) OVER() AS median,
  PERCENTILE_CONT(x, 0.9) OVER() AS percentile90,
  PERCENTILE_CONT(x, 1) OVER() AS max
FROM UNNEST([0, 3, NULL, 1, 2]) AS x LIMIT 1;
```
