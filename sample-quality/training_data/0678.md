--- USER ---

How can I match a variable from a quantified path pattern in a separate MATCH clause?

--- AGENT ---

Variables declared within a quantified path pattern with a search prefix cannot be reused outside the pattern in the same MATCH statement, but they can be matched in a separate MATCH clause to the same statement.

```googlesql
GRAPH FinGraph
MATCH ANY (a:Account {id: 20})->(mid:Account)->(a:Account)->(mid:Account)->(a:Account)
MATCH ALL (p:Person)->(mid)
RETURN p.name
```
