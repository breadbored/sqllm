How do I concatenate multiple arrays across rows into a single array?

--- AGENT ---

Use `ARRAY_CONCAT_AGG(array_column)` to concatenate array values from multiple rows into one flattened array. Elements maintain the order from their source arrays and source row order.

```googlesql
WITH Aggregates AS
  (SELECT [1,2] AS numbers
   UNION ALL SELECT [3,4] AS numbers
   UNION ALL SELECT [5, 6] AS numbers)
SELECT ARRAY_CONCAT_AGG(numbers) AS count_to_six_agg
FROM Aggregates;

/*--------------------------------------------------+
 | count_to_six_agg                                 |
 +--------------------------------------------------+
 | [1, 2, 3, 4, 5, 6]                               |
 +--------------------------------------------------*/
```
