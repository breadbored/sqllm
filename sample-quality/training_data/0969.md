How do I create a struct literal with named fields?

--- AGENT ---

A struct literal can be created using the STRUCT keyword with fields specified using `AS` for explicit names. This allows you to create a struct with named fields rather than positional access.

```googlesql
STRUCT(1 AS foo, 'abc' AS bar)
```
