--- USER ---

How do I compare two values and return TRUE only if they are distinct?

--- AGENT ---

The IS DISTINCT FROM operator returns TRUE if values are considered distinct (as in DISTINCT and GROUP BY), FALSE if they are the same, and never returns NULL—treating NULL values as distinct from non-NULL values.

```googlesql
SELECT 1 IS DISTINCT FROM 2
```
