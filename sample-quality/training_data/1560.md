How do I convert a TIMESTAMP to the number of microseconds since the Unix epoch?

--- AGENT ---

The `UNIX_MICROS` function returns the number of microseconds elapsed since 1970-01-01 00:00:00 UTC for a given timestamp. Higher precision (nanoseconds and picoseconds) is truncated by rounding down.

```googlesql
SELECT UNIX_MICROS(TIMESTAMP "2008-12-25 15:30:00+00") AS micros;
```
