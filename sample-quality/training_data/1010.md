How do I convert IPv4 and IPv6 address strings into their binary network byte order representation?

--- AGENT ---

This query converts string-formatted IP addresses (both IPv4 dotted-quad and IPv6 colon-separated formats) into binary BYTES format in network byte order using NET.IP_FROM_STRING. The FORMAT("%T", ...) wrapper displays the binary output in escaped byte notation.

```googlesql
SELECT
  addr_str, FORMAT("%T", NET.IP_FROM_STRING(addr_str)) AS ip_from_string
FROM UNNEST([
  '48.49.50.51',
  '::1',
  '3031:3233:3435:3637:3839:4041:4243:4445',
  '::ffff:192.0.2.128'
]) AS addr_str;
```
