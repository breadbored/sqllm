--- USER ---

What is the result of creating geography objects with empty geometry, such as POINT EMPTY?

--- AGENT ---

Both POINT EMPTY and GEOMETRYCOLLECTION EMPTY normalize to the same result: an empty geography represented as GEOMETRYCOLLECTION EMPTY. GoogleSQL simplifies geometry to the simplest type possible.

```googlesql
SELECT
  ST_GEOGFROMTEXT('POINT EMPTY') AS a,
  ST_GEOGFROMTEXT('GEOMETRYCOLLECTION EMPTY') AS b
```
