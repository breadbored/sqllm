--- USER ---

For a given client and course container, provide a summary of all sentiment questions with their answer counts, publication dates, and whether each question's course version is currently live.

--- AGENT ---

This query returns aggregated sentiment question metrics for a specific client and container. It joins five tables to correlate answer submissions with question metadata, course assignments, and client relationships. For each unique question name, it calculates the total number of answers received, the count of distinct course versions containing that question, and the earliest question publication date (formatted as MMM-DD-YYYY). It also derives a boolean flag indicating whether the question's associated course is the "live" (currently active) version — determined by checking if the course ID matches the maximum ID within that container.

The inline subquery on the courses table uses a window function to identify the latest course by ID per container, which enables the comparison. The query filters to only published courses within the specified container and client, and returns results ordered by sentiment question ID.

```sql
CREATE OR REPLACE TABLE
  FUNCTION ${self()} (clientId INT64,
    containerId INT64) AS (
  SELECT
    MAX(sq.id) AS sentiment_question_id,
    sq.name AS sentiment_question,
    COUNT(ucsqa.id) AS answers,
    FORMAT_DATE('%b-%d-%Y', MIN(sq.created)) AS publish_date,
    COUNT(DISTINCT ucsqa.course_id) AS versions,
    (MAX(c.id) = MAX(c.course_max_id)) AS live
  FROM
    ${ref("stg_user_course_sentiment_question_answers")} AS ucsqa
  JOIN
    ${ref("stg_sentiment_questions")} sq
  ON
    sq.id = ucsqa.sentiment_question_id
  JOIN (
    SELECT
      id,
      container_id,
      MAX(id) OVER (PARTITION BY container_id) AS course_max_id
    FROM
      ${ref("stg_courses")}
    WHERE
      container_id = containerId
      AND has_been_published = 1) c
  ON
    c.id = ucsqa.course_id
  JOIN
    ${ref("stg_users")} u
  ON
    u.id = ucsqa.user_id
  JOIN
    ${ref("stg_clients_course_containers")} ccc
  ON
    ccc.course_container_id = c.container_id
  WHERE
    client_id = clientId
  GROUP BY
    c.container_id,
    -- sq.id,
    sq.name
  ORDER BY
    sentiment_question_id)
```
