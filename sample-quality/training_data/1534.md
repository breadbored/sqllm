How do I format a timestamp as ISO 8601 date-time with the time zone abbreviation?

--- AGENT ---

The `FORMAT_TIMESTAMP` function can produce ISO 8601 formatted output by using elements like `%Y-%m-%d`, `%H:%M:%S`, and `%Z` for timezone abbreviation. This creates a standardized timestamp representation with timezone information.

```googlesql
SELECT FORMAT_TIMESTAMP("%Y-%m-%dT%H:%M:%S%Z", TIMESTAMP "2050-12-25 15:30:55", "UTC")
  AS formatted;
```
