Generate a BigQuery query that tags user behavioral events by joining app session events with course enrollment data, creating separate tags for general course starts and course-type-specific events, distinguishing between first-time enrollments and repeat courses.

--- AGENT ---

This query joins app session events with course enrollment and course metadata to produce four overlapping event streams with different behavioral classifications. 

**Branch 1** tags sessions with the course type, classifying them as 'content' events, and includes all course enrollments where the enrollment timestamp falls within the event time window of that session.

**Branch 2** duplicates branch 1's structure but applies a fixed 'started_a_course_general' tag classified as 'behavior' instead of course-type-specific 'content'.

**Branch 3** filters to only a user's first course enrollment (using `ROW_NUMBER() = 1`), tags it as 'first_' + course type, and classifies it as 'content'.

**Branch 4** applies the same first-enrollment filter as branch 3 but tags it 'started_first_course_general' and classifies it as 'behavior'.

All branches use `MIN_BY` to extract the earliest app and URL path from events within each session, and apply a HAVING filter to ensure the enrollment date falls within the observed event timestamps for that session. Each row includes the session identifier, tag, classification, earliest app/path, and enrollment timestamp.

```sql
SELECT
  es.sub_session_id AS sub_session_id,
  c.course_type AS tag,
  'content' as class,
  MIN_BY (es.app, es.created) AS app,
  MIN_BY (es.url_path, es.created) AS url_path,
  uc.created AS dt
FROM
  ${ref("base_buckets_events_app_speedy")} AS es
  JOIN ${ref("stg_user_courses")} AS uc ON uc.user_id = es.user_id
  JOIN ${ref("stg_courses")} AS c ON uc.course_id = c.id
GROUP BY
  sub_session_id,
  uc.created,
  tag
HAVING
  uc.created BETWEEN MIN(es.created) AND MAX(es.created)

UNION ALL

SELECT
  es.sub_session_id AS sub_session_id,
  CONCAT('started_a_course_general') AS tag,
  'behavior' as class,
  MIN_BY (es.app, es.created) AS app,
  MIN_BY (es.url_path, es.created) AS url_path,
  uc.created AS dt
FROM
  ${ref("base_buckets_events_app_speedy")} AS es
  JOIN ${ref("stg_user_courses")} AS uc ON uc.user_id = es.user_id
  JOIN ${ref("stg_courses")} AS c ON uc.course_id = c.id
GROUP BY
  sub_session_id,
  uc.created,
  tag
HAVING
  uc.created BETWEEN MIN(es.created) AND MAX(es.created)

UNION ALL

SELECT
  es.sub_session_id AS sub_session_id,
  CONCAT('first_', c.course_type) AS tag,
  'content' as class,
  MIN_BY (es.app, es.created) AS app,
  MIN_BY (es.url_path, es.created) AS url_path,
  uc.created AS dt,
FROM
  ${ref("base_buckets_events_app_speedy")} AS es
JOIN ${ref("stg_user_courses")} AS uc ON uc.user_id = es.user_id
JOIN ${ref("stg_courses")} AS c ON uc.course_id = c.id
GROUP BY
  sub_session_id,
  uc.created,
  tag,
  uc.user_id
HAVING
  uc.created BETWEEN MIN(es.created) AND MAX(es.created)
QUALIFY
  ROW_NUMBER() OVER (
    PARTITION BY
        uc.user_id
    ORDER BY
        uc.created ASC
  ) = 1

UNION ALL

SELECT
  es.sub_session_id AS sub_session_id,
  CONCAT('started_first_course_general') AS tag,
  'behavior' as class,
  MIN_BY (es.app, es.created) AS app,
  MIN_BY (es.url_path, es.created) AS url_path,
  uc.created AS dt,
FROM
  ${ref("base_buckets_events_app_speedy")} AS es
JOIN ${ref("stg_user_courses")} AS uc ON uc.user_id = es.user_id
JOIN ${ref("stg_courses")} AS c ON uc.course_id = c.id
GROUP BY
  sub_session_id,
  uc.created,
  tag,
  uc.user_id
HAVING
  uc.created BETWEEN MIN(es.created) AND MAX(es.created)
QUALIFY
  ROW_NUMBER() OVER (
    PARTITION BY
        uc.user_id
    ORDER BY
        uc.created ASC
  ) = 1
```
