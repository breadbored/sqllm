How do I compute the bounding box of multiple geography rows in an aggregation?

--- AGENT ---

`ST_EXTENT` is the aggregate version of `ST_BOUNDINGBOX`, combining multiple geographies to find their collective bounding box as a struct with xmin, ymin, xmax, ymax.

```googlesql
WITH data AS (
  SELECT 1 id, ST_GEOGFROMTEXT('POLYGON((-125 48, -124 46, -117 46, -117 49, -125 48))') g
  UNION ALL
  SELECT 2 id, ST_GEOGFROMTEXT('POLYGON((172 53, -130 55, -141 70, 172 53))') g
  UNION ALL
  SELECT 3 id, ST_GEOGFROMTEXT('POINT EMPTY') g
)
SELECT ST_EXTENT(g) AS box
FROM data
```
