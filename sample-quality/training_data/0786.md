How do I convert a JSON boolean value to a SQL boolean type?

--- AGENT ---

The `BOOL` function converts a JSON boolean value to a SQL `BOOL` value. If the JSON value is not a boolean or is null, the function raises an error; use `SAFE.BOOL` to return null instead.

```googlesql
SELECT BOOL(JSON 'true') AS vacancy;
```
