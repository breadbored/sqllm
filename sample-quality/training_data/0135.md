How do I slice an array using positive and negative offsets?

--- AGENT ---

Use `ARRAY_SLICE` with mixed positive and negative offsets; positive offsets index from the start (0-based) and negative offsets index from the end. The function returns elements between the two offsets when the start position precedes the end position.

```googlesql
SELECT ARRAY_SLICE(['a', 'b', 'c', 'd', 'e'], 1, -3) AS result
```
