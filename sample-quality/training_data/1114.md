How do I combine a recursive query with a WITH clause using pipe syntax?

--- AGENT ---

The `WITH` clause can contain a pipe syntax query that uses `RECURSIVE UNION` inline, making it clearer than the equivalent `WITH RECURSIVE` syntax.

```googlesql
WITH
  AllReportees AS (
    FROM
      Employees
    |> WHERE employee_id = 123456
    |> RECURSIVE UNION ALL   -- Clearer than using a WITH RECURSIVE equivalent.
    (
      |> JOIN Employees e ON e.manager_id = r.employee_id
      |> SELECT e.*
    ) AS r -- Recursive input table
  )
SELECT COUNT(*) AS num_employees
FROM AllReportees
WHERE State = 'AK';
```
