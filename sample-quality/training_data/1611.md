--- USER ---

How can I rank employees within their department by start date, with the earliest starter getting rank 1?

--- AGENT ---

This query partitions employees by department and orders by start_date, then applies the RANK function to assign a rank to each employee within their department. The rank resets for each department partition.

```googlesql
SELECT name, department, start_date,
  RANK() OVER (PARTITION BY department ORDER BY start_date) AS rank
FROM Employees;
```
