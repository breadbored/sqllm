How can I safely access an array element and return a fallback value if the index doesn't exist?

--- AGENT ---

IFERROR evaluates the array access expression and returns the element if it exists without error. The OFFSET(0) function accesses the first element, which exists, so the catch expression is not evaluated.

```googlesql
SELECT IFERROR((SELECT [1,2,3][OFFSET(0)]), -1) AS result
```
