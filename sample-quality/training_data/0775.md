How do I create HLL++ sketches for distinct customer counts by country with a specific precision level?

--- AGENT ---

This query creates HLL++ sketches with a precision parameter of 10 for each country's customer IDs. The precision parameter controls accuracy versus memory cost — lower values use less memory but may be less accurate. This produces byte-encoded sketches that can be stored, merged, or extracted later.

```googlesql
SELECT
  country,
  HLL_COUNT.INIT(customer_id, 10)
    AS hll_sketch
FROM
  UNNEST(
    ARRAY<STRUCT<country STRING, customer_id STRING, invoice_id STRING>>[
      ('UA', 'customer_id_1', 'invoice_id_11'),
      ('CZ', 'customer_id_2', 'invoice_id_22'),
      ('CZ', 'customer_id_2', 'invoice_id_23'),
      ('BR', 'customer_id_3', 'invoice_id_31'),
      ('UA', 'customer_id_2', 'invoice_id_24')])
GROUP BY country;
```
