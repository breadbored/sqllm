Show me how to filter events from a staging table to include only those from specific domain names.

--- AGENT ---

This query creates a staging table of events by selecting all columns from the `pre_vaas_events` table and filtering to include only rows where the `vaas_url_domain` column matches one of three conditions:

1. Exactly equals `'developers.meta.com'`
2. Matches any subdomain pattern of `dataprotocol.com` (using the LIKE wildcard)
3. Exactly equals `'dataprotocol.com'`

The result is a filtered event set restricted to these three domain sources.

```sql
SELECT
  e.*
FROM
  ${ref("pre_vaas_events")} as e
WHERE
  (
    e.vaas_url_domain = 'developers.meta.com'
  OR
    e.vaas_url_domain LIKE '%.dataprotocol.com'
  OR
    e.vaas_url_domain = 'dataprotocol.com'
  )
```
