## Question
Show me a BigQuery table function that generates a histogram of user course completion progress across different percentage thresholds.

--- AGENT ---

This query creates a table-valued function that produces a distribution histogram of course completion progress. It:

1. Generates 20 progress percentage buckets at 5% intervals (0%, 5%, 10%, ..., 100%), with special handling for values above 90% (treated as 95%)
2. Joins these buckets against a user-courses reporting table, creating pairs of (bucket, user record) where the user's progress meets or exceeds that bucket threshold
3. Counts users for each bucket, labeling the base bucket as "Started" and others as "N%" (e.g., "5%", "10%")
4. Scopes results to a specific client via the clientId function parameter

The output shape is two columns: a bucket name string and an integer count of users reaching that completion level or higher. This creates a reverse-cumulative histogram showing, for each 5% milestone, how many users have completed at least that percentage of their course.

Note: The WHERE clause contains `SELECT * FROM uc.client_id`, which is structurally invalid (attempting to select from a column rather than a table) and would not execute.

```sql
CREATE OR REPLACE TABLE FUNCTION ${self()} (clientId INT64) AS (
  SELECT
    CASE
      WHEN range_hack.bucket = 0 THEN 'Started'
      ELSE CONCAT(range_hack.bucket, '%')
    END AS name,
    COUNT(
      CASE
        WHEN uc.course_progress_percentage >= (
          (
            CASE
              WHEN range_hack.bucket > 90 THEN 95
              ELSE range_hack.bucket
            END
          ) / 100
        ) THEN 1
      END
    ) AS value
  FROM
    (
      SELECT
        h AS bucket
      FROM
        UNNEST (GENERATE_ARRAY(0, 100, 5)) AS h
    ) AS range_hack
    JOIN ${ref("reporting_user_courses")} uc ON uc.course_progress_percentage >= (
      (
        CASE
          WHEN range_hack.bucket > 90 THEN 95
          ELSE range_hack.bucket
        END
      ) / 100
    )
  WHERE
    clientId IN (
      SELECT
        *
      FROM
        uc.client_id
    )
  GROUP BY
    range_hack.bucket
  ORDER BY
    range_hack.bucket ASC
)
```
