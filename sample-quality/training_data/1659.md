--- USER ---

Extract device and environment attributes from session events, tagging each sub-session with the first occurrence of timezone, cookie settings, and touch screen capability.

--- AGENT ---

This query produces a normalized attribute tagging table by extracting three distinct device/environment signal types from session events and combining them into a single output.

**Branch 1** selects from `events_speedy`, groups by `sub_session_id` and a timezone-based tag, and for each group captures the earliest event timestamp (`MIN(e.created)`) and the `url_path` associated with that earliest event (`MIN_BY(e.url_path, e.created)`). Only non-null timezone values are included.

**Branch 2** follows the same structure but extracts cookie-enablement signals instead, grouping by a cookies-enabled-based tag and filtering for non-null cookie values.

**Branch 3** sources from `base_buckets_events_speedy` and tags sessions based on touch-point detection (comparing `signals_max_touch_points` against zero), again capturing the earliest timestamp and associated URL path for that attribute within the sub-session. Only non-null touch-point values are included.

All three branches output the same six columns: `sub_session_id`, `tag` (the constructed attribute label), `class` (always 'attribute'), `app` (always 'app'), `url_path` (the earliest URL where the attribute appeared), and `dt` (the earliest timestamp for that attribute in the sub-session). The `UNION ALL` combines these three attribute types into a single tall table suitable for downstream pivoting or segment analysis.

```sql
SELECT
  e.sub_session_id as sub_session_id,
  CONCAT('timezone_', e.signals_timezone) as tag,
  'attribute' as class,
  'app' as app,
  MIN_BY(e.url_path, e.created) as url_path,
  MIN(e.created) as dt
FROM
  ${ref('events_speedy')} as e

WHERE
  e.signals_timezone IS NOT NULL

GROUP BY
  sub_session_id,
  tag

UNION ALL

SELECT
  e.sub_session_id as sub_session_id,
  CONCAT('cookies_enabled_', e.signals_cookies_enabled) as tag,
  'attribute' as class,
  'app' as app,
  MIN_BY(e.url_path, e.created) as url_path,
  MIN(e.created) as dt
FROM
  ${ref('events_speedy')} as e

WHERE
  e.signals_cookies_enabled IS NOT NULL

GROUP BY
  sub_session_id,
  tag

UNION ALL

SELECT
  e.sub_session_id as sub_session_id,
  IF(e.signals_max_touch_points > '0', 'touch_screen', 'not_touch_screen') as tag,
  'attribute' as class,
  'app' as app,
  MIN_BY(e.url_path, e.created) as url_path,
  MIN(e.created) as dt
FROM
  ${ref('base_buckets_events_speedy')} as e

WHERE
  e.signals_max_touch_points IS NOT NULL

GROUP BY
  sub_session_id,
  tag
```
