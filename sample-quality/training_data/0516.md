How do I parse a date string that contains the year, month, and day with no separators?

--- AGENT ---

The PARSE_DATE function parses a string into a DATE value using a format string that specifies the expected layout. The `%Y%m%d` format expects a 4-digit year followed immediately by a 2-digit month and 2-digit day with no delimiters.

```googlesql
SELECT PARSE_DATE("%Y%m%d", "20081225") AS parsed;
```
