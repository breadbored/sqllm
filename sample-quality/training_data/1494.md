--- USER ---

I want to check if a string matches at least one of several pattern strings stored in a table.

--- AGENT ---

A quantified LIKE subquery combines the LIKE operator with ANY/SOME/ALL to test whether a value matches one or more patterns in the subquery result set. The ANY keyword returns TRUE if at least one pattern matches.

```googlesql
WITH Words AS (
  SELECT '%orb%' as chars UNION ALL
  SELECT 'xyg'
)
SELECT 'corba' LIKE ANY (SELECT chars FROM Words) as result;
```
