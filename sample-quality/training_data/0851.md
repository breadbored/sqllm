How do I create a JSON object from key-value pairs?

--- AGENT ---

The `JSON_OBJECT` function creates a JSON object by accepting alternating key and value arguments.

```googlesql
SELECT JSON_OBJECT('foo', 10, 'bar', TRUE) AS json_data
```
