How do I convert a UINT32 back to INT32 preserving the original bit pattern?

--- AGENT ---

BIT_CAST_TO_INT32 performs a bit-level reinterpretation, converting the maximum UINT32 value back to -1 by preserving the same bit representation.

```googlesql
SELECT BIT_CAST_TO_UINT32(-1) as UINT32_value, BIT_CAST_TO_INT32(BIT_CAST_TO_UINT32(-1)) as bit_cast_value;
```
