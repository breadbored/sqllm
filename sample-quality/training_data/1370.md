How do I format a date using nested FORMAT functions?

--- AGENT ---

FORMAT can be used with the %s specifier to embed the output of FORMAT_DATE, allowing date formatting to be composed within a larger formatted string.

```googlesql
SELECT FORMAT('date: %s!', FORMAT_DATE('%B %d, %Y', date '2015-01-02'));
```
