--- USER ---

How do I execute an inline subquery that finds accounts owned by each matched person, ordered and limited?

--- AGENT ---

The `CALL` statement with an inline subquery lets you declare outer-scoped variables in parentheses before the curly braces, then execute a graph pattern within the subquery. Variables must be explicitly declared to be accessible and multiply-declarable in the subquery's pattern matching.

```googlesql
GRAPH FinGraph
MATCH (p:Person)
CALL (p) {
  MATCH (p)-[:Owns]->(a:Account)
  RETURN a.Id AS account_Id
  ORDER BY account_Id
  LIMIT 2
}
RETURN p.name AS person_name, account_Id
ORDER BY person_name, account_Id;
```
