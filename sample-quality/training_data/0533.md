How do I extract the time zone hour offset from a timestamp while treating it as a specific time zone rather than the session time zone?

--- AGENT ---

The CAST function with the FORMAT clause converts a timestamp to a string and supports the AT TIME ZONE clause to specify which time zone to use. The TZH format element outputs the hour offset of the specified time zone.

```googlesql
SELECT CAST(TIMESTAMP '2008-12-25 00:00:00+00:00' AS STRING FORMAT 'TZH' AT TIME ZONE 'Asia/Kolkata')
AS date_time_to_string;
```
