What is the difference between a NULL array and an empty array in a query result?

--- AGENT ---

In queries, a NULL array and an empty array are two distinct values, though they may appear identical. This query shows both: the first row has an empty array `[]`, and the second row has a NULL array that also displays as `[]` but has `numbers_null = true`.

```googlesql
WITH Items AS (
  SELECT [] AS numbers, "Empty array in query" AS description UNION ALL
  SELECT CAST(NULL AS ARRAY<INT64>), "NULL array in query")
SELECT numbers, description, numbers IS NULL AS numbers_null
FROM Items;
```
