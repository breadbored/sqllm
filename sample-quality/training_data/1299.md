How do I check if two unbounded date ranges overlap?

--- AGENT ---

RANGE_OVERLAPS checks for overlap between ranges including those with unbounded boundaries. This query verifies that [2022-02-01, UNBOUNDED) and [2021-06-15, UNBOUNDED) do overlap because their lower bounds and the unbounded upper bounds create a shared intersection.

```googlesql
SELECT RANGE_OVERLAPS(
  RANGE<DATE> '[2022-02-01, UNBOUNDED)',
  RANGE<DATE> '[2021-06-15, UNBOUNDED)') AS results;
```
