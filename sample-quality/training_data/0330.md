How do I define a primary key on a single column?

--- AGENT ---

A primary key constraint designates one or more columns as the unique identifier for each row. For a single-column primary key, the column value must be unique across all rows in the table.

```googlesql
CREATE TABLE books (title STRING, id STRING, PRIMARY KEY (id));
```
