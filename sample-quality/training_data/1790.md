## Question
Generate a BigQuery table function that maps course progress intervals to lesson content and learning activities, showing how many users have progressed to each point and what interactive elements they encounter.

--- AGENT ---

This table-valued function creates a learning progression map for a course. It works by:

1. **Finding the target course** — retrieves the most recent published course for a given container
2. **Building the lesson timeline** — joins course lessons with video durations and computes cumulative course duration up to each lesson using a window function
3. **Attaching learning activities** — aggregates learning cues (knowledge checks and sentiment questions) for each lesson, rounding their positions to 5-second boundaries aligned with course progress
4. **Generating progress intervals** — creates a complete 5-second interval grid spanning from minimum to maximum user progress in the container
5. **Mapping users to intervals** — counts distinct users at each progress interval and calculates cumulative user count (descending) to show how many have reached or passed each point
6. **Correlating content to progress** — left-joins lessons and cues to intervals using BETWEEN logic to match which lesson (and which cues within that lesson) corresponds to each progress point
7. **Filtering to valid rows** — excludes intervals that do not align with any lesson

The output is a row per valid progress interval, showing cumulative user count, the lesson being viewed, and an array of interactive elements the user encounters at that position.

```sql
CREATE OR REPLACE TABLE
  FUNCTION ${self()} (clientId INT64,
    containerId INT64) AS (
  WITH
    max_course_id AS (
    SELECT
      MAX(c.id) AS course_id
    FROM
      ${ref("stg_courses")} c
    WHERE
      container_id = containerId
      AND published = 1),
    lessons_with_cumulative_duration AS (
    SELECT
      cl.id AS cl_id,
      cl.lesson_id,
      l.name,
      v.duration AS lesson_duration_seconds,
      IFNULL(SUM(v.duration) OVER (ORDER BY cl.id ROWS BETWEEN UNBOUNDED PRECEDING AND 1 PRECEDING), 0) AS current_course_duration
    FROM
      ${ref("stg_course_lessons")} cl
    JOIN
      max_course_id mc
    ON
      mc.course_id = cl.course_id
    JOIN
      ${ref("stg_lessons")} l
    ON
      l.id = cl.lesson_id
    JOIN
      ${ref("stg_videos")} v
    ON
      v.id = l.video_id),
    learning_cues_data AS (
    SELECT
      cl.cl_id,
      cl.lesson_id,
      cl.name,
      cl.lesson_duration_seconds,
      cl.current_course_duration,
      ARRAY_AGG(STRUCT (lc.id AS learning_cue_id,
          lc.start_seconds AS lesson_start_seconds,
          FLOOR((lc.start_seconds + cl.current_course_duration) / 5) * 5 AS course_start_seconds,
          -- Rounding down to nearest 5 seconds
          IFNULL(q.name, sq.name) AS question,
        IF
          (q.name IS NULL, 'Sentiment Question', 'Knowledge Check') AS type)) AS cues
    FROM
      lessons_with_cumulative_duration cl
    JOIN
      ${ref("stg_learning_cues")} lc
    ON
      lc.lesson_id = cl.lesson_id
    LEFT JOIN
      ${ref("stg_knowledge_checks")} kc
    ON
      kc.learning_cue_id = lc.id
    LEFT JOIN
      ${ref("stg_knowledge_check_questions")} kcq
    ON
      kcq.knowledge_check_id = kc.id
    LEFT JOIN
      ${ref("stg_questions")} q
    ON
      q.id = kcq.question_id
    LEFT JOIN
      ${ref("stg_sentiment_questions")} sq
    ON
      sq.learning_cue_id = lc.id
    WHERE
      kc.id IS NOT NULL
      OR sq.id IS NOT NULL
    GROUP BY
      cl.cl_id,
      cl.lesson_id,
      cl.name,
      cl.lesson_duration_seconds,
      cl.current_course_duration),
    intervals AS (
    SELECT
      FLOOR(ucp.course_progress_seconds / 5) * 5 AS progress_interval,
      COUNT(DISTINCT ucp.user_id) AS users
    FROM
      ${ref("agg_user_course_progress")} ucp
    WHERE
      ucp.container_id = containerId
      AND clientId IN UNNEST(ucp.client_id)
    GROUP BY
      progress_interval),
    full_intervals AS (
    SELECT
      interval_value
    FROM
      UNNEST(GENERATE_ARRAY((
          SELECT
            MIN(FLOOR(course_progress_seconds / 5) * 5)
          FROM
            ${ref("agg_user_course_progress")} ucp
          WHERE
            container_id = containerId), (
          SELECT
            MAX(FLOOR(course_progress_seconds / 5) * 5)
          FROM
            ${ref("agg_user_course_progress")} ucp
          WHERE
            container_id = containerId), 5)) AS interval_value),
    final_number_prefilter AS (
    SELECT
      fi.interval_value AS progress_interval,
      COALESCE(SUM(i.users) OVER (ORDER BY fi.interval_value DESC), 0) AS users,
      lc_data_base.cl_id,
      lc_data_base.lesson_id,
      lc_data_base.name,
      lc_data_base.lesson_duration_seconds,
      lc_data_base.current_course_duration,
      ARRAY_AGG(cues_data IGNORE NULLS) AS cues
    FROM
      full_intervals fi
    LEFT JOIN
      intervals i
    ON
      fi.interval_value = i.progress_interval
    LEFT JOIN
      learning_cues_data lc_data_base
    ON
      fi.interval_value BETWEEN lc_data_base.current_course_duration
      AND (lc_data_base.current_course_duration + lc_data_base.lesson_duration_seconds)
    LEFT JOIN
      UNNEST(lc_data_base.cues) AS cues_data
    ON
      fi.interval_value = FLOOR(cues_data.course_start_seconds / 5.0) * 5
    GROUP BY
      fi.interval_value,
      lc_data_base.cl_id,
      lc_data_base.lesson_id,
      lc_data_base.name,
      lc_data_base.lesson_duration_seconds,
      lc_data_base.current_course_duration,
      lc_data_base.cues,
      i.users
    ORDER BY
      fi.interval_value,
      lc_data_base.cl_id)
  SELECT
    *
  FROM
    final_number_prefilter
  WHERE
    cl_id IS NOT NULL )
```
