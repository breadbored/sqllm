## Question
Can you explain what this query does? It seems to be building some kind of denormalized data structure with multiple nested arrays.

--- AGENT ---

This query builds a denormalized summary of a course container with nested deduplication of related entities.

**Step 1: Find the maximum course ID per container**
The `max_course_ids` CTE identifies the highest course ID in each container (excluding "vaas" type courses). This ensures that when you join to course resources and lessons, you're fetching from the latest course version.

**Step 2: Multi-entity aggregation**
The `course_container_courses` CTE performs a complex join that collects:
- All courses in each container, aggregated as an array of structs containing course name, ID, and version
- All associated badges (via learning path course containers), aggregated as structs with badge ID, name, and publication status
- All associated documents (matched by URL path from course and lesson resources), aggregated as structs with doc ID, name, type, and full URL
- All associated clients, aggregated as structs with client ID and client name

The join strategy uses the max_course_ids lookup to narrow resource fetching to the latest course version. The WHERE clause filters out "vaas" type courses. Course status is set to 'published' if any course in the container is published, otherwise 'unpublished'.

**Step 3: Deduplicate nested arrays**
The final SELECT takes each nested array field and re-aggregates it with a GROUP BY on the struct fields. This removes duplicate records within each array while preserving the array structure.

The result is a single row per course container with four deduplicated arrays of related entities.

```sql
WITH
  max_course_ids AS (
  SELECT
    container_id,
    MAX(id) AS max_course_id
  FROM
    ${ref("stg_courses")}
  WHERE
    course_type != "vaas"
  GROUP BY
    container_id),
  course_container_courses AS (
  SELECT
    c.container_id,
    c.course_type,
    CONCAT("https://app.dataprotocol.com/container/", c.container_id) AS course_url,
    CASE
      WHEN MAX(c.published) = 1 THEN 'published'
      ELSE 'unpublished'
  END
    AS course_status,
    ARRAY_AGG(STRUCT (c.name AS course_name,
        c.id AS course_id,
        c.version AS version) IGNORE NULLS
    ORDER BY
      c.id) AS course_details,
    ARRAY_AGG(STRUCT (lp.id AS badge_id,
        lp.name AS badge_name,
        CASE
          WHEN lp.published = 1 THEN 'published'
          WHEN lp.published = 0 THEN 'unpublished'
          ELSE NULL
      END
        AS badge_status) IGNORE NULLS
    ORDER BY
      lp.id) AS badge_details,
    ARRAY_AGG(STRUCT (cd.doc_id AS doc_id,
        cd.name AS doc_name,
        cd.type AS doc_type,
        CONCAT("https://docs.dataprotocol.com", cd.path) AS doc_url ) IGNORE NULLS
    ORDER BY
      cd.doc_id) AS doc_details,
    ARRAY_AGG(STRUCT (ccc.client_id,
        clients.name AS client_name) IGNORE NULLS) AS client_details
  FROM
    ${ref("stg_clients_course_containers")} ccc
  JOIN
    ${ref("stg_clients")} clients
  ON
    clients.id = ccc.client_id
  JOIN
    ${ref("stg_courses")} c
  ON
    c.container_id = ccc.course_container_id
  LEFT JOIN
    ${ref("stg_learning_path_course_containers")} lpcc
  ON
    lpcc.course_container_id = ccc.course_container_id
  JOIN
    max_course_ids mci
  ON
    mci.container_id = c.container_id
  LEFT JOIN
    ${ref("stg_learning_paths")} lp
  ON
    lp.id = lpcc.learning_path_id
    -- and lp.archived = 0
  LEFT JOIN
    ${ref("stg_course_resources")} cr
  ON
    cr.course_id = mci.max_course_id
  LEFT JOIN
    ${ref("stg_course_lessons")} cl
  ON
    cl.course_id = mci.max_course_id
  LEFT JOIN
    ${ref("stg_lesson_resources")} lr
  ON
    lr.lesson_id = cl.lesson_id
  LEFT JOIN
    ${ref("agg_client_documents")} cd
  ON
    cd.path = REGEXP_SUBSTR(cr.url, r'https?://[^/]+(/.*)', 1, 1)
    OR cd.path = REGEXP_SUBSTR(lr.url, r'https?://[^/]+(/.*)', 1, 1)
  WHERE
    course_type != "vaas"
  GROUP BY
    c.container_id,
    c.course_type)
SELECT
  ccc.container_id,
  ccc.course_type,
  ccc.course_status,
  ccc.course_url,
  ARRAY(
  SELECT
    AS STRUCT cd.course_id, cd.course_name, cd.version
  FROM
    ccc.course_details AS cd
  GROUP BY
    cd.course_id, cd.course_name, cd.version) AS course_details,
  ARRAY(
  SELECT
    AS STRUCT bd.badge_id, bd.badge_name, bd.badge_status, CONCAT("https://app.dataprotocol.com/badges/", bd.badge_id) AS badge_url
  FROM
    ccc.badge_details AS bd
  GROUP BY
    bd.badge_id, bd.badge_name, bd.badge_status) AS badge_details,
  ARRAY(
  SELECT
    AS STRUCT dd.doc_id, dd.doc_name, dd.doc_type, dd.doc_url
  FROM
    ccc.doc_details AS dd
  GROUP BY
    dd.doc_id, dd.doc_name, dd.doc_type, dd.doc_url) AS doc_details,
  ARRAY(
  SELECT
    AS STRUCT cd.client_id, cd.client_name
  FROM
    ccc.client_details AS cd
  GROUP BY
    cd.client_id, cd.client_name) AS client_details
FROM
  course_container_courses AS ccc
```
