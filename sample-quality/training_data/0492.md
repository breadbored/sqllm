How can I return a default value when array element access fails because the index is out of bounds?

--- AGENT ---

IFERROR catches the evaluation error from accessing array index 9 when only three elements exist, and returns the fallback value -1.

```googlesql
SELECT IFERROR((SELECT [1,2,3][OFFSET(9)]), -1) AS result
```
