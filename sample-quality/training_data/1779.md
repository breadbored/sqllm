--- AGENT ---

This table function returns assessment question metadata with an aggregated array of answer options and response statistics, filtered by a specific client, container, and assessment question.

The query operates in two layers:

1. **Inner subquery** (`answers`): Aggregates question options with user response statistics. It joins the question set options to question options, then left-joins user course assessment answers (excluding test users) filtered to the specified client and container. The results are grouped by option name, question name, and a correctness flag, producing a count of how many non-test users selected each option.

2. **Outer query**: Joins the question back to this aggregated answer metadata (matching by question name), groups by question name, and packages all answer rows into a single ARRAY_AGG column called `answers`. This produces one row per question with an array of answer option records.

The function parameters (`clientId`, `containerId`, `assessmentQuestionId`) filter to a specific question and count only responses from non-test users in that client/container context.

Here is the original SQL:

```sql
CREATE OR REPLACE TABLE
  FUNCTION ${self()} (clientId INT64,
    containerId INT64,
    assessmentQuestionId INT64) AS (
  SELECT
    q.name,
    ARRAY_AGG(answers) AS `answers`
  FROM
    ${ref("stg_questions")} q
  JOIN (
    SELECT
      MAX(qso.id) AS id,
      qo.name,
      _q.name AS question_name,
      MAX(qso.question_set_id) AS question_id,
      COUNT(ukca.id) AS count,
      ukca.correct
    FROM
      ${ref("stg_question_set_options")} qso
    JOIN
      ${ref("stg_question_options")} qo
    ON
      qo.id = qso.question_option_id
    JOIN
      ${ref("stg_questions")} _q
    ON
      _q.id = qso.question_set_id
    LEFT JOIN (
      SELECT
        uca.id,
        uca.answer_option_id,
        uca.course_id,
        uca.correct
      FROM
        ${ref("stg_user_course_assessment_answers")} uca
      JOIN
        ${ref("stg_users")} AS u
      ON
        uca.user_id = u.id
      WHERE
        u.test_user = 0) AS ukca
    ON
      ukca.answer_option_id = qso.question_option_id
    LEFT JOIN
      ${ref("stg_courses")} c
    ON
      c.id = ukca.course_id
    LEFT JOIN
      ${ref("stg_clients_course_containers")} ccc
    ON
      ccc.course_container_id = c.container_id
      AND container_id = containerId
      AND ccc.client_id = clientId
    WHERE
      _q.id = assessmentQuestionId
    GROUP BY
      qo.name,
      _q.name,
      correct) AS answers
  ON
    answers.question_name = q.name
  WHERE
    q.id = assessmentQuestionId
  GROUP BY
    q.name )
```
