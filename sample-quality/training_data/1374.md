How do I convert hexadecimal-encoded strings to bytes?

--- AGENT ---

The FROM_HEX function converts a hexadecimal string into BYTES format, handling both uppercase and lowercase hex characters.

```googlesql
WITH Input AS (
  SELECT '00010203aaeeefff' AS hex_str UNION ALL
  SELECT '0AF' UNION ALL
  SELECT '666f6f626172'
)
SELECT hex_str, FROM_HEX(hex_str) AS bytes_str
FROM Input;
```
