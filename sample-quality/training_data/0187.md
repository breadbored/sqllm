How do I flatten deeply nested array fields using FLATTEN with dot notation?

--- AGENT ---

Use `FLATTEN()` with chained dot notation to access fields nested multiple levels deep within array structures. This flattens all values from the deeply nested field across all parent elements.

```googlesql
SELECT FLATTEN(items.sales.prices) AS all_prices
FROM ItemsTable

/*------------------------+
 | all_prices             |
 +------------------------+
 | [100, 50, 25, 75, 200] |
 +------------------------*/
```
