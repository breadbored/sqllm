How can I rank a set of numbers where tied values receive the same rank but later ranks skip?

--- AGENT ---

RANK() assigns ordinal ranks starting from 1, with all tied rows receiving the same rank and the next rank value incremented by the count of tied rows (creating gaps). This query ranks simple numbers so that the two 2's both get rank 2, and the next distinct value 5 gets rank 4.

```googlesql
WITH Numbers AS
 (SELECT 1 as x
  UNION ALL SELECT 2
  UNION ALL SELECT 2
  UNION ALL SELECT 5
  UNION ALL SELECT 8
  UNION ALL SELECT 10
  UNION ALL SELECT 10
)
SELECT x,
  RANK() OVER (ORDER BY x ASC) AS rank
FROM Numbers
```
