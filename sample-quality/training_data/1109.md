How do I match columns by name instead of position when combining results with UNION?

--- AGENT ---

The `BY NAME` modifier matches columns by name across the input queries instead of by position. Columns with different orders will still be correctly aligned if they have the same names.

```googlesql
SELECT 1 AS one_digit, 10 AS two_digit
|> UNION ALL BY NAME
    (SELECT 20 AS two_digit, 2 AS one_digit);
```
