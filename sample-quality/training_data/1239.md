How do I convert selected columns into a protocol buffer type?

--- AGENT ---

`SELECT AS typename` produces a value table where the row type is a specific named type, such as a protocol buffer. Each column in the SELECT list must match a field name in the target type.

```googlesql
SELECT AS tests.TestProtocolBuffer mytable.key int64_val, mytable.name string_val
FROM mytable;
```
