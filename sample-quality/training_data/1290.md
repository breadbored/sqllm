How do I check if a specific date value falls within a date range?

--- AGENT ---

RANGE_CONTAINS with a range and a scalar value checks whether that value is contained in the range. This query verifies that April 1, 2022 falls within the range [2022-01-01, 2023-01-01).

```googlesql
SELECT RANGE_CONTAINS(
  RANGE<DATE> '[2022-01-01, 2023-01-01)',
  DATE '2022-04-01') AS results;
```
