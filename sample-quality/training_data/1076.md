How do I add a computed column based on a boolean expression while keeping existing columns?

--- AGENT ---

The `EXTEND` pipe operator adds new computed columns to the input table while keeping all existing columns and rows. This example adds a column that evaluates whether the item is in a list of yellow fruits.

```googlesql
(
  SELECT 'apples' AS item, 2 AS sales
  UNION ALL
  SELECT 'bananas' AS item, 8 AS sales
)
|> EXTEND item IN ('bananas', 'lemons') AS is_yellow;
```
