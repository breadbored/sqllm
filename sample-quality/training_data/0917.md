How do I convert a JSON value to a string with flexible type conversion?

--- AGENT ---

The `LAX_STRING` function converts JSON strings, numbers, and booleans to SQL `STRING` without raising errors on incompatible types.

```googlesql
SELECT LAX_STRING(JSON '"purple"') AS result;

SELECT LAX_STRING(JSON '10') AS result;
```
