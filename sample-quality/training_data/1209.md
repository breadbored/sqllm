--- USER ---

How can I extract and inspect the descriptor options for enum values, including whether they are marked as ideally enabled or in development?

--- AGENT ---

This query retrieves the enum value descriptor proto for each enum value and accesses custom options from those descriptors, using IFNULL to provide defaults for missing options. The function ENUM_VALUE_DESCRIPTOR_PROTO accesses metadata that is normally not available through simple field access.

```googlesql
WITH
  EnabledFeatures AS (
    SELECT CAST(999991 AS googlesql.LanguageFeature) AS feature UNION ALL
    SELECT CAST(999992 AS googlesql.LanguageFeature) AS feature
  )
SELECT
  CAST(feature AS STRING) AS feature_enum_name,
  CAST(feature AS INT64) AS feature_enum_id,
  IFNULL(
    ENUM_VALUE_DESCRIPTOR_PROTO(feature).options.(googlesql.language_feature_options).ideally_enabled,
    TRUE) AS feature_is_ideally_enabled,
  IFNULL(
    ENUM_VALUE_DESCRIPTOR_PROTO(feature).options.(googlesql.language_feature_options).in_development,
    FALSE) AS feature_is_in_development
FROM
  EnabledFeatures;
```
