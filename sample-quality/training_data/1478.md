How do I convert a bytes value to an array of ASCII values?

--- AGENT ---

The TO_CODE_POINTS function on BYTES returns an array of extended ASCII values (0-255) for each byte.

```googlesql
SELECT
  b'\x66\x6f\x6f' AS bytes_value,
  TO_CODE_POINTS(b'\x66\x6f\x6f') AS bytes_value_as_integer
```
