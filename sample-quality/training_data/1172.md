How do I create a loop that executes until a boolean condition becomes true?

--- AGENT ---

Use `REPEAT` to execute a block of statements, then test a condition at the end. The loop continues executing until the condition is true.

```googlesql
DECLARE x INT64 DEFAULT 0;

REPEAT
  SET x = x + 1;
  SELECT x;
  UNTIL x >= 3
END REPEAT;
```
