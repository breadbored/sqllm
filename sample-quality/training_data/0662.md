How do I return the properties of edges in addition to the connected nodes?

--- AGENT ---

Bind a variable to the edge pattern and include that variable's properties in the RETURN clause. This example returns the amount transferred along with the source and destination account IDs.

```googlesql
GRAPH FinGraph
MATCH (src:Account)-[transfer:Transfers]->(dst:Account)
RETURN src.id AS src_id, transfer.amount, dst.id AS dst_id
```
