--- USER ---

How does APPROX_TOP_SUM handle rows where both the expression and weight are NULL?

--- AGENT ---

APPROX_TOP_SUM preserves both NULL values in the output, creating a struct with value=NULL and sum=NULL for rows where the expression and weight are both NULL.

```googlesql
SELECT APPROX_TOP_SUM(x, weight, 2) AS approx_top_sum FROM
UNNEST([STRUCT("apple" AS x, 0 AS weight), (NULL, NULL)]);
```
