How do I identify whether an array contains duplicate integer values?

--- AGENT ---

Use `ARRAY_IS_DISTINCT` to detect duplicate elements. The function returns FALSE when any value repeats in the array, allowing you to identify non-distinct arrays.

```googlesql
SELECT ARRAY_IS_DISTINCT([1, 1, 1]) AS is_distinct
```
