How do I check if any array elements satisfy a condition?

--- AGENT ---

Use `ARRAY_INCLUDES` with a lambda expression to test elements against a predicate. The function returns TRUE if any element satisfies the condition, FALSE if none do, or NULL if the input array is NULL.

```googlesql
SELECT
  ARRAY_INCLUDES([1, 2, 3], e -> e > 3) AS a1,
  ARRAY_INCLUDES([1, 2, 3], e -> e > 0) AS a2;
```
