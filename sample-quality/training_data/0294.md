How do I define a composite primary key as a table constraint?

--- AGENT ---

A primary key constraint can be specified at the table level using the `PRIMARY KEY` clause, which allows you to define a multi-column key and specify sort order (ascending or descending) for each column.

```googlesql
CREATE TABLE books (title STRING, author STRING, PRIMARY KEY (title ASC, author ASC));
```
