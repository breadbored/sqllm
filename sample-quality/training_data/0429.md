How do I calculate the number of days between two dates?

--- AGENT ---

`DATE_DIFF()` returns the number of unit boundaries between two DATE values at a specified granularity. The function counts from the start date to the end date.

```googlesql
SELECT DATE_DIFF(DATE '2010-07-07', DATE '2008-12-25', DAY) AS days_diff;
```
