How do I convert a Unix timestamp in microseconds to a TIMESTAMP value?

--- AGENT ---

The `TIMESTAMP_FROM_UNIX_MICROS` function interprets an integer as the number of microseconds since the Unix epoch (1970-01-01 00:00:00 UTC) and returns the corresponding TIMESTAMP. It can also pass through an existing TIMESTAMP unchanged.

```googlesql
SELECT TIMESTAMP_FROM_UNIX_MICROS(1230219000000000) AS timestamp_value;
```
