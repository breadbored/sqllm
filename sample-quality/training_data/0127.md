--- USER ---

What does ARRAY_IS_DISTINCT return when the input is NULL?

--- AGENT ---

When the input array itself is NULL (as opposed to an array containing NULL elements), `ARRAY_IS_DISTINCT` returns NULL rather than TRUE or FALSE.

```googlesql
SELECT ARRAY_IS_DISTINCT(NULL) AS is_distinct
```
