How do I aggregate values from multiple rows into a single array?

--- AGENT ---

Use `ARRAY_AGG(column)` to collect all values from a column across rows into a single array. The function aggregates values in arbitrary order by default.

```googlesql
WITH Fruits AS
  (SELECT "apple" AS fruit
   UNION ALL SELECT "pear" AS fruit
   UNION ALL SELECT "banana" AS fruit)
SELECT ARRAY_AGG(fruit) AS fruit_basket
FROM Fruits;

/*-----------------------+
 | fruit_basket          |
 +-----------------------+
 | [apple, pear, banana] |
 +-----------------------*/
```
