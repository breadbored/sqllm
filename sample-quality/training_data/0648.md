--- USER ---

How do I query a property graph to return basic information about all nodes, like their name and ID?

--- AGENT ---

To retrieve information from all nodes in a property graph, use a MATCH clause with a single node pattern and specify the properties you want to return. The node pattern `(n)` matches all nodes in the graph.

```googlesql
GRAPH FinGraph
MATCH (n)
RETURN n.name, n.id
```
