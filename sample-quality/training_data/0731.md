--- USER ---

How do I sort query results by a property in descending order and limit the output?

--- AGENT ---

The `ORDER BY` statement sorts results and must be immediately followed by `LIMIT` or `OFFSET` for the sort to take effect. This example returns the three transfers with the largest amounts.

```googlesql
GRAPH FinGraph
MATCH (src_account:Account)-[transfer:Transfers]->(dst_account:Account)
ORDER BY transfer.amount DESC
LIMIT 3
RETURN src_account.id AS account_id, transfer.amount AS transfer_amount
```
