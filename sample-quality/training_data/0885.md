How do I remove JSON null values from objects and arrays?

--- AGENT ---

The `JSON_STRIP_NULLS` function recursively removes JSON null values from JSON objects and arrays.

```googlesql
SELECT JSON_STRIP_NULLS(JSON '{"a": null, "b": "c"}') AS json_data
```
