## Question
Generate a BigQuery SQL query that extracts language attributes from page view events, creating semantic tags for browser language and application language within user sessions.

--- AGENT ---

This query extracts language information from page view events and creates dimensional tags for analytical purposes. It consists of two UNION ALL branches that follow the same pattern:

**First branch (browser language):**
Selects from the events table, filters for `page_view` events, and extracts the `browserLanguage` value from the nested `action_meta` JSON field. For each `sub_session_id`, it creates a `lang_browser:<value>` tag and captures the earliest app and url_path (measured by created timestamp) when that language was observed, along with the earliest timestamp itself. The result includes only non-null and non-empty language values.

**Second branch (application language):**
Identical logic but extracts the `language` field (not `browserLanguage`) from `action_meta`, creating `lang:` prefixed tags instead. Both branches are unioned together.

The output is a set of dimension records where each row represents a language attribute variant observed in a sub_session, marked as an `attribute` class, with the earliest temporal and contextual information (app, url_path, created timestamp) for that variant within that session.

```sql
SELECT
    es.sub_session_id as sub_session_id,
    CONCAT(
        'lang_browser:',
        JSON_EXTRACT_SCALAR(es.action_meta, '$.browserLanguage')
    ) as tag,
    'attribute' as class,
    MIN_BY(es.app, es.created) as app,
    MIN_BY(es.url_path, es.created) as url_path,
    MIN(es.created) as dt
FROM
    ${ref('base_buckets_events_speedy')} as es 
WHERE
    es.name = 'page_view'
AND
    JSON_EXTRACT_SCALAR(es.action_meta, '$.browserLanguage') IS NOT NULL
AND
    JSON_EXTRACT_SCALAR(es.action_meta, '$.browserLanguage') != ''
GROUP BY
    sub_session_id,
    tag

UNION ALL

SELECT
    es.sub_session_id as sub_session_id,
    CONCAT(
        'lang:',
        JSON_EXTRACT_SCALAR(es.action_meta, '$.language')
    ) as tag,
    'attribute' as class,
    MIN_BY(es.app, es.created) as app,
    MIN_BY(es.url_path, es.created) as url_path,
    MIN(es.created) as dt
FROM
    ${ref('base_buckets_events_speedy')} as es 
WHERE
    es.name = 'page_view'
AND
    JSON_EXTRACT_SCALAR(es.action_meta, '$.language') IS NOT NULL
AND
    JSON_EXTRACT_SCALAR(es.action_meta, '$.language') != ''
GROUP BY
    sub_session_id,
    tag
```
