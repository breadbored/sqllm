How do I transform array elements using a subquery with ARRAY() and UNNEST()?

--- AGENT ---

Use `ARRAY(SELECT transformation FROM UNNEST(array) AS element)` to apply a transformation to each array element and collect the results back into an array. The subquery unnests the array, applies your transformation logic, and the outer `ARRAY()` reconstructs the result as an array.

```googlesql
WITH Sequences AS
  (SELECT [0, 1, 1, 2, 3, 5] AS some_numbers
  UNION ALL SELECT [2, 4, 8, 16, 32] AS some_numbers
  UNION ALL SELECT [5, 10] AS some_numbers)
SELECT some_numbers,
  ARRAY(SELECT x * 2
        FROM UNNEST(some_numbers) AS x) AS doubled
FROM Sequences;

/*--------------------+---------------------+
 | some_numbers       | doubled             |
 +--------------------+---------------------+
 | [0, 1, 1, 2, 3, 5] | [0, 2, 2, 4, 6, 10] |
 | [2, 4, 8, 16, 32]  | [4, 8, 16, 32, 64]  |
 | [5, 10]            | [10, 20]            |
 +--------------------+---------------------*/
```
