How do I use ANY_VALUE with a window function to get a value from nearby rows?

--- AGENT ---

`ANY_VALUE` can be used as a window function to return a value from rows within a window frame. This example returns a fruit value from rows ordered by fruit length, within a frame of 1 preceding and current row:

```googlesql
SELECT
  fruit,
  ANY_VALUE(fruit) OVER (ORDER BY LENGTH(fruit) ROWS BETWEEN 1 PRECEDING AND CURRENT ROW) AS any_value
FROM UNNEST(["apple", "banana", "pear"]) as fruit;
```
