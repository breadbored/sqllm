How do I conditionally set a value only if the path exists?

--- AGENT ---

Set `create_if_missing=>false` in `JSON_SET` to only replace existing values and ignore paths that don't exist.

```googlesql
SELECT JSON_SET(
  JSON '{"a": 1}',
  "$.b", 999,
  create_if_missing => false) AS json_data
```
