Show me an example of a multi-level join and aggregation query that combines user course enrollment data with assessment metrics and progress tracking, including associated client information.

--- AGENT ---

This query constructs a denormalized user-course profile table that denormalizes enrollments with assessment completion metrics, learning progress metrics, and client associations.

The query builds three CTEs to shape the data before joining:

1. **user_assessments** — Aggregates assessment-level data at the user-course-assessment grain, counting how many assessments a user started per course and summing completions, passes, and tracking the best score.

2. **container_clients** — For each course container, aggregates the list of associated client IDs and client names into array fields (one row per container).

3. **user_course_progress** — A simple projection that aliases columns from a pre-aggregated course progress table (likely already aggregated at user-course grain).

The main SELECT joins from the user course enrollment table and brings in:
- Course metadata (name, type, version) via direct join to the courses table
- Assessment metrics via left join to the aggregated user_assessments CTE (coalescing null to 0 for assessments_started)
- Progress metrics (minutes, completion percentage, time spent) via left join to the user_course_progress CTE
- Client information via join through the course container
- User metadata (creation date) via direct join to the users table

The output is one row per user-course enrollment, with all associated assessment completions, progress metrics, and clients denormalized onto that row.

```sql
WITH 
user_assessments AS (
  SELECT
    uca.user_id as user_id,
    uca.course_id as course_id,
    uca.assessment_id as assessment_id,
    COUNT(DISTINCT uca.id) as assessments_started,
    SUM(uca.completed) as assessments_finished,
    SUM(uca.completed) as courses_completed,
    SUM(uca.passed) as course_passed,
    MAX(uca.score) as scored
  FROM
    ${ref("stg_user_course_assessments")} as uca
  GROUP BY
    uca.user_id,
    uca.course_id,
    uca.assessment_id
),

container_clients AS (
  SELECT
    ccc.course_container_id as container_id,
    ARRAY_AGG(DISTINCT ccc.client_id IGNORE NULLS ORDER BY ccc.client_id ASC) as client_id,
    ARRAY_AGG(DISTINCT sc.name IGNORE NULLS) as client_names

  FROM
    ${ref("stg_clients_course_containers")} as ccc

  LEFT JOIN
    ${ref("stg_clients")} as sc
  ON
    ccc.client_id = sc.id

  GROUP BY
    ccc.course_container_id
),

user_course_progress AS (
    SELECT
        aucp.user_id as user_id,
        aucp.course_id as course_id,
        aucp.course_updated as course_update,
        aucp.course_started as course_started,
        aucp.course_updated as course_updated_on,
        aucp.course_status as course_status,
        aucp.course_progress_minutes as course_progress_minutes,
        aucp.course_length_minutes as course_length_minutes,
        aucp.course_progress_percentage as course_complete_percent,
        aucp.time_on_course_minutes as time_on_course_minutes,
        aucp.time_on_course as time_on_course_seconds
    FROM
        ${ref("agg_user_course_progress")} as aucp
)

SELECT
  uc.user_id as user_id,
  uc.course_id as course_id,
  cc.container_id as container_id,
  c.version as version,
  c.name as course_name,
  c.course_type as course_type,
  uc.created as course_started,
  uc.updated as course_updated,
  uc.status as course_status,
  ucp.course_progress_minutes as course_progress_minutes,
  ucp.time_on_course_seconds as time_on_course_seconds,
  ucp.course_length_minutes as course_length_minutes,
  ucp.course_complete_percent as course_progress_percentage,
  ucp.time_on_course_minutes as time_on_course_minutes,
  COALESCE(ua.assessments_started, 0) as assessments_started,
  ua.assessments_finished as assessments_finished,
  ua.course_passed as assessments_passed,
  ua.scored as best_assessment_score,
  cc.client_id as client_id,
  cc.client_names as client_name,
  u.created as user_created,

FROM
  ${ref("stg_user_courses")} as uc

LEFT JOIN
  user_assessments as ua
ON
  ua.user_id = uc.user_id
AND
  ua.course_id = uc.course_id

JOIN
  ${ref("stg_courses")} as c
ON
  uc.course_id = c.id

JOIN
  container_clients as cc
ON
  cc.container_id = c.container_id

LEFT JOIN
  user_course_progress as ucp
ON
  ucp.user_id = uc.user_id
AND
  ucp.course_id = uc.course_id

JOIN
  ${ref("stg_users")} as u
ON
  u.id = uc.user_id
```
