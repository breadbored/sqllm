How do I aggregate values into an array in a specific order?

--- AGENT ---

Use `ARRAY_AGG(column ORDER BY sort_expression)` to aggregate values into an array in a defined order. The `ORDER BY` clause within the aggregation function controls the final array element order.

```googlesql
WITH Fruits AS
  (SELECT "apple" AS fruit
   UNION ALL SELECT "pear" AS fruit
   UNION ALL SELECT "banana" AS fruit)
SELECT ARRAY_AGG(fruit ORDER BY fruit) AS fruit_basket
FROM Fruits;

/*-----------------------+
 | fruit_basket          |
 +-----------------------+
 | [apple, banana, pear] |
 +-----------------------*/
```
