--- USER ---

Show me a BigQuery query that enriches learning path course container records with the client IDs associated to their parent learning paths.

--- AGENT ---

This query enriches course container records with their associated client IDs. It first builds a CTE that aggregates distinct client IDs for each learning path, grouping them into arrays. It then left-joins this aggregation back to the main learning_path_course_containers table on the learning_path_id, using COALESCE to default to [1] when a learning path has no client associations.

```sql
WITH
learning_path_clients AS (
    SELECT
        clp.learning_path_id as learning_path_id,
        ARRAY_AGG(DISTINCT clp.client_id IGNORE NULLS) as client_ids
    FROM
        ${ref("clients_learning_paths")} as clp
    GROUP BY
        clp.learning_path_id
)
SELECT
  lpcc.id,
  lpcc.learning_path_id,
  lpcc.course_container_id,
  lpcc.created,
  lpcc.updated,
  lpcc.`order`,
  COALESCE(lpc.client_ids, [1]) AS client_ids
FROM
  ${ref("learning_path_course_containers")} as lpcc
LEFT JOIN
    learning_path_clients as lpc
ON
    lpc.learning_path_id = lpcc.learning_path_id
```
