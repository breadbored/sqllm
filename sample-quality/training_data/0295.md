How do I create a foreign key constraint that references columns in another table?

--- AGENT ---

A foreign key constraint links two tables by referencing one or more columns in another table. The `CONSTRAINT` keyword lets you name the constraint, and the `FOREIGN KEY` clause specifies which columns participate in the relationship.

```googlesql
CREATE TABLE top_authors (
  author_first_name STRING,
  author_last_name STRING,
  CONSTRAINT fk_top_authors_name
    FOREIGN KEY (author_first_name, author_last_name)
    REFERENCES authors (first_name, last_name)
);
```
