How do I reference a specific row from the left table inside a subquery on the right?

--- AGENT ---

`LATERAL` enables the right side of a join (typically a subquery or UNNEST) to reference columns from the left table. The right side is re-evaluated for each row from the left.

```googlesql
SELECT R.SchoolID, OP.LastName AS TopOpPlayer, OP.PointsScored
FROM Roster AS R,
     LATERAL (
      SELECT PS.LastName, PS.PointsScored
      FROM PlayerStats AS PS
      WHERE PS.OpponentID = R.SchoolID
      ORDER BY PointsScored DESC
      LIMIT 1
     ) AS OP
ORDER BY R.SchoolID
```
