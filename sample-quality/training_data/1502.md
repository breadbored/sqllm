How do I create a TVF that calls another TVF and adds an additional filter based on a typed parameter with a default value?

--- AGENT ---

This creates a temporary table-valued function that invokes the CustomerRange TVF and applies an additional WHERE filter on customer type. It demonstrates how TVFs can call other TVFs and accept protocol buffer-typed parameters with default values.

```googlesql
CREATE TEMP TABLE FUNCTION CustomerRangeWithCustomerType(
  MinId INT64,
  MaxId INT64,
  customer_type
    ads.boulder.schema.CustomerType
      DEFAULT 'CUSTOMER_TYPE_ADVERTISER')
AS (
  SELECT *
  FROM CustomerRange(MinId, MaxId)
  WHERE type = customer_type
);
```
