Show me example SQL that enriches individual user assessment answers with question metadata and course-level assessment performance summaries.

--- AGENT ---

This query takes individual user answers to assessment questions as the base fact table and enriches each row with contextual metadata. For each answer, it retrieves: the question name, the text of the answer option the user selected, the course and container information, and the correct answer option for that question. It additionally joins assessment-level performance aggregates (count of assessments started, count of assessments completed, count of assessments passed, and best assessment score) for each user in that course. The result is one row per user answer, filtered to exclude test users (where `test_user = 0`), with all related metadata and summary statistics included.

```sql
SELECT
  ucaa.user_id,
  ucaa.course_id,
  c.container_id,
  c.name                AS course_name,
  ucaa.question_id,
  q.name                AS question_name,
  ucaa.answer_option_id AS user_answer_id,
  qo.name               AS user_answer,
  qs.correct_option_id  AS correct_answer_id,
  con.name              AS correct_answer,
  ucaa.created          AS answer_date,
  ucaa.assessment_id,
  count(uca.completed)  AS assessments_started,
  sum(uca.completed)    AS assessments_finished,
  sum(uca.passed)       AS assessments_passed,
  max(uca.score)        AS best_assessment_score
FROM ${ref("stg_user_course_assessment_answers")} AS ucaa
LEFT JOIN ${ref("stg_questions")} AS q ON ucaa.question_id = q.id
LEFT JOIN ${ref("stg_question_options")} AS qo ON ucaa.answer_option_id = qo.id
LEFT JOIN ${ref("stg_courses")} AS c ON ucaa.course_id = c.id
LEFT JOIN ${ref("stg_question_sets")} AS qs ON ucaa.question_id = qs.question_id
LEFT JOIN (
  SELECT
    qo.name,
    qs.correct_option_id
  FROM ${ref("stg_question_sets")} AS qs
  LEFT JOIN ${ref("stg_question_options")} AS qo
    ON qo.id = qs.correct_option_id
) AS con
  ON qs.correct_option_id = con.correct_option_id
LEFT JOIN ${ref("stg_users")} AS u ON ucaa.user_id = u.id
LEFT JOIN ${ref("stg_user_course_assessments")} AS uca
  ON
    ucaa.user_id = uca.user_id AND ucaa.course_id = uca.course_id
    AND ucaa.assessment_id = uca.assessment_id
WHERE test_user = 0
GROUP BY
  ucaa.user_id, ucaa.course_id, c.container_id, c.name, ucaa.question_id, q.name, ucaa.answer_option_id, qo.name,
  qs.correct_option_id, con.name, ucaa.created, ucaa.assessment_id
```
