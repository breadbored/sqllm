How do I retrieve all rows from a table using pipe syntax?

--- AGENT ---

The simplest pipe query starts with a `FROM` clause and returns all rows and columns. When no pipe operators follow the `FROM` clause, the result is equivalent to `SELECT *` in standard syntax.

```googlesql
FROM Produce;
```
