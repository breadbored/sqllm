--- USER ---

How can I find the most popular item within a three-row window for each category?

--- AGENT ---

This query partitions by category and uses LAST_VALUE with a bounded window frame of 1 row before and after the current row. Unlike calculating across the entire category, this shows the most popular item within a local window.

```googlesql
SELECT item, purchases, category, LAST_VALUE(item)
  OVER (
    PARTITION BY category
    ORDER BY purchases
    ROWS BETWEEN 1 PRECEDING AND 1 FOLLOWING
  ) AS most_popular
FROM Produce
```
