How do I create a JSON object with null values?

--- AGENT ---

The `JSON_OBJECT` function converts SQL `NULL` to JSON `null` and accepts JSON `null` literals as values.

```googlesql
SELECT JSON_OBJECT('a', NULL, 'b', JSON 'null') AS json_data
```
