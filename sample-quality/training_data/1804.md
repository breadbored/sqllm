## Question
I need a BigQuery query that calculates cumulative course engagement metrics by month for a specific client. What would that look like?

--- AGENT ---

This query is a Dataform table function that accepts a clientId parameter and produces a monthly time-series of cumulative course engagement metrics (in hours).

The logic flows in three stages:

1. **MonthlyData CTE**: Aggregates from the reporting_user_courses table, filtering rows where the clientId matches the input parameter. It groups by month-year (formatted as "Mon YY") and computes two sums: total_course_progress_minutes and total_time_on_course_minutes. Results are sorted chronologically.

2. **CumulativeData CTE**: Applies SUM() window functions to the monthly totals, ordered by date, producing cumulative running sums for both metrics across the time period.

3. **Final SELECT**: Converts the cumulative minute values to hours by dividing by 60 and rounding, returning the month-year alongside the two cumulative hour metrics (watch_time and course_time).

The output is a month-by-month breakdown showing how total course progress and total course engagement time accumulate over time for the specified client.

```sql
CREATE OR REPLACE TABLE
  FUNCTION ${self()} (clientId INT64) AS (
  WITH
    MonthlyData AS (
    SELECT
      FORMAT_DATE('%b %y', course_started) AS date,
      SUM(course_progress_minutes) AS total_course_progress_minutes,
      SUM(time_on_course_minutes) AS total_time_on_course_minutes
    FROM
      ${ref("reporting_user_courses")}
    WHERE
      clientId IN UNNEST(client_id)
    GROUP BY
      date
    ORDER BY
      PARSE_DATE('%b %y', date) ),
    CumulativeData AS (
    SELECT
      date,
      SUM(total_course_progress_minutes) OVER (ORDER BY PARSE_DATE('%b %y', date)) AS cumulative_course_progress_minutes,
      SUM(total_time_on_course_minutes) OVER (ORDER BY PARSE_DATE('%b %y', date)) AS cumulative_time_on_course_minutes
    FROM
      MonthlyData )
  SELECT
    date,
    ROUND(cumulative_course_progress_minutes / 60) AS watch_time,
    ROUND(cumulative_time_on_course_minutes / 60) AS course_time
  FROM
    CumulativeData)
```
