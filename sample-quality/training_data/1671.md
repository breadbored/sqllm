--- USER ---

Show me how to extract AB test group assignments from event metadata and create both detailed and generic group labels per session.

--- AGENT ---

This query produces two overlapping sets of tags from event data, both identifying AB test group assignments by session and app.

**First branch:** Extracts the numeric `abGroup` value from the JSON `action_meta` field, maps it to a letter (1→A, 2→B, ..., 7→G), and creates a tag like `AB_group_A`, `AB_group_B`, etc. For each session/app/computed-tag group, it returns the earliest URL path (by creation timestamp) and the earliest event timestamp.

**Second branch:** From the same filtered events, creates a generic `AB_group` tag (constant string, not computed) and groups by session/app. It returns the same columns: earliest URL path and earliest timestamp per group.

The two branches are unioned, so the output contains both granular group assignments (AB_group_A through AB_group_G) and a flat indicator (AB_group) for each session/app combination. Both use `MIN_BY(e.url_path, e.created)` to select the URL path that corresponds to the earliest event in each group.

```sql
SELECT
  e.sub_session_id as sub_session_id,
  CONCAT(
    'AB_group_',
    CASE
      WHEN SAFE_CAST(JSON_EXTRACT_SCALAR(e.action_meta, '$.abGroup') AS INT64) = 1 THEN 'A'
      WHEN SAFE_CAST(JSON_EXTRACT_SCALAR(e.action_meta, '$.abGroup') AS INT64) = 2 THEN 'B'
      WHEN SAFE_CAST(JSON_EXTRACT_SCALAR(e.action_meta, '$.abGroup') AS INT64) = 3 THEN 'C'
      WHEN SAFE_CAST(JSON_EXTRACT_SCALAR(e.action_meta, '$.abGroup') AS INT64) = 4 THEN 'D'
      WHEN SAFE_CAST(JSON_EXTRACT_SCALAR(e.action_meta, '$.abGroup') AS INT64) = 5 THEN 'E'
      WHEN SAFE_CAST(JSON_EXTRACT_SCALAR(e.action_meta, '$.abGroup') AS INT64) = 6 THEN 'F'
      WHEN SAFE_CAST(JSON_EXTRACT_SCALAR(e.action_meta, '$.abGroup') AS INT64) = 7 THEN 'G'
      ELSE 'invalid'
    END
  ) as tag,
  'attribute' as class,
  e.app as app,
  MIN_BY(e.url_path, e.created) as url_path,
  MIN(e.created) as dt
FROM
  ${ref("base_buckets_events_app_speedy")} as e
WHERE
  JSON_EXTRACT_SCALAR(e.action_meta, '$.abGroup') IS NOT NULL
GROUP BY
  e.sub_session_id,
  e.app,
  tag

UNION ALL

SELECT
  e.sub_session_id as sub_session_id,
  'AB_group' as tag,
  'attribute' as class,
  e.app as app,
  MIN_BY(e.url_path, e.created) as url_path,
  MIN(e.created) as dt
FROM 
  ${ref(`base_buckets_events_app_speedy`)} as e
WHERE
  JSON_EXTRACT_SCALAR(e.action_meta, '$.abGroup') IS NOT NULL
GROUP BY
  e.sub_session_id,
  e.app,
  tag
```
