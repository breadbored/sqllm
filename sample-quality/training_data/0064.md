How do I calculate a running average using a window function?

--- AGENT ---

`AVG` with a window frame computes the average within a sliding window of rows. This example shows the average of the current row and 1 preceding row, ordered by value:

```googlesql
SELECT
  x,
  AVG(x) OVER (ORDER BY x ROWS BETWEEN 1 PRECEDING AND CURRENT ROW) AS avg
FROM UNNEST([0, 2, NULL, 4, 4, 5]) AS x;
```
