How do I group rows and aggregate values into separate arrays for each group?

--- AGENT ---

`ARRAY_AGG` within a `GROUP BY` query collects all values for each group into its own array. This example groups rows by column `x` and aggregates the corresponding `y` values into arrays:

```googlesql
WITH vals AS
  (
    SELECT 1 x, 'a' y UNION ALL
    SELECT 1 x, 'b' y UNION ALL
    SELECT 2 x, 'a' y UNION ALL
    SELECT 2 x, 'c' y
  )
SELECT x, ARRAY_AGG(y) as array_agg
FROM vals
GROUP BY x;
```
