How do I calculate the number of hours between two timestamps?

--- AGENT ---

The `TIMESTAMP_DIFF` function computes the number of complete unit boundaries between two timestamps at a specified granularity. Using `HOUR` as the granularity returns the number of whole hours between the end and start timestamps.

```googlesql
SELECT
  TIMESTAMP("2010-07-07 10:20:00+00") AS later_timestamp,
  TIMESTAMP("2008-12-25 15:30:00+00") AS earlier_timestamp,
  TIMESTAMP_DIFF(TIMESTAMP "2010-07-07 10:20:00+00", TIMESTAMP "2008-12-25 15:30:00+00", HOUR) AS hours;
```
