How do I compute a differentially private sum over a view with explicit bounds?

--- AGENT ---

This query applies `SUM` with `DIFFERENTIAL_PRIVACY` to a view, using per-group contribution bounds to clamp values before aggregation. Noise is added based on the epsilon parameter.

```googlesql
SELECT
  WITH DIFFERENTIAL_PRIVACY
    OPTIONS(epsilon=10, delta=.01, max_groups_contributed=1)
    item,
    SUM(quantity, contribution_bounds_per_group=>(0, 100)) quantity
FROM {{USERNAME}}.view_on_professors
GROUP BY item;
```
