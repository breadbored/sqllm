## Question
Show me an example of a complex multi-table event enrichment query that backfills user IDs, forwardfills UTM parameters, detects UTM changes within sessions, and assigns A/B test groups.

--- AGENT ---

This query enriches a base events table with multiple layers of contextual data. It performs several enrichment tasks in sequence:

**Course and Document ID Backfilling**: The `course_id_fill` CTE recovers missing course IDs from the URL path using regex extraction. The `doc_id_fill` CTE joins events to a documents table using path pattern matching to fill in document identifiers.

**GeoIP Aggregation**: The `dd_data_by_ip` CTE groups GeoIP records by IP address and extracts the earliest occurrence of city, country, continent, and coordinates for each IP using `MIN_BY`.

**User and UTM Backfilling/Forwardfilling**: The `combined_event_fills` CTE joins three upstream layers (events_user_backfill_session, events_user_backfill_subsession, and events_start_url_forwardfill) to collect backfilled user IDs and forwardfilled UTM/referrer data.

**UTM Change Detection**: The `utm_changes` CTE uses window functions (LAG) to detect when any of the six UTM fields or referrer change within a session. The `derived_utms_pre` CTE then counts cumulative UTM changes and isolates only the values that changed. The `derived_utms` CTE aggregates these by subsession and app, preserving the earliest change.

**Final Assembly**: The main SELECT joins the base events table to all these enrichments via LEFT JOIN, also pulling in time-on-page, scroll tracking, backlinks, GeoIP, and human-readable descriptions. It concludes by computing A/B test group assignments (binary, ternary, and quaternary) by hashing the session ID.

```sql
/**
* Events Fill
*
* This layer is used to join all the 060 and 070 layers together to get the user ID, path, referrer, and UTM backfills/forwardfills on the event
**/
WITH
  course_id_fill AS (
    SELECT
        e.id as id,
        COALESCE(e.course_id, SAFE_CAST(REGEXP_EXTRACT(e.url_path, r'^/(?:courses|short-codes)/(\d+)', 2, 1) AS INT64)) as course_id
    FROM
        ${ref("events_numbered")} AS e
  ),
  doc_id_fill AS (
    SELECT
        e.id as id,
        MAX(COALESCE(e.doc_id, d.id)) as doc_id
    FROM
        ${ref("events_numbered")} AS e
    LEFT JOIN
        ${ref("documents")} as d
    ON
        e.url_path LIKE CONCAT(d.path, '%')
    GROUP BY
        e.id
  ),
  dd_data_by_ip AS (
    SELECT
        dd.ip_addr as ip_addr,
        MIN_BY(dd.city, dd.`date`) AS dd_geo_city,
        MIN_BY(dd.country_iso_code, dd.`date`) AS dd_geo_country,
        MIN_BY(dd.continent, dd.`date`) AS dd_geo_continent,
        MIN_BY(dd.longitude, dd.`date`) AS dd_geo_long,
        MIN_BY(dd.latitude, dd.`date`) AS dd_geo_lat
    FROM
        ${ref("dd_geo")} AS dd
    GROUP BY
        dd.ip_addr
  ),
  combined_event_fills AS (
    SELECT
      euid.id,
      euid.user_id AS backfilled_user_id,
      eussp.user_id AS subsession_backfilled_id,
      eup.session_start_url_path,
      eup.subsession_start_url_path,
      eup.subsession_start_utm_campaign,
      eup.subsession_start_utm_content,
      eup.subsession_start_utm_medium,
      eup.subsession_start_utm_source,
      eup.subsession_start_utm_term,
      eup.subsession_start_referrer
    FROM
      ${ref("events_user_backfill_session")} AS euid
      LEFT JOIN ${ref("events_user_backfill_subsession")} AS eussp ON euid.id = eussp.id
      LEFT JOIN ${ref("events_start_url_forwardfill")} AS eup ON euid.id = eup.id
  ),
  utm_changes AS (
    SELECT
        ne.id AS id,
        ne.session_id AS session_id,
        ne.sub_session_id AS sub_session_id,
        ne.app AS app,
        ne.created AS created,
        ne.utm_campaign AS utm_campaign,
        ne.utm_content AS utm_content,
        ne.utm_medium AS utm_medium,
        ne.utm_source AS utm_source,
        ne.utm_term AS utm_term,
        ne.referrer AS referrer,
        CASE
            WHEN 
                LAG(ne.utm_campaign) OVER (PARTITION BY ne.session_id ORDER BY ne.created ASC) != ne.utm_campaign 
            OR
                LAG(ne.utm_content) OVER (PARTITION BY ne.session_id ORDER BY ne.created ASC) != ne.utm_content
            OR
                LAG(ne.utm_medium) OVER (PARTITION BY ne.session_id ORDER BY ne.created ASC) != ne.utm_medium
            OR
                LAG(ne.utm_source) OVER (PARTITION BY ne.session_id ORDER BY ne.created ASC) != ne.utm_source
            OR
                LAG(ne.utm_term) OVER (PARTITION BY ne.session_id ORDER BY ne.created ASC) != ne.utm_term
            OR
                LAG(ne.referrer) OVER (PARTITION BY ne.session_id ORDER BY ne.created ASC) != ne.referrer
            THEN true
            ELSE false
        END as utm_changed
    FROM
        ${ref("events_numbered")} AS ne
  ),
  derived_utms_pre AS (
    SELECT
        ne.id,
        ne.session_id as session_id,
        ne.sub_session_id as sub_session_id,
        ne.app as app,
        ne.created as created,
        SUM(
            CASE
                WHEN ne.utm_changed
                THEN 1
                ELSE 0
            END
        ) OVER (PARTITION BY ne.session_id ORDER BY ne.created) AS utm_change_num,
        CASE
            WHEN ne.utm_changed
            THEN ne.utm_campaign
            ELSE NULL
        END AS derived_utm_campaign,
        CASE
            WHEN ne.utm_changed
            THEN ne.utm_content
            ELSE NULL
        END AS derived_utm_content,
        CASE
            WHEN ne.utm_changed
            THEN ne.utm_medium
            ELSE NULL
        END AS derived_utm_medium,
        CASE
            WHEN ne.utm_changed
            THEN ne.utm_source
            ELSE NULL
        END AS derived_utm_source,
        CASE
            WHEN ne.utm_changed
            THEN ne.utm_term
            ELSE NULL
        END AS derived_utm_term,
        CASE
            WHEN ne.utm_changed
            THEN ne.referrer
            ELSE NULL
        END AS derived_referrer
    FROM
        utm_changes AS ne
  ),
  derived_utms AS (
    SELECT
        e.sub_session_id,
        e.app,
        MIN_BY(e.utm_change_num, e.created) as utm_change_num,
        MIN_BY(e.derived_utm_campaign, e.created) as derived_utm_campaign,
        MIN_BY(e.derived_utm_content, e.created) as derived_utm_content,
        MIN_BY(e.derived_utm_medium, e.created) as derived_utm_medium,
        MIN_BY(e.derived_utm_source, e.created) as derived_utm_source,
        MIN_BY(e.derived_utm_term, e.created) as derived_utm_term,
        MIN_BY(e.derived_referrer, e.created) as derived_referrer
    FROM
        derived_utms_pre as e
    GROUP BY
        e.sub_session_id,
        e.app
  )
SELECT
  efill.backfilled_user_id AS backfilled_by_session_user_id,
  efill.subsession_backfilled_id AS backfilled_by_subsession_user_id,
  efill.session_start_url_path AS forwardfilled_by_session_url_path,
  efill.subsession_start_url_path AS forwardfilled_by_subsession_url_path,
  efill.subsession_start_utm_campaign,
  efill.subsession_start_utm_content,
  efill.subsession_start_utm_medium,
  efill.subsession_start_utm_source,
  efill.subsession_start_utm_term,
  efill.subsession_start_referrer,
  _top.time_on_page AS seconds_on_page,
  ROUND(_top.time_on_page / 60, 2) AS minutes_on_page,
  hd.text_description AS text_description,
  ne.*,
  COALESCE(
    cif.course_id,
    CASE WHEN ne.related_entity = 'courses' THEN SAFE_CAST(ne.related_id AS INT64) ELSE NULL END,
    SAFE_CAST(REGEXP_EXTRACT(ne.url_path, r's/(\d+)$', 1, 1) AS INT64)
  ) AS derived_course_id,
  dif.doc_id AS derived_doc_id,
  es.scroll_direction,
  es.scroll_num,
  es.scroll_value,
  es.scroll_changed,
  eb.landed_from AS landed_from,
  ddip.dd_geo_city,
  ddip.dd_geo_continent,
  ddip.dd_geo_country,
  ddip.dd_geo_lat,
  ddip.dd_geo_long,
  dutm.utm_change_num,
  dutm.derived_utm_campaign,
  dutm.derived_utm_content,
  dutm.derived_utm_medium,
  dutm.derived_utm_source,
  dutm.derived_utm_term,
  dutm.derived_referrer,
  cookies.accepted_cookies,
  CASE
    WHEN 2 <= 0 THEN ERROR('Number of groups must be positive')
    WHEN ne.session_id IS NULL 
      OR NOT REGEXP_CONTAINS(ne.session_id, r'(?i)^[0-9a-f]{8}-[0-9a-f]{4}-7[0-9a-f]{3}-[89ab][0-9a-f]{3}-[0-9a-f]{12}$') 
      THEN -1
    ELSE MOD(
      (SELECT CAST(CONCAT('0x', SPLIT(ne.session_id, '-')[OFFSET(4)]) AS INT64)),
      2
    )
  END AS ab_group,
  CASE
    WHEN 3 <= 0 THEN ERROR('Number of groups must be positive')
    WHEN ne.session_id IS NULL 
      OR NOT REGEXP_CONTAINS(ne.session_id, r'(?i)^[0-9a-f]{8}-[0-9a-f]{4}-7[0-9a-f]{3}-[89ab][0-9a-f]{3}-[0-9a-f]{12}$') 
      THEN -1
    ELSE MOD(
      (SELECT CAST(CONCAT('0x', SPLIT(ne.session_id, '-')[OFFSET(4)]) AS INT64)),
      3
    )
  END AS abc_group,
  CASE
    WHEN 4 <= 0 THEN ERROR('Number of groups must be positive')
    WHEN ne.session_id IS NULL 
      OR NOT REGEXP_CONTAINS(ne.session_id, r'(?i)^[0-9a-f]{8}-[0-9a-f]{4}-7[0-9a-f]{3}-[89ab][0-9a-f]{3}-[0-9a-f]{12}$') 
      THEN -1
    ELSE MOD(
      (SELECT CAST(CONCAT('0x', SPLIT(ne.session_id, '-')[OFFSET(4)]) AS INT64)),
      4
    )
  END AS abcd_group
FROM
  ${ref("events_numbered")} AS ne
  LEFT JOIN course_id_fill AS cif
  ON cif.id = ne.id
  LEFT JOIN doc_id_fill AS dif
  ON dif.id = ne.id
  LEFT JOIN derived_utms AS dutm
  ON dutm.sub_session_id = ne.sub_session_id AND dutm.app = ne.app
  LEFT JOIN combined_event_fills AS efill 
  ON ne.id = efill.id
  LEFT JOIN ${ref("events_time_on_page_backfill")} AS _top 
  ON _top.sub_session_id = ne.sub_session_id
  AND _top.page_visit = ne.page_visit
  AND _top.url_path = ne.url_path
  LEFT JOIN ${ref("events_human")} AS hd 
  ON hd.id = ne.id
  LEFT JOIN ${ref("events_backlinks")} as eb 
  ON eb.session_id = ne.session_id AND eb.landing_url = ne.landing_url
  LEFT JOIN ${ref("events_scrolls")} as es 
  ON es.id = ne.id
  LEFT JOIN dd_data_by_ip AS ddip
  ON ddip.ip_addr = ne.ip
  LEFT JOIN ${ref("events_cookies")} as cookies
  ON cookies.id = ne.id
```
