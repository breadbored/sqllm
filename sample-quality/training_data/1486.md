How does TRIM interpret characters as Unicode code points?

--- AGENT ---

The TRIM function operates on Unicode code points rather than bytes; combining diacritics in the character set can affect similar marks elsewhere.

```googlesql
SELECT
  TRIM('abaW̊', 'Y̊') AS a,
  TRIM('W̊aba', 'Y̊') AS b,
  TRIM('abaŪ̊', 'Y̊') AS c,
  TRIM('Ū̊aba', 'Y̊') AS d
```
