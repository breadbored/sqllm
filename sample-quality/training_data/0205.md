How do I filter array elements and return the filtered result as an array?

--- AGENT ---

Use `ARRAY(SELECT element FROM UNNEST(array) AS element WHERE condition)` to apply a filter within the subquery and reconstruct the filtered result as an array. Elements that don't match the condition are excluded from the result.

```googlesql
WITH Sequences AS
  (SELECT [0, 1, 1, 2, 3, 5] AS some_numbers
   UNION ALL SELECT [2, 4, 8, 16, 32] AS some_numbers
   UNION ALL SELECT [5, 10] AS some_numbers)
SELECT
  ARRAY(SELECT x * 2
        FROM UNNEST(some_numbers) AS x
        WHERE x < 5) AS doubled_less_than_five
FROM Sequences;

/*------------------------+
 | doubled_less_than_five |
 +------------------------+
 | [0, 2, 2, 4, 6]        |
 | [4, 8]                 |
 | []                     |
 +------------------------*/
```
