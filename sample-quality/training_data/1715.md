How do I extract the first answer_question event from each user session along with its metadata?

--- AGENT ---

This query identifies all 'answer_question' events from an events table, groups them by sub_session_id, and extracts metadata about the earliest event in each session. For each sub_session that contains answer_question events, it produces a single row capturing:

- The app and url_path values from the earliest event in that session (using MIN_BY to retrieve these values at the minimum created timestamp)
- The creation timestamp of the earliest event
- Static classification tags ('answer_question' and 'behavior') for labeling

The WHERE clause filters for rows where the event name is exactly 'answer_question'. The GROUP BY collapses all events within each sub_session_id into a single aggregated row, effectively creating a tagged event record for each session that performed answer_question actions.

SELECT
    es.sub_session_id as sub_session_id,
    'answer_question' as tag,
    'behavior' as class,
    MIN_BY(es.app, es.created) as app,
    MIN_BY(es.url_path, es.created) as url_path,
    MIN(es.created) as dt

FROM
    ${ref('base_buckets_events_docs_speedy')} as es 

WHERE
    es.name != 'page_view'
AND
    es.name != 'open_link'
AND
    es.name = 'answer_question'

GROUP BY
    sub_session_id,
    tag
