How do I create a value table with a STRUCT that has multiple columns?

--- AGENT ---

The `SELECT AS VALUE` statement wraps query results as a value table, where each row is a single STRUCT value with no separate column names. This example creates a value table with a STRUCT containing two fields.

```googlesql
SELECT * FROM (SELECT AS VALUE STRUCT(123 AS a, FALSE AS b))
```
