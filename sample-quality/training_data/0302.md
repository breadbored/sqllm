How do I combine rows from multiple tables into a single new table?

--- AGENT ---

The `UNION ALL` operator in a `CREATE TABLE AS` query combines the results of multiple `SELECT` statements. Each `SELECT` retrieves rows from a different source table and appends them together.

```googlesql
CREATE TABLE books AS (
  SELECT * FROM old_books UNION ALL
  SELECT * FROM ancient_books);
```
