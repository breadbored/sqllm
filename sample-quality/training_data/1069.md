How do I compute an aggregate result and reuse it in a calculation?

--- AGENT ---

The WITH expression allows you to assign aggregate function results to variables, which can then be used in subsequent expressions without re-executing the aggregation.

```googlesql
SELECT WITH(s AS SUM(input), c AS COUNT(input), s/c)
FROM UNNEST([1.0, 2.0, 3.0]) AS input
```
