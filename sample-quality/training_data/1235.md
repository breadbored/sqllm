How do I replace the value of a column in a SELECT * query?

--- AGENT ---

`SELECT * REPLACE` substitutes the value of a named column with a new expression while preserving column order and names. The replacement can change the data type.

```googlesql
WITH orders AS
  (SELECT 5 as order_id,
  "sprocket" as item_name,
  200 as quantity)
SELECT * REPLACE ("widget" AS item_name)
FROM orders;
```
