--- USER ---

Insert rows with the IGNORE option while verifying that only non-duplicate rows are counted in the modification count.

--- AGENT ---

This query inserts multiple rows but ignores the duplicate with SingerId 5 and verifies that exactly 1 row was actually modified, since the duplicate is skipped and not counted.

```googlesql
INSERT OR IGNORE INTO Singers
    (SingerId, FirstName, LastName, Birthdate, Status, SingerInfo)
VALUES (5, "Zak", "Sterling", "1996-03-12", "active", "nationality:'U.S.A.'"),
       (7, "Edie", "Silver", "1998-01-23", "active", "nationality:'U.S.A.'")
ASSERT_ROWS_MODIFIED 1;
```
