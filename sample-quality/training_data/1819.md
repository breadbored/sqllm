## Question
Provide a BigQuery table function that returns monthly user cohort data — the count of users taking their first course each month, along with a running cumulative total for a specific client.

--- AGENT ---

This BigQuery table function accepts a client ID and returns monthly cohort metrics for that client's user base.

The query works in three layers:

**Layer 1** (`first_course_id`): Identifies each user's first course enrollment by finding the minimum course ID in user_courses, filtering to courses that belong to the specified client (via the client_courses mapping). This produces a normalized set of (user_id, first_course_id) pairs.

**Layer 2** (`first_course`): Retrieves the full course enrollment record for each user's first course by joining back to user_courses on the first course ID.

**Layer 3** (final SELECT): Aggregates enrollment creation dates to the month level, counts distinct users per month, formats the month as text ('Jan 26' etc.), and computes a cumulative running sum of all users who have enrolled by month-end.

The output has three columns:
- `year_month`: Month formatted as 'MMM YY'
- `monthly_users`: Count of users enrolling for the first time in that month
- `cumulative_users`: Running total of all users who have ever enrolled through that month

```sql
CREATE OR REPLACE TABLE
  FUNCTION ${self()} (clientId INT64) AS (
  WITH
    first_course_id AS (
    SELECT
      MIN(uc.id) AS id,
      uc.user_id
    FROM
      ${ref("stg_user_courses")} AS uc
    JOIN
      ${ref("stg_courses")} AS c
    ON
      c.id = uc.course_id
    JOIN
      ${ref("agg_client_courses")} AS cc
    ON
      cc.container_id = c.container_id
    JOIN
      ${ref("stg_users")} u
    ON
      u.id = uc.user_id
    WHERE
      cc.client_id = clientId
    GROUP BY
      uc.user_id),
    first_course AS (
    SELECT
      uc.*
    FROM
      ${ref("stg_user_courses")} AS uc
    JOIN
      first_course_id AS _uc
    ON
      uc.id = _uc.id)
  SELECT
    FORMAT_DATE('%b %y', oc.year_month) AS year_month,
    oc.users AS monthly_users,
    SUM(oc.users) OVER (ORDER BY oc.year_month) AS cumulative_users
  FROM (
    SELECT
      DATETIME_TRUNC(fc.created, MONTH) AS `year_month`,
      COUNT(*) AS users
    FROM
      first_course AS fc
    GROUP BY
      year_month
    ORDER BY
      year_month) AS oc)
```
