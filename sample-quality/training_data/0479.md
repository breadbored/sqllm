How do I parse a string with month, day, year, and time in a custom format into a DATETIME value?

--- AGENT ---

PARSE_DATETIME converts a string to a DATETIME value using a format string. The format string %m/%d/%Y %I:%M:%S %p matches month/day/year and 12-hour time with AM/PM indicator.

```googlesql
SELECT PARSE_DATETIME('%m/%d/%Y %I:%M:%S %p', '8/30/2018 2:23:38 pm') AS datetime;
```
