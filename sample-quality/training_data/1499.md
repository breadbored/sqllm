I need to randomly select approximately half of the players from the Players table.

--- AGENT ---

A volatile subquery produces non-deterministic results because it includes a function like RAND() that doesn't return the same value on each call. The same query executed multiple times may return different rows.

```googlesql
SELECT
  results.username
FROM
  (SELECT * FROM Players WHERE RAND() < 0.5) AS results;
```
