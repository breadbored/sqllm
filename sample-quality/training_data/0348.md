--- USER ---

Delete inventory rows and return the data of all deleted items.

--- AGENT ---

This query removes all inventory rows for the product 'washer' and uses THEN RETURN to retrieve all column values from the deleted rows before they are removed.

```googlesql
DELETE FROM Inventory
WHERE product = 'washer'
THEN RETURN *;
```
