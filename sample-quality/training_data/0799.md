How do I convert a JSON array of integers to a SQL INT32 array?

--- AGENT ---

The `INT32_ARRAY` function converts a JSON array of numbers to a SQL `ARRAY<INT32>` value, provided all values are within the INT32 domain. Decimal values with zero fractional parts are accepted.

```googlesql
SELECT INT32_ARRAY(JSON '[2005, 2003]') AS flight_numbers;
```
