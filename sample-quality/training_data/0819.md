How do I insert an array as a single element into a JSON array?

--- AGENT ---

Set `insert_each_element=>FALSE` in `JSON_ARRAY_INSERT` to insert an entire array as one element rather than expanding it.

```googlesql
SELECT JSON_ARRAY_INSERT(
  JSON '["a", "b", "c"]',
  '$[1]', [1, 2],
  insert_each_element=>FALSE) AS json_data
```
