How do I compute the smallest convex geography that encloses a set of points?

--- AGENT ---

`ST_CONVEXHULL` returns the convex hull, which is the smallest convex geography that contains all points in the input, and can return a point, linestring, or polygon depending on the input.

```googlesql
WITH Geographies AS
 (SELECT ST_GEOGFROMTEXT('POINT(1 1)') AS g UNION ALL
  SELECT ST_GEOGFROMTEXT('LINESTRING(1 1, 2 2)') AS g UNION ALL
  SELECT ST_GEOGFROMTEXT('MULTIPOINT(2 11, 4 12, 0 15, 1 9, 1 12)') AS g)
SELECT
  g AS input_geography,
  ST_CONVEXHULL(g) AS convex_hull
FROM Geographies;
```
