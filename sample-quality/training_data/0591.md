How do I test whether a geography intersects a rectangular region defined by longitude and latitude bounds?

--- AGENT ---

`ST_INTERSECTSBOX` returns TRUE if a geography intersects the rectangular region bounded by the specified west/east longitude and south/north latitude coordinates.

```googlesql
SELECT p, ST_INTERSECTSBOX(p, -90, 0, 90, 20) AS box1,
       ST_INTERSECTSBOX(p, 90, 0, -90, 20) AS box2
FROM UNNEST([ST_GEOGPOINT(10, 10), ST_GEOGPOINT(170, 10),
             ST_GEOGPOINT(30, 30)]) p
```
