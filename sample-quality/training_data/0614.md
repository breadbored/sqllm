How can I check if a graph path has any repeated edges?

--- AGENT ---

The `IS_TRAIL` function returns `TRUE` if a graph path contains no repeated edges, and `FALSE` if any edge appears more than once. This allows you to identify paths that use the same relationship multiple times.

```googlesql
GRAPH FinGraph
MATCH
  p=(a1:Account)-[t1:Transfers]->(a2:Account)-[t2:Transfers]->
    (a3:Account)-[t3:Transfers]->(a4:Account)
WHERE a1.id < a4.id
RETURN
  IS_TRAIL(p) AS is_trail_path, t1.id as t1_id, t2.id as t2_id, t3.id as t3_id
```
