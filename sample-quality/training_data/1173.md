How do I exit a loop immediately when a condition is met?

--- AGENT ---

Use `BREAK` (or its synonym `LEAVE`) inside the loop to exit immediately. This stops execution of the loop body and jumps to the statement after the loop.

```googlesql
DECLARE heads BOOL;
DECLARE heads_count INT64 DEFAULT 0;
LOOP
  SET heads = RAND() < 0.5;
  IF heads THEN
    SELECT 'Heads!';
    SET heads_count = heads_count + 1;
  ELSE
    SELECT 'Tails!';
    BREAK;
  END IF;
END LOOP;
SELECT CONCAT(CAST(heads_count AS STRING), ' heads in a row');
```
