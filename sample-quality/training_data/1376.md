How do I use custom delimiters to define word boundaries in INITCAP?

--- AGENT ---

The INITCAP function accepts an optional second argument specifying custom delimiters to override the default set used to identify word boundaries.

```googlesql
SELECT
  'Apples1oranges2pears' as value,
  '12' AS delimiters,
  INITCAP('Apples1oranges2pears' , '12') AS initcap_value
```
