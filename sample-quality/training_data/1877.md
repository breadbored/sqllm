## Question
Show me an example BigQuery query that retrieves feedback records with associated course and client information, filtered by recency and content constraints.

--- AGENT ---

This query retrieves feedback records enriched with course client information and user data, filtered by specific conditions on recency and content state.

The query works in two stages:

**Stage 1 (CTE `course_clients`):** Pre-aggregates all distinct client IDs associated with each course by joining the clients_course_containers table to courses, producing a denormalized lookup table with one row per course and an array of associated client IDs.

**Stage 2 (Main SELECT):** Joins the feedback table to this pre-aggregated course_clients lookup, adds user information, and optionally includes cleared feedback metadata. It applies field transformations: the response_text is lowercased only when there is no associated cleared feedback record (comment_only is NULL), otherwise it outputs an empty string. The course_completed flag is mapped to a human-readable status. The client_ids array defaults to [1] if no course clients exist.

The WHERE clause filters for feedback that is either: (1) at least 24 hours old, (2) has missing or empty response text, or (3) has no cleared feedback record or a cleared feedback record marked as comment-only.

```sql
WITH
course_clients AS (
  SELECT
    c.id as course_id,
    ARRAY_AGG(DISTINCT ccc.client_id IGNORE NULLS) as client_ids
  FROM
    ${ref("clients_course_containers")} as ccc
  JOIN
    ${ref("courses")} as c
  ON
    c.container_id = ccc.course_container_id
  GROUP BY
    c.id
)
SELECT
  f.id,
  f.response_sentiment,
  LOWER(
    CASE WHEN cf.comment_only IS NULL
      THEN f.response_text
      ELSE ''
    END
  ) AS response_text,
  f.video_seconds,
  f.user_id,
  f.course_id,
  f.lesson_id,
  f.course_completed,
  f.created,
  f.updated,
  CASE
    WHEN f.course_completed = 0 THEN 'in_progress'
    WHEN f.course_completed = 1 THEN 'completed'
  END AS status,
  COALESCE(cc.client_ids, [1]) AS client_ids
FROM
  ${ref("feedback")} AS f
LEFT JOIN
  course_clients as cc
ON
  cc.course_id = f.course_id
JOIN
  ${ref("stg_users")} AS u
ON
  f.user_id = u.id
LEFT JOIN
  ${ref("stg_cleared_feedbacks")} as cf
ON
  cf.feedback_id = f.id
WHERE
  DATETIME_DIFF(CURRENT_DATETIME, f.created, HOUR) >= 24 
OR 
  f.response_text IS NULL 
OR 
  f.response_text = ''
OR
  (cf.comment_only IS NULL OR cf.comment_only = true)
```
