How do I create a table with a named column that contains protocol buffer values?

--- AGENT ---

The `CREATE TABLE AS SELECT` statement without `SELECT AS VALUE` creates a table with an explicit named column containing protocol buffer values. This creates a table where each row has a single column named `albumreviews` holding the full protocol buffer.

```googlesql
CREATE TABLE Reviews AS
SELECT albumreviews FROM AlbumReviewData AS albumreviews;
```
