How do I find the position of a regular expression match?

--- AGENT ---

The REGEXP_INSTR function returns the 1-based starting position of the first match of a regular expression in a string.

```googlesql
SELECT
  REGEXP_INSTR('ab@cd-ef',  '@[^-]*') AS instr_a,
  REGEXP_INSTR('ab@d-ef',   '@[^-]*') AS instr_b,
  REGEXP_INSTR('abc@cd-ef', '@[^-]*') AS instr_c,
  REGEXP_INSTR('abc-ef',    '@[^-]*') AS instr_d,
```
