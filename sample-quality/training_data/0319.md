--- USER ---

How do I store multiple additional columns in an index?

--- AGENT ---

The `STORING` clause can include a comma-separated list of multiple columns, all of which are materialized with the index entry but do not participate in the sort order.

```googlesql
CREATE INDEX i1 ON Books (Title) STORING (First_Name, Last_Name);
```
