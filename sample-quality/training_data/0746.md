How do I use the wildcard with WITH to carry over existing columns while adding new ones?

--- AGENT ---

The `WITH *` clause carries over all existing columns from the working table and can add new computed columns. This example returns all columns plus a new column for the destination account ID.

```googlesql
GRAPH FinGraph
MATCH (src:Account)-[transfer:Transfers]->(dst:Account)
WITH *, dst.id
RETURN dst.id AS destination_id
```
