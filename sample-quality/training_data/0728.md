How do I use OPTIONAL MATCH to include unmatched rows with NULL values?

--- AGENT ---

The `OPTIONAL MATCH` statement returns all rows from the outer query, filling missing matched values with `NULL`. This example finds all people and optionally matches any blocked accounts they own.

```googlesql
GRAPH FinGraph
MATCH (n:Person)
OPTIONAL MATCH (n:Person)-[:Owns]->(a:Account {is_blocked: TRUE})
RETURN n.name, a.id AS blocked_account_id
```
