How can I find which two-day bucket a date falls into, using the default origin point?

--- AGENT ---

This query uses DATE_BUCKET with a 2-day interval width to group dates into buckets, returning the lower bound of each bucket. When no origin date is specified, DATE_BUCKET defaults to 1950-01-01 as the origin point and expands buckets in both directions from it.

```googlesql
WITH some_dates AS (
  SELECT DATE '1949-12-29' AS my_date UNION ALL
  SELECT DATE '1949-12-30' UNION ALL
  SELECT DATE '1949-12-31' UNION ALL
  SELECT DATE '1950-01-01' UNION ALL
  SELECT DATE '1950-01-02' UNION ALL
  SELECT DATE '1950-01-03'
)
SELECT DATE_BUCKET(my_date, INTERVAL 2 DAY) AS bucket_lower_bound
FROM some_dates;
```
