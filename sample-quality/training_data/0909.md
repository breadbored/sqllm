How do I convert a JSON array of strings to a FLOAT array?

--- AGENT ---

The `LAX_FLOAT_ARRAY` function parses each string in the JSON array as a number and converts to `FLOAT`, returning `NULL` for unparseable values.

```googlesql
SELECT LAX_FLOAT_ARRAY(JSON '["10", "1.1", "1.1e2", "+1.5"]') AS result;
```
