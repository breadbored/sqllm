How do I parse a 12-hour time format string like '2:23:38 pm' into a TIME value?

--- AGENT ---

PARSE_TIME() handles 12-hour clock formats using %p for AM/PM designation. This example converts the 12-hour string '2:23:38 pm' to 24-hour TIME representation 14:23:38.

```googlesql
SELECT PARSE_TIME('%I:%M:%S %p', '2:23:38 pm') AS parsed_time;
```
