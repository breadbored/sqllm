How do I flatten a nested array structure to extract specific fields across all nested elements?

--- AGENT ---

Use `FLATTEN` with field access notation (dot notation and array subscripts) to extract a single field from all nested struct elements and concatenate the results into a flat array. The function unwraps nested array layers and collects all matching field values.

```googlesql
WITH t AS (
  SELECT
  [
    STRUCT([STRUCT([1,2,3] AS quantity), STRUCT([4,5,6] AS quantity)] AS sales),
    STRUCT([STRUCT([7,8] AS quantity), STRUCT([] AS quantity)] AS sales)
  ] AS v
)
SELECT FLATTEN(v.sales.quantity) AS all_values
FROM t;
```
