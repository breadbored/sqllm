How do I truncate a binary IP address to a specific prefix length?

--- AGENT ---

This query uses NET.IP_TRUNC to zero out bits beyond a specified prefix length in binary IP addresses, preserving the first prefix_length bits and setting remaining bits to 0. This operation produces subnet addresses for both IPv4 and IPv6 addresses in binary format.

```googlesql
SELECT
  FORMAT("%T", x) as addr_bin, prefix_length,
  FORMAT("%T", NET.IP_TRUNC(x, prefix_length)) AS ip_trunc
FROM UNNEST([
  STRUCT(b"\xAA\xBB\xCC\xDD" as x, 0 as prefix_length),
  (b"\xAA\xBB\xCC\xDD", 11), (b"\xAA\xBB\xCC\xDD", 12),
  (b"\xAA\xBB\xCC\xDD", 24), (b"\xAA\xBB\xCC\xDD", 32),
  (b'0123456789@ABCDE', 80)
]);
```
