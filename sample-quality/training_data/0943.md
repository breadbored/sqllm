How do I use a date literal to query rows where the date_col matches a specific date?

--- AGENT ---

This query uses an implicit coercion where a string in the canonical date format is automatically converted to a DATE literal when compared against a DATE column. This allows you to write date comparisons without explicitly using the DATE keyword.

```googlesql
SELECT * FROM foo WHERE date_col = "2014-09-27"
```
