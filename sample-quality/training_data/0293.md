How do I add column-level options like deprecation warnings to a table definition?

--- AGENT ---

The `OPTIONS` clause on individual column definitions lets you set system-specific metadata for each column. You can also add foreign key references inline using the `REFERENCES` keyword, and mark the primary key inline with `PRIMARY KEY`.

```googlesql
CREATE TABLE books (
  title STRING NOT NULL PRIMARY KEY,
  author STRING
      OPTIONS (is_deprecated=true, comment="Replaced by authorId column"),
  authorId INT64 REFERENCES authors (id),
  category STRING OPTIONS (description="LCC Subclass")
)
```
