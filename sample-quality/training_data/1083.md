How do I rename a column in pipe syntax?

--- AGENT ---

The `RENAME` pipe operator changes the name of specified columns. After renaming, the new name is used in the output, but table aliases still refer to the original column values.

```googlesql
SELECT 1 AS x, 2 AS y, 3 AS z
|> AS t
|> RENAME y AS renamed_y
|> SELECT *, t.y AS t_y;
```
