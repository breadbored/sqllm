What happens if I try to parse a date string with format elements in a different order than the input?

--- AGENT ---

`PARSE_DATE()` produces an error if the format string and input string have misaligned element order, because each element in the format string must correspond to the element in the same position in the date string.

```googlesql
SELECT PARSE_DATE('%Y %A %b %e', 'Thursday Dec 25 2008');
```
