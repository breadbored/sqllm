How can I use aggregate functions on array variables from a quantified path pattern?

--- AGENT ---

Variables declared inside a quantified path pattern bind to arrays outside of the pattern (called group variables) and can be used in aggregate functions. The example uses SUM on an array of edge amounts and ARRAY_LENGTH to count hops.

```googlesql
GRAPH FinGraph
MATCH
  (src:Account {id: 7})-[e:Transfers WHERE e.amount > 100]->{0,2}
  (dst:Account)
WHERE src.id != dst.id
LET total_amount = SUM(e.amount)
RETURN
  src.id AS src_account_id, dst.id AS dst_account_id,
  ARRAY_LENGTH(e) AS number_of_hops, total_amount
```
