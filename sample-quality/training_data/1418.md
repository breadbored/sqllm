How do I name capturing groups in a regular expression?

--- AGENT ---

REGEXP_EXTRACT_GROUPS supports named capturing groups using (?<name>...) syntax, producing a STRUCT with fields named after the groups.

```googlesql
SELECT REGEXP_EXTRACT_GROUPS('2025-09-10', r'(?<year>\d{4})-(?<month>\d{2})-(?<day>\d{2})') AS result
```
