How do I use `CALL PER()` to run a table-valued function on the entire working table instead of each row?

--- AGENT ---

The `CALL PER()` statement executes a named TVF once on all rows in the working table, allowing graph-wide algorithms like PageRank to operate on the complete data structure. The working table resulting from `CALL PER()` contains only the columns specified in `YIELD`, and the input working table columns are not carried over.

```googlesql
GRAPH FinGraph
MATCH (n)
RETURN n

NEXT

-- Assumes PageRank() is a built-in TVF that takes a TABLE of graph elements
-- and returns the same table with an additional 'rank' column.
CALL PER() PageRank() YIELD n, rank
RETURN n.Id AS node_id, rank
ORDER BY rank DESC
LIMIT 5;
```
