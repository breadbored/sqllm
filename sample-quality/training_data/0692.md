How do I query a property graph to find accounts that have received transfers and then look up their owners?

--- AGENT ---

This query uses `NEXT` to chain two linear query statements: the first finds accounts with incoming transfers and counts them, and the second matches those accounts to their owners. The `NEXT` statement allows you to pass results from one query statement to the next, enabling multi-step graph traversal.

```googlesql
GRAPH FinGraph
MATCH (:Account)-[:Transfers]->(account:Account)
RETURN account, COUNT(*) AS num_incoming_transfers
GROUP BY account

NEXT

MATCH (account:Account)<-[:Owns]-(owner:Person)
RETURN
  account.id AS account_id, owner.name AS owner_name,
  num_incoming_transfers
```
