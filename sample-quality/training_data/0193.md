How do I convert an array literal into individual rows while preserving the original array order?

--- AGENT ---

Use `UNNEST()` on an array literal and apply `WITH OFFSET` to capture the zero-based position of each element, then `ORDER BY` the offset to restore the original array order.

```googlesql
SELECT *
FROM UNNEST(['foo', 'bar', 'baz', 'qux', 'corge', 'garply', 'waldo', 'fred'])
  AS element
WITH OFFSET AS offset
ORDER BY offset;

/*----------+--------+
 | element  | offset |
 +----------+--------+
 | foo      | 0      |
 | bar      | 1      |
 | baz      | 2      |
 | qux      | 3      |
 | corge    | 4      |
 | garply   | 5      |
 | waldo    | 6      |
 | fred     | 7      |
 +----------+--------*/
```
