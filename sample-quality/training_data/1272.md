How do I define a temporary table that references itself?

--- AGENT ---

A `RECURSIVE` CTE references itself to generate a sequence of rows. The base term provides the first set of rows, and the recursive term adds more rows until iteration produces no new rows.

```googlesql
WITH RECURSIVE
  T1 AS (
    (SELECT 1 AS n) UNION ALL
    (SELECT n + 1 AS n FROM T1 WHERE n < 3))
SELECT * FROM T1 ORDER BY n
```
