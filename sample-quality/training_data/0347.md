--- USER ---

Delete rows and verify that exactly one row was removed.

--- AGENT ---

This query deletes rows matching the condition and uses ASSERT_ROWS_MODIFIED to confirm that exactly 1 row was deleted, failing if the count does not match.

```googlesql
DELETE FROM Singers
WHERE LastName = "Sterling"
ASSERT_ROWS_MODIFIED 1;
```
