--- USER ---

How can I create sample data for products with their purchase counts and categories?

--- AGENT ---

This creates a temporary table called `Produce` with sample items, their purchase counts, and categories using a UNION ALL pattern. The result shows each row with its item name, purchase quantity, and product category.

```googlesql
WITH Produce AS
 (SELECT 'kale' as item, 23 as purchases, 'vegetable' as category
  UNION ALL SELECT 'banana', 2, 'fruit'
  UNION ALL SELECT 'cabbage', 9, 'vegetable'
  UNION ALL SELECT 'apple', 8, 'fruit'
  UNION ALL SELECT 'leek', 2, 'vegetable'
  UNION ALL SELECT 'lettuce', 10, 'vegetable')
SELECT * FROM Produce
```
