How do I create a simple JavaScript function that adds one to a numeric input?

--- AGENT ---

This query creates a minimal JavaScript UDF that performs a single arithmetic operation on the input parameter. Since the function body contains no escape sequences or special characters, it can use a regular quoted string literal.

```googlesql
CREATE TEMP FUNCTION PlusOne(x DOUBLE)
RETURNS DOUBLE
LANGUAGE js
AS 'return x+1;';

SELECT val, PlusOne(val) AS result
FROM UNNEST([1, 2, 3]) AS val;
```
