How do I extract a JSON array field using double-quoted escaping?

--- AGENT ---

The `JSON_QUERY` function uses double quotes to escape invalid JSONPath characters in key names.

```googlesql
SELECT
  JSON_QUERY(
    '{"class": {"students": [{"name": "Jane"}]}}',
    '$.class."students"') AS student_names;
```
