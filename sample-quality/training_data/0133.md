How do I extract a contiguous subsequence from an array?

--- AGENT ---

Use `ARRAY_SLICE` with a start and end offset to extract consecutive elements. Both offsets are inclusive, support positive (0-based from start) and negative (from end) indexing, and out-of-bounds values are clamped to array boundaries.

```googlesql
SELECT ARRAY_SLICE(['a', 'b', 'c', 'd', 'e'], 1, 3) AS result
```
