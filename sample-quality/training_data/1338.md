How does population variance handle NULL values in the input?

--- AGENT ---

The `VAR_POP` function ignores NULL values during aggregation. In this example, the NULL is skipped and the variance is computed from the two non-NULL values 10 and 14.

```googlesql
SELECT VAR_POP(x) AS results FROM UNNEST([10, 14, NULL]) AS x
```
