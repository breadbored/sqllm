How do I return an entire graph node variable from GRAPH_TABLE so I can access its properties in the SELECT clause?

--- AGENT ---

When you return the full node variable (not individual properties) in the RETURN statement, you can reference that node in the outer SELECT and access its properties using dot notation. This approach works when you want to conditionally select or transform node properties after the GRAPH_TABLE operation.

```googlesql
SELECT n.name, n.id
FROM GRAPH_TABLE(
  FinGraph
  MATCH (n:Person)
  RETURN n
);
```
