How do I create an array containing arrays from a subquery?

--- AGENT ---

Wrap array columns in a struct using `SELECT AS STRUCT` so the `ARRAY` function can accept them without triggering the nested array error. This packages each array into a struct field, allowing you to construct an array of struct-wrapped arrays.

```googlesql
SELECT ARRAY
  (SELECT AS STRUCT [1, 2, 3] UNION ALL
   SELECT AS STRUCT [4, 5, 6]) AS new_array;
```
