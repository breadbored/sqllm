--- USER ---

When a protocol buffer has a oneof field, how can I determine which of the mutually exclusive options is currently set?

--- AGENT ---

This query uses EXTRACT with ONEOF_CASE to return the name of the field that is set within the group_name oneof. It returns 'solo' or 'band' depending on which field was populated in the proto.

```googlesql
SELECT EXTRACT(ONEOF_CASE(group_name) FROM album_col) AS artist_type
FROM AlbumList;
```
