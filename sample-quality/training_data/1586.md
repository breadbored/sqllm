--- USER ---

How do I nest Lua functions so that the output of one function becomes the input to another?

--- AGENT ---

This query demonstrates composing Lua functions by passing the output of DivideByTwo directly as arguments to MultiplyInputs. Function composition allows you to chain multiple operations together, building more complex calculations from simpler building blocks.

```googlesql
CREATE TEMP FUNCTION MultiplyInputs(x DOUBLE, y DOUBLE)
RETURNS DOUBLE
LANGUAGE lua
AS r'''
  return x*y;
''';

CREATE TEMP FUNCTION DivideByTwo(x DOUBLE)
RETURNS DOUBLE
LANGUAGE lua
AS r'''
  return x/2;
''';

WITH numbers AS
  (SELECT 1 AS x, 5 as y
  UNION ALL
  SELECT 2 AS x, 10 as y
  UNION ALL
  SELECT 3 as x, 15 as y)
SELECT x,
  y,
  MultiplyInputs(DivideByTwo(x), DivideByTwo(y)) as half_product
FROM numbers;
```
