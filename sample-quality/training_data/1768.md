--- USER ---

Show me a BigQuery SQL query that computes course performance metrics for a specific client, including pass rates, average scores, and user counts aggregated at the course container level.

--- AGENT ---

This query creates a table function that accepts a `clientId` parameter and returns course performance metrics for that client.

The query builds the result through four CTEs:

1. **max_course_ids** — identifies the latest course version within each container by finding the maximum course ID per container.

2. **course_names** — retrieves the name of the latest course in each container by joining the courses table to max_course_ids.

3. **best_scores** — filters user assessments for the specified client and computes each user's best performance within each container (maximum passed status and maximum score), grouped by user and container.

4. **aggregate_scores** — summarizes container-level metrics from best_scores: total user count, count of users who passed, count who didn't pass, and average score/pass rate across those best scores.

The final SELECT joins all user assessments (not filtered to best scores) back to the course containers, course names, and pre-calculated aggregates. It then computes overall pass rates (sum of all passed flags divided by distinct user count, capped at 100%) and overall average score per container, grouping by container, course name, and the aggregate fields.

**Original SQL:**

```sql
CREATE OR REPLACE TABLE
  FUNCTION ${self()} (clientId INT64) AS (
  WITH
    max_course_ids AS (
    SELECT
      container_id,
      MAX(id) AS max_id
    FROM
      ${ref("stg_courses")}
    GROUP BY
      container_id),
    course_names AS (
    SELECT
      c.container_id,
      c.name
    FROM
      ${ref("stg_courses")} c
    JOIN
      max_course_ids mci
    ON
      c.id = mci.max_id),
    best_scores AS (
    SELECT
      c.container_id,
      uca.user_id,
      MAX(uca.passed) AS passed,
      MAX(uca.score) AS score
    FROM
      ${ref("stg_user_course_assessments")} uca
    JOIN
      ${ref("stg_courses")} c
    ON
      c.id = uca.course_id
    JOIN
      ${ref("stg_users")} u
    ON
      u.id = uca.user_id
    JOIN
      ${ref("stg_clients_course_containers")} ccc
    ON
      ccc.course_container_id = c.container_id
    WHERE
      ccc.client_id = clientId
    GROUP BY
      uca.user_id,
      container_id),
    aggregate_scores AS (
    SELECT
      container_id,
      COUNT(DISTINCT user_id) AS total_users,
      COUNT(DISTINCT
        CASE
          WHEN passed = 1 THEN user_id
      END
        ) AS users_passed,
      COUNT(DISTINCT
        CASE
          WHEN passed = 0 THEN user_id
      END
        ) AS users_left,
      ROUND(AVG(score)) AS avg_score,
      ROUND(AVG(passed) * 100) AS pass_rate,
    FROM
      best_scores
    GROUP BY
      container_id)
  SELECT
    c.container_id,
    cn.name,
    agg.total_users,
    agg.users_passed,
    --        agg.users_left,
    --        COUNT(uca.id)     AS                                               assessment_starts,
    LEAST(ROUND(SUM(passed) / COUNT(DISTINCT uca.user_id) * 100), 100) total_pass_rate,
    ROUND(AVG(score)) AS total_avg_score,
    --        agg.avg_score     as                                               best_avg_score,
    --        agg.pass_rate     as                                               best_pass_rate,
  FROM
    ${ref("stg_user_course_assessments")} uca
  JOIN
    ${ref("stg_courses")} c
  ON
    c.id = uca.course_id
  JOIN
    course_names cn
  ON
    cn.container_id = c.container_id
  JOIN
    ${ref("stg_users")} u
  ON
    u.id = uca.user_id
  JOIN
    aggregate_scores agg
  ON
    agg.container_id = c.container_id
  GROUP BY
    c.container_id,
    cn.name,
    agg.total_users,
    agg.users_passed,
    agg.users_left,
    agg.avg_score,
    agg.pass_rate )
```
