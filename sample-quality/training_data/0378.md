--- USER ---

Why does using ROWNUM fail when the value table is produced by a subquery?

--- AGENT ---

A subquery that produces a value table returns only the value without row metadata, so pseudocolumns like ROWNUM are no longer available. This query fails because the subquery `SELECT a FROM AlbumReviewData AS a` returns a value table that lacks the ROWNUM pseudocolumn.

```googlesql
SELECT a.ROWNUM, a.albumtitle AS title FROM (SELECT a FROM AlbumReviewData AS a)
```
