How do I extract a nested field from a specific array element in JSON?

--- AGENT ---

The `JSON_EXTRACT` function can navigate through arrays and objects using combined dot and bracket notation.

```googlesql
SELECT JSON_EXTRACT(
  '{"class": {"students": [{"name": "John"}, {"name": "Jamie"}]}}',
  '$.class.students[1].name') AS second_student;
```
