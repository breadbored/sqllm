Can you explain what this data filtering query does? It seems to be cleaning event data and flagging bots, but I want to understand the structure and what columns it adds.

--- AGENT ---

This query filters and enriches event data from two separate databases, applying consistent data quality rules and bot detection across both sources.

**First UNION ALL branch** selects from the production events table (dp-dwh.events_sources.dp_events) where environment is 'production' or client is 'devalliance'. It applies a comprehensive WHERE clause that rejects any record where the session_id is not a valid UUID, or where type, name, app, environment, client, UTM parameters, URL fields, action, click_target, landing_path, and url_path_version contain characters outside their expected character sets (using REGEXP_CONTAINS). It also validates the JSON structure in action_meta by checking specific extraction paths.

**Second UNION ALL branch** applies nearly identical validation rules to the developersalliance.events_sources.dp_events table, but without the production environment filter (that condition is commented out).

Both branches add two computed columns to the output:
1. **user_agent** (anonymized) — the original plaintext user agent is replaced with its SHA1 hash
2. **is_bot** — a boolean flag set to true if the user agent matches any of 25+ patterns associated with automated tools, crawlers, API clients, and bot services (including patterns for bot keywords, headless browsers, testing frameworks like Selenium and Postman, language-specific user agents for Python/Java/Go, and known bot services from Vercel, AWS, Facebook, and Google)

All other columns from the source table are preserved as-is (using `e.* EXCEPT(user_agent)`).

/**
* Filter
*
* This layer is used to filter out any garbage data that could be the result of the wrong enviornment,
* manually injected values, or other issues. This is the first layer of filtering that should be done.
**/
SELECT
  -- COALESCE(sm.master_id, e.session_id) as session_id,
  -- Leave off user_agent
  e.* EXCEPT(user_agent),
  -- We only care about matching user agents, 
  -- we do not care what is actually in them for now
  -- Later we may parse them at the endpoint level with a library
  SHA1(e.user_agent) as user_agent,
  CASE 
    -- User-Agent based bot filtering
    WHEN 
         user_agent LIKE '%bot%' 
      OR user_agent LIKE '%crawl%' 
      OR user_agent LIKE '%spider%' 
      OR user_agent LIKE '%slurp%' 
      OR user_agent LIKE '%headless%' 
      OR user_agent LIKE '%phantomjs%' 
      OR user_agent LIKE '%selenium%' 
      OR user_agent LIKE '%wget%' 
      OR user_agent LIKE '%curl%' 
      OR user_agent LIKE '%python-requests%'
      OR user_agent LIKE '%java%' 
    --   OR user_agent LIKE '%nexus%'        -- Nexus device, often used in emulation
      OR user_agent LIKE '%emulator%' 
      OR user_agent LIKE '%postman%'      -- Postman API client
      OR user_agent LIKE '%python%'       -- Generic Python user agents
      OR user_agent LIKE '%scrapy%'       -- Scrapy, a web crawling framework
      OR user_agent LIKE '%httpclient%'   -- Java HTTP client libraries
      OR user_agent LIKE '%libwww-perl%'  -- Perl HTTP client
      OR user_agent LIKE '%go-http-client%' -- Go HTTP client
      OR user_agent LIKE '%puffin%'       -- Puffin Browser, sometimes used in automation
      OR user_agent LIKE '%okhttp%'       -- OkHttp, a popular HTTP client for Android and Java
      OR user_agent LIKE '%axios%'        -- Axios, a JavaScript HTTP client
      OR user_agent LIKE '%insomnia%'     -- Insomnia, another HTTP client
      OR user_agent LIKE '%webdriver%'    -- Generic WebDriver for automated browsers
      OR user_agent LIKE '%chrome-lighthouse%' -- Chrome's Lighthouse tool, often run headlessly
      OR user_agent LIKE '%appengine%'    -- Google App Engine
      OR user_agent LIKE '%gsa_crawler%'  -- Google Search Appliance
      OR user_agent = 'vercel-screenshot/1.0' -- Vercel
      OR user_agent = 'vercel-favicon/1.0'
      OR user_agent = 'amazon-kendra'     -- AWS
      OR user_agent LIKE 'facebookexternalhit%' -- Facebook Link Bot

    THEN true
    ELSE false
  END as is_bot

    OR (NOT REGEXP_CONTAINS(e.type, '^([a-zA-Z0-9_-]+)?$') AND e.type IS NOT NULL)
    OR (NOT REGEXP_CONTAINS(e.name, '^([a-zA-Z0-9_-]+)?$') AND e.name IS NOT NULL)

    OR (NOT REGEXP_CONTAINS(utm_medium, '^([^;()]+)?$') AND utm_medium IS NOT NULL)

    OR (NOT REGEXP_CONTAINS(utm_content, '^([^;()]+)?$') AND utm_content IS NOT NULL)
    OR (NOT REGEXP_CONTAINS(utm_source, '^([^;()]+)?$') AND utm_source IS NOT NULL)
    OR (NOT REGEXP_CONTAINS(referrer, '^([^;()]+)?$') AND referrer IS NOT NULL)
    OR (NOT REGEXP_CONTAINS(related_id, '^([a-zA-Z0-9_-]+)?$') AND related_id IS NOT NULL)
    OR (NOT REGEXP_CONTAINS(related_entity, '^([a-zA-Z0-9_-]+)?$') AND related_entity IS NOT NULL)
    OR (NOT REGEXP_CONTAINS(url, '^([^;()]+)?$') AND url IS NOT NULL)
    OR (NOT REGEXP_CONTAINS(url_path, '^([^;()]+)?$') AND url_path IS NOT NULL)
    OR (NOT REGEXP_CONTAINS(e.action, '^([^;()]+)?$') AND e.action IS NOT NULL)
    OR (NOT REGEXP_CONTAINS(click_target, '^([^;()]+)?$') AND click_target IS NOT NULL)
