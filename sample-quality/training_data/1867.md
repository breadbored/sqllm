Can you show me a BigQuery query that retrieves document badge information and enriches it with associated client IDs?

--- AGENT ---

This query retrieves all document badge records and enriches each one with the list of clients associated with its document. The logic works in two steps: first, a CTE aggregates all distinct client IDs for each document into an array (ignoring null values). Then the main query selects all columns from the document_badges table and left joins to this aggregated client list, returning the client_ids array where a match exists, or defaulting to [1] if no clients are found for that document.

WITH
doc_clients AS (
  SELECT
    dc.doc_id as doc_id,
    ARRAY_AGG(DISTINCT dc.client_id IGNORE NULLS) as client_ids
  FROM
    ${ref("documents_clients")} as dc
  GROUP BY
    dc.doc_id
)
SELECT
  db.id,
  db.document_id,
  db.learning_path_id,
  db.`order`,
  db.created,
  db.updated,
  COALESCE(dc.client_ids, [1]) AS client_ids
FROM
  ${ref("document_badges")} as db
LEFT JOIN
  doc_clients as dc
ON
  dc.doc_id = db.document_id
