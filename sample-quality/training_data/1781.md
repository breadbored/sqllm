## Question
Show me a summary of assessment questions used within a specific client's course container, including how many course versions use each question, whether it's in the live/latest published version, and the total answer counts.

--- AGENT ---

This query is a table function that generates an assessment question summary report for a given client and course container. It joins together user assessment answers, question metadata, and course versioning information to produce per-question metrics.

The query aggregates across all non-test user responses to each assessment question, and determines two key things about each question: (1) how many distinct course versions in the container reference it, and (2) whether the question exists in the live/latest published course version for that container (by comparing the current course ID to the maximum published course ID). The output includes the question identifier and name, the earliest publish date the question appeared, the version count, live status, and total answer count, sorted by question ID.

```sql
CREATE OR REPLACE TABLE
  FUNCTION ${self()} (clientId INT64,
    containerId INT64) AS (
  SELECT
    q.id AS assessment_question_id,
    q.name AS assessment_question,
    FORMAT_DATE('%b-%d-%Y', MIN(aq.created)) AS publish_date,
    COUNT(DISTINCT ucaa.course_id) AS versions,
    (MAX(c.id) = MAX(c.course_max_id)) AS live,
    COUNT(ucaa.id) AS answers
  FROM
    ${ref("stg_user_course_assessment_answers")} ucaa
  JOIN
    ${ref("stg_assessments_questions")} aq
  ON
    aq.question_id = ucaa.question_id
  JOIN
    ${ref("stg_questions")} q
  ON
    q.id = aq.question_id
  JOIN (
    SELECT
      c.id,
      c.container_id,
      cm.course_max_id
    FROM
      ${ref("stg_courses")} c
    JOIN
      ${ref("stg_clients_course_containers")} ccc
    ON
      ccc.course_container_id = c.container_id
    JOIN (
      SELECT
        container_id,
        MAX(id) AS course_max_id
      FROM
        ${ref("stg_courses")}
      WHERE
        has_been_published = 1
      GROUP BY
        container_id) cm
    ON
      cm.container_id = c.container_id
    WHERE
      c.container_id = containerId
      AND client_id = clientId) c
  ON
    c.id = ucaa.course_id
  JOIN
    ${ref("stg_users")} u
  ON
    u.id = ucaa.user_id
  WHERE
    u.test_user = 0
  GROUP BY
    q.id,
    q.name
  ORDER BY
    assessment_question_id)
```
