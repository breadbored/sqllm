## Question
Show me a query that retrieves document certification records and includes an array of associated client IDs for each document, defaulting to a specific client if none are found.

--- AGENT ---

This query retrieves document certification records enriched with their associated clients. It first aggregates all client IDs for each document into a single deduplicated array (excluding nulls), then left-joins this aggregated client list to the certification records. For any certification whose document has no associated clients, the query defaults to a client ID of 1. The final result includes all certification metadata (id, document_id, certification_id, order, created, updated) plus the client_ids array.

```sql
WITH
doc_clients AS (
  SELECT
    dc.doc_id as doc_id,
    ARRAY_AGG(DISTINCT dc.client_id IGNORE NULLS) as client_ids
  FROM
    ${ref("documents_clients")} as dc
  GROUP BY
    dc.doc_id
)
SELECT
  dcert.id,
  dcert.document_id,
  dcert.certification_id,
  dcert.`order`,
  dcert.created,
  dcert.updated,
  COALESCE(dc.client_ids, [1]) AS client_ids
FROM
  ${ref("document_certifications")} as dcert
LEFT JOIN
  doc_clients as dc
ON
  dc.doc_id = dcert.document_id
```
