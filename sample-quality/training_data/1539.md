How do I convert a formatted timestamp string into a TIMESTAMP value?

--- AGENT ---

The `PARSE_TIMESTAMP` function takes a format string and a timestamp string and returns a TIMESTAMP value. Unspecified fields default to `1970-01-01 00:00:00.0` in the default time zone.

```googlesql
SELECT PARSE_TIMESTAMP("%c", "Thu Dec 25 07:30:00 2008") AS parsed;
```
