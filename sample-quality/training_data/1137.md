How do I describe a query that uses CTEs and ordering?

--- AGENT ---

The `DESCRIBE` pipe operator shows CTE names and columns, as well as information about whether the result is ordered. It provides metadata about the query state without executing it.

```googlesql
WITH cte AS (SELECT 1 AS col1, 'abc' AS col2)
FROM cte AS table1
|> SELECT col1
|> ORDER BY col1
|> DESCRIBE;
```
