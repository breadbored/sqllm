--- USER ---

Insert a singer record, but replace the entire existing row if the primary key already exists.

--- AGENT ---

This query inserts a row and uses the REPLACE keyword to completely replace any existing row with the same primary key, clearing unspecified columns to their default values.

```googlesql
INSERT OR REPLACE INTO Singers
    (FirstName, LastName, BirthDate, Status, SingerInfo)
VALUES ("Zak", "Sterling", "1996-03-12", "active", "nationality:'U.S.A.'");
```
