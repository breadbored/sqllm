--- USER ---

How do you combine two graph paths that share a common node using the concatenation operator?

--- AGENT ---

The graph concatenation operator (||) combines multiple graph paths into one, preserving the original order of nodes and edges. The last node of the first path must match the first node of the second path for the concatenation to succeed.

```googlesql
-- This successfully produces the concatenated path called `full_path`.
MATCH
  p=(src:Account)-[t1:Transfers]->(mid:Account),
  q=(mid)-[t2:Transfers]->(dst:Account)
LET full_path = p || q
```
