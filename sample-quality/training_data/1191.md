How do I get the count of elements in a repeated protocol buffer field?

--- AGENT ---

This query uses the `ARRAY_LENGTH` function to count the number of elements in a repeated field. Since repeated fields are represented as arrays, you can apply array functions like `ARRAY_LENGTH` to determine how many elements are present.

```googlesql
SELECT
  c.Orders.order_number,
  ARRAY_LENGTH(c.Orders.line_item)
FROM
  Customers AS c;
```
