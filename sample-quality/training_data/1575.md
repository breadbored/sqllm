--- USER ---

How do I create a temporary JavaScript function that multiplies two numbers and use it in a query?

--- AGENT ---

This query creates a temporary JavaScript UDF that exists only for the current session and multiplies two inputs. The temporary function follows the same syntax and calling behavior as a permanent function, but is dropped when the session ends.

```googlesql
CREATE TEMP FUNCTION MultiplyInputs(x DOUBLE, y DOUBLE)
RETURNS DOUBLE
LANGUAGE js
AS r'''
  return x*y;
''';

WITH numbers AS
  (SELECT 1 AS x, 5 as y
  UNION ALL
  SELECT 2 AS x, 10 as y
  UNION ALL
  SELECT 3 as x, 15 as y)
SELECT x, y, MultiplyInputs(x, y) as product
FROM numbers;
```
