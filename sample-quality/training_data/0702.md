How do I nest multiple subqueries with variable scoping to perform conditional aggregation?

--- AGENT ---

Nested `CALL` statements with explicit variable scope lists allow you to pass multiple variables from outer scopes into inner subqueries. This example counts internal transfers (between accounts of the same person) using nested scoping.

```googlesql
GRAPH FinGraph
MATCH (p:Person {Id:2})
CALL (p) {
  MATCH (p)-[:Owns]->(a:Account)
  OPTIONAL CALL (p, a) {
    MATCH (a)-[t:Transfers]->(other:Account), (p)-[:Owns]->(other)
    RETURN count(t) AS num_internal_transfers
  }
  RETURN a.Id AS account_Id, COALESCE(num_internal_transfers, 0) AS internal_transfers
}
RETURN p.name, account_Id, internal_transfers
ORDER BY account_Id;
```
