--- USER ---

How do I extract just the minute component from a time value as a 2-digit string?

--- AGENT ---

The CAST function with the FORMAT clause converts a time to a string. The MI format element outputs the minute (00-59) as a 2-digit number.

```googlesql
SELECT CAST(TIME '21:30:00' AS STRING FORMAT 'MI') AS date_time_to_string;
```
