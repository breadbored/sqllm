How do I extract the labels that are assigned to a graph node or edge?

--- AGENT ---

The `LABELS` function returns an array of string labels associated with a graph element, preserving the original case of each label. This is helpful for determining the types or categories of nodes and edges in your graph.

```googlesql
GRAPH FinGraph
MATCH (n:Person|Account)
RETURN LABELS(n) AS label, n.id
```
