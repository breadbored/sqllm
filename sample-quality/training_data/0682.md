How can I match paths that do not visit the same node twice?

--- AGENT ---

Use the ACYCLIC path mode to filter out any paths that visit the same node more than once. This produces paths without repeated nodes.

```googlesql
GRAPH FinGraph
MATCH
  ACYCLIC (a1:Account)-[t1:Transfers]->(a2:Account)-[t2:Transfers]->
  (a3:Account)
RETURN
  a1.id as account1_id, a2.id as account2_id,
  a3.id as account3_id
```
