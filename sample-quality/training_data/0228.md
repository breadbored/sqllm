--- USER ---

How can I sort an array of strings case-insensitively when one element is explicitly marked with a case-insensitive collation?

--- AGENT ---

When a collation specification is applied to one element in a set of strings, that collation propagates through the ORDER BY clause and applies to the entire sort operation. This query demonstrates case-insensitive sorting where the explicitly collated 'B' causes all strings to be sorted case-insensitively, placing 'a' before 'B' and 'b'.

```googlesql
SELECT *
FROM UNNEST([
  COLLATE('B', 'und:ci'),
  'b',
  'a'
]) AS character
ORDER BY character
```
