--- USER ---

How can I rank a set of numbers with consecutive ranks that don't skip when there are ties?

--- AGENT ---

DENSE_RANK() assigns ordinal ranks starting from 1, with all tied rows receiving the same rank and the next distinct rank always incrementing by 1 (unlike RANK, which skips rank values). This query ranks simple numeric values without requiring partitioning or complex window definitions.

```googlesql
WITH Numbers AS
 (SELECT 1 as x
  UNION ALL SELECT 2
  UNION ALL SELECT 2
  UNION ALL SELECT 5
  UNION ALL SELECT 8
  UNION ALL SELECT 10
  UNION ALL SELECT 10
)
SELECT x,
  DENSE_RANK() OVER (ORDER BY x ASC) AS dense_rank
FROM Numbers
```
