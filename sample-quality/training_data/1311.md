How do I calculate the Pearson correlation coefficient between two numeric columns?

--- AGENT ---

The `CORR` function computes the Pearson coefficient of correlation between pairs of values. The return value ranges from -1 to 1, where 0 indicates no correlation. Pairs containing NULL are ignored, and the function returns NULL if fewer than two valid pairs exist.

```googlesql
SELECT CORR(y, x) AS results
FROM
  UNNEST(
    [
      STRUCT(1.0 AS y, 5.0 AS x),
      (3.0, 9.0),
      (4.0, 7.0)]);
```
