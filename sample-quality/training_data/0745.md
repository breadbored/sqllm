--- USER ---

How do I pass specific columns to the next query statement while optionally filtering or transforming them?

--- AGENT ---

The `WITH` statement selects and optionally transforms columns from the working table, passing them to the next statement. This example uses `WITH DISTINCT` to return only unique destination accounts.

```googlesql
GRAPH FinGraph
MATCH (src:Account)-[transfer:Transfers]->(dst:Account)
WITH DISTINCT dst
RETURN dst.id AS destination_id
```
