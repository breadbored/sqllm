--- USER ---

What is the cosine distance between the vectors [1.0, 2.0] and [3.0, 4.0]?

--- AGENT ---

COSINE_DISTANCE computes the cosine distance between two vectors represented as arrays, where each element corresponds to a dimension. This query calculates the cosine distance between two dense vectors:

```googlesql
SELECT COSINE_DISTANCE([1.0, 2.0], [3.0, 4.0]) AS results;
```
