What does ARRAY_SLICE return when both offsets are negative and start comes before end?

--- AGENT ---

Use `ARRAY_SLICE` with both negative offsets where the start offset is less negative (closer to the end) than the end offset. Since the start position comes after the end position in the array, the function returns an empty array.

```googlesql
SELECT ARRAY_SLICE(['a', 'b', 'c', 'd', 'e'], -1, -3) AS result
```
