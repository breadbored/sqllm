How do I use a named window to compute the most popular item within a local range for each category?

--- AGENT ---

This query defines a named window `ItemWindow` in the WINDOW clause that partitions by category and orders by purchases, then applies a bounded frame in the OVER clause. It produces the same results as the previous example but uses a named window for reusability.

```googlesql
SELECT item, purchases, category, LAST_VALUE(item)
  OVER (
    ItemWindow
    ROWS BETWEEN 1 PRECEDING AND 1 FOLLOWING
  ) AS most_popular
FROM Produce
WINDOW ItemWindow AS (
  PARTITION BY category
  ORDER BY purchases)
```
