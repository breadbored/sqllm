--- USER ---

How do I declare the same label name multiple times in different, non-overlapping blocks?

--- AGENT ---

Labels can be reused in non-overlapping sections of code. Each labeled block or loop can have the same label name as long as they do not nest.

```googlesql
label_1: BEGIN
  BREAK label_1;
END;

label_2: BEGIN
  BREAK label_2;
END;

label_1: BEGIN
  BREAK label_1;
END;
```
