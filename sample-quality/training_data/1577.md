--- USER ---

How do I compose two JavaScript functions so that the output of one becomes the input to another?

--- AGENT ---

This query demonstrates function composition by nesting UDF calls—the result of DivideByTwo is passed directly to MultiplyInputs as arguments. This allows you to chain transformations and build more complex calculations from simpler functions.

```googlesql
CREATE TEMP FUNCTION MultiplyInputs(x DOUBLE, y DOUBLE)
RETURNS DOUBLE
LANGUAGE js
AS r'''
  return x*y;
''';

CREATE TEMP FUNCTION DivideByTwo(x DOUBLE)
RETURNS DOUBLE
LANGUAGE js
AS r'''
  return x/2;
''';

WITH numbers AS
  (SELECT 1 AS x, 5 as y
  UNION ALL
  SELECT 2 AS x, 10 as y
  UNION ALL
  SELECT 3 as x, 15 as y)
SELECT x,
  y,
  MultiplyInputs(DivideByTwo(x), DivideByTwo(y)) as half_product
FROM numbers;
```
