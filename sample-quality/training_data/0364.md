--- USER ---

Perform nested DELETE and UPDATE operations with ASSERT_ROWS_MODIFIED clauses.

--- AGENT ---

This query performs nested DML operations with assertions on each nested statement to ensure that the DELETE removes one matching row and the UPDATE finds one matching row to modify.

```googlesql
UPDATE Singers
SET
    (DELETE FROM AlbumTitles a WHERE a.AlbumTitles = "Too Many Vegetables"
        ASSERT_ROWS_MODIFIED 1),
    (UPDATE AlbumTitles a SET a.AlbumTitles = "Album IV"
        WHERE a.AlbumTitles = "Album Four"),
    (INSERT AlbumTitles VALUES ("The Sloth and the Tiger"));
```
