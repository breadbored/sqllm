How do I sum all the numeric values in an array?

--- AGENT ---

Use `ARRAY_SUM` to add all non-NULL numeric values in an array. The function returns NULL if the array is empty, contains only NULLs, or if the input array itself is NULL.

```googlesql
SELECT ARRAY_SUM([1, 2, 3, 4, 5, NULL, 4, 3, 2, 1]) as sum
```
