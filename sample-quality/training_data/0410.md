How do I create a nested protocol buffer extension when constructing a protocol buffer?

--- AGENT ---

When an extension points to a nested protocol buffer, you can construct the nested buffer using NEW and pass it as the extension value with a parenthesized path as the field name.

```googlesql
SELECT
 NEW googlesql.examples.music.Album(
   'New Moon' AS album_name,
   NEW googlesql.examples.music.AlbumExtension(
    DATE(1956,1,1) AS release_date
   )
 AS (googlesql.examples.music.AlbumExtension.album_extension));
```
