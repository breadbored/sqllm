How do I create a typed array literal with string elements?

--- AGENT ---

An array literal can be explicitly typed using the ARRAY keyword with angle brackets specifying the element type. This ensures type safety and clarity about the array's content type.

```googlesql
ARRAY<string>['x', 'y', 'xy']
```
