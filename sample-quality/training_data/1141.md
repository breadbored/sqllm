How do I create a table and populate it from a pipe query result?

--- AGENT ---

The `CREATE TABLE` terminal pipe operator creates a new table and inserts the input table's rows into it. This is a terminal operator—no operators can follow it.

```googlesql
FROM Produce
|> AGGREGATE SUM(sales) AS total_sales GROUP BY category
|> CREATE TEMP TABLE ShopSales;
```
