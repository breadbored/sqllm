How do I join two tables while keeping all rows from the left table, even if they don't match?

--- AGENT ---

`LEFT OUTER JOIN` returns all rows from the left table and matching rows from the right table, filling with NULL values for non-matching right table columns.

```googlesql
SELECT Roster.LastName, TeamMascot.Mascot
FROM Roster LEFT JOIN TeamMascot ON Roster.SchoolID = TeamMascot.SchoolID;
```
