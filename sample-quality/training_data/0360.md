Perform a nested UPDATE within an UPDATE to modify data deep in a proto hierarchy.

--- AGENT ---

This query performs a nested UPDATE that modifies the Chart repeated field of a Song within an Album, adding a new chart entry for a specific song within a specific album of a specific singer.

```googlesql
UPDATE Singers s
SET (UPDATE s.Albums.Song so
    SET (INSERT INTO so.Chart
         VALUES ("chartname: 'Galaxy Top 100', rank: 5"))
    WHERE so.songtitle = "Bonus Track")
WHERE s.SingerId = 5
ASSERT_ROWS_MODIFIED 1;
```
