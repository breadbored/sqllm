What does GENERATE_TIMESTAMP_ARRAY return when the start is later than the end?

--- AGENT ---

Use `GENERATE_TIMESTAMP_ARRAY` with a positive interval when the start timestamp is later than the end timestamp. Since there are no timestamps between them with a forward step, the function returns an empty array.

```googlesql
SELECT GENERATE_TIMESTAMP_ARRAY('2016-10-06 00:00:00', '2016-10-05 00:00:00',
                                INTERVAL 1 HOUR) AS timestamp_array;
```
