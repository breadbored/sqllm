How do I generate an array of timestamps at daily intervals?

--- AGENT ---

Use `GENERATE_TIMESTAMP_ARRAY` with a start and end timestamp and an `INTERVAL` clause specifying the increment. This produces an array of timestamps spaced by the given interval.

```googlesql
SELECT GENERATE_TIMESTAMP_ARRAY('2016-10-05 00:00:00', '2016-10-07 00:00:00',
                                INTERVAL 1 DAY) AS timestamp_array;
```
