--- USER ---

How can I find the minimum, median, and maximum string values in a column while ignoring NULL entries?

--- AGENT ---

PERCENTILE_DISC returns the actual value at a given percentile (not interpolated) and by default ignores NULLs. For string values, it returns the sorted value whose cumulative distribution meets or exceeds the specified percentile threshold.

```googlesql
SELECT
  x,
  PERCENTILE_DISC(x, 0) OVER() AS min,
  PERCENTILE_DISC(x, 0.5) OVER() AS median,
  PERCENTILE_DISC(x, 1) OVER() AS max
FROM UNNEST(['c', NULL, 'b', 'a']) AS x;
```
