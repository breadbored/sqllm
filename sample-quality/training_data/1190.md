How do I retrieve repeated fields from a protocol buffer as arrays?

--- AGENT ---

This query returns a repeated field, `line_item`, from a protocol buffer column. Repeated fields in protocol buffers are automatically converted to and returned as arrays in GoogleSQL.

```googlesql
SELECT
  c.Orders.line_item
FROM
  Customers AS c;
```
