How do I limit the number of rows returned by a query?

--- AGENT ---

`LIMIT` restricts the result set to a maximum number of rows. When used with ORDER BY, it returns the first N rows in the specified sort order.

```googlesql
SELECT *
FROM UNNEST(ARRAY<STRING>['a', 'b', 'c', 'd', 'e']) AS letter
ORDER BY letter ASC LIMIT 2;
```
