--- USER ---

How do I find the account with the most outgoing transfers to a blocked account and then identify its owner?

--- AGENT ---

By chaining queries with `NEXT`, you can aggregate results in one query and use them as input to a subsequent query. This example finds the source account with the most transfers to blocked accounts, then matches it to its owner.

```googlesql
GRAPH FinGraph
MATCH (src_account:Account)-[transfer:Transfers]->(dst_account:Account {is_blocked:true})
RETURN src_account, COUNT(transfer) as total_transfers
ORDER BY total_transfers
LIMIT 1
NEXT
MATCH (src_account:Account)<-[owns:Owns]-(owner:Person)
RETURN src_account.id AS account_id, owner.name AS owner_name
```
