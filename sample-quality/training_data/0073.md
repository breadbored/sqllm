How do I count distinct values that meet a specific condition?

--- AGENT ---

Combine `COUNT(DISTINCT ...)` with `IF` to count only distinct values where a condition is true. This example counts distinct positive integers:

```googlesql
SELECT COUNT(DISTINCT IF(x > 0, x, NULL)) AS distinct_positive
FROM UNNEST([1, -2, 4, 1, -5, 4, 1, 3, -6, 1]) AS x;
```
