How do I parse a timestamp from a full English date-time representation string?

--- AGENT ---

The PARSE_TIMESTAMP function parses a string into a TIMESTAMP value using a format string. The `%c` format element matches the full locale-specific date and time representation (for example, "Thu Dec 25 07:30:00 2008") and converts it to a timestamp with the session's time zone.

```googlesql
SELECT PARSE_TIMESTAMP("%c", "Thu Dec 25 07:30:00 2008") AS parsed;
```
