How do I format a date to display the full month name in uppercase?

--- AGENT ---

The CAST function with the FORMAT clause converts a date to a string. The MONTH format element outputs the full month name, and case matching rules apply: uppercase format elements produce uppercase output.

```googlesql
SELECT CAST(DATE '2018-01-30' AS STRING FORMAT 'MONTH') AS date_time_to_string;
```
