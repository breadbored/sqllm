--- USER ---

Show me a BigQuery query that counts click events on documents by document metadata, click action details, chapter section, and date, with session-level deduplication.

--- AGENT ---

This query aggregates click events from a document tracking system. It joins the event stream (`stg_events_speedy`) with a document catalog (`agg_client_documents`) to attach document metadata (name and URL path). The query filters for click events on the 'docs' app, then groups by document, event description, click target, action, and chapter/section information extracted from a JSON metadata field. For each distinct group, it counts the number of unique sessions that performed that event. The chapter and section fields are extracted from the `action_meta` JSON object, and a combined chapter indicator is constructed by concatenating chapter and section numbers (with defaults 'X' and '0' where missing). Results are truncated to day-level granularity.

```sql
SELECT
  cd.name AS doc_name,
  cd.path AS url_path,
  e.text_description,
  e.name,
  e.type,
  click_target,
  action,
  JSON_EXTRACT_SCALAR(action_meta, '$.chapter_name') AS chapter_name,
  JSON_EXTRACT_SCALAR(action_meta, '$.chapter_number') AS chapter,
  JSON_EXTRACT_SCALAR(action_meta, '$.section_number') AS section,
  CONCAT(IFNULL(JSON_EXTRACT_SCALAR(action_meta, '$.chapter_number'), 'X'), '.', IFNULL(JSON_EXTRACT_SCALAR(action_meta, '$.section_number'), '0')) AS combined_chapter,
  COUNT(DISTINCT session_id) AS event_count,
  DATETIME_TRUNC(e.created, DAY) AS date
FROM
  ${ref("stg_events_speedy")} e
INNER JOIN
  ${ref("agg_client_documents")} cd
ON
  e.doc_id = cd.doc_id
WHERE
  e.app = 'docs'
  AND (e.type = 'click')
GROUP BY
  cd.name,
  cd.path,
  e.text_description,
  e.name,
  e.type,
  click_target,
  action,
  JSON_EXTRACT_SCALAR(action_meta, '$.chapter_name'),
  JSON_EXTRACT_SCALAR(action_meta, '$.chapter_number'),
  JSON_EXTRACT_SCALAR(action_meta, '$.section_number'),
  CONCAT(IFNULL(JSON_EXTRACT_SCALAR(action_meta, '$.chapter_number'), 'X'), '.', IFNULL(JSON_EXTRACT_SCALAR(action_meta, '$.section_number'), '0')),
  DATETIME_TRUNC(e.created, DAY)
```
