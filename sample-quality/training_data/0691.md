How can I combine a path search prefix with a path mode by placing the mode in a subpath?

--- AGENT ---

Use the path search prefix on the outer path pattern and place the path mode inside a subpath pattern. This allows the prefix and mode to coexist, with the prefix controlling which paths are returned and the mode filtering within those results.

```googlesql
GRAPH FinGraph
MATCH ANY SHORTEST (TRAIL ->{1,4})
RETURN COUNT(1) as num_paths
```
