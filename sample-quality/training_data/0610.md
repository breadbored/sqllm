How can I determine what table an element in a graph came from?

--- AGENT ---

The `ELEMENT_DEFINITION_NAME` function returns the name of the underlying table (or element definition) that a graph element belongs to. This is useful for understanding the structural types of nodes and edges in a matched pattern.

```googlesql
GRAPH FinGraph
MATCH (p:Person)-[o:Owns]->(:Account)
RETURN
  p.name AS name,
  ELEMENT_DEFINITION_NAME(p) AS node_element_definition_name,
  ELEMENT_DEFINITION_NAME(o) AS edge_element_definition_name
```
