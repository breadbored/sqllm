--- USER ---

Create a BigQuery query that tags VAAs video viewing events based on muting behavior, capturing both session-level behavioral metadata and course-enriched content classifications.

--- AGENT ---

This query produces a union of two tagging datasets from VAAs video events, both filtered for sessions where muting settings were present.

The first branch tags events at the behavioral level: it groups muting events by sub_session_id, the muted setting value, and URL path, creating tags of the form `vaas_muted:<muted_value>` and classifying them as 'behavior' class events. It returns the earliest timestamp for each group.

The second branch enriches the same muting events with course context: it joins the events table to a courses table on video ID, then groups by the same dimensions (sub_session_id, URL path) but creates tags that combine the muted setting with the course name (`vaas_muted:<muted_value>:<course_name>`), classifying these as 'content' class events. It also returns the earliest timestamp per group.

The final result stacks both classifications into a single dataset where each row represents either a behavioral muting tag or a content-enriched muting tag for a given sub-session and URL, with an `app` field hardcoded to 'vaas' and a `dt` field set to the minimum event timestamp within each group.

```sql
SELECT
  e.sub_session_id as sub_session_id,
  CONCAT('vaas_muted:', e.vaas_settings_muted) as tag,
  'behavior' as class,
  'vaas' as app,
  e.vaas_url_url_path as url_path,
  MIN(e.created) as dt
FROM
  ${ref("base_buckets_events_vaas_speedy")} as e
WHERE
  e.vaas_settings_muted IS NOT NULL
GROUP BY
  sub_session_id,
  tag,
  url_path

UNION ALL

SELECT
  e.sub_session_id as sub_session_id,
  CONCAT('vaas_muted:', e.vaas_settings_muted, ':', c.name) as tag,
  'content' as class,
  'vaas' as app,
  e.vaas_url_url_path as url_path,
  MIN(e.created) as dt
FROM
  ${ref("base_buckets_events_vaas_speedy")} as e
JOIN
  ${ref("courses")} as c
ON
  c.vaas_id = e.vaas_url_video_id
WHERE
  e.vaas_settings_muted IS NOT NULL
GROUP BY
  sub_session_id,
  tag,
  url_path
```
