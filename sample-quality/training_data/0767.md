How do I generate a unique fingerprint for each row based on multiple columns combined together?

--- AGENT ---

The FARM_FINGERPRINT function computes a 64-bit fingerprint from the concatenated string representation of multiple columns, producing a consistent INT64 value that uniquely identifies each row combination. This is useful for detecting duplicate rows or creating row identifiers without needing to concatenate strings explicitly.

```googlesql
WITH example AS (
  SELECT 1 AS x, "foo" AS y, true AS z UNION ALL
  SELECT 2 AS x, "apple" AS y, false AS z UNION ALL
  SELECT 3 AS x, "" AS y, true AS z
)
SELECT
  *,
  FARM_FINGERPRINT(CONCAT(CAST(x AS STRING), y, CAST(z AS STRING)))
    AS row_fingerprint
FROM example;
```
