What happens if I include one field and then try to explicitly exclude another field that was already implicitly excluded?

--- AGENT ---

This query produces an error because once you include year as the first field, all other fields are implicitly excluded—including month. You cannot then explicitly exclude a field that has already been implicitly excluded.

```googlesql
SELECT FILTER_FIELDS(award_col, +year, -month) AS filtered_fields
FROM MusicAwards

-- Error
```
