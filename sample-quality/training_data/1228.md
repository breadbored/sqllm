How can I unset a field in a protocol buffer by replacing it with NULL?

--- AGENT ---

This query uses REPLACE_FIELDS to set the details field to NULL, effectively removing it from the proto while preserving other fields like title.

```googlesql
SELECT REPLACE_FIELDS(
  NEW Book("The Hummingbird" AS title,
    NEW BookDetails(10 AS chapters) AS details),
  NULL AS details)
AS proto;
```
