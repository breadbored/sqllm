How do I create a buffer zone around a point, controlling the number of segments used to approximate the circular boundary?

--- AGENT ---

`ST_BUFFER` creates a buffer geography around an input geography by a specified radius in meters, using a parameter to control the number of line segments that approximate the buffer curve.

```googlesql
SELECT
  -- num_seg_quarter_circle=2
  ST_NUMPOINTS(ST_BUFFER(ST_GEOGFROMTEXT('POINT(1 2)'), 50, 2)) AS eight_sides,
  -- num_seg_quarter_circle=8, since 8 is the default
  ST_NUMPOINTS(ST_BUFFER(ST_GEOGFROMTEXT('POINT(100 2)'), 50)) AS thirty_two_sides;
```
