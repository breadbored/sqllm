How do I filter array rows based on whether the array contains a specific value?

--- AGENT ---

Use `ARRAY(SELECT element FROM UNNEST(array) WHERE value IN UNNEST(array))` to filter elements based on whether a specific value exists anywhere in the array. The `IN UNNEST()` clause checks for presence of that value.

```googlesql
WITH Sequences AS
  (SELECT [0, 1, 1, 2, 3, 5] AS some_numbers
   UNION ALL SELECT [2, 4, 8, 16, 32] AS some_numbers
   UNION ALL SELECT [5, 10] AS some_numbers)
SELECT
   ARRAY(SELECT x
         FROM UNNEST(some_numbers) AS x
         WHERE 2 IN UNNEST(some_numbers)) AS contains_two
FROM Sequences;

/*--------------------+
 | contains_two       |
 +--------------------+
 | [0, 1, 1, 2, 3, 5] |
 | [2, 4, 8, 16, 32]  |
 | []                 |
 +--------------------*/
```
