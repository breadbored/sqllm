How do I create a table with a generated column that concatenates two other columns?

--- AGENT ---

A generated column is defined using the `GENERATED ALWAYS AS` clause with a scalar expression. The expression can reference other columns and call built-in functions like `CONCAT`. The `HIDDEN` attribute hides a column from `SELECT *` expansions.

```googlesql
CREATE TABLE Authors (
  first_name STRING HIDDEN,
  last_name STRING HIDDEN,
  full_name STRING GENERATED ALWAYS AS (CONCAT(first_name, ' ', last_name)));
```
