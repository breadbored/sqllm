How do I format a timestamp as a date in the pattern month-day-year?

--- AGENT ---

The `FORMAT_TIMESTAMP` function uses format elements like `%b` for abbreviated month, `%d` for day, and `%Y` for four-digit year. Combining these with literal hyphens produces a compact month-day-year format.

```googlesql
SELECT FORMAT_TIMESTAMP("%b-%d-%Y", TIMESTAMP "2050-12-25 15:30:55+00")
  AS formatted;
```
