How do I match nodes with a WHERE condition on properties?

--- AGENT ---

Node patterns can include a `WHERE` clause to filter nodes based on more complex conditions. This example matches `Person` nodes with a birthday before a specific date.

```googlesql
GRAPH FinGraph
MATCH (p:Person WHERE p.birthday < '1990-01-10')
RETURN p.name
```
