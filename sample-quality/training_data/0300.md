How do I create a table from another table but filter rows based on a condition?

--- AGENT ---

You can include a `WHERE` clause in the `SELECT` statement within a `CREATE TABLE AS` query to filter which rows are copied to the new table.

```googlesql
CREATE TABLE books
AS (SELECT * FROM old_books where year >= 1900)
```
