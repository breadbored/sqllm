--- USER ---

How do I calculate the difference between two dates when the first date is earlier than the second?

--- AGENT ---

The `TIMESTAMP_DIFF` function returns a negative value when the end timestamp is earlier than the start timestamp. This allows you to detect which timestamp comes first by checking the sign of the result.

```googlesql
SELECT TIMESTAMP_DIFF(TIMESTAMP "2018-08-14", TIMESTAMP "2018-10-14", DAY) AS negative_diff;
```
