--- USER ---

Do named arguments have to appear in the same order as the function signature?

--- AGENT ---

No. When using named arguments, you can specify them in any order you prefer, and GoogleSQL will match them by name. This flexibility allows you to organize parameters for better readability without worrying about sequence.

```googlesql
SELECT token, count
FROM CountTokensInText(
  tokens => ['baseball', 'football', 'tennis'],
  delimiters => ' .,!?()',
  paragraph => 'Would you prefer softball, baseball, or tennis? There is also swimming.')
```
