How do I write a JavaScript function that uses a regular expression to extract matching characters from a string?

--- AGENT ---

This query creates a JavaScript UDF that applies a regex pattern to find all lowercase letters in the input string. The function body requires a raw string literal to preserve the regex syntax without escaping.

```googlesql
CREATE TEMP FUNCTION ExtractLetters(x STRING)
RETURNS STRING
LANGUAGE js
AS r'''
var re = /[a-z]/g;
return x.match(re);
''';

SELECT val, ExtractLetters(val) AS result
FROM UNNEST(['ab-c', 'd_e', '!']) AS val;
```
