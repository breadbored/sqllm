How can I bucket datetimes into 7-day intervals aligned to a specific datetime reference point?

--- AGENT ---

This query uses DATETIME_BUCKET to group datetimes into 7-day buckets with a custom origin of 2000-12-22 12:00:00. The origin point acts as the boundary reference, and all buckets expand symmetrically from this point, allowing you to align bucket boundaries to a specific moment in time.

```googlesql
WITH some_datetimes AS (
  SELECT DATETIME '2000-12-20 00:00:00' AS my_datetime UNION ALL
  SELECT DATETIME '2000-12-21 00:00:00' UNION ALL
  SELECT DATETIME '2000-12-22 00:00:00' UNION ALL
  SELECT DATETIME '2000-12-23 00:00:00' UNION ALL
  SELECT DATETIME '2000-12-24 00:00:00' UNION ALL
  SELECT DATETIME '2000-12-25 00:00:00'
)
SELECT DATETIME_BUCKET(
  my_datetime,
  INTERVAL 7 DAY,
  DATETIME '2000-12-22 12:00:00') AS bucket_lower_bound
FROM some_datetimes;
```
