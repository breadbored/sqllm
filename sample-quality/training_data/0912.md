How do I convert a JSON boolean to an INT32?

--- AGENT ---

The `LAX_INT32` function converts JSON booleans to `INT32`: `true` becomes 1, `false` becomes 0.

```googlesql
SELECT LAX_INT32(JSON 'true') AS result;

SELECT LAX_INT32(JSON 'false') AS result;
```
