How do I construct a bytes value with multiple token chunks in different formats?

--- AGENT ---

Bytes literals can be built from multiple chunks using the `b` prefix, and can mix regular and triple-quoted formats. GoogleSQL concatenates all chunks into a single bytes value.

```googlesql
SELECT b'\x41' b'''\x42''' b"""\x41"""
```
