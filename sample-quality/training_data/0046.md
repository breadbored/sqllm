How can I get a non-deterministic value from a group of items?

--- AGENT ---

`ANY_VALUE` returns an arbitrary value from a group, useful when you don't care which specific row's value you retrieve. Here's a simple example that returns one fruit from the list:

```googlesql
SELECT ANY_VALUE(fruit) as any_value
FROM UNNEST(["apple", "banana", "pear"]) as fruit;
```
