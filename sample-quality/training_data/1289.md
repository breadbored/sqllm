How do I verify that one date range does not fall within another date range?

--- AGENT ---

RANGE_CONTAINS checks whether an inner range is contained in an outer range and returns FALSE if it is not. This query shows that the range [2023-01-01, 2023-04-01) is not contained in [2022-01-01, 2023-01-01) because the inner range starts at the upper boundary of the outer range.

```googlesql
SELECT RANGE_CONTAINS(
  RANGE<DATE> '[2022-01-01, 2023-01-01)',
  RANGE<DATE> '[2023-01-01, 2023-04-01)') AS results;
```
