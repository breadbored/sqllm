How do I check if a JSON array contains another array's elements?

--- AGENT ---

The `JSON_CONTAINS` function returns `true` if the first array contains all elements of the second array, regardless of order or duplicates.

```googlesql
SELECT
    JSON_CONTAINS(JSON '[1, 2, 3]', JSON '[2]') AS result1,
    JSON_CONTAINS(JSON '[1, 2, 3]', JSON '2') AS result2;
```
