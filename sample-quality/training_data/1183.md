How do I change an existing value in a protocol buffer map field?

--- AGENT ---

This query uses `PROTO_MODIFY_MAP` to update an existing map entry by passing the same key ('A') with a new value (6). When a key already exists, calling `PROTO_MODIFY_MAP` replaces the old value.

```googlesql
SELECT
  PROTO_MODIFY_MAP(m.purchased, 'A', 6) AS result_map
FROM
  (SELECT AS VALUE CAST("purchased { key: 'A' value: 22 }" AS Item)) AS m;
```
