--- USER ---

How do I create a simple SQL function that takes two integer inputs and returns a calculation result?

--- AGENT ---

This query creates a temporary SQL UDF that performs a calculation on input parameters and then calls the function on a set of test values. SQL UDFs execute the expression in the function body once per row, returning a single scalar value each time.

```googlesql
CREATE TEMP FUNCTION AddFourAndDivide(x INT64, y INT64)
RETURNS DOUBLE
AS (
  (x + 4) / y
);

WITH
  numbers AS (
    SELECT 1 AS val UNION ALL
    SELECT 3 AS val UNION ALL
    SELECT 4 AS val UNION ALL
    SELECT 5 AS val
  )
SELECT val, AddFourAndDivide(val, 2) AS result
FROM numbers;
```
