How do I get distinct combinations of multiple columns using the SELECT operator?

--- AGENT ---

`SELECT DISTINCT` followed by column names returns only distinct rows based on the specified columns. This is one way to get unique combinations of values.

```googlesql
FROM table
|> SELECT DISTINCT a, b, c;
```
