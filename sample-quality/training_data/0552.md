How do I call a function using named arguments?

--- AGENT ---

Named arguments allow you to specify parameters explicitly by name using the `=>` syntax. This makes it clear which value corresponds to which parameter and can improve code readability.

```googlesql
SELECT token, count
FROM CountTokensInText(
  paragraph => 'Would you prefer softball, baseball, or tennis? There is also swimming.',
  tokens => ['baseball', 'football', 'tennis'],
  delimiters => ' .,!?()')
```
