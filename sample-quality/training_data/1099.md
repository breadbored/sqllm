How do I call a table-valued function in standard SQL syntax, passing the result of another function as an argument?

--- AGENT ---

In standard SQL, table-valued functions can be nested by passing one function's output as the `TABLE` argument to another function. The outer function processes the result of the inner function.

```googlesql
-- Call the TVFs without using the CALL operator.
SELECT *
FROM
  tvf2(arg2, arg3, TABLE tvf1(TABLE input_table, arg1));
```
