--- USER ---

How do I filter nodes to only return those with a specific label, like Person?

--- AGENT ---

Add a label condition to the node pattern using a colon followed by the label name. The pattern `(n:Person)` matches only nodes with the Person label.

```googlesql
GRAPH FinGraph
MATCH (n:Person)
RETURN n.name, n.id
```
