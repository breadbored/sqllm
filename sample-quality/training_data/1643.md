--- USER ---

Show me a BigQuery query that identifies when users complete learning paths, combining both active course completions and legacy badge records, with deduplication across sources.

--- AGENT ---

This query merges two independent sources of learning path completion records and deduplicates them by user and learning path.

**First branch** aggregates current course completions:
- Joins users through user courses (grouped by user and course container) to learning path containers and learning paths
- Counts the total courses per learning path
- Flags a learning path as completed when the count of 'completed' statuses equals the total course count
- Marks these records as non-legacy (`legacy_badge = false`)
- Groups by user, learning path, and path name

**Second branch** pulls legacy badge records:
- Joins users directly to user badges
- Extracts the most recent badge creation timestamp per user-learning_path
- Marks these records as completed and legacy (`did_complete = true`, `legacy_badge = true`)
- Groups by user, learning path id, and path name

**Outer deduplication**:
- Unions both branches to combine the sources
- For each user-learning_path pair, uses `MIN_BY` to select the earliest completion date and associated values across duplicates
- Filters to exclude test users in both branches

The result is one row per user-learning_path combination with the earliest completion metadata.

```sql
SELECT
  both.user_id,
  both.learning_path_id,
  MIN_BY(both.learning_path_name, both.completed_date) AS learning_path_name,
  MIN(both.completed_date) AS completed_date,
  MIN_BY(both.did_complete, both.completed_date) AS did_complete,
  MIN_BY(both.legacy_badge, both.completed_date) AS legacy_badge
FROM
  (
    -- Badge completions by current courses
    SELECT
      u.id                                AS user_id,
      lp.id                               AS learning_path_id,
      lp.name                             AS learning_path_name,
      max(ucc.updated)                    AS completed_date, -- This is assuming that the user_courses entry will be updated only to change the status to 'completed'
      sum(
        CASE WHEN ucc.status = 'completed' THEN 1 ELSE 0 END
      ) = max(lp_num_courses.num_courses) AS did_complete,
      false AS legacy_badge
    FROM
      ${ref("stg_users")} AS u
    INNER JOIN
      (
        SELECT
          _uc.user_id,
          min(_uc.status)                 AS status,
          min_by(_uc.updated, _uc.status) AS updated,
          _c.container_id
        FROM
          ${ref("stg_user_courses")} AS _uc
        INNER JOIN
          ${ref("stg_courses")} AS _c
          ON
            _uc.course_id = _c.id
        GROUP BY
          _uc.user_id,
          _c.container_id
      ) AS ucc
      ON
        u.id = ucc.user_id
    INNER JOIN
      ${ref("stg_learning_path_course_containers")} AS lpcc
      ON
        ucc.container_id = lpcc.course_container_id
    INNER JOIN
      ${ref("stg_learning_paths")} AS lp
      ON
        lpcc.learning_path_id = lp.id
    INNER JOIN
      (
        SELECT
          _lp.id,
          count(_lpcc.id) AS num_courses
        FROM
          ${ref("stg_learning_path_course_containers")} AS _lpcc
        INNER JOIN
          ${ref("stg_learning_paths")} AS _lp
          ON
            _lpcc.learning_path_id = _lp.id
        GROUP BY
          _lp.id
      ) AS lp_num_courses
      ON
        lp.id = lp_num_courses.id
    WHERE
      u.test_user = 0
    GROUP BY
      u.id,
      lp.id,
      lp.name

    UNION ALL

    -- Existing badge entries
    SELECT
      u.id                                AS user_id,
      ub.learning_path_id                 AS learning_path_id,
      ub.name                             AS learning_path_name,
      max(ub.created)                     AS completed_date,
      true AS did_complete,
      true AS legacy_badge
    FROM
      ${ref("stg_users")} AS u
    JOIN
      ${ref("stg_user_badges")} as ub
    ON
      u.id = ub.user_id
    WHERE
      u.test_user = 0
    GROUP BY
      u.id,
      ub.learning_path_id,
      ub.name
  ) as both
GROUP BY
  both.user_id,
  both.learning_path_id
```
