How do I set an extension field on a protocol buffer?

--- AGENT ---

Extension fields are set using a parenthesized path (extension name) as the field name, with a comma preceding the extension field in the map constructor.

```googlesql
SELECT
  NEW googlesql.examples.music.Album {
    album_name: 'New Moon',
    (googlesql.examples.music.downloads): 30
  }
```
