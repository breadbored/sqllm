How do I set a system variable to store a configuration value as a literal string?

--- AGENT ---

System variables are prefixed with @@ and can be set to literal values using the SET statement. This example sets the system variable @@system_var_a to the string value "TEST".

```googlesql
SET @@system_var_a = "TEST";
```
