How do I count non-null values without noise on a view for validation?

--- AGENT ---

This query shows the un-noised version of counting non-NULL expression values with differential privacy applied to a view, using a high epsilon to suppress noise.

```googlesql
SELECT
  WITH DIFFERENTIAL_PRIVACY
    OPTIONS(epsilon=1e20, delta=.01, max_groups_contributed=1)
    item,
    COUNT(item, contribution_bounds_per_group=>(0, 100)) times_requested
FROM {{USERNAME}}.view_on_professors
GROUP BY item;
```
