How do I use pseudocolumns like ROWNUM with protocol buffer data in a value table?

--- AGENT ---

Pseudocolumns work with value tables when the source is aliased, as the alias represents the scanned rows rather than the value itself. This query successfully retrieves the row number and album title because the alias gives access to the underlying row's ROWNUM pseudocolumn.

```googlesql
SELECT a.ROWNUM, a.albumtitle AS title FROM AlbumReviewData AS a
```
