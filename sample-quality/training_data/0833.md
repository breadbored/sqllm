How do I extract string values from a JSON array while removing quotes?

--- AGENT ---

Combine `JSON_EXTRACT_ARRAY` with `JSON_EXTRACT_SCALAR` and `UNNEST` to extract and unquote string values from a JSON array.

```googlesql
SELECT ARRAY(
  SELECT JSON_EXTRACT_SCALAR(string_element, '$')
  FROM UNNEST(JSON_EXTRACT_ARRAY('["apples","oranges","grapes"]','$')) AS string_element
) AS string_array;
```
