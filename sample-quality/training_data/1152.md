How do I declare a variable and infer its type from a query result?

--- AGENT ---

When you omit the `variable_type` and provide a `DEFAULT` clause with a subquery, the variable's type is inferred from the query result. The variable can then hold a row as a `STRUCT` of the query's columns.

```googlesql
DECLARE item DEFAULT (SELECT item FROM schema1.products LIMIT 1);
```
