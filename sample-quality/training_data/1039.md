How do I add days to a date?

--- AGENT ---

The date arithmetic operator allows you to add or subtract an integer number of days to or from a date using the + and - operators.

```googlesql
SELECT DATE "2020-09-22" + 1 AS day_later, DATE "2020-09-22" - 7 AS week_ago
```
