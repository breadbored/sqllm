How can I get the unique identifier of the source node of an edge in a graph query?

--- AGENT ---

The `SOURCE_NODE_ID` function returns a string identifier for the source node of a graph edge. This identifier is unique within the scope of the current query and identifies where the edge originates.

```googlesql
GRAPH FinGraph
MATCH (p:Person)-[o:Owns]->(:Account)
RETURN p.name AS name, SOURCE_NODE_ID(o) AS source_node_id
```
