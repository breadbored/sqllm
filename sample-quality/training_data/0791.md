How do I convert a large JSON number to DOUBLE with rounding instead of an error?

--- AGENT ---

Set `wide_number_mode=>'round'` to round a JSON number that cannot be represented as a DOUBLE without loss of precision, rather than raising an error.

```googlesql
SELECT DOUBLE(JSON '18446744073709551615', wide_number_mode=>'round') as result;
```
