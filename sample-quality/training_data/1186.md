How do I create a table with a column that stores protocol buffer values?

--- AGENT ---

This query creates a `Customers` table with a column `Orders` that holds protocol buffer values of type `examples.shipping.Order`. The table uses `Id` as its primary key and can store complex nested protocol buffer messages in the `Orders` column.

```googlesql
CREATE TABLE Customers (
  Id INT64,
  Orders examples.shipping.Order,
) PRIMARY KEY(Id);
```
