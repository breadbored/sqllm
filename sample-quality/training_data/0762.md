--- USER ---

How do I check whether a person named "Lee" owns any account?

--- AGENT ---

This query uses an `EXISTS` subquery to verify whether at least one matching row exists. The subquery searches for a person named "Lee" with an ownership relationship to an account, and returns `TRUE` if found.

```googlesql
GRAPH FinGraph
RETURN EXISTS {
  MATCH (p:Person {Name: "Lee"})-[o:Owns]->(a:Account)
  RETURN p.Name
  LIMIT 1
} AS results;
```
