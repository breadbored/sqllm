--- USER ---

How do I match paths where two variables could bind to the same node or different nodes?

--- AGENT ---

Use different variable names for each node in the path. By default, different variables are independent and can bind to the same node or different nodes.

```googlesql
GRAPH FinGraph
MATCH (a:Account)-[t1:Transfers]->(mid:Account)-[t2:Transfers]->(a2)
RETURN a.id AS a_id, a2.id AS a2_id
```
