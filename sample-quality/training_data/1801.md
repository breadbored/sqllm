## Question
Can you show me a BigQuery table function that counts the distinct number of courses offered by a given client?

--- AGENT ---

This query defines a BigQuery table function that accepts a `clientId` parameter and returns the count of distinct course containers offered by that client.

The logic:
1. Joins `stg_clients_course_containers` (aliased as `ccc`) with `stg_courses` (aliased as `c`) on the condition that `c.container_id` matches `ccc.course_container_id`
2. Filters rows where the `client_id` column equals the input parameter `clientId`
3. Counts the distinct values of `course_container_id` and names the result `courses_offered`
4. Returns a single row with the aggregated count

The result is a function that can be invoked as `${self()}(some_client_id)` to retrieve the number of distinct courses offered by a specific client.

```sql
CREATE OR REPLACE TABLE
  FUNCTION ${self()} (clientId INT64) AS (
  SELECT
    COUNT(distinct course_container_id) AS courses_offered
  FROM
    ${ref("stg_clients_course_containers")} ccc
  JOIN
    ${ref("stg_courses")} c
  ON
    c.container_id = ccc.course_container_id
  WHERE
    client_id = clientId)
```
