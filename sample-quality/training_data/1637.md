--- USER ---

Provide me with a BigQuery SQL query that aggregates daily active user metrics to a monthly level.

--- AGENT ---

This query aggregates daily-level user activity metrics to monthly granularity. It reads from the `agg_daily_active_users` source table, truncates the date column to month boundaries, and sums the three user count metrics (`active_users`, `returning_users`, `new_users`) across all days within each month. The output is a monthly time-series with one row per month showing the total active sessions, returning user sessions, and new user sessions, ordered chronologically by month.

```sql
SELECT
  datetime_trunc(date, MONTH) AS year_month,
  sum(active_users)           AS total_active_sessions,
  sum(returning_users)        AS returning_user_sessions,
  sum(new_users)              AS new_user_sessions
FROM ${ref("agg_daily_active_users")}
GROUP BY year_month
ORDER BY year_month
```
