--- USER ---

Extract survey responses from question events, enrich them with course and document metadata, and create a summary field showing all answer variations given within each session for that content.

--- AGENT ---

This query processes question-type events in two aggregation stages:

**Stage 1 (base CTE):** Extracts course-questions and docs-questions events, unnesting their answer arrays from JSON metadata. For each answer option, it derives an answer_value from structured JSON or marks optional text as 'true'/'false'. It left-joins to course and document lookup tables for naming, and to client mapping tables for course containers and documents. After filtering for valid (non-empty, non-false) answers with at least one client association, it groups by event and collects all associated client IDs into an array.

**Stage 2 (answer_combos CTE):** Re-aggregates the base results by (sub_session_id, course_id, doc_id), creating a comma-separated string of all distinct answers per unique session-content combination.

**Final Select:** Unnests the client_ids array from base, joins to the answer_combos aggregation on sub_session and matching course or document, then re-aggregates to collect distinct client IDs back into an array. Returns the flattened event-level data with the composite answer_combo field showing all responses given in that session for the associated content.

```sql
WITH
base AS (
    SELECT
      vals.id,
      vals.session_id,
      vals.sub_session_id,
      vals.app,
      vals.event_name,
      vals.course_id,
      vals.doc_id,
      vals.in_popup,
      c.name as course_name,
      d.name as doc_name,
      vals.question,
      vals.answer,
      vals.answer_value,
      vals.other_text,
      vals.utm_campaign,
      vals.utm_content,
      vals.utm_medium,
      vals.utm_source,
      ARRAY_CONCAT(ARRAY_AGG(ccc.client_id), ARRAY_AGG(dc.client_id)) as client_ids
    FROM
      (
        SELECT
          e.id,
          e.app,
          e.name as event_name,
          e.session_id,
          e.sub_session_id,
          e.derived_course_id as course_id,
          e.derived_doc_id as doc_id,
          CASE
            WHEN
              e.utm_source = 'mh_docs'
            AND
              e.utm_medium = 'banner'
            AND
              e.utm_campaign = 'devevice_setup_native_openxr'
            THEN true
            ELSE false
          END as in_popup,
          JSON_EXTRACT_SCALAR(e.action_meta, '$.question') as question,
          a as answer,
          IF(a = 'other', IF(JSON_EXTRACT_SCALAR(e.action_meta, '$.other_text') != '', 'true', 'false'), JSON_EXTRACT_SCALAR(PARSE_JSON(JSON_EXTRACT(e.action_meta, '$.answers'))[a], '$')) as answer_value,
          JSON_EXTRACT_SCALAR(e.action_meta, '$.other_text') as other_text,
          e.utm_campaign,
          e.utm_content,
          e.utm_medium,
          e.utm_source,
          e.utm_term,
          e.created,
        FROM
          ${ref("all_events")} as e, UNNEST(ARRAY_CONCAT(JSON_KEYS(PARSE_JSON(JSON_EXTRACT(e.action_meta, '$.answers'))), ["other"])) as a
        WHERE
          e.name = 'course-questions'
        OR
          e.name = 'docs-questions'
        ORDER BY
          created ASC
      ) as vals
    LEFT JOIN
      ${ref("stg_courses")} as c
    ON
      c.id = vals.course_id
    LEFT JOIN
      ${ref("stg_clients_course_containers")} as ccc
    ON
      ccc.course_container_id = c.container_id
    LEFT JOIN
      ${ref("stg_documents")} as d
    ON
      d.id = vals.doc_id
    LEFT JOIN
      ${ref("stg_documents_clients")} as dc
    ON
      d.id = dc.doc_id
    WHERE
      answer_value IS NOT NULL
    AND
      answer_value != 'false'
    AND
      answer_value != ''
    AND
      (dc.client_id IS NOT NULL OR ccc.client_id IS NOT NULL)
    GROUP BY
      vals.id,
      vals.session_id,
      vals.sub_session_id,
      vals.app,
      vals.event_name,
      vals.course_id,
      vals.doc_id,
      vals.in_popup,
      vals.question,
      vals.answer,
      vals.answer_value,
      vals.other_text,
      vals.utm_campaign,
      vals.utm_content,
      vals.utm_medium,
      vals.utm_source,
      c.name,
      d.name
),
answer_combos AS (
  SELECT
    sub_session_id,
    course_id,
    doc_id,
    ARRAY_TO_STRING(
        ARRAY_AGG(
            DISTINCT answer
            ORDER BY answer
        ), 
        ', '
    ) AS answer_combo
  FROM
    base
  GROUP BY
    sub_session_id,
    course_id,
    doc_id
)
SELECT
  w.id,
  w.session_id,
  w.sub_session_id,
  w.app,
  w.event_name,
  w.course_id,
  w.doc_id,
  w.in_popup,
  w.course_name,
  w.doc_name,
  w.question,
  w.answer,
  w.answer_value,
  ac.answer_combo,
  w.other_text,
  w.utm_campaign,
  w.utm_content,
  w.utm_medium,
  w.utm_source,
  ARRAY_AGG(DISTINCT cids IGNORE NULLS) as client_ids
FROM
  base as w, w.client_ids as cids
JOIN
    answer_combos as ac
ON
    ac.sub_session_id = w.sub_session_id
AND
    ((ac.course_id IS NOT NULL AND ac.course_id = w.course_id) OR (ac.doc_id IS NOT NULL AND ac.doc_id = w.doc_id))
GROUP BY
  w.id,
  w.session_id,
  w.sub_session_id,
  w.app,
  w.event_name,
  w.course_id,
  w.doc_id,
  w.in_popup,
  w.course_name,
  w.doc_name,
  w.question,
  w.answer,
  ac.answer_combo,
  w.answer_value,
  w.other_text,
  w.utm_campaign,
  w.utm_content,
  w.utm_medium,
  w.utm_source
```
