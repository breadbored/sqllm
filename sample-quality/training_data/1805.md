## Question
Show me a BigQuery example that demonstrates how to combine a subquery to find the latest record with aggregation functions to gather all matching records in an array, for a specific client.

--- AGENT ---

This is a BigQuery table function that retrieves course containers for a given client, combining a subquery pattern with array aggregation.

The query's logic:

1. **Filters to a specific client** — starts with the input `clientId` parameter to find that client's course containers

2. **Uses a subquery to identify the latest course per container** — the inline view `mcid` finds the `MAX(id)` for each `container_id` in the courses table, treating the highest ID as the most recent course

3. **Joins through the container hierarchy** — connects the client to their course containers, then to all published courses in those containers

4. **Re-joins to the max course** — brings in the metadata of the latest course (specifically its name) via `co2`

5. **Aggregates all published courses** — groups by container and max-course details, collecting arrays of all published course versions and IDs within that container

**Output**: One row per course container showing:
- The container and client identifiers
- The latest course's ID and name (as scalar columns)
- Two arrays containing all versions and IDs of all published courses in that container

```sql
CREATE OR REPLACE TABLE
  FUNCTION ${self()} (clientId INT64) AS (
  SELECT
    ccc.client_id dashboard_id,
    mcid.id course_id,
    mcid.container_id,
    co2.name,
    ARRAY_AGG(co.version) AS versions,
    ARRAY_AGG(co.id) AS ids
  FROM
    ${ref("stg_clients")} c
  JOIN
    ${ref("stg_clients_course_containers")} ccc
  ON
    ccc.client_id = c.id
  JOIN
    ${ref("stg_courses")} co
  ON
    co.container_id = ccc.course_container_id
  JOIN (
    SELECT
      MAX(id) id,
      container_id
    FROM
      ${ref("stg_courses")}
    GROUP BY
      container_id) mcid
  ON
    mcid.container_id = ccc.course_container_id
  JOIN
    ${ref("stg_courses")} AS co2
  ON
    co2.id = mcid.id
  WHERE
    c.id = clientId
    AND co.published = 1
  GROUP BY
    c.id,
    ccc.course_container_id,
    ccc.client_id,
    mcid.id,
    mcid.container_id,
    co2.name)
```
