How do I lift an expression outside of a horizontal aggregation to avoid type errors?

--- AGENT ---

When you need to use the entire array in a calculation before aggregating, assign it to a separate variable first, then use that variable in the aggregation. This prevents the system from treating the array incorrectly in the aggregate function.

```googlesql
GRAPH FinGraph
LET arr1 = [1, 2, 3]
LET len = ARRAY_LENGTH(arr1)
LET avg_val = SUM(arr1 / len)
RETURN avg_val
```
