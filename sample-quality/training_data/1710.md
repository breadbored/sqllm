--- USER ---

Extract course content interaction tags from user session events, differentiating between published course containers and versioned course variants, with validation that lesson progress timestamps align with the session window.

--- AGENT ---

This query produces two parallel streams of course content tags, both anchored to user sessions and lesson progress data.

**First branch (Course Container)** — Tags courses by their published name using `MAX_BY(c.name, c.published)` to select the most recently published version. The tag format is `"course:latest:<course_name>"`. It groups by `sub_session_id`, `url_path`, and `container_id`, and applies a HAVING filter that ensures the minimum lesson progress timestamp falls within the event session's time window.

**Second branch (Course versions)** — Tags courses by their specific version number, producing tags like `"course:v<version>:<name>"`. It follows the same grouping and filtering logic but groups by `sub_session_id`, `url_path`, and the constructed `tag` field instead of `container_id`.

Both branches perform the same three-table join (events → lesson progress → course metadata), both filter by matching user_id and date, and both extract a URL path and earliest timestamp from the lesson progress records. The UNION ALL combines both tagged variants into a single result set.

---

```sql
-- Course Container (published course name)
SELECT
  es.sub_session_id AS sub_session_id,
  CONCAT("course:latest:", MAX_BY(c.name, c.published)) as tag, -- `courses:Course Name`
  'content' as class,
  'app' AS app,
  CONCAT("/courses/", uclp.course_id) as url_path,
  MIN(uclp.created) AS dt,
FROM
  ${ref('base_buckets_events_app_speedy')} AS es
JOIN
  ${ref('user_course_lesson_progress')} AS uclp
ON
  uclp.user_id = es.user_id
AND
  DATE(uclp.created) = DATE(es.created)
-- Join this on to get course data
JOIN
  ${ref("stg_courses")} as c
ON
  c.id = uclp.course_id
GROUP BY
  sub_session_id,
  url_path,
  c.container_id
HAVING
  MIN(uclp.created) BETWEEN MIN(es.created) AND MAX(es.created)

UNION ALL

-- Course versions
SELECT
  es.sub_session_id AS sub_session_id,
  CONCAT("course:v", c.version, ":", c.name) as tag, -- `courses:Course Name`
  'content' as class,
  'app' AS app,
  CONCAT("/courses/", uclp.course_id) as url_path,
  MIN(uclp.created) AS dt,
FROM
  ${ref('base_buckets_events_app_speedy')} AS es
JOIN
  ${ref('user_course_lesson_progress')} AS uclp
ON
  uclp.user_id = es.user_id
AND
  DATE(uclp.created) = DATE(es.created)
-- Join this on to get course data
JOIN
  ${ref("stg_courses")} as c
ON
  c.id = uclp.course_id
GROUP BY
  sub_session_id,
  url_path,
  tag
HAVING
  MIN(uclp.created) BETWEEN MIN(es.created) AND MAX(es.created)
```
