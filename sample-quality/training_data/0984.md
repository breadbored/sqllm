Does element ordering affect cosine distance with sparse vector pairs?

--- AGENT ---

For sparse vectors represented as dimension-magnitude pairs, reordering the pairs does not affect the cosine distance result. This query shows that sparse vectors with reordered dimension-magnitude pairs produce the same distance:

```googlesql
SELECT COSINE_DISTANCE([(1, 1.0), (2, 2.0)], [(1, 3.0), (2, 4.0)]) AS results;
```
