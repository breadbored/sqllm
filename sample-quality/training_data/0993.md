How can I group students into age brackets based on predefined boundary values?

--- AGENT ---

RANGE_BUCKET scans a sorted array of boundaries and returns the 0-based position representing the upper bound for a given value, making it useful for creating histograms and bucketing data into groups. This query groups students into age categories based on boundaries at ages 10, 20, and 30:

```googlesql
WITH students AS
(
  SELECT 9 AS age UNION ALL
  SELECT 20 AS age UNION ALL
  SELECT 25 AS age UNION ALL
  SELECT 31 AS age UNION ALL
  SELECT 32 AS age UNION ALL
  SELECT 33 AS age
)
SELECT RANGE_BUCKET(age, [10, 20, 30]) AS age_group, COUNT(*) AS count
FROM students
GROUP BY 1
```
