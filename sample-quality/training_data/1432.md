How do I extract a substring using a regex pattern with the REGEXP_SUBSTR synonym?

--- AGENT ---

The REGEXP_SUBSTR function is a synonym for REGEXP_EXTRACT, extracting text matching a regular expression with optional position and occurrence parameters.

```googlesql
WITH example AS
(SELECT 'Hello World Helloo' AS value, 'H?ello+' AS regex, 1 AS position, 1 AS
occurrence
)
SELECT value, regex, position, occurrence, REGEXP_SUBSTR(value, regex,
position, occurrence) AS regexp_value FROM example;
```
