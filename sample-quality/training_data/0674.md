How do I retrieve a shortest path between two nodes using a path search prefix?

--- AGENT ---

Use the ANY SHORTEST prefix before the path pattern to return one shortest path (fewest edges) from each pair of endpoints. The prefix is placed before the opening parenthesis of the path pattern.

```googlesql
GRAPH FinGraph
MATCH ANY SHORTEST (a:Account {is_blocked:true})-[t:Transfers]->{1, 4} (b:Account)
LET path_length = COUNT(t)
RETURN a.id AS a_id, path_length, b.id AS b_id
```
