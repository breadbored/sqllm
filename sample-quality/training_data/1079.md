How do I replace the value of a column based on an expression?

--- AGENT ---

The `SET` pipe operator replaces the values of specified columns. The expression can reference the current column values—in this case, squaring the x value and setting y to a constant.

```googlesql
(
  SELECT 1 AS x, 11 AS y
  UNION ALL
  SELECT 2 AS x, 22 AS y
)
|> SET x = x * x, y = 3;
```
