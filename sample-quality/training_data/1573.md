--- USER ---

How do I create a JavaScript function that returns different greetings based on the time of day?

--- AGENT ---

This query defines a JavaScript UDF with conditional logic that checks the current hour and returns a different message accordingly. Multi-statement JavaScript function bodies require a raw string literal to properly handle the code formatting.

```googlesql
CREATE TEMP FUNCTION CustomGreeting(a STRING)
RETURNS STRING
LANGUAGE js
AS r'''
  var d = new Date();
  if (d.getHours() < 12) {
    return 'Good Morning, ' + a + '!';
  } else {
    return 'Good Evening, ' + a + '!';
  }
''';

SELECT CustomGreeting(names) as everyone
FROM UNNEST(["Hannah", "Max", "Jakob"]) AS names;
```
