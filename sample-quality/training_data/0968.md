How do I create a struct literal using tuple syntax?

--- AGENT ---

A struct literal can be created using parentheses with comma-separated values. This tuple syntax infers the struct type from the element types, and requires at least two fields to distinguish it from a simple parenthesized expression.

```googlesql
(1, 2, 3)
```
