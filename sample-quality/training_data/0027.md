How do I compute a differentially private percentile using the deprecated ANON_PERCENTILE_CONT?

--- AGENT ---

This query demonstrates the deprecated `ANON_PERCENTILE_CONT` function with the `ANONYMIZATION` clause, computing the 50th percentile with explicit per-value bounds using `CLAMPED BETWEEN` syntax.

```googlesql
SELECT
  WITH ANONYMIZATION
    OPTIONS(epsilon=10, delta=.01, max_groups_contributed=1)
    item,
    ANON_PERCENTILE_CONT(quantity, 0.5 CLAMPED BETWEEN 0 AND 100) percentile_requested
FROM {{USERNAME}}.view_on_professors
GROUP BY item;
```
