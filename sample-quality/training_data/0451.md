What happens if a required date component is missing from the format string?

--- AGENT ---

`PARSE_DATE()` produces an error when the format string is missing elements that appear in the input string, because every element in the input must have a corresponding format element in the correct position.

```googlesql
SELECT PARSE_DATE('%A %b %e', 'Thursday Dec 25 2008');
```
