How do I replace text matching a regex pattern with captured groups?

--- AGENT ---

The REGEXP_REPLACE function replaces all matches of a regular expression with a replacement string that can reference captured groups using \1, \2, etc., or \0 for the entire match.

```googlesql
SELECT REGEXP_REPLACE('# Heading', r'^# ([a-zA-Z0-9\s]+$)', '<h1>\\1</h1>') AS html
```
