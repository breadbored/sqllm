## Question
Build a session-level summary table that captures the initial landing page, total page views, session duration, and UTM tracking parameters for each user session.

--- AGENT ---

This query constructs a **session-level aggregate** from an events table, with special handling to identify and anchor each session to its first page view.

**Structure:**

The query uses a three-level join strategy:

1. **Innermost subquery**: Finds the earliest `timestamp` per `session_id`
2. **Middle subquery (`first_view`)**: Joins events back to that earliest timestamp to identify the first event with a non-null `url_path` for each session, then groups to ensure one row per session/path/timestamp combination
3. **Main query**: Joins the full events table to `first_view` on `session_id`, ensuring we only process sessions that have a valid initial page view

**Output:**

One row per session, containing:
- `session_id`: unique session identifier
- `initial_path`: the `url_path` from the first event in the session
- `page_views`: distinct count of unique `url_path` values visited in the session
- `created`: earliest timestamp in the session
- `session_duration`: time elapsed between first and last event, in seconds
- `action_count` and `event_count`: raw counts of events (column names suggest a categorical distinction between "action" and "event" types, though the column types are not visible)
- Geographic/device fields: `country`, `city`, `device_type` (preserved as single values per session via `any_value()`)
- UTM parameters: `utm_host`, `utm_id`, `utm_source`, `utm_campaign`, `utm_medium`, `utm_term` (all aggregated with `max()`, suggesting these may be sparse or repeated across events)
- `user_agent` and `ip`: preserved per session (assumes these are constant within a session)

```sql
SELECT
  de.session_id,
  first_view.url_path         AS inital_path,
  count(DISTINCT de.url_path) AS page_views,
  min(de.timestamp)           AS created,
  timestamp_diff(
    timestamp(max(de.timestamp)),
    timestamp(min(de.timestamp)),
    SECOND
  )                           AS session_duration,
  count(action)               AS action_count,
  count(event)                AS event_count,
  country,
  any_value(city)             AS city,
  any_value(device_type)      AS device,
  max(referrer_host)          AS utm_host,
  max(utm_id)                 AS utm_id,
  max(utm_source)             AS utm_source,
  max(utm_campaign)           AS utm_campaign,
  max(utm_medium)             AS utm_medium,
  max(utm_term)               AS utm_term,
  user_agent,
  ip
FROM ${ref("stg_dd_docs_events")} AS de
INNER JOIN (
  SELECT
    de.session_id,
    de.url_path,
    de.timestamp
  FROM ${ref("stg_dd_docs_events")} AS de
  INNER JOIN (
    SELECT
      session_id,
      min(timestamp) AS timestamp
    FROM ${ref("stg_dd_docs_events")} AS de
    GROUP BY session_id
  ) AS first_load
    ON de.session_id = first_load.session_id AND de.timestamp = first_load.timestamp
  WHERE url_path IS NOT NULL
  GROUP BY de.session_id, de.url_path, de.timestamp
) AS first_view
  ON de.session_id = first_view.session_id
GROUP BY de.session_id, first_view.url_path, country, user_agent, ip
```
