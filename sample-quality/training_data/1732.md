--- USER ---

Identify fullscreen viewing events in the VAAS application, creating behavior tags and content tags based on course enrollment, aggregated by session.

--- AGENT ---

This query produces a unified dataset of fullscreen viewing behavior in a VAAS application, combining two complementary views:

**First branch — behavior-level tagging:**
Captures fullscreen viewing events from the events table where the fullscreen setting is explicitly enabled. Groups by session and URL path, recording the earliest timestamp of that fullscreen viewing session. All records are tagged as 'behavior' class with the generic tag 'vaas_fullscreen'.

**Second branch — content-level tagging:**
Takes the same fullscreen events but enriches them by joining to a courses table on video ID. This allows each fullscreen session to be tagged with the course name (e.g., 'vaas_fullscreen:course_name'), classified as 'content' instead of 'behavior'. Also groups by session and URL, recording the earliest timestamp.

**Combined output:**
The UNION ALL merges both branches into a single result set where each row represents a unique sub_session_id, url_path combination with its earliest fullscreen timestamp. The first branch produces generic behavior records; the second produces course-specific content records. Both share the same column structure and app identifier.

```sql
SELECT
  e.sub_session_id as sub_session_id,
  'vaas_fullscreen' as tag,
  'behavior' as class,
  'vaas' as app,
  e.vaas_url_url_path as url_path,
  MIN(e.created) as dt
FROM
  ${ref("base_buckets_events_vaas_speedy")} as e
WHERE
  e.vaas_settings_fullscreen IS NOT NULL
AND
  e.vaas_settings_fullscreen = 'fullscreen'
GROUP BY
  sub_session_id,
  url_path

UNION ALL

SELECT
  e.sub_session_id as sub_session_id,
  CONCAT('vaas_fullscreen:', c.name) as tag,
  'content' as class,
  'vaas' as app,
  e.vaas_url_url_path as url_path,
  MIN(e.created) as dt
FROM
  ${ref("base_buckets_events_vaas_speedy")} as e
JOIN
  ${ref("courses")} as c
ON
  c.vaas_id = e.vaas_url_video_id
WHERE
  e.vaas_settings_fullscreen IS NOT NULL
AND
  e.vaas_settings_fullscreen = 'fullscreen'
GROUP BY
  sub_session_id,
  tag,
  url_path
```
