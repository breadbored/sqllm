How do I compute a differentially private sum of quantities per item with contribution bounds?

--- AGENT ---

This query uses `SUM` with `DIFFERENTIAL_PRIVACY` to compute the sum of quantities per item, with explicit contribution bounds applied per group. Noise is added based on the epsilon parameter for privacy.

```googlesql
SELECT
  WITH DIFFERENTIAL_PRIVACY
    OPTIONS(epsilon=10, delta=.01, max_groups_contributed=1, privacy_unit_column=id)
    item,
    SUM(quantity, contribution_bounds_per_group => (0,100)) quantity
FROM professors
GROUP BY item;
```
