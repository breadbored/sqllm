How do I rename a TVF output column to avoid conflicts with existing columns?

--- AGENT ---

The `YIELD` clause with `AS` allows you to rename TVF output columns to prevent naming conflicts with columns in the outer scope. This example renames the TVF's `Id` column to `tvf_Id` to distinguish it from the person's `Id`.

```googlesql
GRAPH FinGraph
MATCH (p:Person {Id: 1})
CALL my_tvf(p) YIELD Id AS tvf_Id
RETURN p.Id AS person_Id, tvf_Id;
```
