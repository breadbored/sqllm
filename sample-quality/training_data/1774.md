--- USER ---

How can I calculate the average time spent on courses per unique user for a specific client?

--- AGENT ---

This table function accepts a client ID parameter and returns the average time on course per unique user for that client. It sums all time_on_course_minutes from the reporting_user_courses table and divides by the count of distinct user_ids, filtering to include only rows where the provided clientId parameter appears within the client_id array column of each row.

```sql
CREATE OR REPLACE TABLE
  FUNCTION ${self()} (clientId INT64) AS (
  SELECT
    (SUM(time_on_course_minutes) / COUNT(DISTINCT user_id)) AS avg_time_on_course
  FROM
    ${ref("reporting_user_courses")}
  WHERE
    clientId IN UNNEST(client_id))
```
