--- USER ---

How can I match multiple independent path patterns in a single query to combine results from different relationship types?

--- AGENT ---

Specify multiple path patterns separated by commas in the MATCH clause. Each pattern is evaluated independently, and the results are combined. In this example, the first pattern traces transfers between accounts, and the second pattern finds who owns the middle account.

```googlesql
GRAPH FinGraph
MATCH
  (src:Account)-[t1:Transfers]->(mid:Account)-[t2:Transfers]->(dst:Account),
  (mid)<-[:Owns]-(p:Person)
RETURN
  p.name, src.id AS src_account_id, mid.id AS mid_account_id,
  dst.id AS dst_account_id
```
