--- USER ---

How do I create a block of statements with a variable scope that ends when the block ends?

--- AGENT ---

Use `BEGIN...END` to define a block where variables declared inside the block are only visible within that block. Variables declared in an outer scope can be referenced inside, but inner variables are not accessible after `END`.

```googlesql
DECLARE x INT64 DEFAULT 10;
BEGIN
  DECLARE y INT64;
  SET y = x;
  SELECT y;
END;
SELECT x;
```
