--- USER ---

How do I match nodes that have a specific label using a label expression?

--- AGENT ---

Use the label syntax with a colon in a node pattern. A label expression can consist of a single label name that nodes must match.

```googlesql
GRAPH FinGraph
MATCH (n:Person)
RETURN n.name
```
