How do I add 5 days to a date value?

--- AGENT ---

`DATE_ADD()` adds a specified time interval to a DATE value. The interval is specified using the INTERVAL keyword with a number and a date part like DAY, WEEK, or MONTH.

```googlesql
SELECT DATE_ADD(DATE '2008-12-25', INTERVAL 5 DAY) AS five_days_later;
```
