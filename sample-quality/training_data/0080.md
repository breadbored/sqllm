How do I check if all values in a group satisfy a condition?

--- AGENT ---

`LOGICAL_AND` returns true only if all non-NULL expressions in a group are true. This example checks whether all values are less than 3:

```googlesql
SELECT LOGICAL_AND(x < 3) AS logical_and FROM UNNEST([1, 2, 4]) AS x;
```
