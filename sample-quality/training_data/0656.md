How do I find all edges with a specific label, like Owns?

--- AGENT ---

Add a label condition to an edge pattern using a colon followed by the label name. The pattern `-[e:Owns]->` matches only edges that have the Owns label.

```googlesql
GRAPH FinGraph
MATCH -[e:Owns]->
RETURN e.id
```
