Show me an example BigQuery query that selects and transforms columns from a user-client mapping table.

--- AGENT ---

This query selects several columns from the client_users table and applies transformations. It retrieves: id, client_id, and user_id columns as-is; transforms saml_id by converting it to lowercase; includes created and updated columns; and constructs an array field named client_ids using a subquery that selects client_id. The result is a flattened view of client user records with the saml_id normalized to lowercase.

```sql
SELECT
  id,
  client_id,
  user_id,
  LOWER(saml_id) AS saml_id,
  created,
  updated,
  ARRAY(SELECT client_id) as client_ids,
FROM
  ${ref("client_users")}
```
