--- USER ---

How do I compute hyperbolic cosecant values, including a safe version for problematic inputs?

--- AGENT ---

CSCH computes the hyperbolic cosecant of an angle in radians, while SAFE.CSCH returns NULL when the computation would error. This query demonstrates hyperbolic cosecant values for multiple inputs, with a safe evaluation of zero:

```googlesql
SELECT CSCH(0.5) AS a, CSCH(-2) AS b, SAFE.CSCH(0) AS c;
```
