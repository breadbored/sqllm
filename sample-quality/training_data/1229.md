How do I convert a native GoogleSQL DATE type into a protocol buffer date message?

--- AGENT ---

This query uses TO_PROTO to convert a DATE value into a google.type.Date proto message, returning the date as a structured proto with year, month, and day fields.

```googlesql
SELECT TO_PROTO(DATE '2019-10-30')
```
