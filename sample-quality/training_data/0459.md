How do I calculate the number of days between two DATETIME values?

--- AGENT ---

DATETIME_DIFF returns the number of unit boundaries between two DATETIME values at the specified granularity. This example counts the number of days between the two datetimes.

```googlesql
SELECT
  DATETIME "2010-07-07 10:20:00" as first_datetime,
  DATETIME "2008-12-25 15:30:00" as second_datetime,
  DATETIME_DIFF(DATETIME "2010-07-07 10:20:00",
    DATETIME "2008-12-25 15:30:00", DAY) as difference;
```
