How do I find the number of elements in an array?

--- AGENT ---

Use `ARRAY_LENGTH` to return the count of elements in an array. The function returns 0 for an empty array, NULL if the input array is NULL, and handles NULL elements within the array normally.

```googlesql
SELECT
  ARRAY_LENGTH(["coffee", NULL, "milk" ]) AS size_a,
  ARRAY_LENGTH(["cake", "pie"]) AS size_b;
```
