How do you find transfers where a specific node is the source sending the transaction?

--- AGENT ---

The IS SOURCE predicate checks whether a node is the source of an edge. This query returns transfer records where node 'a' is the source, meaning 'a' is sending the transfer to 'b'.

```googlesql
GRAPH FinGraph
MATCH (a:Account)-[transfer:Transfers]-(b:Account)
WHERE a IS SOURCE of transfer
RETURN a.id AS a_id, b.id AS b_id
```
