How do I calculate a three-row moving average (one row before, current row, one row after)?

--- AGENT ---

This query creates a window frame that includes one row before, the current row, and one row after based on the ORDER BY sort. The AVG function then calculates the average purchases for these three rows at each position.

```googlesql
SELECT item, purchases, category, AVG(purchases)
  OVER (
    ORDER BY purchases
    ROWS BETWEEN 1 PRECEDING AND 1 FOLLOWING
  ) AS avg_purchases
FROM Produce
```
