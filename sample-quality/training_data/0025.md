How do I count non-null values using the deprecated ANON_COUNT with clamping?

--- AGENT ---

This query demonstrates the second signature of the deprecated `ANON_COUNT` function, which counts non-NULL values in an expression using `CLAMPED BETWEEN` syntax for explicit bounds.

```googlesql
SELECT
  WITH ANONYMIZATION
    OPTIONS(epsilon=10, delta=.01, max_groups_contributed=1)
    item,
    ANON_COUNT(item CLAMPED BETWEEN 0 AND 100) times_requested
FROM {{USERNAME}}.view_on_professors
GROUP BY item;
```
