## Question
Show me a BigQuery query that extracts and classifies user interaction events from a documentation platform, generating different behavior tags based on whether a user's action includes click target information.

--- AGENT ---

This query produces behavioral event tags from a stream of `expand_section` events in a documentation platform. It creates three variants of the same underlying event, each with increasing specificity:

**Branch 1** captures basic `expand_section` events without additional interaction details. For each sub_session and URL path, it extracts the earliest app that recorded the event (using `MIN_BY`) and tags it simply as `expand_section`.

**Branch 2** filters to only `expand_section` events where a user clicked a target element. It constructs a more specific tag by appending the action type (e.g., `expand_section_click`), and hardcodes the app as `docs`.

**Branch 3** further refines Branch 2 by adding click target details to the tag (e.g., `expand_section_click_header`). This captures the most granular behavioral variant when both action and target are recorded.

All three branches are unioned together, producing a single behavioral event stream where each row represents a distinct interaction pattern, classified by tag specificity and grouped by session, URL path, and event timestamp.

```sql
SELECT
    e.sub_session_id as sub_session_id,
    'expand_section' as tag,
    'behavior' as class,
    MIN_BY(e.app, e.created) as app,
    e.url_path as url_path,
    e.created as dt
FROM
    ${ref("base_buckets_events_docs_speedy")} as e
WHERE
    e.name = 'expand_section'

GROUP BY
    sub_session_id,
    tag,
    url_path,
    dt

UNION ALL

SELECT
  e.sub_session_id as sub_session_id,
  CONCAT('expand_section_', e.action) as tag,
  'behavior' as class,
  'docs' as app,
  e.url_path as url_path,
  e.created as dt
FROM
  ${ref('base_buckets_events_docs_speedy')} as e
WHERE
  e.click_target IS NOT NULL
AND
  e.name = 'expand_section'
GROUP BY
  sub_session_id,
  tag,
  url_path,
  dt

UNION ALL

SELECT
  e.sub_session_id as sub_session_id,
  CONCAT('expand_section_', e.action, '_', e.click_target) as tag,
  'behavior' as class,
  'docs' as app,
  e.url_path as url_path,
  e.created as dt
FROM
  ${ref('base_buckets_events_docs_speedy')} as e
WHERE
  e.click_target IS NOT NULL
AND
  e.name = 'expand_section'
GROUP BY
  sub_session_id,
  tag,
  url_path,
  dt
```
