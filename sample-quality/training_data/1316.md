What does the correlation function return when all values have zero variance?

--- AGENT ---

When all input values in either variable have zero variance (all values are identical), `CORR` returns NaN because the correlation coefficient cannot be meaningfully computed without variation in the data.

```googlesql
SELECT CORR(x, y) AS results
FROM
  (
    SELECT 0 AS x, 0 AS y
    UNION ALL
    SELECT 0 AS x, 0 AS y
  )
```
