How do I generate an array of integers from a starting value to an ending value with a specific step?

--- AGENT ---

The GENERATE_ARRAY function creates an array of values between two numbers with a specified step interval. A positive step generates values in ascending order.

```googlesql
SELECT GENERATE_ARRAY(11, 33, 2) AS odds;
```
