How do I parse a datetime range string into a RANGE<DATETIME> value?

--- AGENT ---

CAST to RANGE<DATETIME> interprets a string containing two datetime values separated by a comma and converts it to a RANGE type with DATETIME boundaries.

```googlesql
SELECT CAST(
  '[2014-09-27 12:30:00.45, 2016-10-17 11:15:00.33)'
  AS RANGE<DATETIME>) AS string_to_range
```
