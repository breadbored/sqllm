How do I apply case-insensitive collation to a column during sorting?

--- AGENT ---

The 'und:ci' Unicode collation specification performs case-insensitive comparison following the Unicode Collation Algorithm. This query applies case-insensitive collation to the Place column in the ORDER BY clause, making the sort ignore differences between uppercase and lowercase letters.

```googlesql
SELECT Place
FROM Locations
ORDER BY Place COLLATE 'und:ci'
```
