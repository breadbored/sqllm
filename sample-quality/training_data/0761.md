--- USER ---

How do I create an array of outgoing transfer amounts for each account owned by a person?

--- AGENT ---

This query uses an `ARRAY` subquery to collect all transfer amounts from each account into an array. For each person and their owned account, the `ARRAY` expression executes a nested graph query that matches all transfers from that account and returns their amounts as a list.

```googlesql
GRAPH FinGraph
MATCH (p:Person)-[:Owns]->(account:Account)
RETURN
 p.name, account.id AS account_id,
 ARRAY {
   MATCH (a:Account)-[transfer:Transfers]->(:Account)
   WHERE a = account
   RETURN transfer.amount AS transfers
 } AS transfers;
```
