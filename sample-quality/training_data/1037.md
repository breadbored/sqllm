How do I flatten all values from a deeply nested array structure?

--- AGENT ---

The array elements field access operator with FLATTEN traverses nested arrays and structs to extract all values at a specified field level, combining them into a single array.

```googlesql
SELECT FLATTEN(my_array.sales.prices) AS all_prices FROM SalesTable
```
