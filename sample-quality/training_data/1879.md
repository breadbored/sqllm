--- USER ---

For each knowledge check question, retrieve the associated client IDs based on the course container hierarchy, or default to a fallback client if none are found.

--- AGENT ---

This query builds a mapping of knowledge checks to their associated clients by traversing the course structure hierarchy, then joins that mapping back to knowledge check questions.

The CTE `knowledge_check_clients` performs a multi-table join chain: it connects `clients_course_containers` through `courses`, `course_lessons`, `lessons`, `learning_cues`, and finally to `knowledge_checks`. For each knowledge check, it aggregates all distinct client IDs that flow through this hierarchy into an array.

The main query selects all columns from `knowledge_check_questions` and left joins to the CTE on `knowledge_check_id`. The `COALESCE` function ensures that if a knowledge check question has no associated clients (NULL array from the left join), it defaults to `[1]`.

The net effect is a row per knowledge check question with its full metadata plus an array of client IDs, or a default array containing just `1` if no clients are found.

```sql
WITH
knowledge_check_clients AS (
    SELECT
        kc.id as knowledge_check_id,
        ARRAY_AGG(DISTINCT ccc.client_id IGNORE NULLS) as client_ids
    FROM
        ${ref("clients_course_containers")} as ccc
    JOIN
        ${ref("courses")} as c
    ON
        c.container_id = ccc.course_container_id
    JOIN
        ${ref("course_lessons")} as cl
    ON
        cl.course_id = c.id
    JOIN
        ${ref("lessons")} as l
    ON 
        l.id = cl.lesson_id
    JOIN
        ${ref("learning_cues")} as lc
    ON
        lc.lesson_id = l.id
    JOIN
        ${ref("knowledge_checks")} as kc
    ON
        kc.learning_cue_id = lc.id
    GROUP BY
        kc.id
)
SELECT
  kcq.id,
  kcq.knowledge_check_id,
  kcq.question_id,
  kcq.created,
  kcq.updated,
  COALESCE(kcc.client_ids, [1]) AS client_ids
FROM
  ${ref("knowledge_check_questions")} as kcq
LEFT JOIN
    knowledge_check_clients as kcc
ON
    kcc.knowledge_check_id = kcq.knowledge_check_id
```
