How do I filter edges using a WHERE clause with multiple conditions, like a date range?

--- AGENT ---

Place a WHERE clause inside the edge pattern and use AND to combine multiple conditions. This example filters edges whose creation time falls within a specific range.

```googlesql
GRAPH FinGraph
MATCH -[e:Owns WHERE e.create_time > '2020-01-14'
                 AND e.create_time < '2020-05-14']->
RETURN e.id
```
