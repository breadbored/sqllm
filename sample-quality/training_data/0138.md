How do I slice an array when both start and end point to the same position?

--- AGENT ---

Use `ARRAY_SLICE` with identical start and end offsets. The function returns an array containing only the single element at that position.

```googlesql
SELECT ARRAY_SLICE(['a', 'b', 'c', 'd', 'e'], 3, 3) AS result
```
