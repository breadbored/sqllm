How can I check if a graph path contains any repeated nodes?

--- AGENT ---

The `IS_ACYCLIC` function returns `TRUE` if a graph path has no repeating nodes, and `FALSE` if any node appears more than once in the path. This is useful for filtering paths that form cycles.

```googlesql
GRAPH FinGraph
MATCH p=(src:Account)-[t1:Transfers]->(mid:Account)-[t2:Transfers]->(dst:Account)
RETURN src.id AS source_account_id, IS_ACYCLIC(p) AS is_acyclic_path
```
