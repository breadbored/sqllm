How do I calculate week differences using different week start days?

--- AGENT ---

DATETIME_DIFF supports WEEK (which defaults to Sunday), WEEK(MONDAY), and ISOWEEK (which uses ISO 8601 standards starting on Monday). This example shows how the same two dates yield different results depending on which week definition is used.

```googlesql
SELECT
  DATETIME_DIFF('2017-12-18', '2017-12-17', WEEK) AS week_diff,
  DATETIME_DIFF('2017-12-18', '2017-12-17', WEEK(MONDAY)) AS week_weekday_diff,
  DATETIME_DIFF('2017-12-18', '2017-12-17', ISOWEEK) AS isoweek_diff;
```
