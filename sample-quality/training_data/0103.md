How do I calculate approximate quantiles with only distinct values while keeping NULLs in the result?

--- AGENT ---

Combining DISTINCT and RESPECT NULLS removes duplicate values while still including NULL in the quantile calculation, affecting both the count of unique values and where NULL appears in the output boundaries.

```googlesql
SELECT APPROX_QUANTILES(DISTINCT x, 2 RESPECT NULLS) AS approx_quantiles
FROM UNNEST([NULL, NULL, 1, 1, 1, 4, 5, 6, 7, 8, 9, 10]) AS x;
```
