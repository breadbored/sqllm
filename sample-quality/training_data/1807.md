Show me a BigQuery example of how to aggregate user performance metrics across courses, filtering to the latest attempt per question and aggregating at the course container level.

--- AGENT ---

This query is a parameterized table function that produces aggregated performance metrics filtered to a specific client. Here's how it works:

**CTEs:**
- `max_course_ids`: For each course container, identifies the course with the highest ID value
- `course_names`: Retrieves the name of that highest-ID course per container

**Main query logic:**
- Starts from stg_user_knowledge_check_answers (answer records) and joins with stg_users, stg_courses, and stg_clients_course_containers to correlate the data
- The LEFT JOIN to the subquery `ba` filters the dataset to include only rows where ukca.id is the maximum id within each (user_id, course_id, knowledge_check_id, question_id) grouping — effectively selecting one latest attempt per question per user per course
- Joins in course names indexed by container
- Filters to a specific clientId parameter and excludes records where the test_user field equals 0

**Output aggregation** at the (container_id, course_name) level:
- `total_users`: COUNT(DISTINCT user_id) — unique users represented
- `questions_answered`: COUNT(ukca.id) — total row count
- `total_correct_rate`: ROUND(AVG(ba.correct) * 100) — average of the correct field, expressed as a percentage

```sql
CREATE OR REPLACE TABLE
  FUNCTION ${self()} (clientId INT64) AS (
  WITH
    max_course_ids AS (
    SELECT
      container_id,
      MAX(id) AS max_id
    FROM
      ${ref("stg_courses")}
    GROUP BY
      container_id ),
    course_names AS (
    SELECT
      c.container_id,
      c.name
    FROM
      ${ref("stg_courses")} c
    JOIN
      max_course_ids mci
    ON
      c.id = mci.max_id )
  SELECT
    c.container_id,
    cn.name,
    COUNT(DISTINCT ukca.user_id) AS total_users,
    COUNT(ukca.id) AS questions_answered,
    ROUND(AVG(ba.correct) * 100) AS total_correct_rate
  FROM
    ${ref("stg_user_knowledge_check_answers")} ukca
  JOIN
    ${ref("stg_users")} u
  ON
    u.id = ukca.user_id
  JOIN
    ${ref("stg_courses")} c
  ON
    c.id = ukca.course_id
  JOIN
    ${ref("stg_clients_course_containers")} ccc
  ON
    ccc.course_container_id = c.container_id
  LEFT JOIN (
    SELECT
      MAX(ukca.id) AS id,
      user_id,
      course_id,
      knowledge_check_id,
      question_id,
      MAX(correct) AS correct
    FROM
      ${ref("stg_user_knowledge_check_answers")} ukca
    JOIN
      ${ref("stg_users")} u
    ON
      u.id = ukca.user_id
    JOIN
      ${ref("stg_courses")} c
    ON
      c.id = ukca.course_id
    JOIN
      ${ref("stg_clients_course_containers")} ccc
    ON
      ccc.course_container_id = c.container_id
    WHERE
      ccc.client_id = clientId
      AND u.test_user = 0
    GROUP BY
      user_id,
      course_id,
      knowledge_check_id,
      question_id ) AS ba
  ON
    ba.id = ukca.id
    AND ba.user_id = ukca.user_id
    AND ba.course_id = ukca.course_id
    AND ba.knowledge_check_id = ukca.knowledge_check_id
    AND ba.question_id = ukca.question_id
  JOIN
    course_names cn
  ON
    cn.container_id = c.container_id
  WHERE
    ccc.client_id = clientId
    AND u.test_user = 0
  GROUP BY
    c.container_id,
    cn.name)
```
