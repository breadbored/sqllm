--- USER ---

How do I check if a person named "Lee" owns an account without explicitly returning any columns from the match?

--- AGENT ---

This query uses an `EXISTS` subquery with only a `MATCH` statement, omitting the `RETURN` clause. The subquery still evaluates to `TRUE` if the pattern matches at least one row, even without specifying what to return.

```googlesql
GRAPH FinGraph
RETURN EXISTS {
  MATCH (p:Person {Name: "Lee"})-[o:Owns]->(a:Account)
} AS results;
```
