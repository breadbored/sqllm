How do I specify the privacy unit column when my data source is a subquery with an alias?

--- AGENT ---

Reference the privacy unit column through the subquery alias by qualifying the column name with that alias (members.id). The privacy_unit_column option in the differential privacy clause accepts the fully qualified aliased reference.

```googlesql
SELECT WITH DIFFERENTIAL_PRIVACY
  OPTIONS (epsilon=10, delta=.01, privacy_unit_column=members.id)
  item,
  COUNT(*, contribution_bounds_per_group=>(0, 100))
FROM (SELECT * FROM students) AS members;
```
