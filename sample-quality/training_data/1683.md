Provide me with a BigQuery SQL query that creates behavioral tags at the sub-session level, marking logged-in user sessions and capturing the initial app and URL from the earliest event in each session.

--- AGENT ---

This query creates behavioral tag records at the sub-session level by aggregating event data for authenticated users. For each unique sub_session_id where user_id is present, it:

1. Assigns a fixed tag label ('ever_logged_in') and classification ('behavior')
2. Uses MIN_BY to capture the app and url_path values from the event with the earliest timestamp in that session
3. Records the minimum (earliest) timestamp as the session's start time

The result is one row per sub_session_id containing the session identifier, tag metadata, the first app and URL path encountered, and the session start time.

```sql
SELECT
  es.sub_session_id as sub_session_id,
  'ever_logged_in' as tag,
  'behavior' as class,
  MIN_BY(es.app, es.created) as app,
  MIN_BY(es.url_path, es.created) as url_path,
  MIN(es.created) as dt

FROM
  ${ref('base_buckets_events_app_speedy')} as es

WHERE
  es.user_id IS NOT NULL

GROUP BY
  es.sub_session_id
```
