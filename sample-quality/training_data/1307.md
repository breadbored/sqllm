--- USER ---

How do I find all nodes that can be reached from a specific starting node in a directed graph?

--- AGENT ---

A recursive CTE can explore graph reachability by starting with a base node, then iteratively joining the intermediate result against the graph edges to find newly reachable nodes. Each iteration of the recursive term discovers one more hop of connected nodes until no new nodes can be reached. This query finds all nodes reachable from node 5 in a directed acyclic graph by joining the running set of known nodes against the edge table:

```googlesql
WITH RECURSIVE
  GraphData AS (
    --    1          5
    --   / \        / \
    --  2 - 3      6   7
    --      |       \ /
    --      4        8
    SELECT 1 AS from_node, 2 AS to_node UNION ALL
    SELECT 1, 3 UNION ALL
    SELECT 2, 3 UNION ALL
    SELECT 3, 4 UNION ALL
    SELECT 5, 6 UNION ALL
    SELECT 5, 7 UNION ALL
    SELECT 6, 8 UNION ALL
    SELECT 7, 8
  ),
  R AS (
    (SELECT 5 AS node)
    UNION ALL
    (
      SELECT GraphData.to_node AS node
      FROM R
      INNER JOIN GraphData
        ON (R.node = GraphData.from_node)
    )
  )
SELECT DISTINCT node FROM R ORDER BY node;
```
