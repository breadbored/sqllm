--- USER ---

How can I compute cosecant values for different angles, including safely handling inputs that would error?

--- AGENT ---

CSC computes the cosecant of an angle in radians, and SAFE.CSC prevents errors for problematic inputs like 0 by returning NULL instead. This query shows cosecant values for multiple inputs:

```googlesql
SELECT CSC(100) AS a, CSC(-1) AS b, SAFE.CSC(0) AS c;
```
