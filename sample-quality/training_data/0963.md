--- USER ---

How do I construct an interval spanning from hours to seconds with mixed values?

--- AGENT ---

An INTERVAL with a datetime part range uses a string with components separated by colons, along with starting and ending datetime part keywords. This allows you to express intervals that span multiple time units from the string to the colon-separated components.

```googlesql
INTERVAL '10:20:30.52' HOUR TO SECOND
```
