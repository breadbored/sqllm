How do I access a struct field by its position number instead of by name?

--- AGENT ---

The struct subscript operator retrieves values by position within a struct using bracket notation with OFFSET or ORDINAL keywords. OFFSET uses zero-based indexing and ORDINAL uses one-based indexing.

```googlesql
SELECT
  STRUCT<INT64, STRING, BOOL>(23, "tea", FALSE)[0] AS field_index,
  STRUCT<INT64, STRING, BOOL>(23, "tea", FALSE)[OFFSET(0)] AS field_offset,
  STRUCT<INT64, STRING, BOOL>(23, "tea", FALSE)[ORDINAL(1)] AS field_ordinal
```
