What does GENERATE_ARRAY return when the direction conflicts with the step sign?

--- AGENT ---

Use `GENERATE_ARRAY` with a positive step when the start value exceeds the end value. Since the direction conflicts with the step direction, the function returns an empty array instead of attempting to generate values.

```googlesql
SELECT GENERATE_ARRAY(10, 0, 3) AS example_array;
```
