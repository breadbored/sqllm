--- USER ---

Show me a BigQuery query that extracts the earliest app and URL path for each device category within a sub-session, and creates a tag describing that device category.

--- AGENT ---

This query groups event records from the `base_buckets_events_app_speedy` table by sub_session_id and device_category. For each group, it uses `MIN_BY` to extract the app and url_path values that correspond to the earliest created timestamp, effectively capturing the first app and first URL path observed in each sub_session/device_category combination. It also constructs a tag by concatenating 'device_category_' with the device_category value, assigns a fixed class label of 'attribute' to each row, and includes the minimum (earliest) created timestamp for the group.

```sql
SELECT
  es.sub_session_id as sub_session_id,
  CONCAT('device_category_', es.device_category) as tag,
  'attribute' as class,
  MIN_BY(es.app, es.created) as app,
  MIN_BY(es.url_path, es.created) as url_path,
  MIN(es.created) as dt

FROM
  ${ref('base_buckets_events_app_speedy')} as es 

GROUP BY
    sub_session_id,
    tag
```
