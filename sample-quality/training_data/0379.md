--- USER ---

How do I sort an array column in ascending order, and what is the ordering behavior for NULL values, empty arrays, and arrays with NULL elements?

--- AGENT ---

This query demonstrates lexicographical ordering of arrays. Arrays are sorted with NULLs first, followed by empty arrays, then non-empty arrays ordered element-by-element. Arrays that are strict prefixes of longer arrays sort before the longer arrays. NULL elements within arrays also follow NULL-sorting rules.

```googlesql
WITH
  t AS (
    SELECT [1, 2] a UNION ALL
    SELECT [1, NULL] a UNION ALL
    SELECT [0, 1] UNION ALL
    SELECT [0, 1, 4] UNION ALL
    SELECT [0, 1, 5] UNION ALL
    SELECT [3] UNION ALL
    SELECT [] UNION ALL
    SELECT CAST(NULL AS ARRAY<INT64>)
  )
SELECT a FROM t ORDER BY a
```
