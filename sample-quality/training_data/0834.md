How do I extract only a specific field from multiple JSON objects in an array?

--- AGENT ---

The `JSON_EXTRACT_ARRAY` function with a specific JSONPath extracts only the array at that path from within a larger JSON object.

```googlesql
SELECT JSON_EXTRACT_ARRAY(
  '{"fruit": [{"apples": 5, "oranges": 10}, {"apples": 2, "oranges": 4}], "vegetables": [{"lettuce": 7, "kale": 8}]}',
  '$.fruit'
) AS string_array;
```
