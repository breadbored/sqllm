How do I query protocol buffer map fields by searching the underlying repeated structure?

--- AGENT ---

This query treats the protocol buffer map field as an underlying repeated field with `key` and `value` subfields, then uses an `EXISTS` subquery to check if any map entry matches a condition. This approach allows filtering based on map key-value pairs.

```googlesql
SELECT
  C.Orders.order_number
FROM
  Customers AS C
WHERE
  EXISTS(
    SELECT
      *
    FROM
      C.Orders.Labels AS label
    WHERE
      label.key = 'color' AND label.value = 'red'
  );
```
