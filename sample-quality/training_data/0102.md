How do I include NULL values when calculating approximate quantile boundaries?

--- AGENT ---

RESPECT NULLS includes NULL as a value in the approximate quantile calculation rather than filtering it out, so NULL can appear in the returned array of boundaries if it falls on a quantile boundary.

```googlesql
SELECT APPROX_QUANTILES(x, 2 RESPECT NULLS) AS approx_quantiles
FROM UNNEST([NULL, NULL, 1, 1, 1, 4, 5, 6, 7, 8, 9, 10]) AS x;
```
