How do I check if an email address matches a pattern?

--- AGENT ---

The REGEXP_CONTAINS function checks if a string partially matches a regular expression pattern, returning TRUE for valid email-like strings.

```googlesql
SELECT
  'foo@example.com' AS email,
  REGEXP_CONTAINS('foo@example.com', r'@[a-zA-Z0-9-]+\.[a-zA-Z0-9-.]+') AS is_valid
```
