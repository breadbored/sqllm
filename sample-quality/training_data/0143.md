How do I extract the first element only when using negative offsets?

--- AGENT ---

Use `ARRAY_SLICE` with a negative start offset clamped to the array beginning and a negative end offset that points before the start. The function returns only the first element.

```googlesql
SELECT ARRAY_SLICE(['a', 'b', 'c', 'd', 'e'], -30, -5) AS result
```
