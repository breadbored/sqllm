How do I filter window function results without using GROUP BY?

--- AGENT ---

The `QUALIFY` clause filters rows based on window function results, allowing you to select top-N rows per group or apply other ranking-based filters without aggregating.

```googlesql
SELECT
  item,
  RANK() OVER (PARTITION BY category ORDER BY purchases DESC) as rank
FROM Produce
WHERE Produce.category = 'vegetable'
QUALIFY rank <= 3
```
