How do I explicitly specify that an array should contain DOUBLE values even when the literals are integers?

--- AGENT ---

You can declare the array type before the bracket notation using angle brackets. This forces the elements to be cast to the specified type.

```googlesql
SELECT ARRAY<DOUBLE>[1, 2, 3] AS floats;
```
