How do I convert a JSON string array to a typed SQL integer array?

--- AGENT ---

Combine `JSON_EXTRACT_ARRAY` with `CAST` and `UNNEST` to extract and convert JSON array elements to a specific SQL type.

```googlesql
SELECT ARRAY(
  SELECT CAST(integer_element AS INT64)
  FROM UNNEST(
    JSON_EXTRACT_ARRAY('[1,2,3]','$')
  ) AS integer_element
) AS integer_array;
```
