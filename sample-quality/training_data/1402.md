How do I verify that already-normalized spaces remain unchanged?

--- AGENT ---

The NORMALIZE function with NFKC returns the same string when it contains regular spaces, confirming that normalization is idempotent.

```googlesql
SELECT NORMALIZE('Raha Mahan', NFKC) AS normalized_name
```
