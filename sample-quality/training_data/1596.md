How can I assign the results of multiple aggregates to multiple variables in a single SET statement?

--- AGENT ---

Multiple runtime variables can be assigned together from a SELECT AS STRUCT query, allowing you to populate several variables from a single query result. This example assigns a COUNT(DISTINCT ...) and a SUM(...) result to the corpus_count and word_count variables, using a runtime variable in the WHERE clause to filter the input data.

```googlesql
SET (corpus_count, word_count) = (
  SELECT AS STRUCT COUNT(DISTINCT corpus), SUM(word_count)
  FROM shakespeare
  WHERE LOWER(word) = target_word
);
```
