How do I format a time value to show only the hour and minute?

--- AGENT ---

The FORMAT_TIME function formats a TIME value using format elements. The `%R` element produces the time in hour:minute format without seconds.

```googlesql
SELECT FORMAT_TIME("%R", TIME "15:30:00") as formatted_time;
```
