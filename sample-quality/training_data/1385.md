How do I find the second overlapping occurrence of a pattern?

--- AGENT ---

The INSTR function allows you to locate subsequent overlapping occurrences by incrementing the occurrence parameter.

```googlesql
SELECT
  'helloooo' AS value, 'oo' AS subvalue, 1 AS position, 2 AS occurrence,
  INSTR('helloooo', 'oo', 1, 2) AS instr;
```
