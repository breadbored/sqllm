How do I use a CASE expression to compare a single column value against multiple options and return different strings for each match?

--- AGENT ---

A CASE expr expression evaluates the first argument against each WHEN value in sequence and returns the result for the first match; if no match is found, it returns the ELSE result. This is useful for simple lookups where you want to map a column's value to different outputs.

```googlesql
WITH Numbers AS (
  SELECT 90 as A, 2 as B UNION ALL
  SELECT 50, 8 UNION ALL
  SELECT 60, 6 UNION ALL
  SELECT 50, 10
)
SELECT
  A,
  B,
  CASE A
    WHEN 90 THEN 'red'
    WHEN 50 THEN 'blue'
    ELSE 'green'
    END
    AS result
FROM Numbers
```
