How do I format large numbers to include a thousands separator?

--- AGENT ---

The CAST function with the FORMAT clause converts a number to a formatted string. The comma (,) format element acts as a thousands separator, grouping digits in sets of three from right to left.

```googlesql
SELECT CAST(1234 AS STRING FORMAT '999,999') as a;
```
