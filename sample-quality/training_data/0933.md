How do I convert a large integer to a JSON string?

--- AGENT ---

Set `stringify_wide_numbers=>TRUE` in `TO_JSON` to encode large numeric values outside the `DOUBLE` domain as JSON strings.

```googlesql
SELECT TO_JSON(9007199254740993, stringify_wide_numbers=>TRUE) as stringify_on;
```
