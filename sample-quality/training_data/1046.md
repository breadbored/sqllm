How do I check if a subquery returns any rows?

--- AGENT ---

The EXISTS operator returns TRUE if a subquery produces one or more rows and FALSE if it produces zero rows. This is useful for conditional logic based on the presence or absence of matching data.

```googlesql
WITH Words AS (
  SELECT 'Intend' as value, 'east' as direction UNION ALL
  SELECT 'Secure', 'north' UNION ALL
  SELECT 'Clarity', 'west'
 )
SELECT EXISTS( SELECT value FROM Words WHERE direction = 'south' ) as result
```
