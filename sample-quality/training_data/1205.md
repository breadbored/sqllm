How do I find rows where a protocol buffer repeated field contains a specific value?

--- AGENT ---

This query uses an EXISTS subquery to scan inside a repeated field and return only rows where at least one element matches a condition. The subquery unnests the repeated field `line_item` and filters for the product name 'Foo'.

```googlesql
SELECT
  C.Id
FROM
  Customers AS C
WHERE
  EXISTS(
    SELECT
      *
    FROM
      C.Orders.line_item AS item
    WHERE
      item.product_name = 'Foo'
  );
```
