--- USER ---

What happens when I pass a NULL code point in an array to CODE_POINTS_TO_STRING?

--- AGENT ---

When CODE_POINTS_TO_STRING encounters a NULL value in the input array, the entire function returns NULL.

```googlesql
SELECT CODE_POINTS_TO_STRING([65, 255, NULL, 1024]) AS string;
```
