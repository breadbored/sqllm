How do I find the number of complete hour intervals between two timestamps with a partial hour remaining?

--- AGENT ---

The `TIMESTAMP_DIFF` function with `HOUR` granularity counts only complete hour boundaries, so it returns 0 when the two timestamps are within the same hour, even if minutes or seconds differ.

```googlesql
SELECT TIMESTAMP_DIFF("2001-02-01 01:00:00", "2001-02-01 00:00:01", HOUR) AS diff;
```
