How do I return different values based on whether a comparison expression evaluates to true or false?

--- AGENT ---

The IF function evaluates a boolean condition as its first argument and returns the second argument if TRUE, or the third argument if FALSE, without evaluating the branch that was not taken. This is useful for inline conditional logic that doesn't require a full CASE statement.

```googlesql
SELECT
  10 AS A,
  20 AS B,
  IF(10 < 20, 'true', 'false') AS result
```
