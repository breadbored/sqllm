How do I convert an array to a delimited string, replacing NULLs with a placeholder?

--- AGENT ---

Use `ARRAY_TO_STRING` with a delimiter and a `null_text` parameter. Elements are joined with the delimiter, and any NULL values in the array are replaced by the specified null_text string before concatenation.

```googlesql
SELECT ARRAY_TO_STRING(['coffee', 'tea', 'milk', NULL], '--', 'MISSING') AS text
```
