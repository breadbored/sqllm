How do I sort results and skip to the nth item in the sorted order?

--- AGENT ---

The `ORDER BY` statement combined with `OFFSET` skips N rows from the sorted result set. This example returns all transfers after skipping the first (largest) transfer when sorted by amount.

```googlesql
GRAPH FinGraph
MATCH (src_account:Account)-[transfer:Transfers]->(dst_account:Account)
ORDER BY transfer.amount DESC
OFFSET 1
RETURN src_account.id AS account_id, transfer.amount AS transfer_amount
```
