How do I call multiple table-valued functions in sequence using pipe syntax?

--- AGENT ---

The `CALL` pipe operator lets you chain multiple table-valued function calls. The input table is passed as the first table argument to each function, and you omit it from the argument list.

```googlesql
-- Call the same TVFs with the CALL operator.
FROM input_table
|> CALL tvf1(arg1)
|> CALL tvf2(arg2, arg3);
```
