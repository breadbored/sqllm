How do I check if a value matches patterns from an array?

--- AGENT ---

The quantified LIKE operator with UNNEST checks if a value matches any pattern in an array, treating the array elements as a set of patterns to search against.

```googlesql
SELECT * FROM Words WHERE value LIKE ANY UNNEST(['%ion%', '%and%'])
```
