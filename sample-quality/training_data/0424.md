How do I get the current date in a specific time zone like Los Angeles?

--- AGENT ---

`CURRENT_DATE()` accepts a time zone expression to return the current date in that zone. The function converts the current time to the specified time zone before extracting the date.

```googlesql
SELECT CURRENT_DATE('America/Los_Angeles') AS the_date;
```
