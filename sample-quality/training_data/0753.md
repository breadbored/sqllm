--- USER ---

How do I create a complete property graph with both nodes and edges, specifying all types and their references?

--- AGENT ---

The CREATE OR REPLACE PROPERTY GRAPH statement defines the complete structure of a property graph with node tables, edge tables, and all their relationships. Use OR REPLACE to update an existing graph or create it if it doesn't exist.

```googlesql
CREATE OR REPLACE PROPERTY GRAPH FinGraph
  NODE TABLES (
    Account,
    Person
  )
  EDGE TABLES (
    PersonOwnAccount
      SOURCE KEY (id) REFERENCES Person (id)
      DESTINATION KEY (account_id) REFERENCES Account (id)
      LABEL Owns,
    AccountTransferAccount
      SOURCE KEY (id) REFERENCES Account (id)
      DESTINATION KEY (to_id) REFERENCES Account (id)
      LABEL Transfers
  );
```
