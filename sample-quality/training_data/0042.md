How do I compute the average precipitation for just the earliest year in my dataset?

--- AGENT ---

`HAVING MIN` filters to rows with the minimum value of the specified column before aggregation. This query finds rows with the minimum `year` value, then averages the `inches` column for those rows only.

```googlesql
WITH
  Precipitation AS (
    SELECT 2001 AS year, 'spring' AS season, 9 AS inches
    UNION ALL
    SELECT 2001, 'winter', 1
    UNION ALL
    SELECT 2000, 'fall', 3
    UNION ALL
    SELECT 2000, 'summer', 5
    UNION ALL
    SELECT 2000, 'spring', 7
    UNION ALL
    SELECT 2000, 'winter', 2
  )
SELECT AVG(inches HAVING MIN year) AS average FROM Precipitation;
```
