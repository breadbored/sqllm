--- USER ---

Show me a daily client activity report that segments users into new and returning cohorts based on their first interaction date with each client.

--- AGENT ---

This query produces a daily cohort analysis grouped by client, identifying which users are active, new, and returning for each (day, client) combination.

The CTE `client_transactions_min` establishes each user's first interaction date with each client. It unnests the client_id array to create one row per user-client pair, then finds the minimum day value — this becomes the user's "new user date" for that client.

The main query then:
1. Unnests the client_id array again to get individual client IDs for each transaction row
2. Joins to the CTE to lookup each user's first-interaction day with each client
3. Joins to the client lookup table to get client names
4. Groups by (day, client_id, client_name)
5. Within each group, uses ARRAY_AGG to create three distinct-user arrays:
   - `active_users`: all users with transactions that day
   - `new_users`: users whose first_day equals the current day (flagged with CASE WHEN)
   - `returning_users`: users whose first_day is earlier than the current day (flagged with CASE WHEN)

The IGNORE NULLS clause in the CASE statements ensures only matching users appear in the new/returning arrays.

**Original SQL:**

```sql
WITH
  client_transactions_min AS (
    -- Get the first day each user participated with each client
  SELECT
    ct.user_id,
    cid AS client_id,
    MIN(ct.day) AS first_day
  FROM
    ${ref("agg_client_transactions")} AS ct,
    UNNEST(ct.client_id) AS cid
    -- Flatten client_id array
  GROUP BY
    ct.user_id,
    cid)
SELECT
  ct.day,
  cid AS client_id,
  sc.name AS client_name,
  ARRAY_AGG(DISTINCT ct.user_id) AS active_users,
  ARRAY_AGG(DISTINCT
    CASE
      WHEN ctm.first_day = ct.day THEN ct.user_id
  END
    IGNORE NULLS) AS new_users,
  ARRAY_AGG(DISTINCT
    CASE
      WHEN ctm.first_day < ct.day THEN ct.user_id
  END
    IGNORE NULLS) AS returning_users
FROM
  ${ref("agg_client_transactions")} AS ct
LEFT JOIN
  UNNEST(ct.client_id) AS cid
LEFT JOIN
  client_transactions_min AS ctm
ON
  ct.user_id = ctm.user_id
  AND cid = ctm.client_id
LEFT JOIN
  ${ref("stg_clients")} AS sc
ON
  cid = sc.id
GROUP BY
  ct.day,
  cid,
  sc.name
```
