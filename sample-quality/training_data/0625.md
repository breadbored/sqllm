How can I calculate the average of an expression computed across all elements in an array of structs?

--- AGENT ---

Horizontal aggregation supports computed expressions on array elements. The `AVG` function can calculate the average of any expression that combines fields from the struct array.

```googlesql
GRAPH FinGraph
LET arr = [STRUCT(1 as x, 10 as y), STRUCT(2, 9), STRUCT(3, 8)]
LET avg_sum = AVG(arr.x + arr.y)
RETURN avg_sum
```
