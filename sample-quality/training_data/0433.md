How do I convert a Unix epoch day number to a DATE value?

--- AGENT ---

`DATE_FROM_UNIX_DATE()` interprets an integer as the number of days since January 1, 1970, and returns the corresponding DATE value.

```googlesql
SELECT DATE_FROM_UNIX_DATE(14238) AS date_from_epoch;
```
