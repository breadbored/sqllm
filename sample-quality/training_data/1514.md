How can I convert a string like '15' into a TIME value representing 15:00:00?

--- AGENT ---

PARSE_TIME() converts a STRING to a TIME using a format string. Unspecified time fields default to 00, so the format %H with input '15' creates a TIME of 15:00:00.

```googlesql
SELECT PARSE_TIME("%H", "15") as parsed_time;
```
