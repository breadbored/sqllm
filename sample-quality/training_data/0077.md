How do I use COUNTIF within a window frame?

--- AGENT ---

`COUNTIF` with an `OVER` clause counts true conditions within a window frame. This example counts negative values in a sliding window ordered by absolute value:

```googlesql
SELECT
  x,
  COUNTIF(x<0) OVER (ORDER BY ABS(x) ROWS BETWEEN 1 PRECEDING AND 1 FOLLOWING) AS num_negative
FROM UNNEST([5, -2, 3, 6, -10, NULL, -7, 4, 0]) AS x;
```
