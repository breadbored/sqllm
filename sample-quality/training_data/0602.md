How do I combine two linestrings into a single union geography?

--- AGENT ---

`ST_UNION` with two linestring arguments returns a geometry representing the point set union of both linestrings combined.

```googlesql
SELECT ST_UNION(
  ST_GEOGFROMTEXT('LINESTRING(-122.12 47.67, -122.19 47.69)'),
  ST_GEOGFROMTEXT('LINESTRING(-122.12 47.67, -100.19 47.69)')
) AS results
```
