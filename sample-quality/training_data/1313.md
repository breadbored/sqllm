--- USER ---

What does the correlation function return when there is only one valid pair after removing NULLs?

--- AGENT ---

When `CORR` has fewer than two valid pairs (after filtering out NULL values), it returns NULL since correlation cannot be computed without at least two pairs.

```googlesql
SELECT CORR(y, x) AS results
FROM UNNEST([STRUCT(1.0 AS y, NULL AS x),(9.0, 3.0)])
```
