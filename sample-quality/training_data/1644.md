--- USER ---

Show me a query that calculates course completion progress for each user, including the total course duration, time spent, and percentage complete.

--- AGENT ---

This query builds a user course progress dataset with one row per (user_id, course_id) combination. It performs a multi-level aggregation:

**Lesson-level aggregation**: For each user-lesson pair in the course, it captures the maximum progress seconds (indicating furthest point reached) and sums the interval seconds spent on that lesson.

**Course-level aggregation**: It then aggregates lesson-level metrics up to the course level by summing progress and time metrics across all lessons the user has touched.

**Course duration calculation**: A separate subquery traverses the lesson-to-video mapping and sums video durations per course to compute total course length in minutes.

**Progress percentage**: Divides aggregated user progress by course duration and caps the result at 100% using LEAST.

**Client association**: Collects all distinct client IDs linked to the course via the container relationship using ARRAY_AGG.

**Filtering**: The WHERE clause ensures only enrollments with actual lesson progress are included in the output.

The query uses ANY_VALUE on deterministic columns (those that have the same value per grouping key) for efficiency, and MAX for the updated timestamp.

```sql
SELECT
  uc.user_id,
  uc.course_id,
  c.container_id,
  c.name AS course_name,
  ANY_VALUE(uc.created) AS course_started,
  MAX(uc.updated) AS course_updated,
  ANY_VALUE(uc.status) AS course_status,
  ANY_VALUE(ulp.user_total_lesson_progress_seconds) AS course_progress_seconds,
  ANY_VALUE(ulp.user_total_lesson_progress) AS course_progress_minutes,
  ANY_VALUE(cd.course_duration_minutes) AS course_length_minutes,
  LEAST( ROUND(((ANY_VALUE(ulp.user_total_lesson_progress) / ANY_VALUE(cd.course_duration_minutes))), 2), 1 ) AS course_progress_percentage,
  ANY_VALUE(ROUND((ulp.time_on_course / 60), 3)) AS time_on_course_minutes,
  ANY_VALUE(ulp.time_on_course) AS time_on_course,
  ARRAY_AGG(DISTINCT clients.id IGNORE NULLS
  ORDER BY
    clients.id) AS client_id
FROM
  ${ref("stg_user_courses")} AS uc
INNER JOIN
  ${ref("stg_courses")} AS c
ON
  uc.course_id = c.id
INNER JOIN
  ${ref("stg_users")} AS u
ON
  uc.user_id = u.id
LEFT JOIN (
  SELECT
    lesson_progress.user_id,
    lesson_progress.course_id,
    ROUND((SUM(lesson_progress.user_total_lesson_progress) / 60), 3) AS user_total_lesson_progress,
    (SUM(lesson_progress.user_total_lesson_progress)) AS user_total_lesson_progress_seconds,
    SUM(time_on_lesson) AS time_on_course,
  FROM (
    SELECT
      user_id,
      course_id,
      MAX(progress_seconds) AS user_total_lesson_progress,
      SUM(interval_seconds) AS time_on_lesson,
    FROM
      ${ref("stg_user_course_lesson_progress")} AS uclp
    WHERE
      user_id IS NOT NULL
    GROUP BY
      user_id,
      course_id,
      lesson_id ) AS lesson_progress
  GROUP BY
    lesson_progress.user_id,
    lesson_progress.course_id) AS ulp
ON
  uc.course_id = ulp.course_id
  AND uc.user_id = ulp.user_id
LEFT JOIN (
  SELECT
    cl.course_id,
    ROUND((SUM(v.duration) / 60), 3) AS course_duration_minutes
  FROM
    ${ref("stg_lessons")} AS l
  LEFT JOIN
    ${ref("stg_videos")} AS v
  ON
    l.video_id = v.id
  LEFT JOIN
    ${ref("stg_course_lessons")} AS cl
  ON
    l.id = cl.lesson_id
  GROUP BY
    course_id
  ORDER BY
    course_id ) AS cd
ON
  uc.course_id = cd.course_id
LEFT JOIN
  ${ref("stg_clients_course_containers")} AS ccc
ON
  c.container_id = ccc.course_container_id
LEFT JOIN
  ${ref("stg_clients")} AS clients
ON
  clients.id = ccc.client_id
WHERE
  ulp.user_id IS NOT NULL
GROUP BY
  uc.user_id,
  uc.course_id,
  c.container_id,
  c.name
```
