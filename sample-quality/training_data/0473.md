How do I get the last day of the month containing a given DATETIME value?

--- AGENT ---

LAST_DAY returns the last day of the specified period. When called with MONTH, it returns the last day of the month containing the given datetime.

```googlesql
SELECT LAST_DAY(DATETIME '2008-11-25', MONTH) AS last_day
```
