How do I get the JSON type of a value?

--- AGENT ---

The `JSON_TYPE` function returns the JSON type name (object, array, string, number, boolean, null) as a SQL string.

```googlesql
SELECT json_val, JSON_TYPE(json_val) AS type
FROM
  UNNEST(
    [
      JSON '"apple"',
      JSON '10',
      JSON '3.14',
      JSON 'null',
      JSON '{"city": "New York", "State": "NY"}',
      JSON '["apple", "banana"]',
      JSON 'false'
    ]
  ) AS json_val;
```
