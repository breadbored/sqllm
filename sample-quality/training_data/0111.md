How do I create an array of structs from a multi-column subquery?

--- AGENT ---

Use `SELECT AS STRUCT` within the `ARRAY` function to wrap multiple columns into a struct for each row. This allows `ARRAY` to accept subqueries with more than one column by packaging each row's values into a single struct element.

```googlesql
SELECT
  ARRAY
    (SELECT AS STRUCT 1, 2, 3
     UNION ALL SELECT AS STRUCT 4, 5, 6) AS new_array;
```
