How do I create a JSON object with array values using TO_JSON?

--- AGENT ---

Use `TO_JSON` to explicitly convert SQL arrays and values to JSON format before passing them to `JSON_OBJECT`.

```googlesql
SELECT
  JSON_OBJECT(
    ['a', 'b'],
    [TO_JSON(10), TO_JSON(['foo', 'bar'])])
    AS json_data
```
