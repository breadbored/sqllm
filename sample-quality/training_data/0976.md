How do I compute cotangent values, including a safe variant that handles division by zero?

--- AGENT ---

COT computes the cotangent of an angle in radians, while SAFE.COT returns NULL instead of raising an error when the input would cause a division by zero. This query evaluates COT(1) and safely handles COT(0):

```googlesql
SELECT COT(1) AS a, SAFE.COT(0) AS b;
```
