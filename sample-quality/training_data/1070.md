How do I combine a WITH expression with a WITH clause to organize complex queries?

--- AGENT ---

You can use a WITH clause to define common table expressions alongside a WITH expression in the SELECT statement, allowing you to combine intermediate table definitions with local variable assignments.

```googlesql
WITH my_table AS (
  SELECT 1 AS x, 2 AS y
  UNION ALL
  SELECT 3 AS x, 4 AS y
  UNION ALL
  SELECT 5 AS x, 6 AS y
)
SELECT WITH(a AS SUM(x), b AS COUNT(x), a/b) AS avg_x, AVG(y) AS avg_y
FROM my_table
WHERE x > 1
```
