How do I filter grouped results, similar to SQL's HAVING clause?

--- AGENT ---

The `FILTER` statement can be used after an aggregation step with `GROUP BY` to filter based on aggregate values, functioning like SQL's `HAVING` clause. This example finds source accounts with more than one outgoing transfer.

```googlesql
GRAPH FinGraph
MATCH (source:Account)-[e:Transfers]->(dest:Account)
RETURN source, COUNT(e) AS num_transfers
GROUP BY source

NEXT

FILTER WHERE num_transfers > 1
RETURN source.id AS source_id, num_transfers
```
