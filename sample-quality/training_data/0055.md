How do I aggregate distinct values into an array in a specific order?

--- AGENT ---

`ARRAY_AGG` with both `DISTINCT` and `ORDER BY` returns unique values sorted as specified. This example aggregates distinct integers sorted in ascending order:

```googlesql
WITH vals AS
  (
    SELECT 1 x UNION ALL
    SELECT -2 x UNION ALL
    SELECT 3 x UNION ALL
    SELECT -2 x UNION ALL
    SELECT 1 x
  )
SELECT ARRAY_AGG(DISTINCT x ORDER BY x) as array_agg
FROM vals;
```
