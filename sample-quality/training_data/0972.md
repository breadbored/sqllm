--- USER ---

How do I pass a named query parameter to substitute a value in a WHERE clause?

--- AGENT ---

A named query parameter is specified using the `@` character followed by an identifier. The parameter is replaced at execution time with a bound value, allowing you to write queries that accept dynamic input without string concatenation.

```googlesql
SELECT * FROM Roster WHERE LastName = @myparam
```
