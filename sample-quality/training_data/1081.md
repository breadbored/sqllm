How do I remove multiple columns from a table?

--- AGENT ---

The `DROP` pipe operator removes the specified columns from the input table, keeping all other columns. This is similar to using `SELECT * EXCEPT` in standard syntax.

```googlesql
SELECT 'apples' AS item, 2 AS sales, 'fruit' AS category
|> DROP sales, category;
```
