How do I match all unique simple paths between two nodes without repeated edges?

--- AGENT ---

The `MATCH ANY` modifier returns only the first matching path per result, avoiding duplicate paths and keeping results unique. This example finds unique destination accounts reachable within one or two transfers.

```googlesql
GRAPH FinGraph
MATCH ANY (src:Account {id: 7})-[e:Transfers]->{1,2}(dst:Account)
LET ids_in_path = ARRAY_CONCAT(ARRAY_AGG(e.Id), [dst.Id])
RETURN src.id AS source_account_id, dst.id AS destination_account_id, ids_in_path
```
