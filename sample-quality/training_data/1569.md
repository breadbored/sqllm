--- USER ---

How can I create a function that accepts arguments of different types without specifying a particular data type?

--- AGENT ---

This query demonstrates the ANY TYPE templated parameter, which allows a function to accept arguments of any data type at call time. The function body is validated only when the function is invoked, so the same function can work with integers, floats, or other compatible types.

```googlesql
CREATE TEMP FUNCTION AddFourAndDivideAny(x ANY TYPE, y ANY TYPE)
AS (
  (x + 4) / y
);

SELECT
  AddFourAndDivideAny(3, 4) AS integer_input,
  AddFourAndDivideAny(1.59, 3.14) AS floating_point_input;
```
