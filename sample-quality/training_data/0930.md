How do I extract a string value from a JSON object?

--- AGENT ---

Use `STRING` with `JSON_QUERY` to extract a specific string field from a JSON object.

```googlesql
SELECT STRING(JSON_QUERY(JSON '{"name": "sky", "color": "blue"}', "$.color")) AS color;
```
