How do I test whether an array value is NULL?

--- AGENT ---

Use the IS NULL operator to check if an array is NULL. Note that a NULL array is distinct from an empty array in queries, though both display as empty arrays when written to a table.

```googlesql
SELECT CAST(NULL AS ARRAY<INT64>) IS NULL AS array_is_null;
```
