How do I convert a UINT64 to INT64 while keeping the same bit pattern?

--- AGENT ---

BIT_CAST_TO_INT64 preserves the bit representation of the input value rather than reinterpreting it as a numeric value. When you cast the maximum UINT64 value back to INT64 using bit casting, you recover the original -1.

```googlesql
SELECT BIT_CAST_TO_UINT64(-1) as UINT64_value, BIT_CAST_TO_INT64(BIT_CAST_TO_UINT64(-1)) as bit_cast_value;
```
