--- USER ---

How do you query all edges that represent either Transfers or Owns relationships?

--- AGENT ---

The IS LABELED predicate on an edge can be used with a label expression containing multiple labels separated by the pipe operator (|) to match edges with any of the specified labels.

```googlesql
GRAPH FinGraph
MATCH (a)-[e]-(b:Account)
WHERE e IS LABELED Transfers | Owns
RETURN a.Id as a_id, Labels(e) AS labels, b.Id as b_id
ORDER BY a_id, b_id
```
