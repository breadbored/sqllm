How do I use COUNT as a window function to count rows in partitions?

--- AGENT ---

`COUNT` with an `OVER` clause counts rows within partitions defined by the window. This example partitions by modulo 3 and shows the total row count and distinct count per partition:

```googlesql
SELECT
  x,
  COUNT(*) OVER (PARTITION BY MOD(x, 3)) AS count_star,
  COUNT(DISTINCT x) OVER (PARTITION BY MOD(x, 3)) AS count_dist_x
FROM UNNEST([1, 4, 4, 5]) AS x;
```
