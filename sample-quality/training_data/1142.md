How do I write a CREATE TABLE statement using pipe syntax?

--- AGENT ---

In pipe syntax, the `CREATE TABLE` terminal operator creates a table from the pipe input. The data comes from the pipeline instead of an `AS query` clause, making the syntax more concise.

```googlesql
CREATE TEMP TABLE ShopSales AS
FROM Produce
|> AGGREGATE SUM(sales) AS total_sales GROUP BY category;
```
