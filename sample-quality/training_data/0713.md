--- USER ---

How do I use a variable defined and returned from one linear query statement in a subsequent statement?

--- AGENT ---

A variable defined in one linear query statement and included in its `RETURN` statement becomes available to the next linear query statement. This example defines `a` in the first statement and uses it in a `LET` statement in the second.

```googlesql
GRAPH FinGraph
MATCH (source:Account)-[e:Transfers]->(destination:Account)
LET a = source
RETURN a

NEXT

LET b = a -- Valid: 'a' is defined and returned from the linear query statement above.
RETURN b.id AS b_id
```
