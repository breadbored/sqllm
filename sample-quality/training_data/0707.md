How do I unnest an array expression and expand rows based on array elements?

--- AGENT ---

The `FOR` statement unnests an `ARRAY` expression and expands the working table with a new column for each element. The optional `WITH OFFSET` clause adds an offset column that indexes each array element.

```googlesql
GRAPH FinGraph
MATCH (p:Person)-[o:Owns]->(a:Account)
FOR element in ["all","some"] WITH OFFSET
RETURN p.Id, element as alert_type, offset
ORDER BY p.Id, element, offset
```
