--- USER ---

How do I execute a subquery that doesn't reference any variables from the outer scope?

--- AGENT ---

An empty variable scope list `CALL ()` in a subquery indicates the subquery is self-contained and cannot access any outer variables. The result is the same for all rows because the subquery executes independently on the graph.

```googlesql
GRAPH FinGraph
MATCH (p:Person)
CALL () {
  MATCH (n:Person)
  RETURN count(n) AS total_persons
}
RETURN p.name, total_persons
ORDER BY p.name;
```
