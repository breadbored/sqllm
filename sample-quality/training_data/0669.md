--- USER ---

How can I match paths of variable length, such as accounts that are 1 to 3 transfers away from a source account?

--- AGENT ---

Use a quantified path pattern with a bounded quantifier in curly braces: `{lower, upper}`. This applies the quantifier to an edge pattern to match paths with between lower and upper repetitions of that edge.

```googlesql
GRAPH FinGraph
MATCH (src:Account {id: 7})-[e:Transfers]->{1, 3}(dst:Account)
WHERE src != dst
RETURN ARRAY_LENGTH(e) AS hops, dst.id AS dst_account_id
```
