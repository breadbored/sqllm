How do I extract a numeric value from a JSON object and convert it to FLOAT?

--- AGENT ---

Use `FLOAT` with `JSON_QUERY` to extract a specific numeric field from a JSON object and convert it to a SQL FLOAT value.

```googlesql
SELECT FLOAT(JSON_QUERY(JSON '{"vo2_max": 39.1, "age": 18}', "$.vo2_max")) AS vo2_max;
```
