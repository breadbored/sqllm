How do I sort strings using their raw Unicode code point values?

--- AGENT ---

The 'binary' collation specification sorts strings in Unicode code point order without applying any language-specific rules or normalization. This query uses binary collation in the ORDER BY clause to sort the Place column by the numeric values of the Unicode code points that make up each string.

```googlesql
SELECT Place
FROM Locations
ORDER BY Place COLLATE 'binary'
```
