How do I construct a JSON literal from multiple concatenated string parts?

--- AGENT ---

A JSON literal can span multiple string tokens of different types, including raw strings. GoogleSQL concatenates all the parts to form a single JSON value, allowing you to mix literal JSON with raw strings to avoid excessive escaping.

```googlesql
SELECT JSON '{"name": "my_file.md", "regex": ' /*start*/ r' "\\d+"' /*end*/ '}'
```
