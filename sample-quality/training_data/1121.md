How do I find unique rows from the left query that do not appear in the right query?

--- AGENT ---

The `EXCEPT DISTINCT` pipe operator returns rows that appear in the input query but not in the specified query, removing duplicates from the result.

```googlesql
SELECT * FROM UNNEST(ARRAY<INT64>[1, 2, 3, 3, 4]) AS number
|> EXCEPT DISTINCT
    (SELECT * FROM UNNEST(ARRAY<INT64>[1, 2]) AS number);
```
