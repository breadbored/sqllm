How do I compute the frequency of each letter in a set of words?

--- AGENT ---

You can combine CODE_POINTS_TO_STRING with TO_CODE_POINTS and UNNEST to break words into individual characters, then aggregate to count letter frequencies.

```googlesql
WITH Words AS (
  SELECT word
  FROM UNNEST(['foo', 'bar', 'baz', 'giraffe', 'llama']) AS word
)
SELECT
  CODE_POINTS_TO_STRING([code_point]) AS letter,
  COUNT(*) AS letter_count
FROM Words,
  UNNEST(TO_CODE_POINTS(word)) AS code_point
GROUP BY 1
ORDER BY 2 DESC;
```
