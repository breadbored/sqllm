--- USER ---

Does FROM_PROTO work on native GoogleSQL types, or only on protocol buffer types?

--- AGENT ---

FROM_PROTO accepts both protocol buffer types and their GoogleSQL equivalents as input. When passed a native DATE type, it returns the same DATE value unchanged.

```googlesql
SELECT FROM_PROTO(DATE '2019-10-30')
```
