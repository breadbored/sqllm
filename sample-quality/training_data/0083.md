How do I use MAX as a window function to find the maximum in each partition?

--- AGENT ---

`MAX` with an `OVER` clause returns the maximum value within each window partition. This example partitions by modulo 2 and shows the maximum value in each partition:

```googlesql
SELECT x, MAX(x) OVER (PARTITION BY MOD(x, 2)) AS max
FROM UNNEST([8, NULL, 37, 55, NULL, 4]) AS x;
```
