How do I extract and unquote string values from a JSON array?

--- AGENT ---

Combine `JSON_QUERY_ARRAY` with `JSON_VALUE` and `UNNEST` to extract strings and remove their quotes.

```googlesql
SELECT ARRAY(
  SELECT JSON_VALUE(string_element, '$')
  FROM UNNEST(JSON_QUERY_ARRAY('["apples", "oranges", "grapes"]', '$')) AS string_element
) AS string_array;
```
