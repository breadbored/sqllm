How do I count distinct IDs that exist in both tables using a subquery?

--- AGENT ---

Use `COUNT(DISTINCT IF(...))` with a subquery to count distinct values from one table that also appear in another table. This example counts IDs from the vendors table that also exist in the customers table:

```googlesql
WITH
  customers AS (
    SELECT 1934 AS id, 'a' AS team UNION ALL
    SELECT 2991, 'b' UNION ALL
    SELECT 3988, 'c'),
  vendors AS (
    SELECT 1934 AS id, 'd' AS team UNION ALL
    SELECT 2991, 'e' UNION ALL
    SELECT 4366, 'f')
SELECT
  COUNT(DISTINCT IF(id IN (SELECT id FROM customers), id, NULL)) AS result
FROM vendors;
```
