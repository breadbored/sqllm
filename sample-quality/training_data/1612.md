--- USER ---

How do I define a named window that includes partitioning, ordering, and a frame clause all in one place?

--- AGENT ---

This query defines a named window `ItemWindow` that includes the PARTITION BY, ORDER BY, and window frame clause together. When referenced in the OVER clause, all these specifications are applied simultaneously.

```googlesql
SELECT item, purchases, category, LAST_VALUE(item)
  OVER (ItemWindow) AS most_popular
FROM Produce
WINDOW ItemWindow AS (
  PARTITION BY category
  ORDER BY purchases
  ROWS BETWEEN 2 PRECEDING AND 2 FOLLOWING)
```
