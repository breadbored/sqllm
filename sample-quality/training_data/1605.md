--- USER ---

How can I sum only the two rows preceding each row, excluding the current row itself?

--- AGENT ---

This query uses a window frame that spans from the beginning of the table to two rows before the current row. When a boundary extends beyond the partition, it includes only rows within that partition. The first two rows have NULL because there are no preceding rows.

```googlesql
SELECT item, purchases, category, SUM(purchases)
  OVER (
    ORDER BY purchases
    ROWS BETWEEN UNBOUNDED PRECEDING AND 2 PRECEDING
  ) AS total_purchases
FROM Produce;
```
