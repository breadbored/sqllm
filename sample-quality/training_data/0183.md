How do I access individual elements from an array using different indexing methods?

--- AGENT ---

You can access array elements using zero-based indexing with bracket notation, `OFFSET()` for zero-based indexing, or `ORDINAL()` for one-based indexing. Each method retrieves the element at the specified position, with `ORDINAL()` being useful when you think in one-based index terms.

```googlesql
SELECT
  some_numbers,
  some_numbers[0] AS index_0,
  some_numbers[OFFSET(1)] AS offset_1,
  some_numbers[ORDINAL(1)] AS ordinal_1
FROM Sequences

/*--------------------+---------+----------+-----------+
 | some_numbers       | index_0 | offset_1 | ordinal_1 |
 +--------------------+---------+----------+-----------+
 | [0, 1, 1, 2, 3, 5] | 0       | 1        | 0         |
 | [2, 4, 8, 16, 32]  | 2       | 4        | 2         |
 | [5, 10]            | 5       | 10       | 5         |
 +--------------------+---------+----------+-----------*/
```
