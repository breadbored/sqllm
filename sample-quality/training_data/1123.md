How do I control the order of set operations in EXCEPT using parentheses?

--- AGENT ---

Parentheses in `EXCEPT` operations control which query is subtracted from which. Without parentheses, operations are left-associative; with parentheses, you can group operations differently to change the result.

```googlesql
SELECT * FROM UNNEST(ARRAY<INT64>[1, 2, 3, 3, 4]) AS number
|> EXCEPT DISTINCT
(
  SELECT * FROM UNNEST(ARRAY<INT64>[1, 2]) AS number
  |> EXCEPT DISTINCT
      (SELECT * FROM UNNEST(ARRAY<INT64>[1, 4]) AS number)
);
```
