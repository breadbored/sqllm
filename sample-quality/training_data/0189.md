How can I flatten nested array data into individual rows using UNNEST?

--- AGENT ---

You can flatten nested arrays into rows using three equivalent syntaxes: explicit `FLATTEN()` within `UNNEST()`, implicit `FLATTEN()` by passing the array directly to `UNNEST()`, or direct field access in the `FROM` clause with dot notation. All three approaches produce identical results, with the `FROM` clause method being the most concise.

```googlesql
-- In UNNEST (FLATTEN used explicitly):
SELECT colors
FROM ItemsTable
INNER JOIN UNNEST(FLATTEN(items.color)) AS colors;

-- In UNNEST (FLATTEN used implicitly):
SELECT colors
FROM ItemsTable
INNER JOIN UNNEST(items.color) AS colors;

-- In the FROM clause (UNNEST used implicitly):
SELECT colors
FROM ItemsTable
INNER JOIN ItemsTable.items.color AS colors;

/*--------+
 | colors |
 +--------+
 | red    |
 | green  |
 | orange |
 +--------*/
```
