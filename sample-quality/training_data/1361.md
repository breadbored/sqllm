How do I compare string character weights using the default collation?

--- AGENT ---

Without explicitly applying a collation, string comparisons use the default collation specification which assigns different weights to lowercase and uppercase letters.

```googlesql
WITH Words AS (
  SELECT
    'a' AS char1,
    'Z' AS char2
)
SELECT ( Words.char1 < Words.char2 ) AS a_less_than_Z
FROM Words;
```
