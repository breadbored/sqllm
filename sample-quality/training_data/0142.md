How do I extract all elements from an array using out-of-bounds offsets?

--- AGENT ---

Use `ARRAY_SLICE` with a very negative start offset and a very positive end offset. Both bounds are clamped to valid array boundaries, returning all elements from start to finish.

```googlesql
SELECT ARRAY_SLICE(['a', 'b', 'c', 'd', 'e'], -30, 30) AS result
```
