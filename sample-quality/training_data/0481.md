--- USER ---

I need to verify that my data source contains more than 5 rows as a quality check.

--- AGENT ---

The ASSERT statement evaluates the condition COUNT(*) > 5 and succeeds silently if true. If the condition is false or NULL, it generates an error with the custom description provided in the AS clause. This example demonstrates a successful assertion—the dataset has 6 rows, so the condition is satisfied and no error is produced.

```googlesql
ASSERT (
  (SELECT COUNT(*) > 5 FROM UNNEST([1, 2, 3, 4, 5, 6]))
) AS 'Table must contain more than 5 rows.';
```
