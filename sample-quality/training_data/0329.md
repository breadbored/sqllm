How do I add a structured column with nested fields to an existing table?

--- AGENT ---

The `ALTER TABLE ... ADD COLUMN` statement supports adding a column with a `STRUCT` type. The struct definition specifies nested fields with their own types and optional constraints.

```googlesql
ALTER TABLE mydataset.mytable
ADD COLUMN A STRUCT<
               B GEOGRAPHY,
               C ARRAY<INT64>,
               D INT64 NOT NULL,
               E TIMESTAMP OPTIONS(description="creation time")
             >
```
