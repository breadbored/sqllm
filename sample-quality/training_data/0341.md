--- USER ---

Insert inventory items and return all columns plus a computed column based on the inserted values.

--- AGENT ---

This query inserts three rows into the Inventory table and uses THEN RETURN to return all columns along with a computed column that multiplies quantity by 10 for each inserted row.

```googlesql
INSERT Inventory (product, quantity)
VALUES ('washer', 20),
       ('dryer', 30),
       ('oven', 5)
THEN RETURN *, quantity * 10 AS new_quantity;
```
