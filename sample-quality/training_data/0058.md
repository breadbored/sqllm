How do I concatenate multiple arrays into a single array?

--- AGENT ---

`ARRAY_CONCAT_AGG` merges multiple arrays into one flat result array, preserving NULL elements within arrays. This example concatenates four separate arrays into a single array:

```googlesql
SELECT ARRAY_CONCAT_AGG(x) AS array_concat_agg FROM (
  SELECT [NULL, 1, 2, 3, 4] AS x
  UNION ALL SELECT NULL
  UNION ALL SELECT [5, 6]
  UNION ALL SELECT [7, 8, 9]
);
```
