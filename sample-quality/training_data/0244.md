How do I replace a NULL value with a default value in a single function call?

--- AGENT ---

IFNULL evaluates its first argument and returns it if it is not NULL; if it is NULL, it returns the second argument instead. This is a shorthand for the common pattern of checking whether a value is NULL and providing a fallback.

```googlesql
SELECT IFNULL(NULL, 0) as result
```
