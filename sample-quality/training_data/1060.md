How do I check if a value matches any pattern from a list of patterns?

--- AGENT ---

The quantified LIKE operator with ANY checks if a value matches at least one pattern in a set of patterns, returning TRUE if any pattern matches, FALSE if none match, or NULL if the value is NULL.

```googlesql
SELECT * FROM Words WHERE value LIKE ANY ('Intend%', '%intention%')
```
