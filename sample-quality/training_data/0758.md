How do I query a property graph using GRAPH_TABLE and select specific properties from the matched nodes?

--- AGENT ---

The GRAPH_TABLE operator allows you to match nodes in a property graph using GQL syntax and return specific properties in the result set. By aliasing properties in the RETURN statement with AS, you can control which columns appear in the output table that can then be selected in the outer SELECT clause.

```googlesql
SELECT name, id
FROM GRAPH_TABLE(
  FinGraph
  MATCH (n:Person)
  RETURN n.name AS name, n.id AS id
);
```
