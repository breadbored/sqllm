What happens to the correlation calculation when one value in a pair is NULL?

--- AGENT ---

The `CORR` function skips any pair containing a NULL value during aggregation. In this example, the pair (4.0, NULL) is ignored, leaving only two valid pairs to compute correlation.

```googlesql
SELECT CORR(y, x) AS results
FROM
  UNNEST(
    [
      STRUCT(1.0 AS y, 5.0 AS x),
      (3.0, 9.0),
      (4.0, NULL)]);
```
