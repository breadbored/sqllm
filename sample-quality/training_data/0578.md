How do I parse a hexadecimal WKB string directly without using the FROM_HEX function?

--- AGENT ---

`ST_GEOGFROM` accepts hexadecimal-encoded WKB strings directly as input, without requiring a separate `FROM_HEX` call.

```googlesql
SELECT ST_GEOGFROM('010100000000000000000000400000000000001040') AS WKB_format;
```
