How do I insert a value into a JSON null to create an array?

--- AGENT ---

The `JSON_ARRAY_INSERT` function replaces a JSON null with a JSON array of the appropriate size, padded with nulls, and inserts the value at the specified index.

```googlesql
SELECT JSON_ARRAY_INSERT(JSON '{"a": null}', '$.a[2]', 10) AS json_data
```
