How do I skip the remaining statements in a loop and continue to the next iteration?

--- AGENT ---

Use `CONTINUE` (or its synonym `ITERATE`) to skip remaining statements in the current iteration and jump back to the start of the loop.

```googlesql
DECLARE heads BOOL;
DECLARE heads_count INT64 DEFAULT 0;
LOOP
  SET heads = RAND() < 0.5;
  IF heads THEN
    SELECT 'Heads!';
    SET heads_count = heads_count + 1;
    CONTINUE;
  END IF;
  SELECT 'Tails!';
  BREAK;
END LOOP;
SELECT CONCAT(CAST(heads_count AS STRING), ' heads in a row');
```
