--- USER ---

Show me a BigQuery query that tags app events with the client associated with a user's course progress on the same day, capturing the earliest event details per sub-session.

--- AGENT ---

This query correlates app events with course progress data to assign client ownership at the sub-session level.

**What it does:**

1. **Joins events to progress**: Connects app events from `base_buckets_events_app_speedy` to user progress records in `user_course_lesson_progress` using three conditions: same user, same date, and course membership (with three matching strategies — direct course_id, derived_course_id, or course_id extracted from the URL path).

2. **Traces to client**: Follows the course through its container to identify which client owns that course, creating a tag as `client_<name>`.

3. **Captures first event**: Within each sub-session and client pair, extracts the earliest event's app and URL path (using `MIN_BY` on the created timestamp) plus the session's earliest timestamp.

4. **Groups by session and client**: Produces one row per unique sub_session + client combination, with the earliest event's metadata and a fixed `content` class marker.

**The SQL:**

```sql
SELECT
    es.sub_session_id,
    CONCAT('client_', cl.name) as tag,
    'content' as class,
    MIN_BY(es.app, es.created) as app,
    MIN_BY(es.url_path, es.created) as url_path,
    MIN(es.created) as dt
FROM
    ${ref("base_buckets_events_app_speedy")} as es

JOIN
  ${ref('user_course_lesson_progress')} AS uclp
ON
  uclp.user_id = es.user_id
AND
  DATE(uclp.created) = DATE(es.created)
AND
    (
        uclp.course_id = es.course_id
    OR
        uclp.course_id = es.derived_course_id
    OR
        SAFE_CAST(REGEXP_EXTRACT(es.url_path, r's/(\d+)$', 1, 1) AS INT64) = uclp.course_id
    )

JOIN
    ${ref("courses")} as c
ON
    uclp.course_id = c.id

JOIN
    ${ref("clients_course_containers")} as ccc
ON
    ccc.course_container_id = c.container_id

JOIN
    ${ref("clients")} as cl
ON
    cl.id = ccc.client_id

GROUP BY
    sub_session_id,
    tag
```
