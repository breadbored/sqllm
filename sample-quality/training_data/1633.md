Show me a BigQuery query that extracts cookie banner events along with user IDs and event metadata.

--- AGENT ---

This query filters the events table to extract only cookie banner events. It selects four columns for each matching event: the user_id (identifying which user triggered the event), type and action (event metadata describing the nature of the interaction), and created (the event timestamp). The WHERE clause restricts results to rows where the name column equals "cookie_banner", effectively extracting a subset of all events.

```sql
SELECT
  user_id,
  type,
  action,
  created
FROM ${ref("stg_events_speedy")}
WHERE name = "cookie_banner"
```
