## Question
Show me how to compute which clients are associated with each sentiment question through a course hierarchy, then enrich the sentiment question options table with that information.

--- AGENT ---

This query enriches a sentiment question options table by computing which clients are associated with each sentiment question through a multi-level relationship chain.

The CTE `stg_sentiment_question_options` starts from the clients_course_containers table and joins through courses, course_lessons, lessons, learning_cues, and sentiment_questions. For each sentiment question ID, it uses ARRAY_AGG(DISTINCT ...) to collect all client IDs that can be reached through these relationships, producing one row per sentiment question with an array of associated client IDs.

The final SELECT enriches the source sentiment_question_options table by left-joining with this CTE on sentiment_question_id. It uses COALESCE to handle cases where a sentiment question has no associated clients: if the left join finds no match, the client_ids defaults to [1].

The result is a table containing all columns from sentiment_question_options plus a new client_ids column—either the computed array of associated clients or the default value [1].

```sql
WITH
stg_sentiment_question_options AS (
    SELECT
        sq.id as sentiment_question_id,
        ARRAY_AGG(DISTINCT ccc.client_id IGNORE NULLS) as client_ids
    FROM
        ${ref("clients_course_containers")} as ccc
    JOIN
        ${ref("courses")} as c
    ON
        c.container_id = ccc.course_container_id
    JOIN
        ${ref("course_lessons")} as cl
    ON
        cl.course_id = c.id
    JOIN
        ${ref("lessons")} as l
    ON 
        l.id = cl.lesson_id
    JOIN
        ${ref("learning_cues")} as lc
    ON
        lc.lesson_id = l.id
    JOIN
        ${ref("sentiment_questions")} as sq
    ON
        sq.learning_cue_id = lc.id
    GROUP BY
        sq.id
)
SELECT
    sqo.*,
    COALESCE(ssqo.client_ids, [1]) AS client_ids
FROM
    ${ref("sentiment_question_options")} as sqo
LEFT JOIN
    stg_sentiment_question_options as ssqo
ON
    ssqo.sentiment_question_id = sqo.sentiment_question_id
```
