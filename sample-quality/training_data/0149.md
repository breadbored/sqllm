How do I join array elements with a custom delimiter and handle missing values?

--- AGENT ---

Use `ARRAY_TO_STRING` with both a delimiter string and a replacement text for NULLs. This allows you to create a readable string representation with explicit handling of empty or missing array values.

```googlesql
SELECT ARRAY_TO_STRING(['cake', 'pie', NULL], '--', 'MISSING') AS text
```
