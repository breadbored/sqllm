How do I define reusable window specifications by name?

--- AGENT ---

The `WINDOW` clause defines named window specifications that can be referenced in multiple window functions, reducing repetition and improving readability.

```googlesql
SELECT item, purchases, category, LAST_VALUE(item)
  OVER (item_window) AS most_popular
FROM Produce
WINDOW item_window AS (
  PARTITION BY category
  ORDER BY purchases
  ROWS BETWEEN 2 PRECEDING AND 2 FOLLOWING)
```
