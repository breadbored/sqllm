How do I check if two date ranges overlap at any point?

--- AGENT ---

RANGE_OVERLAPS returns TRUE if any part of one range intersects with another range. This query checks whether [2022-02-01, 2022-09-01) and [2021-06-15, 2022-04-15) overlap, which they do between 2022-02-01 and 2022-04-15.

```googlesql
SELECT RANGE_OVERLAPS(
  RANGE<DATE> '[2022-02-01, 2022-09-01)',
  RANGE<DATE> '[2021-06-15, 2022-04-15)') AS results;
```
