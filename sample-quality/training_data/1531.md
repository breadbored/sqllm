How do I format a timestamp into a human-readable date string with the locale style?

--- AGENT ---

The `FORMAT_TIMESTAMP` function converts a timestamp into a formatted string using format elements; the `%c` element produces a locale-appropriate date and time representation. You can optionally specify a time zone to display the timestamp in that zone.

```googlesql
SELECT FORMAT_TIMESTAMP("%c", TIMESTAMP "2050-12-25 15:30:55+00", "UTC")
  AS formatted;
```
