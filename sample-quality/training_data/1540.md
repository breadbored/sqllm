How do I convert a TIMESTAMP value to a string representation with a specific time zone?

--- AGENT ---

The `STRING` function converts a TIMESTAMP to its string representation and supports an optional time zone parameter to display the timestamp in that zone. Without a time zone, it uses the default time zone.

```googlesql
SELECT STRING(TIMESTAMP "2008-12-25 15:30:00+00", "UTC") AS string;
```
