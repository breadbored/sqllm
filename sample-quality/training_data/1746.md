--- USER ---

Extract the first-occurring event's URL, landing page, and UTM parameters within each session-segment combination, excluding internal referrer sources.

--- AGENT ---

This query groups event records by `session_id` and `segment_id`, then extracts columns from the event with the minimum `event_num` within each group (the first event). It uses the `MIN_BY` function to capture the values of several columns (utm_visit, url_path, landing_path, referrer, utm_campaign, utm_content, utm_medium, utm_source, utm_term, and several "last_utm_*" and "last_referrer" columns) at the event with the lowest event number.

The query applies conditional logic to the utm and referrer columns: if the referrer value does not contain the string 'dataprotocol.com', those utm and referrer fields are included in the output; otherwise they are set to NULL. 

The WHERE clause filters the input to only events where `url_path` is not null and the `referrer` does not match any of four internal domain patterns (dataprotocol.com, dataprotocol.dev, data-protocol.vercel.app, or localhost). The output contains one row per session-segment combination.

```sql
/**
* Start URLs
*
* This layer is used to get the first URL, referrer, and UTMs in a subsession.
**/
SELECT
  ne.session_id,
  MIN(ne.event_num) AS event_num,
  ne.segment_id,
  MIN_BY(ne.utm_visit, ne.event_num) AS utm_visit,
  MIN_BY(ne.url_path, ne.event_num) AS start_url_path,
  MIN_BY(ne.landing_path, ne.event_num) AS start_landing_url_path,
  CASE WHEN MIN_BY(ne.referrer, ne.event_num) NOT LIKE '%dataprotocol.com%' THEN MIN_BY(ne.utm_campaign, ne.event_num) ELSE NULL END AS start_utm_campaign,
  CASE WHEN MIN_BY(ne.referrer, ne.event_num) NOT LIKE '%dataprotocol.com%'THEN MIN_BY(ne.utm_content, ne.event_num) ELSE NULL END AS start_utm_content,
  CASE WHEN MIN_BY(ne.referrer, ne.event_num) NOT LIKE '%dataprotocol.com%'THEN MIN_BY(ne.utm_medium, ne.event_num) ELSE NULL END AS start_utm_medium,
  CASE WHEN MIN_BY(ne.referrer, ne.event_num) NOT LIKE '%dataprotocol.com%'THEN MIN_BY(ne.utm_source, ne.event_num) ELSE NULL END AS start_utm_source,
  CASE WHEN MIN_BY(ne.referrer, ne.event_num) NOT LIKE '%dataprotocol.com%'THEN MIN_BY(ne.utm_term, ne.event_num) ELSE NULL END AS start_utm_term,
  CASE WHEN MIN_BY(ne.referrer, ne.event_num) NOT LIKE '%dataprotocol.com%'THEN MIN_BY(ne.referrer, ne.event_num) ELSE NULL END AS start_referrer,
  MIN_BY(ne.last_utm_campaign, ne.event_num) AS last_utm_campaign,
  MIN_BY(ne.last_utm_content, ne.event_num) AS last_utm_content,
  MIN_BY(ne.last_utm_medium, ne.event_num) AS last_utm_medium,
  MIN_BY(ne.last_utm_source, ne.event_num) AS last_utm_source,
  MIN_BY(ne.last_utm_term, ne.event_num) AS last_utm_term,
  MIN_BY(ne.last_referrer, ne.event_num) AS last_referrer
FROM
  ${ref("events_numbered")} AS ne
WHERE
  ne.url_path IS NOT NULL 
  AND ne.referrer NOT LIKE '%dataprotocol.com%' 
  AND ne.referrer NOT LIKE '%dataprotocol.dev%'
  AND ne.referrer NOT LIKE '%data-protocol.vercel.app%'
  AND ne.referrer NOT LIKE '%localhost%'
GROUP BY
  ne.session_id,
  ne.segment_id
```
