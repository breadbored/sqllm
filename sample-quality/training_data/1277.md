How do I reference all columns of a table alias as a single STRUCT value?

--- AGENT ---

A range variable (table alias) can be referenced directly in the SELECT list to return a STRUCT containing all columns from that table. This is useful for preserving row structure in value tables.

```googlesql
WITH Grid AS (SELECT 1 x, 2 y)
SELECT Coordinate FROM Grid AS Coordinate;
```
