What does GENERATE_DATE_ARRAY return when the direction conflicts with the interval sign?

--- AGENT ---

Use `GENERATE_DATE_ARRAY` with a positive interval when the start date is earlier than the end date but the interval direction conflicts. Since the direction is inconsistent with the interval, the function returns an empty array.

```googlesql
SELECT GENERATE_DATE_ARRAY('2016-10-05',
  '2016-10-01', INTERVAL 1 DAY) AS example;
```
