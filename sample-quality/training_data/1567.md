--- USER ---

How do I create a custom aggregate function that computes a sum and then divides it by a constant divisor that is the same for all rows in the group?

--- AGENT ---

This example creates a user-defined aggregate function that accepts an aggregate parameter (dividend) and a non-aggregate parameter (divisor marked with `NOT AGGREGATE`). The function sums the dividend values across the group and divides the result by the constant divisor. The SELECT query demonstrates calling this UDA, summing (1 + 3 + 5) and dividing by 2 to produce 4.5.

```googlesql
CREATE TEMP AGGREGATE FUNCTION ScaledSum(
  dividend DOUBLE,
  divisor DOUBLE NOT AGGREGATE)
RETURNS DOUBLE
AS (
  SUM(dividend) / divisor
);

SELECT ScaledSum(col1, 2) AS scaled_sum
FROM (
  SELECT 1 AS col1 UNION ALL
  SELECT 3 AS col1 UNION ALL
  SELECT 5 AS col1
);
```
