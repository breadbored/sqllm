How do I remove duplicate elements from an array?

--- AGENT ---

Use `ARRAY(SELECT DISTINCT element FROM UNNEST(array) AS element)` to remove duplicates from an array. The `SELECT DISTINCT` clause filters out duplicate values, and `ARRAY()` reconstructs the unique elements as an array.

```googlesql
WITH Sequences AS
  (SELECT [0, 1, 1, 2, 3, 5] AS some_numbers)
SELECT ARRAY(SELECT DISTINCT x
             FROM UNNEST(some_numbers) AS x) AS unique_numbers
FROM Sequences;

/*-----------------+
 | unique_numbers  |
 +-----------------+
 | [0, 1, 2, 3, 5] |
 +-----------------*/
```
