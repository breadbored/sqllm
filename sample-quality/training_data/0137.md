How do I extract the last three elements of an array using negative indices?

--- AGENT ---

Use `ARRAY_SLICE` with negative indices that target the end of the array. A start offset of -3 selects the third-from-last element, and an end offset of -1 selects the last element, returning all three together.

```googlesql
SELECT ARRAY_SLICE(['a', 'b', 'c', 'd', 'e'], -3, -1) AS result
```
