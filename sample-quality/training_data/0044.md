How do I compute a running average that includes the current row and one row before it, ordered by a column?

--- AGENT ---

The `OVER` clause turns an aggregate function into a window function, computing the aggregation over a specified frame of rows rather than all rows. This query calculates `AVG` for each row using a window containing that row and the one preceding it, ordered by `x`.

```googlesql
SELECT
  x,
  AVG(x) OVER (ORDER BY x ROWS BETWEEN 1 PRECEDING AND CURRENT ROW) AS avg
FROM UNNEST([0, 2, 4, 4, 5]) AS x;
```
