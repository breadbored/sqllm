Does element order within vectors affect cosine distance calculations?

--- AGENT ---

COSINE_DISTANCE produces the same result regardless of element order within each vector, since corresponding dimensions are determined by position. This query demonstrates that reordering elements produces the same distance as vectors with different element order:

```googlesql
SELECT COSINE_DISTANCE([2.0, 1.0], [4.0, 3.0]) AS results;
```
