How do I convert integer values to binary IPv4 addresses?

--- AGENT ---

This query converts IPv4 addresses stored as INT64 integers to their binary BYTES representation in network byte order using NET.IPV4_FROM_INT64, where the least significant bit of the integer maps to the least significant bit of the IP address. The query demonstrates positive, hexadecimal, and negative integer inputs.

```googlesql
SELECT x, x_hex, FORMAT("%T", NET.IPV4_FROM_INT64(x)) AS ipv4_from_int64
FROM (
  SELECT CAST(x_hex AS INT64) x, x_hex
  FROM UNNEST(["0x0", "0xABCDEF", "0xFFFFFFFF", "-0x1", "-0x2"]) AS x_hex
);
```
