This query creates a table-valued function that accepts an integer `clientId` parameter and returns a single aggregated metric: the average of a `course_progress_percentage` column. It reads from a `reporting_user_courses` table and filters records using the provided `clientId` parameter through a WHERE clause that matches the parameter value against a column reference in a subquery.

```sql
CREATE OR REPLACE TABLE
  FUNCTION ${self()} (clientId INT64) AS (
  SELECT
    AVG(course_progress_percentage) AS avg_course_progress
  FROM
    ${ref("reporting_user_courses")} uc
  WHERE
    clientId IN (
    SELECT
      *
    FROM
      uc.client_id) )
```
