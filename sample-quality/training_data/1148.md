How do I create a table in a TEE subpipeline while continuing with the main pipeline?

--- AGENT ---

The `TEE` semi-terminal operator can run terminal operators like `CREATE TABLE` in a subpipeline while the main pipeline continues and returns results. Side effects are applied atomically.

```googlesql
WITH
  Fruit AS (
    SELECT 'apple' AS item, 100 AS sales
    UNION ALL
    SELECT 'banana' AS item, 150 AS sales
  )
FROM Fruit
|> TEE (
     |> AGGREGATE COUNT(*) AS count_by_item GROUP BY item
     |> CREATE TABLE FruitCounts
   )
|> SELECT *;
```
