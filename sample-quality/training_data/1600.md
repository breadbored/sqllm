--- USER ---

How do I create a sample dataset of farm animals with their population counts and categories?

--- AGENT ---

This creates a temporary `Farm` table containing animal records with their population count and category (mammal or bird). Each UNION ALL statement adds another animal to the dataset.

```googlesql
WITH Farm AS
 (SELECT 'cat' as animal, 23 as population, 'mammal' as category
  UNION ALL SELECT 'duck', 3, 'bird'
  UNION ALL SELECT 'dog', 2, 'mammal'
  UNION ALL SELECT 'goose', 1, 'bird'
  UNION ALL SELECT 'ox', 2, 'mammal'
  UNION ALL SELECT 'goat', 2, 'mammal')
SELECT * FROM Farm
```
