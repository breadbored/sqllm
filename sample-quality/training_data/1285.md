How do I construct a timestamp range with timezone information?

--- AGENT ---

RANGE constructs a range value from two timestamp bounds with timezone context. This query creates a timestamp range with both bounds specified in America/Los_Angeles timezone, though the result is normalized to UTC.

```googlesql
SELECT RANGE(TIMESTAMP '2022-10-01 14:53:27 America/Los_Angeles',
             TIMESTAMP '2022-10-01 16:00:00 America/Los_Angeles') AS results;
```
