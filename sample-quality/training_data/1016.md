How do I safely convert IP address strings to binary format, handling invalid input gracefully?

--- AGENT ---

This query uses NET.SAFE_IP_FROM_STRING to convert IP address strings to binary format, returning NULL for invalid inputs (such as CIDR notation, incomplete addresses, or malformed IPv6) instead of throwing an error. This function is useful when processing untrusted or variable-quality IP data.

```googlesql
SELECT
  addr_str,
  FORMAT("%T", NET.SAFE_IP_FROM_STRING(addr_str)) AS safe_ip_from_string
FROM UNNEST([
  '48.49.50.51',
  '::1',
  '3031:3233:3435:3637:3839:4041:4243:4445',
  '::ffff:192.0.2.128',
  '48.49.50.51/32',
  '48.49.50',
  '::wxyz'
]) AS addr_str;
```
