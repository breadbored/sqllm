How do I create a new database with configuration options like base directory and owner?

--- AGENT ---

The `CREATE DATABASE` statement creates a database and lets you specify schema options like base directory and owner through the `OPTIONS` clause. These options are system-specific and follow GoogleSQL hint syntax.

```googlesql
CREATE DATABASE library OPTIONS(
  base_dir=`/city/orgs`,
  owner='libadmin'
);
```
