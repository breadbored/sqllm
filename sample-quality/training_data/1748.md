--- USER ---

Provide me with a BigQuery SQL query that extracts scroll event information and associates the most recent scroll action with each event in a user session.

--- AGENT ---

This query extracts scroll events and enriches every event with the most recent scroll information that occurred at or before it within the same session, sub-session, page, and URL context.

The `scrolls` CTE uses a window function to number scroll events (identified by regex pattern matching on the `action` field) within partitions defined by session_id, sub_session_id, page_visit, and url_path. For each scroll event, it extracts:
- The numeric scroll value from the action string
- The scroll direction (up or down) by checking for `_up` in the action
- A running count of scrolls within the partition
- A boolean flag indicating whether the event represents a scroll or page_view action

The main SELECT performs a LEFT JOIN between the events and the scrolls CTE, using a range condition (`e.event_num >= ees.event_num`) to find all scroll events that occurred at or before each event. The `MAX_BY` function then extracts the most recent scroll information (by event_num) for each event. Null values are replaced with defaults: "down" for direction, 0 for scroll count and value, and true for the changed flag. The result is grouped by event ID.

```sql
/**
* Last Scroll
*
* This layer is just to get the scroll numbering for a later forwardfill
**/
WITH scrolls AS (
  SELECT
    SUM(
      CASE
        WHEN 
          REGEXP_CONTAINS(ee.action, r'^scroll(?:_up)?_(\d+)$')
        THEN 1
        ELSE 0
      END
    ) OVER (
      PARTITION BY
        ee.session_id,
        ne.sub_session_id,
        ee.page_visit,
        ee.url_path
      ORDER BY
        ee.created ASC
    ) AS scroll_num,
    REGEXP_CONTAINS(ee.action, r'^scroll(?:_up)?_(\d+)$') OR REGEXP_CONTAINS(ee.action, r'^page_view$') as scroll_changed,
    CASE
      WHEN REGEXP_CONTAINS(ee.action, r'^scroll(?:_up)?_(\d+)$') THEN CAST(REGEXP_EXTRACT(ee.action, r'(\d+)$') AS INT64)
      ELSE NULL
    END as scroll_value,
    CASE
      WHEN REGEXP_CONTAINS(ee.action, r'^scroll(?:_up)?_(\d+)$') 
        THEN 
          CASE WHEN REGEXP_EXTRACT(ee.action, r'scroll(_up)?') IS NOT NULL
            THEN 'up'
            ELSE 'down'
          END
      ELSE NULL
    END as scroll_direction,
    ee.session_id AS session_id,
    ee.page_visit AS page_visit,
    ee.url_path AS url_path,
    ne.sub_session_id AS sub_session_id,
    ne.event_num AS event_num,
    ee.id AS id
  FROM
    ${ref("events_paths")} AS ee
  JOIN
    ${ref("events_numbered")} AS ne
  ON
    ne.id = ee.id
)
SELECT
  IFNULL(MAX_BY(ees.scroll_direction, ees.event_num), "down") AS scroll_direction,
  IFNULL(MAX_BY(ees.scroll_num, ees.event_num), 0) AS scroll_num,
  IFNULL(MAX_BY(ees.scroll_value, ees.event_num), 0) AS scroll_value,
  IFNULL(MAX_BY(ees.scroll_changed, ees.event_num), true) AS scroll_changed,
  e.id AS id
FROM
  ${ref("events_numbered")} AS e
LEFT JOIN
  scrolls AS ees
ON
  e.session_id = ees.session_id
AND
  e.sub_session_id = ees.sub_session_id
AND
  e.page_visit = ees.page_visit
AND
  e.url_path = ees.url_path
AND
  e.event_num >= ees.event_num
AND
  ees.scroll_changed

GROUP BY
  e.id
```
