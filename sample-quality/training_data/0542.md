How do I parse a time string with AM/PM indicator when the format uses A.M./P.M. notation?

--- AGENT ---

The CAST function with the FORMAT clause parses a string into a time using the specified format. The A.M. format element matches either "A.M." or "P.M." in the input and adjusts the 12-hour time to the correct value.

```googlesql
SELECT CAST('03:30 P.M.' AS TIME FORMAT 'HH:MI A.M.') AS string_to_date_time
```
