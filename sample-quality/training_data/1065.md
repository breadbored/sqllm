How do I create a protocol buffer with named fields using a map-style constructor?

--- AGENT ---

The NEW operator with map constructor syntax creates a protocol buffer by specifying field names and their values, supporting nested structures and arrays.

```googlesql
NEW Universe {
  name: "Sol"
  closest_planets: ["Mercury", "Venus", "Earth" ]
  star {
    radius_miles: 432,690
    age: 4,603,000,000
  }
  constellations: [{
    name: "Libra"
    index: 0
  }, {
    name: "Scorpio"
    index: 1
  }]
  all_planets: (SELECT planets FROM SolTable)
}
```
