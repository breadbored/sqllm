How do I see the difference between string and bytes code point representations?

--- AGENT ---

When a Unicode character spans multiple bytes, TO_CODE_POINTS on BYTES returns multiple values while TO_CODE_POINTS on STRING returns one code point per character.

```googlesql
SELECT TO_CODE_POINTS(b'Ā') AS b_result, TO_CODE_POINTS('Ā') AS s_result;
```
