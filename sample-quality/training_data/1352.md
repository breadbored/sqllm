How do I convert Unicode code points to their corresponding characters?

--- AGENT ---

The CHR function converts Unicode code points into their corresponding characters, accepting values within valid Unicode ranges.

```googlesql
SELECT CHR(65) AS A, CHR(255) AS B, CHR(513) AS C, CHR(1024)  AS D;
```
