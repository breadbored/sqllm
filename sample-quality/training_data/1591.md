How do I read the value of a system variable back in a query?

--- AGENT ---

System variables can be referenced directly in queries using the @@ prefix. This example selects the current value of the system variable @@system_var_c, though support for reading system variables depends on the implementation you use.

```googlesql
SELECT @@system_var_c;
```
