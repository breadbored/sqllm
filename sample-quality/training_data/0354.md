--- USER ---

Swap the values of two columns in an UPDATE statement.

--- AGENT ---

This query swaps the FirstName and LastName columns by assigning FirstName to the old LastName value and LastName to the old FirstName value simultaneously.

```googlesql
UPDATE Singers s
SET s.FirstName = s.LastName,
    s.LastName = s.FirstName
WHERE true;
```
