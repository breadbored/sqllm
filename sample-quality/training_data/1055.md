How do I verify that two NULL values are the same?

--- AGENT ---

The IS NOT DISTINCT FROM operator returns TRUE when both values are NULL, treating NULL as equal to itself—unlike the = operator which returns NULL in this case.

```googlesql
SELECT NULL IS NOT DISTINCT FROM NULL
```
