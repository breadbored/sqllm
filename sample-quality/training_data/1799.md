This query creates a table function that aggregates course watch time for a specific client and container. The function accepts two parameters: `clientId` (a client identifier) and `containerId` (a container identifier).

The function sums the `course_progress_minutes` column from the `reporting_user_courses` table, returning the result as `course_watch_time`. It filters the results using two conditions:

1. The `clientId` parameter must be present within an array-valued `client_id` column (using `IN UNNEST`), indicating that the source table stores multiple client IDs as an array in each row
2. The `container_id` column must exactly match the `containerId` parameter

The function returns a single row with the aggregated watch time for the specified client and container combination.

```sql
CREATE OR REPLACE TABLE
  FUNCTION ${self()} (clientId INT64,
    containerId INT64) AS (
  SELECT
    SUM(course_progress_minutes) AS course_watch_time
  FROM
    ${ref("reporting_user_courses")}
  WHERE
    clientId IN UNNEST(client_id)
    AND container_id = containerId )
```
