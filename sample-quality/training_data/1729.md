--- USER ---

Generate a BigQuery SQL query that tags user sub-sessions based on caption settings from video events, distinguishing between user behavior preferences and content-specific attributes.

--- AGENT ---

This query creates event-based tags for user sub-sessions by extracting caption configuration signals from a video event stream. It produces two distinct tag classes:

**First branch (behavior):** Identifies when caption settings are actively enabled (not null and not 'off'), creating a tag from the caption setting value alone. It groups by sub_session_id and URL path to consolidate multiple caption events into a single row per session-URL pair, recording the earliest event timestamp.

**Second branch (content):** For the same caption-enabled events, joins with a courses table to associate caption settings with specific course content (via vaas_id matching). This branch creates a richer tag that includes both the caption setting and the course name, allowing segmentation by which content was accessed with which caption preferences.

Both branches share the same output schema (sub_session_id, tag, class, app, url_path, dt) and union together to produce a single result set. The distinction between 'behavior' and 'content' class labels suggests downstream use cases that care about user preference signals separately from content metadata.

```sql
SELECT
  e.sub_session_id as sub_session_id,
  CONCAT('vaas_captions:', e.vaas_settings_captions) as tag,
  'behavior' as class,
  'vaas' as app,
  e.vaas_url_url_path as url_path,
  MIN(e.created) as dt
FROM
  ${ref("base_buckets_events_vaas_speedy")} as e
WHERE
  e.vaas_settings_captions IS NOT NULL
AND
  e.vaas_settings_captions != 'off'
GROUP BY
  sub_session_id,
  tag,
  url_path

UNION ALL

SELECT
  e.sub_session_id as sub_session_id,
  CONCAT('vaas_captions:', e.vaas_settings_captions, ':', c.name) as tag,
  'content' as class,
  'vaas' as app,
  e.vaas_url_url_path as url_path,
  MIN(e.created) as dt
FROM
  ${ref("base_buckets_events_vaas_speedy")} as e
JOIN
  ${ref("courses")} as c
ON
  c.vaas_id = e.vaas_url_video_id
WHERE
  e.vaas_settings_captions IS NOT NULL
AND
  e.vaas_settings_captions != 'off'
GROUP BY
  sub_session_id,
  tag,
  url_path
```
