How do I horizontally aggregate a specific field from an array of structs?

--- AGENT ---

You can use horizontal aggregation on a field extracted from an array of structs by referencing the field directly. The aggregate function applies the operation to that field across all struct elements in the array.

```googlesql
GRAPH FinGraph
LET arr = [STRUCT(1 as x, 10 as y), STRUCT(2, 9), STRUCT(3, 8)]
LET total = SUM(arr.x)
RETURN total
```
