--- USER ---

How can I reuse endpoint variables from one path pattern in another MATCH clause in the same statement?

--- AGENT ---

Variables that represent endpoints (first or last nodes) of a path with a search prefix can be reused in other MATCH clauses in the same statement. Non-endpoint variables cannot be reused outside their path pattern.

```googlesql
GRAPH FinGraph
MATCH
  ANY (a:Account {id: 20})->(mid:Account)->(a:Account)->(mid:Account)->(a:Account),
  ALL (p:Person)->(a)
RETURN p.name
```
