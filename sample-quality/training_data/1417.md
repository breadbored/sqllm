How do I extract all capturing groups from a single pattern match?

--- AGENT ---

The REGEXP_EXTRACT_GROUPS function returns a STRUCT where each field corresponds to a capturing group in the pattern, returning the text matched by each group.

```googlesql
SELECT REGEXP_EXTRACT_GROUPS('abc123xyz', r'([a-z]+)([0-9]+)([a-z]+)') AS result
```
