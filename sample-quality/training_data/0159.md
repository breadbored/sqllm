How do I flatten nested arrays and extract a specific element from each inner array?

--- AGENT ---

Use `FLATTEN` with array offset notation to select a single element from each nested array. The `[OFFSET(1)]` syntax extracts the element at index 1 from every nested array, and `FLATTEN` collects all results into a single flat array.

```googlesql
WITH t AS (
  SELECT
  [
    STRUCT([STRUCT([1,2,3] AS quantity), STRUCT([4,5,6] AS quantity)] AS sales),
    STRUCT([STRUCT([7,8,9] AS quantity), STRUCT([10,11,12] AS quantity)] AS sales)
  ] AS v
)
SELECT FLATTEN(v.sales.quantity[OFFSET(1)]) AS second_values
FROM t;
```
