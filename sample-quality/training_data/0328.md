How do I add multiple columns to an existing table, some conditionally?

--- AGENT ---

The `ALTER TABLE ... ADD COLUMN` statement can add multiple columns in a single statement. The `IF NOT EXISTS` modifier on individual columns prevents errors if those columns already exist.

```googlesql
ALTER TABLE mydataset.mytable
    ADD COLUMN A STRING,
    ADD COLUMN IF NOT EXISTS B GEOGRAPHY,
    ADD COLUMN C ARRAY<NUMERIC>,
    ADD COLUMN D DATE OPTIONS(description="my description")
```
