--- USER ---

How do I calculate the sum of numeric values in a group?

--- AGENT ---

`SUM` returns the total of all non-NULL values in an aggregation. This example sums a list of integers:

```googlesql
SELECT SUM(x) AS sum
FROM UNNEST([1, 2, 3, 4, 5, 4, 3, 2, 1]) AS x;
```
