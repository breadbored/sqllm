--- USER ---

Modify a chart rank value deep within a three-level nested proto structure.

--- AGENT ---

This query performs multiple levels of nested UPDATEs to find a specific song in a specific album and update the rank value of a specific chart within that song, with assertions at each level.

```googlesql
UPDATE Singers s
SET (UPDATE s.Albums.Song so
     SET (UPDATE so.Chart c
          SET c.rank = 2
          WHERE c.chartname = "Galaxy Top 100"
          ASSERT_ROWS_MODIFIED 1)
     WHERE so.songtitle = "Bonus Track"
     ASSERT_ROWS_MODIFIED 1)
WHERE s.SingerId = 5
ASSERT_ROWS_MODIFIED 1;
```
