How do I declare a variable with an explicit default value based on a function?

--- AGENT ---

Use the `DEFAULT` clause with a function expression to initialize a variable when it is declared. The variable is set to the result of that function.

```googlesql
DECLARE d DATE DEFAULT CURRENT_DATE();
```
