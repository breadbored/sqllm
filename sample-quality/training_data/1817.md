## Question
Show me an example of a BigQuery table function that counts distinct users for a specific client.

--- AGENT ---

This query defines a parameterized BigQuery table function that accepts a single `clientId` integer argument and returns the count of distinct user IDs from the `reporting_user_courses` table, filtered to include only rows where the provided `clientId` matches a value in the `client_id` column.

The `UNNEST` operation in the WHERE clause indicates that `client_id` is stored as an ARRAY type in the source table, so the function unpacks that array to perform the membership check.

The output is a single column named `unique_users` containing the count.

```sql
CREATE OR REPLACE TABLE
  FUNCTION ${self()} (clientId INT64) AS (
  SELECT
    COUNT(DISTINCT user_id) AS unique_users
  FROM
    ${ref("reporting_user_courses")}
  WHERE
    clientId IN UNNEST(client_id) )
```
