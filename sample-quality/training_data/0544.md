How do I format multiple numeric values to display currency with thousands separators?

--- AGENT ---

The CAST function with the FORMAT clause converts numeric values to formatted strings. The format string '$999,999.999' specifies a dollar sign, digit placeholders (9), a comma as thousands separator, and decimal places. The query uses UNNEST to format multiple values in a single result set.

```googlesql
SELECT input, CAST(input AS STRING FORMAT '$999,999.999') AS output
FROM UNNEST([1.2, 12.3, 123.456, 1234.56, -12345.678, 1234567.89]) AS input
```
