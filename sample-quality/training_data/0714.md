--- USER ---

How do I define multiple variables sequentially when one depends on another?

--- AGENT ---

You can use multiple `LET` statements in sequence to define variables that depend on previously defined variables. Each `LET` statement can reference variables from earlier `LET` statements in the same query.

```googlesql
GRAPH FinGraph
MATCH (source:Account)-[e:Transfers]->(destination:Account)
LET a = source
LET b = a
RETURN b.id AS b_id
```
