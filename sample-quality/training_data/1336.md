What does sample standard deviation return when an input value is infinity?

--- AGENT ---

The `STDDEV_SAMP` function returns NaN when any input value is positive or negative infinity, as these extreme values make standard deviation undefined.

```googlesql
SELECT STDDEV_SAMP(x) AS results FROM UNNEST([10, 14, CAST('Infinity' as DOUBLE)]) AS x
```
