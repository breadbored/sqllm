## Question
Write a query that enriches resource access events with user course progress metrics and client information, showing what resources were opened by each user along with their progress and time spent in each course.

--- AGENT ---

This query generates a detailed log of resource access events (specifically when users opened resources) enriched with course progress context and client associations.

**How it works:**

1. **Main source**: Events from `agg_resource_events` filtered to only `opened_resource` actions
2. **Resource metadata**: LEFT JOINs to course and lesson resource tables to retrieve resource names and URLs (preferring lesson-level URLs, falling back to course-level)
3. **User progress calculation** (via `ulp` CTE): For each user-course pair, it aggregates lesson-level progress by:
   - Taking the MAX progress_seconds per lesson (a single snapshot of each lesson's progress)
   - Summing across all lessons and converting to minutes
   - Also summing time spent on each lesson
4. **Course duration** (via `cd` CTE): Sums video durations across all lessons in each course to establish total course length in minutes
5. **Progress metrics**: Calculates a course progress percentage by dividing accumulated progress by total duration, capped at 100%
6. **Current context**: INNER JOINs to `agg_user_course_progress` to include current snapshot course progress data
7. **Client associations**: INNER JOINs through container and client tables to collect all client IDs and names associated with each course container
8. **Aggregation**: Groups by user, course, resource, and temporal details; uses ANY_VALUE for progress fields (suggesting a single value per group) and ARRAY_AGG to collect distinct client IDs and names

```sql
SELECT
  e.user_id,
  e.course_id,
  ucp.container_id,
  ucp.course_name,
  e.action,
  e.related_id AS resource_id,
  e.course_resource_id,
  e.lesson_resource_id,
  LOWER(JSON_EXTRACT_SCALAR(action_meta, '$.resource_name')) AS resource_name,
  COALESCE(lr.url, cr.url, lr.file, cr.file) AS resource_url,
  ANY_VALUE(ulp.user_total_lesson_progress) AS course_progress_minutes,
  ANY_VALUE(cd.course_duration_minutes) AS course_length_minutes,
  LEAST( ROUND(((ANY_VALUE(ulp.user_total_lesson_progress) / ANY_VALUE(cd.course_duration_minutes))), 2), 1 ) AS course_progress_percentage,
  ANY_VALUE(ROUND((ulp.time_on_course / 60), 3)) AS time_on_course_minutes,
  ucp.course_status,
  ucp.course_progress_minutes AS current_course_progress_minutes,
  ucp.course_progress_percentage AS current_course_progress_percentage,
  ucp.course_length_minutes AS current_course_length_minutes,
  ucp.time_on_course_minutes AS current_time_on_course_minutes,
  e.created,
  ARRAY_AGG(DISTINCT ccc.client_id) AS client_id,
  ARRAY_AGG(DISTINCT clients.name) AS client_name
FROM
  ${ref("agg_resource_events")} AS e
LEFT JOIN
  ${ref("stg_course_resources")}cr
ON
  cr.id = e.course_resource_id
LEFT JOIN
  ${ref("stg_lesson_resources")} AS lr
ON
  e.lesson_resource_id = lr.id
  -- get course progress at time of resource click
INNER JOIN (
  SELECT
    lesson_progress.user_id,
    lesson_progress.course_id,
    ROUND((SUM(lesson_progress.user_total_lesson_progress) / 60), 3) AS user_total_lesson_progress,
    SUM(time_on_lesson) AS time_on_course
  FROM (
    SELECT
      uclp.user_id,
      uclp.course_id,
      MAX(progress_seconds) AS user_total_lesson_progress,
      SUM(interval_seconds) AS time_on_lesson
    FROM
      ${ref("stg_user_course_lesson_progress")} AS uclp
    INNER JOIN
      ${ref("stg_events_app_speedy")} AS e
    ON
      uclp.user_id = e.user_id
      AND uclp.course_id = e.course_id
    WHERE
      uclp.user_id IS NOT NULL
      AND uclp.created <= e.created
    GROUP BY
      uclp.user_id,
      uclp.course_id,
      uclp.lesson_id ) AS lesson_progress
  GROUP BY
    lesson_progress.user_id,
    lesson_progress.course_id ) AS ulp
ON
  e.course_id = ulp.course_id
  AND e.user_id = ulp.user_id
  --get course duration
INNER JOIN (
  SELECT
    cl.course_id,
    ROUND((SUM(v.duration) / 60), 3) AS course_duration_minutes
  FROM
    ${ref("stg_lessons")} AS l
  LEFT JOIN
    ${ref("stg_videos")} AS v
  ON
    l.video_id = v.id
  LEFT JOIN
    ${ref("stg_course_lessons")} AS cl
  ON
    l.id = cl.lesson_id
  GROUP BY
    course_id
  ORDER BY
    course_id ) AS cd
ON
  e.course_id = cd.course_id
INNER JOIN
  ${ref("agg_user_course_progress")} AS ucp
ON
  e.course_id = ucp.course_id
  AND e.user_id = ucp.user_id
INNER JOIN
  ${ref("stg_clients_course_containers")} AS ccc
ON
  ucp.container_id = ccc.course_container_id
INNER JOIN
  ${ref("stg_clients")} AS clients
ON
  clients.id = ccc.client_id
WHERE
  e.name = 'opened_resource'
GROUP BY
  e.user_id,
  e.course_id,
  ucp.container_id,
  ucp.course_name,
  e.action,
  e.related_id,
  e.course_resource_id,
  e.lesson_resource_id,
  LOWER(JSON_EXTRACT_SCALAR(action_meta, '$.resource_name')),
  COALESCE(lr.url, cr.url, lr.file, cr.file),
  ucp.course_status,
  ucp.course_progress_minutes,
  ucp.course_progress_percentage,
  ucp.course_length_minutes,
  ucp.time_on_course_minutes,
  e.created
```
