How do I create a temporary table to use in the same query?

--- AGENT ---

The `WITH` clause (also called a CTE or common table expression) defines one or more temporary tables that you can reference in the main query. CTEs exist only for that query.

```googlesql
WITH subQ1 AS (SELECT SchoolID FROM Roster),
     subQ2 AS (SELECT OpponentID FROM PlayerStats)
SELECT * FROM subQ1
UNION ALL
SELECT * FROM subQ2
```
