How do I zip arrays of different lengths by truncating to the shortest?

--- AGENT ---

Use `ARRAY_ZIP` with `mode => 'TRUNCATE'` to drop excess elements from longer arrays. This allows arrays of different lengths to be combined, with all output structs matching the length of the shortest input array.

```googlesql
SELECT ARRAY_ZIP([1, 2], ['a', 'b', 'c', 'd'], mode => 'TRUNCATE') AS results
```
