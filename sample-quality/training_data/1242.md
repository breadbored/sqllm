How do I rotate rows into columns using a pivot table?

--- AGENT ---

`PIVOT` rotates unique values from a specified column into separate output columns, with an aggregate function applied to their values. Rows are implicitly grouped by all non-pivoted columns.

```googlesql
SELECT * FROM
  (SELECT product, sales, quarter FROM Produce)
  PIVOT(SUM(sales) FOR quarter IN ('Q1', 'Q2', 'Q3', 'Q4'))
```
