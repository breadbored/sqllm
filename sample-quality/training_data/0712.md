How do I define a variable and use it in the same linear query statement?

--- AGENT ---

The `LET` statement defines a variable that can be referenced in the same linear query statement. This example defines variable `a` as a reference to the matched source account and then returns it.

```googlesql
GRAPH FinGraph
MATCH (source:Account)-[e:Transfers]->(destination:Account)
LET a = source
RETURN a.id AS a_id
```
