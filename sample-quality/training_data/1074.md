How do I rename a column using the SELECT pipe operator?

--- AGENT ---

The `SELECT` pipe operator produces a new table with only the listed columns. You can rename a column by using the `AS` keyword to assign a new alias to the column in the output.

```googlesql
FROM (SELECT 'apples' AS item, 2 AS sales)
|> SELECT item AS fruit_name;
```
