How do I create a polymorphic TVF that accepts arguments of any type and conditionally returns rows from an input table?

--- AGENT ---

This creates a temporary table-valued function using templated function parameters (ANY TYPE and ANY TABLE) that allows the function to accept arguments and tables of any compatible type. The function returns all rows from the input table where a comparison predicate on the typed arguments evaluates to true.

```googlesql
CREATE TEMP TABLE FUNCTION MyFunction(
     first_value ANY TYPE,
     second_value ANY TYPE,
     MyInputTable ANY TABLE)
   AS
     SELECT *
     FROM MyInputTable
     WHERE first_value > second_value;
```
