How do I generate an array of dates within a range?

--- AGENT ---

Use `GENERATE_DATE_ARRAY` with a start and end date to create an array of consecutive dates. The default increment is 1 day, producing a daily sequence from start to end date inclusive.

```googlesql
SELECT GENERATE_DATE_ARRAY('2016-10-05', '2016-10-08') AS example;
```
