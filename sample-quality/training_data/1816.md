Write a BigQuery table function that calculates the total watch time across all courses for a given client ID.

--- AGENT ---

This is a BigQuery table function that accepts a single client ID as input and returns that client's total cumulative watch time. The function queries the `reporting_user_courses` table and filters for rows where the input `clientId` matches a value in the `client_id` field (which is an array type, requiring UNNEST to unnest it for comparison). It then sums all `course_progress_minutes` for matching records and returns this aggregated value as `total_watch_time`. The function can be invoked by passing a client ID and will efficiently compute the sum in a single aggregation step.

```sql
CREATE OR REPLACE TABLE
  FUNCTION ${self()} (clientId INT64) AS (
  SELECT
    SUM(course_progress_minutes) AS total_watch_time
  FROM
    ${ref("reporting_user_courses")}
  WHERE
    clientId IN UNNEST(client_id))
```
