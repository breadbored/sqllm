How do I construct a date range with a start and end date?

--- AGENT ---

RANGE constructs a range value from two temporal bounds of the same type. This query creates a date range from December 1, 2022 to December 31, 2022, using the half-open interval notation [start, end).

```googlesql
SELECT RANGE(DATE '2022-12-01', DATE '2022-12-31') AS results;
```
