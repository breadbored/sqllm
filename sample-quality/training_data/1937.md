## 5. Plan CTE layers bottom-up

1. **`rankedlessons`** — assigns a row number to each course-lesson ordering row, partitioned by course, ordered by `id`, to identify the Nth lesson.
2. **`secondlesson`** — filters `rankedlessons` to `rn = 2` to isolate the second lesson's `(course_id, id)`.
3. **First lesson descriptor (inline subquery `fld`)** — identifies the first lesson per course via `MIN(id)`, resolves to `lesson_id` and `duration` through the lessons→videos chain.
4. **First lesson progress (inline subquery `uflp`)** — joins user progress to `fld`, aggregates `MAX(progress_seconds)` per `(user_id, course_id, lesson_id)`, computes capped percentage.
5. **Second lesson descriptor (inline subquery `sld`)** — identifies the second lesson per course via the `secondlesson` CTE, resolves to `lesson_id` and `duration`.
6. **Second lesson progress (inline subquery `uslp`)** — joins user progress to `sld`, aggregates `MAX(progress_seconds)` per `(user_id, course_id, lesson_id)`, computes capped percentage.
7. **Final SELECT** — joins `agg_user_course_progress` to `uflp` (INNER) and `uslp` (LEFT) on `(user_id, course_id)`.

## 6. Write each CTE with justification

**`rankedlessons`** — Needed to identify lesson position within a course by ordering. `ROW_NUMBER()` partitioned by `course_id` and ordered by `id` gives a stable ordinal position for each lesson.

**`secondlesson`** — Filters to `rn = 2` so downstream joins can resolve the second lesson's `lesson_id` and video duration. Separated as a CTE because it is referenced once in the second-lesson descriptor subquery.

**First lesson identification uses `MIN(id)` inline** rather than the CTE approach — this is an equivalent but different pattern for finding the first row. It avoids the CTE by using a simple aggregate.

**The `ucl` subquery (completed lessons count)** is joined in both branches but produces no columns in the output. It exists structurally but is unused in the final SELECT — likely a remnant from an earlier version that filtered or annotated based on completion count.

## 7. Compose the final query

```sql
WITH
rankedlessons AS (
  SELECT
    course_id,
    id,
    row_number() OVER (PARTITION BY course_id ORDER BY id) AS rn
  FROM
    ${ref("stg_course_lessons")}
),

secondlesson AS (
  SELECT
    rl.course_id,
    rl.id
  FROM
    rankedlessons AS rl
  WHERE
    rl.rn = 2
)

SELECT
  ucp.*,
  uflp.first_lesson_length,
  uflp.user_first_lesson_progress,
  uflp.first_lesson_progress_percentage,
  uslp.second_lesson_length,
  uslp.user_second_lesson_progress,
  uslp.second_lesson_progress_percentage
FROM
  ${ref("agg_user_course_progress")} AS ucp

  -- first lesson
INNER JOIN (
  SELECT
    uclp.user_id,
    uclp.course_id,
    uclp.lesson_id,
    max(uclp.progress_seconds)                                                   AS user_first_lesson_progress,
    fld.first_lesson_length,
    least(round(((max(uclp.progress_seconds) / fld.first_lesson_length)), 2), 1) AS first_lesson_progress_percentage
  FROM
    ${ref("stg_user_course_lesson_progress")} AS uclp
  INNER JOIN (
    SELECT
      cl.course_id,
      cl.lesson_id AS first_lesson_id,
      v.duration   AS first_lesson_length
    FROM
      ${ref("stg_course_lessons")} AS cl
    INNER JOIN (
      SELECT
        course_id,
        min(id) AS id
      FROM
        ${ref("stg_course_lessons")}
      GROUP BY
        course_id
    ) AS fl
      ON
        cl.id = fl.id
        AND cl.course_id = fl.course_id
    INNER JOIN
      ${ref("stg_lessons")} AS l
      ON
        cl.lesson_id = l.id
    INNER JOIN
      ${ref("stg_videos")} AS v
      ON
        l.video_id = v.id
  ) AS fld
    ON
      uclp.course_id = fld.course_id
      AND uclp.lesson_id = fld.first_lesson_id
  LEFT JOIN (
    SELECT
      ucp.user_id,
      ucp.course_id,
      count(DISTINCT ufcl.lesson_id) AS completed_lessons
    FROM
      ${ref("agg_user_course_progress")} AS ucp
    LEFT JOIN
      ${ref("stg_user_finished_course_lessons")} AS ufcl
      ON
        ucp.course_id = ufcl.course_id
        AND ucp.user_id = ufcl.user_id
    GROUP BY
      ucp.user_id,
      ucp.course_id
  ) AS ucl
    ON
      uclp.user_id = ucl.user_id
      AND uclp.course_id = ucl.course_id
  GROUP BY
    uclp.user_id,
    uclp.course_id,
    lesson_id,
    fld.first_lesson_length
) AS uflp
  ON
    ucp.user_id = uflp.user_id
    AND ucp.course_id = uflp.course_id
-- second lesson
LEFT JOIN (
  SELECT
    uclp.user_id,
    uclp.course_id,
    uclp.lesson_id,
    max(uclp.progress_seconds)                                                    AS user_second_lesson_progress,
    sld.second_lesson_length,
    least(round(((max(uclp.progress_seconds) / sld.second_lesson_length)), 2), 1) AS second_lesson_progress_percentage
  FROM
    ${ref("stg_user_course_lesson_progress")} AS uclp
  INNER JOIN (
    SELECT
      cl.course_id,
      cl.lesson_id AS second_lesson_id,
      v.duration   AS second_lesson_length
    FROM
      ${ref("stg_course_lessons")} AS cl
      -- get second lesson
    INNER JOIN
      secondlesson AS sl
      ON
        cl.id = sl.id
        AND cl.course_id = sl.course_id
    INNER JOIN
      ${ref("stg_lessons")} AS l
      ON
        cl.lesson_id = l.id
    INNER JOIN
      ${ref("stg_videos")} AS v
      ON
        l.video_id = v.id
  ) AS sld
    ON
      uclp.course_id = sld.course_id
      AND uclp.lesson_id = sld.second_lesson_id
  LEFT JOIN (
    SELECT
      ucp.user_id,
      ucp.course_id,
      count(DISTINCT ufcl.lesson_id) AS completed_lessons
    FROM
      ${ref("agg_user_course_progress")} AS ucp
    LEFT JOIN
      ${ref("stg_user_finished_course_lessons")} AS ufcl
      ON
        ucp.course_id = ufcl.course_id
        AND ucp.user_id = ufcl.user_id
    GROUP BY
      ucp.user_id,
      ucp.course_id
  ) AS ucl
    ON
      uclp.user_id = ucl.user_id
      AND uclp.course_id = ucl.course_id
  GROUP BY
    uclp.user_id,
    uclp.course_id,
    lesson_id,
    sld.second_lesson_length
) AS uslp
  ON
    ucp.user_id = uslp.user_id
    AND ucp.course_id = uslp.course_id
```
