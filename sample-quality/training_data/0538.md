How do I parse a date string that contains the full month name, day, and year?

--- AGENT ---

The CAST function with the FORMAT clause parses a string into a date using the specified format. The MONTH format element matches the full month name, DD matches the 2-digit day, and YYYY matches the 4-digit year.

```googlesql
SELECT CAST('DECEMBER 03, 2018' AS DATE FORMAT 'MONTH DD, YYYY') AS string_to_date
```
