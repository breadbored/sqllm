How do I construct an interval value from individual year, month, day, hour, minute, and second components?

--- AGENT ---

MAKE_INTERVAL takes INT64 values for year, month, day, hour, minute, and second (all optional, defaulting to 0) and constructs an interval object. You can pass arguments positionally or as named arguments using the `=>` syntax.

```googlesql
SELECT
  MAKE_INTERVAL(1, 6, 15) AS i1,
  MAKE_INTERVAL(hour => 10, second => 20) AS i2,
  MAKE_INTERVAL(1, minute => 5, day => 2) AS i3
```
