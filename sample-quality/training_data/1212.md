How do I check whether a specific field has been set in a protocol buffer without retrieving its value?

--- AGENT ---

This query uses EXTRACT with HAS to test field presence, returning TRUE if the field is set and FALSE if it is unset or NULL. This is useful for understanding which optional fields in proto messages have been populated.

```googlesql
SELECT EXTRACT(HAS(date) FROM chart_col) AS has_release_date
FROM AlbumList
```
