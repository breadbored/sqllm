How do I define a recursive CTE inline in a pipe query?

--- AGENT ---

The `WITH RECURSIVE` clause in pipe syntax defines a CTE that recursively unions a base query with a recursive query. The recursive CTE can be used in downstream pipe operators.

```googlesql
SELECT 1 AS key
|> WITH RECURSIVE t AS (
    SELECT 2 AS key
    |> UNION ALL (
      SELECT key + 1 AS key
      FROM t
      WHERE key <= 3
    )
  )
|> UNION ALL (FROM t)
```
