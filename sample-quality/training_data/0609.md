How do I extract all the edges from a matched graph path as an array?

--- AGENT ---

The `EDGES` function returns an array of all edges in a graph path while preserving their original order. You can then operate on these edges directly, such as extracting properties or transforming them with other functions.

```googlesql
GRAPH FinGraph
MATCH p=(src:Account)-[t1:Transfers]->(mid:Account)-[t2:Transfers]->(dst:Account)
LET es = EDGES(p)
RETURN ARRAY_CONCAT(ARRAY_TRANSFORM(es, e -> e.Id), [dst.Id]) as ids_in_path
```
