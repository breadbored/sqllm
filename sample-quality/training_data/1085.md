How do I filter rows based on a condition using the pipe syntax?

--- AGENT ---

The `WHERE` pipe operator filters the input table by a boolean expression, removing rows where the condition is false. It behaves the same as the `WHERE` clause in standard syntax.

```googlesql
(
  SELECT 'apples' AS item, 2 AS sales
  UNION ALL
  SELECT 'bananas' AS item, 5 AS sales
  UNION ALL
  SELECT 'carrots' AS item, 8 AS sales
)
|> WHERE sales >= 3;
```
