How do I write a CTE definition with a trailing comma in the WITH pipe operator?

--- AGENT ---

The `WITH` pipe operator allows a trailing comma after the last CTE definition, providing flexibility in formatting multi-line CTE definitions.

```googlesql
SELECT 1 AS key
|> WITH t1 AS (
     SELECT 2
   ), t2 AS (
     SELECT 3
   ),
|> UNION ALL (FROM t1), (FROM t2)
```
