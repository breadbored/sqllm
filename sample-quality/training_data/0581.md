How do I repair invalid WKT polygons that have self-intersections or unclosed rings?

--- AGENT ---

`ST_GEOGFROMTEXT` with `make_valid => TRUE` attempts to fix invalid polygons by repairing topological issues like self-intersections and unclosed rings.

```googlesql
WITH data AS (
  SELECT 'POLYGON((0 -1, 2 1, 2 -1, 0 1))' wkt)
SELECT
  SAFE.ST_GEOGFROMTEXT(wkt) as geom,
  SAFE.ST_GEOGFROMTEXT(wkt, make_valid => TRUE) as valid_geom
FROM data
```
