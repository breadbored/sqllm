How do I extract a numeric value from a JSON object and convert it to UINT32?

--- AGENT ---

Use `UINT32` with `JSON_QUERY` to extract a specific numeric field from a JSON object and convert it to a SQL UINT32 value.

```googlesql
SELECT UINT32(JSON_QUERY(JSON '{"gate": "A4", "flight_number": 2005}', "$.flight_number")) AS flight_number;
```
