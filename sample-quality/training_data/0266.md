How do I convert an INTERVAL of 3 days to a STRING?

--- AGENT ---

CAST from INTERVAL to STRING produces a standardized interval string representation in the format 'Y-M D H:M:S', showing years-months, days, and hours:minutes:seconds.

```googlesql
SELECT CAST(INTERVAL 3 DAY AS STRING) AS interval_to_string
```
