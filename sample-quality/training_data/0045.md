How do I calculate separate averages for precipitation in wet seasons versus dry seasons?

--- AGENT ---

The `WHERE` clause filters the input rows to an aggregate function before aggregation occurs. This query uses `WHERE` to compute two separate averages: one for spring and summer months, and another for fall and winter months.

```googlesql
WITH
  Precipitation AS (
    SELECT 2001 AS year, 'spring' AS season, 9 AS inches
    UNION ALL
    SELECT 2001, 'winter', 1
    UNION ALL
    SELECT 2000, 'fall', 3
    UNION ALL
    SELECT 2000, 'summer', 5
    UNION ALL
    SELECT 2000, 'spring', 7
    UNION ALL
    SELECT 2000, 'winter', 2
  )
SELECT
  AVG(inches WHERE season IN ('spring', 'summer')) AS wet_seasons_avg,
  AVG(inches WHERE season IN ('fall', 'winter')) AS dry_seasons_avg
FROM Precipitation;
```
