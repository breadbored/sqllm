--- USER ---

How do I merge per-country HLL++ sketches into a single combined sketch without extracting a final count?

--- AGENT ---

This query creates per-country HLL++ sketches and then merges them into a new sketch using MERGE_PARTIAL, which returns a byte-encoded sketch rather than a numeric cardinality. This is useful when you need to preserve the merged sketch for further operations like additional merges or storage rather than immediately extracting a count.

```googlesql
SELECT HLL_COUNT.MERGE_PARTIAL(HLL_sketch) AS distinct_customers_with_open_invoice
FROM
  (
    SELECT
      country,
      HLL_COUNT.INIT(customer_id) AS hll_sketch
    FROM
      UNNEST(
        ARRAY<STRUCT<country STRING, customer_id STRING, invoice_id STRING>>[
          ('UA', 'customer_id_1', 'invoice_id_11'),
          ('BR', 'customer_id_3', 'invoice_id_31'),
          ('CZ', 'customer_id_2', 'invoice_id_22'),
          ('CZ', 'customer_id_2', 'invoice_id_23'),
          ('BR', 'customer_id_3', 'invoice_id_31'),
          ('UA', 'customer_id_2', 'invoice_id_24')])
    GROUP BY country
  );
```
