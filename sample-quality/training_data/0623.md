How can I sum the values in a simple array using horizontal aggregation?

--- AGENT ---

Horizontal aggregation allows you to use aggregate functions directly on array values within a `LET` clause. The `SUM` function sums all elements in an array without requiring a row-by-row aggregation.

```googlesql
GRAPH FinGraph
LET arr = [1, 2, 3]
LET total = SUM(arr)
RETURN total
```
