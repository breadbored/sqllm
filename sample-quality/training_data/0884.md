How do I set multiple values in a JSON structure?

--- AGENT ---

The `JSON_SET` function accepts multiple path-value pairs that are applied left to right to the same JSON value.

```googlesql
SELECT JSON_SET(
  JSON '{"a": 1, "b": {"c":3}, "d": [4]}',
  '$.a', 'v1',
  '$.b.e', 'v2',
  '$.d[2]', 'v3') AS json_data
```
