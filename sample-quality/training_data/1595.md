How do I declare runtime variables in a GoogleSQL procedural script, including variables with default values?

--- AGENT ---

Runtime variables must be declared with DECLARE statements before use in a script. This example declares one string variable with a default value and two integer variables without defaults, showing both single-declaration and multi-declaration syntax.

```googlesql
DECLARE target_word STRING DEFAULT 'methinks';
DECLARE corpus_count, word_count INT64;
```
