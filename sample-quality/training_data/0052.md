How do I aggregate values into an array while ignoring NULL values?

--- AGENT ---

`ARRAY_AGG` with the `IGNORE NULLS` clause excludes NULL values from the result array. This example collects integers while filtering out any NULL entries:

```googlesql
SELECT ARRAY_AGG(x IGNORE NULLS) AS array_agg
FROM UNNEST([NULL, 1, -2, 3, -2, 1, NULL]) AS x;
```
