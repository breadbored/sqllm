How do I explicitly match all paths when the default behavior would not match all?

--- AGENT ---

Use the ALL prefix before the path pattern to explicitly request all matching paths. When no prefix is specified, ALL is the default, so this prefix is primarily used for clarity.

```googlesql
GRAPH FinGraph
MATCH ALL (a:Account {id: 20})-[t:Transfers]->(b:Account)
RETURN a.id AS a_id, t.amount, b.id AS b_id
```
