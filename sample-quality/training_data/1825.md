## Tables
### agg_client_transactions
| column | type |
|---|---|
| user_id | INT64 |
| user_created | DATETIME |
| day | DATETIME |
| client_id | ARRAY<INT64> |

## Question
Generate a BigQuery SQL query that produces daily user engagement metrics (new vs. returning users) for each client, identifying which users are interacting with a client for the first time on a given day versus returning users.

--- AGENT ---

This query produces a daily user engagement summary per client by comparing each transaction against the user's first interaction date with that client.

**Step 1: Establish baseline dates**
The CTE `client_transactions_min` finds the earliest day each user transacted with each client. It unnests the `client_id` array (indicating users can interact with multiple clients in a single transaction) and groups by user-client pair to find MIN(day).

**Step 2: Classify users by interaction status**
The main query joins transactions back to this baseline, unnesting `client_id` again to match the CTE structure. For each transaction on each day:
- If the transaction day matches the user's first interaction day with that client → classify as "new"
- If the transaction day is after the first interaction → classify as "returning"

**Step 3: Aggregate metrics**
The query counts:
- `total_users`: distinct users with any transaction that day
- `new_users`: distinct users whose first interaction with the client occurred that day
- `returning_users`: distinct users with prior interactions who transacted again
- `total_sessions`: raw transaction count
- `returning_user_sessions`: session count from returning users
- Results are grouped by date and client_id and ordered by date

```sql
WITH
  client_transactions_min AS (
  SELECT
    ct.user_id,
    cid AS client_id,
    MIN(ct.`day`) AS min_day
  FROM
    ${ref("agg_client_transactions")} AS ct,
    UNNEST(ct.client_id) AS cid
  GROUP BY
    ct.user_id,
    cid )
SELECT
  COUNT(DISTINCT ct.user_id) AS total_users,
  COUNT( DISTINCT
    CASE
      WHEN DATE_TRUNC(ctm.min_day, DAY) = DATE_TRUNC(ct.`day`, DAY) THEN ct.user_id
  END
    ) AS new_users,
  COUNT( DISTINCT
    CASE
      WHEN DATE_TRUNC(ctm.min_day, DAY) != DATE_TRUNC(ct.`day`, DAY) THEN ct.user_id
  END
    ) AS returning_users,
  COUNT(ct.user_id) AS total_sessions,
  COUNT(
    CASE
      WHEN DATE_TRUNC(ctm.min_day, DAY) != DATE_TRUNC(ct.`day`, DAY) THEN ct.user_id
  END
    ) AS returning_user_sessions,
  DATE_TRUNC(ct.`day`, DAY) AS date,
  cl AS client_id
FROM
  ${ref("agg_client_transactions")} AS ct
CROSS JOIN
  UNNEST(ct.client_id) AS cl
INNER JOIN
  client_transactions_min AS ctm
ON
  ct.user_id = ctm.user_id
  AND ctm.client_id = cl
GROUP BY
  DATE_TRUNC(ct.`day`, DAY),
  cl
ORDER BY
  date
```
