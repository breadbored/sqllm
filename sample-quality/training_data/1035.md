How do I safely access a protocol buffer map field when the key might not exist?

--- AGENT ---

Use the SAFE_KEY keyword with the map subscript operator to return NULL if the requested key is not present in the protocol buffer map, instead of producing an error.

```googlesql
SELECT
  CAST(m.purchased[SAFE_KEY('B')] AS safe_key_missing
FROM
  (SELECT AS VALUE CAST("purchased { key: 'A' value: 2 }" AS Item)) AS m
```
