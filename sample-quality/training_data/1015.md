How do I convert binary IPv4 addresses to integer format?

--- AGENT ---

This query converts binary IPv4 addresses in network byte order back to INT64 integer format using NET.IPV4_TO_INT64, where the least significant bit of the IP address maps to the least significant bit of the output integer. The output is formatted as hexadecimal for clarity.

```googlesql
SELECT
  FORMAT("%T", x) AS addr_bin,
  FORMAT("0x%X", NET.IPV4_TO_INT64(x)) AS ipv4_to_int64
FROM
UNNEST([b"\x00\x00\x00\x00", b"\x00\xab\xcd\xef", b"\xff\xff\xff\xff"]) AS x;
```
