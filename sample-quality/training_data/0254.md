How do I cast a hexadecimal string like '0x123' to an INT64 integer?

--- AGENT ---

CAST supports converting hex strings to integers by recognizing the '0x' prefix and interpreting the following characters as hexadecimal digits. The string '0x123' converts to 291 in decimal.

```googlesql
SELECT '0x123' as hex_value, CAST('0x123' as INT64) as hex_to_int;
```
