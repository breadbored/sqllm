--- USER ---

How do I retrieve a single person's name from those whose country property is "Australia"?

--- AGENT ---

This query uses a `VALUE` subquery to extract a single scalar value from the subquery result. The subquery matches a person with `country: "Australia"` and returns their name; the `LIMIT 1` ensures exactly one row is produced, which becomes the scalar result.

```googlesql
GRAPH FinGraph
RETURN VALUE {
  MATCH (p:Person {country: "Australia"})
  RETURN p.name
  LIMIT 1
} AS results;
```
