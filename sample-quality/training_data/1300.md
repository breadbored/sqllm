How do I group overlapping date ranges by employee and create merged session ranges?

--- AGENT ---

RANGE_SESSIONIZE groups ranges within partitions and merges those that meet or overlap according to the specified option. This query sessionizes ranges by employee ID using the default MEETS behavior, combining overlapping or adjacent ranges into single session ranges for each employee.

```googlesql
SELECT
  emp_id, duration, session_range
FROM
  RANGE_SESSIONIZE(
    TABLE mydataset.my_sessionized_range_table,
    'duration',
    ['emp_id'])
ORDER BY emp_id;
```
