--- USER ---

How do I match edges that flow from one node to another in the opposite of the default left-to-right direction?

--- AGENT ---

Use a left-directed edge pattern with the arrow pointing left: `<-`. This abbreviated pattern matches edges whose direction is reversed from left to right.

```googlesql
GRAPH FinGraph
MATCH (account:Account)<-(person:Person)
RETURN account.id, person.name
```
