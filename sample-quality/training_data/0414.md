How do I construct a timestamp using an explicit UTC offset instead of a named timezone?

--- AGENT ---

You can specify a timestamp with a UTC offset like -08:00. This offset is equivalent to using the named timezone that has the same offset for that date and time.

```googlesql
SELECT UNIX_MILLIS(TIMESTAMP '2008-12-25 15:30:00-08:00') AS millis;
```
