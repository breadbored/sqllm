This query creates a table function that computes the average course watch time per user for a given client. It takes a single parameter (`clientId`) and queries the `reporting_user_courses` table, filtering to rows where the `clientId` parameter exists within the `client_id` column (which appears to be an array). The function returns a single computed metric: the total sum of `course_progress_minutes` divided by the count of distinct `user_id` values, aliased as `avg_watch_time_on_course`. This yields the mean watch time across all users associated with that client.

```sql
CREATE OR REPLACE TABLE
  FUNCTION ${self()} (clientId INT64) AS (
  SELECT
    (SUM(course_progress_minutes) / COUNT(DISTINCT user_id)) AS avg_watch_time_on_course
  FROM
    ${ref("reporting_user_courses")}
  WHERE
    clientId IN UNNEST(client_id) )
```
