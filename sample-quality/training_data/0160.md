How do I flatten an array of structs and extract a scalar field from each struct?

--- AGENT ---

Use `FLATTEN` with dot notation to select a single scalar field from all struct elements in an array. The function extracts that field value from every struct and returns all results in a single flat array.

```googlesql
WITH t AS (
  SELECT
  [
    STRUCT(1 AS price, 2 AS quantity),
    STRUCT(10 AS price, 20 AS quantity)
  ] AS v
)
SELECT FLATTEN(v.price) AS all_prices
FROM t;
```
