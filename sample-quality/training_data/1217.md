How do I include a parent field but exclude a specific nested field within it?

--- AGENT ---

This query first includes both year and type fields, then explicitly excludes type.award_name, keeping the type messages but with only their category fields visible.

```googlesql
SELECT FILTER_FIELDS(award_col, +year, +type, -type.award_name) AS filtered_fields
FROM MusicAwards
```
