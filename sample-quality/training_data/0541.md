How do I parse a time string with fractional seconds in a specific precision?

--- AGENT ---

The CAST function with the FORMAT clause parses a string into a time using the specified format. The FF1 format element matches 1 digit of fractional seconds and sets the fractional part of the result, padding or rounding as needed.

```googlesql
SELECT CAST('01:05:07.16' AS TIME FORMAT 'HH24:MI:SS.FF1') AS string_to_date_time
```
