How do I extract different capturing groups and handle optional matches?

--- AGENT ---

REGEXP_EXTRACT with multiple patterns and quantifiers (?, +) shows how it returns the captured text or NULL for optional groups that don't match.

```googlesql
SELECT
  REGEXP_EXTRACT('ab', '.b') AS result_a,
  REGEXP_EXTRACT('ab', '(.)b') AS result_b,
  REGEXP_EXTRACT('xyztb', '(.)+b') AS result_c,
  REGEXP_EXTRACT('ab', '(z)?b') AS result_d
```
