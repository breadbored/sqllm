## Question
I need to retrieve video metadata along with the list of clients that have access to each video through their course enrollments. Please provide a BigQuery query that enriches video records with aggregated client associations.

--- AGENT ---

This query enriches the videos table with client associations derived from course structures. A staging CTE (`stg_videos`) traverses the course hierarchy—joining clients_course_containers through courses and course_lessons to lessons—and aggregates distinct client IDs for each video into an array using ARRAY_AGG. The main query selects all video columns, applies LOWER() normalization to several string fields (key, optimize_error, and various video format identifiers), and left-joins the staging CTE to attach client arrays to each video. For videos with no associated clients (resulting in NULL from the left join), the COALESCE expression defaults the array to [1].

WITH
stg_videos AS (
    SELECT
        l.video_id as video_id,
        ARRAY_AGG(DISTINCT ccc.client_id IGNORE NULLS) as client_ids
    FROM
        ${ref("clients_course_containers")} as ccc
    JOIN
        ${ref("courses")} as c
    ON
        c.container_id = ccc.course_container_id
    JOIN
        ${ref("course_lessons")} as cl
    ON
        cl.course_id = c.id
    JOIN
        ${ref("lessons")} as l
    ON 
        l.id = cl.lesson_id
    GROUP BY
        l.video_id
)

SELECT
    v.id,
    LOWER(key)                 AS key,
    v.created,
    v.updated,
    v.duration,
    v.optimized,
    v.processing,
    LOWER(optimize_error)      AS optimize_error,
    LOWER(hd480)               AS hd480,
    LOWER(hd720)               AS hd720,
    LOWER(hd1080)              AS hd1080,
    LOWER(srt_key)             AS srt_key,
    LOWER(cloudflare_video_id) AS cloudflare_video_id,
    v.creator_admin_id,
    v.updated_admin_id,
    COALESCE(sv.client_ids, [1]) AS client_ids
FROM
  ${ref("videos")} as v
LEFT JOIN
    stg_videos as sv
ON
    sv.video_id = v.id
