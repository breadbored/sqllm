How do I count non-null values in a column using differential privacy with contribution bounds?

--- AGENT ---

This signature of `COUNT` counts non-NULL values in an expression instead of all rows, using `DIFFERENTIAL_PRIVACY` to add noise for privacy. Contribution bounds per group are applied before aggregating across privacy units.

```googlesql
SELECT
  WITH DIFFERENTIAL_PRIVACY
    OPTIONS(epsilon=10, delta=.01, max_groups_contributed=1, privacy_unit_column=id)
    item,
    COUNT(item, contribution_bounds_per_group => (0,100)) times_requested
FROM professors
GROUP BY item;
```
