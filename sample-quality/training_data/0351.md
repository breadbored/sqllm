--- USER ---

Update inventory quantities by adding amounts from a related table using a correlated subquery.

--- AGENT ---

This query updates the Inventory table by adding quantities from the NewArrivals table for matching products, and sets supply_constrained to false for affected rows.

```googlesql
UPDATE dataset.Inventory
SET quantity = quantity +
  (SELECT quantity FROM dataset.NewArrivals
   WHERE Inventory.product = NewArrivals.product),
    supply_constrained = false
WHERE product IN (SELECT product FROM dataset.NewArrivals)
```
