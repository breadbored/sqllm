How do I expand the columns of a table alias using SELECT with dot notation?

--- AGENT ---

`SELECT alias.*` produces one output column for each column in the table or STRUCT referenced by the alias. This is useful when you want to unpack all fields from an aliased table or STRUCT field.

```googlesql
WITH groceries AS
  (SELECT "milk" AS dairy,
   "eggs" AS protein,
   "bread" AS grain)
SELECT g.*
FROM groceries AS g;
```
