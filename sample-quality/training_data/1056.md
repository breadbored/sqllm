How do I find rows where a string matches a pattern?

--- AGENT ---

The LIKE operator checks if a string matches a pattern, where % matches any sequence of characters and _ matches a single character. Returns TRUE for matches, FALSE for non-matches, and NULL for NULL inputs.

```googlesql
SELECT 'apple' LIKE 'a%'
```
