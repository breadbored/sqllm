How do I call a stored procedure and retrieve an output parameter?

--- AGENT ---

Use `CALL` with the procedure name and arguments. For output parameters, declare a variable and pass it as an argument; the procedure populates the variable, which you can then select.

```googlesql
DECLARE retCode INT64;
-- Procedure signature: (IN account_id STRING, OUT retCode INT64)
CALL mySchema.UpdateSomeTables('someAccountId', retCode);
SELECT retCode;
```
