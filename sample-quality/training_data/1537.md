What happens when I try to parse a timestamp format that is missing a required element?

--- AGENT ---

The `PARSE_TIMESTAMP` function produces an error if the format string is missing elements that appear in the timestamp string. All elements present in the input must have corresponding format specifiers.

```googlesql
SELECT PARSE_TIMESTAMP("%a %b %e %I:%M:%S", "Thu Dec 25 07:30:00 2008");
```
