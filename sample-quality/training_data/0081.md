How do I check if at least one value in a group satisfies a condition?

--- AGENT ---

`LOGICAL_OR` returns true if any non-NULL expression in a group is true. This example checks whether at least one value is less than 3:

```googlesql
SELECT LOGICAL_OR(x < 3) AS logical_or FROM UNNEST([1, 2, 4]) AS x;
```
