How do I parse a time string in 12-hour format with minutes, seconds, and AM/PM indicator?

--- AGENT ---

The PARSE_TIME function parses a string into a TIME value using a format string. The format '%I:%M:%S %p' expects a 2-digit hour in 12-hour format, minutes, seconds, and an AM/PM indicator. The function converts the 12-hour time to 24-hour format internally.

```googlesql
SELECT PARSE_TIME('%I:%M:%S %p', '2:23:38 pm') AS parsed_time
```
