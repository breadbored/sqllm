How do I create a table with a primary key column that cannot be null?

--- AGENT ---

Column attributes like `PRIMARY KEY` and `NOT NULL` can be specified inline on a column definition. `PRIMARY KEY` designates the column as the unique identifier for a row, and `NOT NULL` requires a value to be present.

```googlesql
CREATE TABLE books (title STRING, author STRING, isbn INT64 PRIMARY KEY NOT NULL);
```
