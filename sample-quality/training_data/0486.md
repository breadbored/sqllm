--- USER ---

How can I return an error message conditionally when processing data that contains unexpected values?

--- AGENT ---

This query uses a CASE statement with ERROR in the ELSE clause to return a validation message based on the input value, or error if an unexpected value is found. The FROM clause generates test data with three rows.

```googlesql
SELECT
  CASE
    WHEN value = 'foo' THEN 'Value is foo.'
    WHEN value = 'bar' THEN 'Value is bar.'
    ELSE ERROR(CONCAT('Found unexpected value: ', value))
  END AS new_value
FROM (
  SELECT 'foo' AS value UNION ALL
  SELECT 'bar' AS value UNION ALL
  SELECT 'baz' AS value);
```
