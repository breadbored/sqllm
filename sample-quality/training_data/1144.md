How do I split a pipeline into multiple independent subpipelines and return all results?

--- AGENT ---

The `FORK` terminal pipe operator ends the main pipeline and splits it into multiple subpipelines. The input table is computed once and a copy is passed to each subpipeline. All output tables are returned in order.

```googlesql
WITH
  Fruit AS (
    SELECT 'apple' AS item, 100 AS sales
    UNION ALL
    SELECT 'banana' AS item, 150 AS sales
  )
FROM Fruit
|> FORK
    (|> AGGREGATE COUNT(*) AS total_fruits),
    (|> AGGREGATE COUNT(*) AS count_by_item GROUP BY item),
    (|> SELECT item, sales);
```
