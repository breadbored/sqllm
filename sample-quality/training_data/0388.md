Can I use GENERATE_ARRAY to create an array in descending order?

--- AGENT ---

Yes, use a negative step value to generate an array in descending order from the start value down to the end value.

```googlesql
SELECT GENERATE_ARRAY(21, 14, -1) AS countdown;
```
