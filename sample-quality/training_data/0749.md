How do I chain multiple set operations of the same type together?

--- AGENT ---

Multiple instances of the same set operator (e.g., `UNION ALL` with `UNION ALL`) can be chained to combine more than two query results. This example chains three `UNION ALL` operations to combine three groups of person records.

```googlesql
GRAPH FinGraph
MATCH (p:Person)
RETURN p.name, 1 AS group_id
UNION ALL
MATCH (p:Person)
RETURN 2 AS group_id, p.name
UNION ALL
MATCH (p:Person)
RETURN 3 AS group_id, p.name
```
