How do I get the last node in a matched graph path?

--- AGENT ---

The `PATH_LAST` function returns the final node in a graph path as a graph element. This is useful when you need to access properties of the destination node in a path.

```googlesql
GRAPH FinGraph
MATCH p=(src:Account)-[t1:Transfers]->(mid:Account)-[t2:Transfers]->(dst:Account)
LET f = PATH_LAST(p)
RETURN
  LABELS(f) AS labels,
  f.nick_name AS nick_name;
```
