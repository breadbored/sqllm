How do I generate an array of dates using month intervals?

--- AGENT ---

Use `GENERATE_DATE_ARRAY` with `INTERVAL n MONTH` to step by months instead of days. This allows you to generate dates spaced one or more months apart across a date range.

```googlesql
SELECT GENERATE_DATE_ARRAY('2016-01-01',
  '2016-12-31', INTERVAL 2 MONTH) AS example;
```
