What happens when I create an array from columns with different numeric types like INT64 and DOUBLE?

--- AGENT ---

GoogleSQL coerces the columns to a common supertype. In this case, INT64 and DOUBLE both coerce to DOUBLE, so the resulting array has a DOUBLE element type.

```googlesql
SELECT [a, b, c]
FROM
  (SELECT CAST(5 AS INT64) AS a,
          CAST(37 AS DOUBLE) AS b,
          406 AS c);
```
