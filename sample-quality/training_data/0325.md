How do I create a procedure that accepts input parameters and produces output?

--- AGENT ---

The `CREATE PROCEDURE` statement defines a reusable block of SQL statements. Parameter modes (`IN`, `OUT`, `INOUT`) specify whether each parameter is used for input, output, or both. The procedure body contains SQL statements wrapped in a `BEGIN...END` block.

```sql
CREATE PROCEDURE my_procedure(IN x INT64, OUT y STRING)
BEGIN
  IF x > 0 THEN
    SET y = 'positive';
  ELSE
    SET y = 'non-positive';
  END IF;
END;
```
