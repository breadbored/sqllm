--- USER ---

How do I distinguish between a NULL placeholder and an actual NULL value using GROUPING?

--- AGENT ---

When `GROUPING` returns 1 for a column, the NULL in that column is a placeholder representing aggregation, not an actual NULL from the data. This example shows how GROUPING identifies whether NULL represents aggregation or actual data:

```googlesql
WITH
  Products AS (
    SELECT 'shirt' AS product_type, 't-shirt' AS product_name, 3 AS product_count UNION ALL
    SELECT 'shirt', 't-shirt', 8 UNION ALL
    SELECT NULL, 'polo', 25 UNION ALL
    SELECT 'pants', 'jeans', 6
  )
SELECT
  product_type,
  product_name,
  SUM(product_count) AS product_sum,
  GROUPING(product_type) AS product_type_is_aggregated
FROM Products
GROUP BY GROUPING SETS(product_type, product_name)
ORDER BY product_name;
```
