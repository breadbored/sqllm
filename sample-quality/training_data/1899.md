Provide an example BigQuery SQL query that enriches user assessment records with their associated client IDs, combining a default client ID with any external client ID they may have.

--- AGENT ---

This query enriches user assessment records with client ID information. It first builds a mapping of users to their client IDs, creating an array containing the default client ID (1) and each user's external client ID if available. It then left-joins this mapping to the assessment records, returning all assessment data with the appropriate client_ids array—either the mapped array for users with external client IDs, or [1] as a fallback for users without one. The status field is normalized to lowercase.

```sql
WITH
user_clients AS (
  SELECT
    u.id as user_id,
    [1, u.external_client_id] as client_ids
  FROM
    ${ref("users")} as u
  WHERE
    u.external_client_id IS NOT NULL
)
SELECT
  _utb.id,
  _utb.user_id,
  _utb.course_id,
  _utb.assessment_id,
  LOWER(_utb.status) AS status,
  _utb.completed,
  _utb.score,
  _utb.passed,
  _utb.created,
  _utb.updated,
  COALESCE(ucl.client_ids, [1]) as client_ids
FROM
  ${ref("user_course_assessments")} as _utb
LEFT JOIN
  user_clients as ucl
ON
  ucl.user_id = _utb.user_id
```
