How do I construct a protocol buffer with nested message fields and arrays of sub-messages?

--- AGENT ---

Nested messages are set using nested braces, and array fields use square brackets containing multiple struct constructors for each element.

```googlesql
SELECT
  NEW googlesql.examples.music.Album {
    album_name: 'New Moon'
    singer {
      nationality: 'Canadian'
      residence: [ { city: 'Victoria' }, { city: 'Toronto' } ]
    }
    song: ['Sandstorm', 'Wait']
  }
```
