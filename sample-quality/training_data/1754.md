--- AGENT ---

This query transforms raw event records from the `events_fill` table into a finalized, enriched events dataset. The transformation performs several operations:

1. **Safe Type Casting**: Converts string columns to their target types (INT64, BOOL, DATETIME) using SAFE_CAST to handle conversion failures gracefully.

2. **JSON Field Extraction**: Extracts nested data from three JSON objects:
   - `user_agent_json`: Parses browser, device, engine, and OS information
   - `geo`: Parses geographic data (city, country, continent, coordinates, timezone, postal code, metro code, region)
   - `signals`: Extracts browser fingerprinting signals (canvas, fonts, screen dimensions, WebGL properties, storage availability, etc.)

3. **Window Functions**: Computes time deltas and sequential relationships:
   - Uses `LAG()` to retrieve the previous event's app value within each session and subsession
   - Uses `LEAD()` to calculate minutes and seconds until the next event in the session

4. **Data Enrichment**: Applies COALESCE logic to prioritize geographic data from Datadog sources over JSON-embedded values, and establishes fallback app values.

5. **Column Aliasing**: Renames and reorganizes columns for downstream consumption, including computed fields like `minutes_until_next_event`, `seconds_from_last_event`, and various session/subsession metrics.
