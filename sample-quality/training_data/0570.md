How do I break down a complex geography into its individual component parts?

--- AGENT ---

`ST_DUMP` decomposes a geography into an array of simple geometries (points, linestrings, or polygons), with an optional dimension filter to return only components of a specific type.

```googlesql
WITH example AS (
  SELECT ST_GEOGFROMTEXT('POINT(0 0)') AS geography
  UNION ALL
  SELECT ST_GEOGFROMTEXT('MULTIPOINT(0 0, 1 1)') AS geography
  UNION ALL
  SELECT ST_GEOGFROMTEXT('GEOMETRYCOLLECTION(POINT(0 0), LINESTRING(1 2, 2 1))'))
SELECT
  geography AS original_geography,
  ST_DUMP(geography) AS dumped_geographies
FROM example
```
