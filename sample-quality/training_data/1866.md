## Question
Show me an example BigQuery query that demonstrates how to use subqueries within array aggregation and simple column transformations.

--- AGENT ---

This query selects all rows from the `doc_topics` table, transforming the `name` column to lowercase and wrapping all client IDs from the `clients` table into a single array column. The ARRAY subquery collects every `id` value from `clients` without filtering, producing a full cross-product relationship where each doc topic row contains the complete set of client IDs as an array. The comment indicates this table is deprecated and likely empty.

Note: The trailing comma after the `client_ids` array definition suggests this may be incomplete or template code.

```sql
-- EMPTY TABLE - DEPRECATED

SELECT
  dt.id,
  LOWER(dt.name) AS name,
  dt.created,
  dt.updated,
  ARRAY(SELECT id FROM ${ref("clients")}) as client_ids,
FROM
  ${ref("doc_topics")} as dt
```
