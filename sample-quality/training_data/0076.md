How do I count how many values satisfy different conditions in a single query?

--- AGENT ---

`COUNTIF` returns the count of rows where an expression is true. This example counts negative and positive values separately:

```googlesql
SELECT COUNTIF(x<0) AS num_negative, COUNTIF(x>0) AS num_positive
FROM UNNEST([5, -2, 3, 6, -10, -7, 4, 0]) AS x;
```
