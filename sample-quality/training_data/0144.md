What does ARRAY_SLICE return when the start offset exceeds the array length?

--- AGENT ---

Use `ARRAY_SLICE` with a positive start offset beyond the array bounds. Since there are no elements at or after that position, the function returns an empty array.

```googlesql
SELECT ARRAY_SLICE(['a', 'b', 'c', 'd', 'e'], 5, 30) AS result
```
