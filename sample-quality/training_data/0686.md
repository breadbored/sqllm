How can I apply a path mode to a quantified edge pattern?

--- AGENT ---

Specify the path mode before the path pattern that includes the quantified edges. The mode applies to the entire expanded pattern created by the quantifier.

```googlesql
GRAPH FinGraph
MATCH TRAIL (a1:Account)-[t1:Transfers]->{4}(a5:Account)
RETURN COUNT(1) as num_paths
```
