How do I flatten a single field from nested array data using the FLATTEN operator?

--- AGENT ---

Use the `FLATTEN()` operator with array element field access (dot notation) to extract and flatten a specific field from nested array data. This creates a single flat array containing all values from that field across all nested elements.

```googlesql
SELECT FLATTEN(items.color) AS colors
FROM ItemsTable

/*----------------------+
 | colors               |
 +----------------------+
 | [red, green, orange] |
 +----------------------*/
```
