How can I get the S2 cell ID that covers a specific geographic point, with support for different precision levels?

--- AGENT ---

`S2_CELLIDFROMPOINT` returns the S2 cell ID covering a point geography, optionally at a specified precision level. The SAFE prefix handles non-point inputs by returning NULL instead of erroring.

```googlesql
WITH data AS (
  SELECT 1 AS id, ST_GEOGPOINT(-122, 47) AS geo
  UNION ALL
  -- empty geography isn't supported
  SELECT 2 AS id, ST_GEOGFROMTEXT('POINT EMPTY') AS geo
  UNION ALL
  -- only points are supported
  SELECT 3 AS id, ST_GEOGFROMTEXT('LINESTRING(1 2, 3 4)') AS geo
)
SELECT id,
       SAFE.S2_CELLIDFROMPOINT(geo) cell30,
       SAFE.S2_CELLIDFROMPOINT(geo, level => 10) cell10
FROM data;
```
