How do I use a label to mark a block so I can exit it early?

--- AGENT ---

Add a label before `BEGIN` and use `BREAK label_name` to exit the block immediately. This is useful for early termination from nested structures.

```googlesql
label_1: BEGIN
  SELECT 1;
  BREAK label_1;
  SELECT 2; -- Unreached
END;
```
