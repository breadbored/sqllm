How do I get an approximate count of distinct values in a dataset?

--- AGENT ---

APPROX_COUNT_DISTINCT returns an approximate count of distinct values using significantly less memory than the exact COUNT(DISTINCT) function, though with some statistical uncertainty. This is useful for large datasets where an exact count would be too memory-intensive to compute.

```googlesql
SELECT APPROX_COUNT_DISTINCT(x) as approx_distinct
FROM UNNEST([0, 1, 1, 2, 3, 5]) as x;
```
