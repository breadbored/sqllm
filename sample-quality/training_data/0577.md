How do I convert a hex-encoded WKB (Well-Known Binary) string to a point geography?

--- AGENT ---

`ST_GEOGFROM` converts hexadecimal-encoded WKB format to a GEOGRAPHY value, allowing binary geometry representations to be parsed.

```googlesql
SELECT ST_GEOGFROM(FROM_HEX('010100000000000000000000400000000000001040')) AS WKB_format;
```
