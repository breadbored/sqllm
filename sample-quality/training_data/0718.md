How do I match nodes that have one of several possible labels?

--- AGENT ---

The `MATCH` statement supports label alternation using the pipe operator `|`, allowing you to match nodes with any of the specified labels. This example matches all nodes that are either `Person` or `Account` nodes.

```googlesql
GRAPH FinGraph
MATCH (n:Person|Account)
RETURN LABELS(n) AS label, n.id
```
