How do I cast an array from one element type to another?

--- AGENT ---

Use `CAST(array AS ARRAY<new_type>)` to convert an array's element type, provided the source type is castable to the target type. This operation preserves array values while changing their data type.

```googlesql
SELECT CAST(int_array AS ARRAY<DOUBLE>) AS double_array
FROM (SELECT ARRAY<INT32>[1, 2, 3] AS int_array);

/*--------------+
 | double_array |
 +--------------+
 | [1, 2, 3]    |
 +--------------*/
```
