How do I get the first element from an array?

--- AGENT ---

Use `ARRAY_FIRST` to extract the first element of an array. The function returns NULL if the input array is NULL, and produces an error if the array is empty.

```googlesql
SELECT ARRAY_FIRST(['a','b','c','d']) as first_element
```
