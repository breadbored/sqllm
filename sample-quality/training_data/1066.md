How do I create a protocol buffer using a parenthesized argument list?

--- AGENT ---

The NEW operator with parenthesized syntax creates a protocol buffer by passing positional arguments, optionally with AS aliases to map arguments to specific fields.

```googlesql
SELECT
  key,
  name,
  NEW googlesql.examples.music.Chart(key AS rank, name AS chart_name)
FROM
  (SELECT 1 AS key, "2" AS name)
```
