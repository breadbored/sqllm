How do I add a new key-value pair to a protocol buffer map field?

--- AGENT ---

This query uses the `PROTO_MODIFY_MAP` function to add a new key-value pair to a map field. The function takes the map field, a key ('B'), and a value (11) as arguments. The query shows that both the existing key 'A' and the newly added key 'B' appear in the result.

```googlesql
SELECT
  PROTO_MODIFY_MAP(m.purchased, 'B', 11) AS result_map
FROM
  (SELECT AS VALUE CAST("purchased { key: 'A' value: 22 }" AS Item)) AS m;
```
