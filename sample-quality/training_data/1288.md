How do I check if one date range is completely contained within another date range?

--- AGENT ---

RANGE_CONTAINS with two range arguments checks whether the inner range is fully contained in the outer range, returning TRUE if every value in the inner range exists in the outer range. This query checks if the range [2022-04-01, 2022-07-01) falls entirely within [2022-01-01, 2023-01-01).

```googlesql
SELECT RANGE_CONTAINS(
  RANGE<DATE> '[2022-01-01, 2023-01-01)',
  RANGE<DATE> '[2022-04-01, 2022-07-01)') AS results;
```
