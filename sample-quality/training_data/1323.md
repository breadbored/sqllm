How does sample covariance differ from population covariance when one pair contains a NULL?

--- AGENT ---

The `COVAR_SAMP` function ignores pairs with NULL values just like `COVAR_POP`, but uses the sample (unbiased) formula instead of the population formula. Here, the pair (NULL, 3.0) is skipped, leaving four valid pairs for calculation.

```googlesql
SELECT COVAR_SAMP(y, x) AS results
FROM
  UNNEST(
    [
      STRUCT(1.0 AS y, 1.0 AS x),
      (2.0, 6.0),
      (9.0, 3.0),
      (2.0, 6.0),
      (NULL, 3.0)])
```
