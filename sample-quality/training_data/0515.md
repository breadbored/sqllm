How do I format a timestamp to show the abbreviated month, year, and RFC 3339 time zone offset?

--- AGENT ---

The FORMAT_TIMESTAMP function formats a TIMESTAMP value using format elements. The `%b` element outputs the abbreviated month name, `%Y` outputs the year, and `%Ez` outputs the time zone offset in RFC 3339 format (±HH:MM).

```googlesql
SELECT FORMAT_TIMESTAMP("%b %Y %Ez", TIMESTAMP "2008-12-25 15:30:00+00")
  AS formatted;
```
