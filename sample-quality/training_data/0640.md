--- USER ---

How do you find transfers where node 'b' is the account receiving the transfer?

--- AGENT ---

The IS DESTINATION predicate filters the matched transfer edges to return only those where the specified node is the destination of the edge.

```googlesql
GRAPH FinGraph
MATCH (a:Account)-[transfer:Transfers]-(b:Account)
WHERE b IS DESTINATION of transfer
RETURN a.id AS a_id, b.id AS b_id
```
