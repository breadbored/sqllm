--- USER ---

Provide a BigQuery SQL example that aggregates raw event data into session-level engagement metrics, enriched with document metadata and associated clients.

--- AGENT ---

This query constructs a session-level analytics view that combines behavioral engagement metrics with document and client information.

The first CTE, `doc_events`, aggregates raw event records by session and sub-session. It extracts the earliest document ID, landing path, and language (using MIN_BY ordered by creation time); calculates time on page as the duration between the first and last events in that session; tracks scroll behavior (maximum depth reached, number of scroll events); counts non-generic clicks on the landing page; and normalizes UTM parameters (converting underscores, plus signs, URL-encoded spaces, and dashes to spaces, then lowercasing) extracted from the earliest URL in the session.

The second CTE, `subsession_data`, analyzes each sub-session's engagement. It counts distinct pages viewed and total clicks, calculates session duration, and marks a session as a bounce if it is under 30 seconds, has only one page view, and zero clicks; otherwise it is marked as engaged. It also flags whether the landing URL originated from app.dataprotocol.com.

The third CTE, `went_to_app`, identifies whether each sub-session started in an "app" value or visited an "app" value at any point during the session.

The final SELECT joins `doc_events` against four tables: `stg_documents` (matching on landing path to get document name and type), `stg_documents_clients` and `stg_clients` (to retrieve clients associated with the document), and includes special conditional join logic for collection paths to associate them with the Meta Horizon client. It applies business logic to display collection names and marks collection paths as type 'collection'. It aggregates client IDs and names into arrays, and the final GROUP BY produces one row per session-subsession combination with engagement metrics, referral source, UTM attribution, and client associations.

```sql
WITH
  doc_events AS (
  SELECT
    e.session_id,
    e.sub_session_id,
    MAX(e.user_id) AS user_id,
    MAX(e.current_user_id) AS current_user_id,
    MIN_BY(e.derived_doc_id, e.created) AS doc_id,
    REGEXP_REPLACE(MIN_BY(e.url_path, e.created), r'/([a-z]{2})$', '') AS landing_path,
    REGEXP_EXTRACT(MIN_BY(e.url_path, e.created), r'/([a-z]{2})$') AS lang,
    DATETIME_DIFF(MAX(e.created), MIN(e.created), SECOND) AS time_on_page,
    MAX(e.scroll_value) AS max_scroll_depth,
    MAX(e.scroll_num) AS scroll_events_count,
    COUNTIF(e.type = "click"
      AND e.name != "any_click") AS landing_page_click_count,
    MIN_BY(e.derived_referrer, e.created) AS referrer,
    LOWER(REGEXP_REPLACE(REGEXP_REPLACE(REGEXP_REPLACE(REGEXP_REPLACE( REGEXP_EXTRACT(MIN_BY(url, created), r'utm_source=([^&]*)'), r'_', ' '), r'\+', ' '), r'%20', ' '), r'-', ' ')) AS utm_source,
    LOWER(REGEXP_REPLACE(REGEXP_REPLACE(REGEXP_REPLACE(REGEXP_REPLACE( REGEXP_EXTRACT(MIN_BY(url, created), r'utm_medium=([^&]*)'), r'_', ' '), r'\+', ' '), r'%20', ' '), r'-', ' ')) AS utm_medium,
    LOWER(REGEXP_REPLACE(REGEXP_REPLACE(REGEXP_REPLACE(REGEXP_REPLACE( REGEXP_EXTRACT(MIN_BY(url, created), r'utm_campaign=([^&]*)'), r'_', ' '), r'\+', ' '), r'%20', ' '), r'-', ' ')) AS utm_campaign,
    LOWER(REGEXP_REPLACE(REGEXP_REPLACE(REGEXP_REPLACE(REGEXP_REPLACE( REGEXP_EXTRACT(MIN_BY(url, created), r'utm_content=([^&]*)'), r'_', ' '), r'\+', ' '), r'%20', ' '), r'-', ' ')) AS utm_content,
    LOWER(REGEXP_REPLACE(REGEXP_REPLACE(REGEXP_REPLACE(REGEXP_REPLACE( REGEXP_EXTRACT(MIN_BY(url, created), r'utm_term=([^&]*)'), r'_', ' '), r'\+', ' '), r'%20', ' '), r'-', ' ')) AS utm_term,
    MAX(e.landed_from) as landed_from,
    MIN_BY(e.geo_country, e.created) AS geo_country,
    MIN_BY(e.geo_city, e.created) AS geo_city,
    CONCAT(MIN_BY(e.geo_lat, e.created), ",", MIN_BY(e.geo_long, e.created)) AS lat_long,
    MIN(e.created) AS created
  FROM
    ${ref("stg_events_docs_speedy")} e
  GROUP BY
    e.session_id,
    e.sub_session_id),
  subsession_data AS (
  SELECT
    sub_session_id,
    COUNT(DISTINCT url_path) AS page_views,
    DATETIME_DIFF(MAX(created), MIN(created), SECOND) AS session_duration,
    COUNTIF(type = "click") AS session_click_count,
    CASE
      WHEN DATETIME_DIFF(MAX(created), MIN(created), SECOND) < 30 AND COUNT(DISTINCT url_path) = 1 AND COUNTIF(type = "click") = 0 THEN FALSE
    -- Bounce criteria
      ELSE TRUE
    -- Engaged if not a bounce
  END
    AS engaged,
    CASE
      WHEN MIN_BY(landing_url, created) LIKE '%app.dataprotocol.com%' THEN TRUE
      ELSE FALSE
  END
    AS from_app
  FROM
    ${ref("stg_events_docs_speedy")}
  GROUP BY
    sub_session_id),
  went_to_app AS (
  SELECT
    e.sub_session_id,
    MIN_BY(e.app, e.created) = 'app' AS started_at_app,
    MIN(e.app) = 'app' AS went_to_app
  FROM
    ${ref("stg_events_speedy")} AS e
  GROUP BY
    e.sub_session_id)
SELECT
  e.session_id,
  e.sub_session_id,
  e.user_id,
  e.current_user_id,
  d.id AS doc_id,
  CASE
    WHEN e.landing_path = '/collections/meta-quest' OR e.landing_path = '/collections/meta-horizon/' THEN 'Meta Horizon Collection'
    WHEN e.landing_path LIKE '/collections/%' THEN INITCAP( REPLACE( REGEXP_EXTRACT(e.landing_path, r'/collections/[^/]+/([^/]+)'), '-', ' ' ) )
    ELSE d.name
END
  AS doc_name,
  CASE
    WHEN e.landing_path LIKE '/collections/%' THEN 'collection'
    ELSE d.type
END
  AS doc_type,
  e.landing_path,
  e.lang,
  e.time_on_page,
  e.landing_page_click_count,
  e.max_scroll_depth,
  e.scroll_events_count,
  COALESCE(MAX(wta.started_at_app), FALSE) AS started_at_app,
  COALESCE(MAX(wta.went_to_app), FALSE) AS went_to_app,
  sd.page_views,
  sd.session_duration,
  sd.session_click_count,
  sd.engaged,
  e.referrer,
  e.utm_source,
  e.utm_medium,
  e.utm_campaign,
  e.utm_content,
  e.landed_from,
  e.created,
  e.geo_country AS country,
  e.geo_city AS city,
  e.lat_long,
  ARRAY_AGG(DISTINCT clients.id IGNORE NULLS) AS client_id,
  ARRAY_AGG(DISTINCT clients.name IGNORE NULLS) AS client_name
FROM
  doc_events e
LEFT JOIN
  ${ref("stg_documents")} d
ON
  e.landing_path = d.path
LEFT JOIN
  ${ref("stg_documents_clients")} dc
ON
  dc.doc_id = d.id
LEFT JOIN
  ${ref("stg_clients")} clients
ON
  clients.id = dc.client_id
  OR ( e.landing_path LIKE '/collections/%'
    AND ( ( e.landing_path LIKE '/collections/meta-quest/%'
        AND clients.name = 'Meta Horizon' )
      OR ( e.landing_path LIKE '/collections/meta-horizon/%'
        AND clients.name = 'Meta Horizon' ) ) )
LEFT JOIN
  subsession_data AS sd
ON
  sd.sub_session_id = e.sub_session_id
LEFT JOIN
  went_to_app AS wta
ON
  wta.sub_session_id = e.sub_session_id
GROUP BY
  e.session_id,
  e.sub_session_id,
  e.user_id,
  e.current_user_id,
  d.id,
  doc_name,
  doc_type,
  e.landing_path,
  e.lang,
  e.time_on_page,
  e.landing_page_click_count,
  e.max_scroll_depth,
  e.scroll_events_count,
  sd.from_app,
  sd.page_views,
  sd.session_duration,
  sd.session_click_count,
  sd.engaged,
  e.referrer,
  e.utm_source,
  e.utm_medium,
  e.utm_campaign,
  e.utm_content,
  e.landed_from,
  e.created,
  country,
  city,
  lat_long
```
