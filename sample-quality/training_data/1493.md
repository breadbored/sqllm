--- USER ---

I need to check whether the username 'corba' exists in the Players table.

--- AGENT ---

An IN subquery tests whether a value is found in the set of rows returned by the subquery. It returns TRUE if a match is found, FALSE if no rows are returned.

```googlesql
SELECT
  'corba' IN (SELECT username FROM Players) AS result;
```
