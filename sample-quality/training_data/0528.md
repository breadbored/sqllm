How do I extract the second component from a time value?

--- AGENT ---

The CAST function with the FORMAT clause converts a time to a string. The SS format element outputs the second (00-59) as a 2-digit number.

```googlesql
SELECT CAST(TIME '21:30:25.16' AS STRING FORMAT 'SS') AS date_time_to_string;
```
