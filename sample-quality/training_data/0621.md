How do I get all the property names associated with a graph element?

--- AGENT ---

The `PROPERTY_NAMES` function returns an array of property names for a graph element while preserving the original case of each name. This allows you to introspect what properties are available on a node or edge.

```googlesql
GRAPH FinGraph
MATCH (n:Person|Account)
RETURN PROPERTY_NAMES(n) AS property_names, n.id
```
