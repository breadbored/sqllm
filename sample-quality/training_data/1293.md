How do I get the upper bound from a date range that has no upper limit?

--- AGENT ---

RANGE_END returns NULL when the upper bound of the range is unbounded. This query attempts to extract the end date from a range with no upper boundary, which results in NULL.

```googlesql
SELECT RANGE_END(RANGE<DATE> '[2022-12-01, UNBOUNDED)') AS results;
```
