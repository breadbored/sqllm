--- USER ---

Show me how to calculate the maximum time spent on each page within a session, grouped by subsession and page visit.

--- AGENT ---

This query calculates the maximum time spent on each unique page visit within a subsession. It groups events by three dimensions—subsession ID, page visit identifier, and URL path—and then aggregates to find the maximum value of `seconds_since_url_path` for each group. The result is a deduplicated set of page visits with their peak time-on-page metric, structured to be backfilled to earlier events in the same subsession.

```sql
SELECT
  e.sub_session_id AS sub_session_id,
  e.page_visit AS page_visit,
  e.url_path AS url_path,
  MAX(e.seconds_since_url_path) AS time_on_page
FROM
  ${ref("events_numbered")} AS e
GROUP BY
  e.sub_session_id,
  e.page_visit,
  e.url_path
```
