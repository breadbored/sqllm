How do I generate network masks for different IP address byte lengths and prefix lengths?

--- AGENT ---

This query demonstrates NET.IP_NET_MASK creating byte sequences where the first prefix_length bits are set to 1 and remaining bits are 0, supporting both IPv4 (4 bytes) and IPv6 (16 bytes) mask generation. The function produces masks that can be used for IP filtering and subnet operations.

```googlesql
SELECT x, y, FORMAT("%T", NET.IP_NET_MASK(x, y)) AS ip_net_mask
FROM UNNEST([
  STRUCT(4 as x, 0 as y),
  (4, 20),
  (4, 32),
  (16, 0),
  (16, 1),
  (16, 128)
]);
```
