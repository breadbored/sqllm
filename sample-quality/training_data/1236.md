How do I remove duplicate rows from a query result?

--- AGENT ---

`SELECT DISTINCT` eliminates duplicate rows and returns only unique rows. It cannot be used with PROTO or GRAPH_ELEMENT column types.

```googlesql
WITH PlayerStats AS (
  SELECT ['Coolidge', 'Adams'] as Name, 3 as PointsScored UNION ALL
  SELECT ['Adams', 'Buchanan'], 0 UNION ALL
  SELECT ['Coolidge', 'Adams'], 1 UNION ALL
  SELECT ['Kiran', 'Noam'], 1)
SELECT DISTINCT Name
FROM PlayerStats;
```
