--- USER ---

Provide an example BigQuery SQL query that summarizes daily page-level engagement metrics by counting sessions, measuring average time spent, and tracking action volume.

--- AGENT ---

This query aggregates session-level engagement data to daily page summaries in two stages.

The CTE `session_durations` calculates per-session metrics:
- It groups events by URL path, session ID, and calendar day
- For each session, it computes duration as the time difference (in seconds) between the maximum and minimum event timestamps
- It counts the total number of actions recorded in that session
- Only sessions with non-null URL paths are included

The outer query then aggregates these session-level metrics to daily page-level summaries:
- It groups by URL path and day
- It counts the number of distinct sessions per page per day
- It calculates the average session duration across all sessions on that page, rounded to the nearest second
- It sums the total action count across all sessions on that page per day
- Results are ordered by session count in descending order

The output is one row per (url_path, day) pair, showing how many sessions visited each page that day, how long they stayed on average, and how many actions were performed in total.

```sql
WITH session_durations AS (
  SELECT
    de.url_path,
    de.session_id,
    datetime_trunc(cast(timestamp AS timestamp), DAY) AS day,
    timestamp_diff(
      timestamp(max(de.timestamp)),
      timestamp(min(de.timestamp)),
      SECOND
    )                                                 AS session_duration,
    count(de.action)                                  AS action_count
  FROM ${ref("stg_dd_docs_events")} AS de
  WHERE url_path IS NOT NULL
  GROUP BY de.url_path, de.session_id, datetime_trunc(cast(timestamp AS timestamp), DAY)
)

SELECT
  url_path,
  day,
  count(DISTINCT session_id)   AS sessions,
  round(avg(session_duration)) AS avg_session_duration,
  sum(action_count)            AS total_action_count
FROM session_durations
GROUP BY url_path, day
ORDER BY sessions DESC
```
