How do I create a loop that executes until a `BREAK` statement is encountered?

--- AGENT ---

Use `LOOP` to create an indefinite loop. The loop body executes repeatedly until you explicitly issue a `BREAK` or `LEAVE` statement.

```googlesql
DECLARE x INT64 DEFAULT 0;
LOOP
  SET x = x + 1;
  IF x >= 10 THEN
    LEAVE;
  END IF;
END LOOP;
SELECT x;
```
