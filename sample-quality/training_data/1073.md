How do I join two tables in pipe syntax and filter the joined result?

--- AGENT ---

In pipe syntax, you can use the `FROM` clause to set up joins, then apply pipe operators to filter and select. The `JOIN` happens in the `FROM` clause, and the `WHERE` and `SELECT` operators apply to the joined rows.

```googlesql
-- Join tables in the FROM clause and then apply pipe operators.
FROM
  Produce AS p1
  JOIN Produce AS p2
    USING (item)
|> WHERE item = 'bananas'
|> SELECT p1.item, p2.sales;
```
