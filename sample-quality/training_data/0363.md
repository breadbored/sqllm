--- USER ---

Perform multiple nested operations on the same proto field using DELETE, UPDATE, and INSERT in the correct sequence.

--- AGENT ---

This query performs three ordered nested operations on the Residence repeated field: deleting residences in Seattle, updating the end year for residences in Eugene, and inserting a new song.

```googlesql
UPDATE Singers s
SET
    (DELETE FROM s.SingerInfo.Residence r WHERE r.City = 'Seattle'),
    (UPDATE s.SingerInfo.Residence r SET r.end_year = 2015 WHERE r.City = 'Eugene'),
    (INSERT s.Albums.Song VALUES ("songtitle: 'The Second Best Song'"))
WHERE SingerId = 3 AND s.Albums.title = 'Go! Go! Go!';
```
