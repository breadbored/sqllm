How do I create a buffer around a point with a specified tolerance instead of specifying the number of segments?

--- AGENT ---

`ST_BUFFERWITHTOLERANCE` creates a buffer geography like `ST_BUFFER`, but allows you to specify a tolerance value in meters instead of the number of segments, controlling how much the approximated buffer can deviate from the ideal radius.

```googlesql
SELECT
  -- tolerance_meters=25, or 25% of the buffer radius.
  ST_NumPoints(ST_BUFFERWITHTOLERANCE(ST_GEOGFROMTEXT('POINT(1 2)'), 100, 25)) AS five_sides,
  -- tolerance_meters=1, or 1% of the buffer radius.
  st_NumPoints(ST_BUFFERWITHTOLERANCE(ST_GEOGFROMTEXT('POINT(100 2)'), 100, 1)) AS twenty_four_sides;
```
