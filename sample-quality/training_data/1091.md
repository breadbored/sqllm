How do I get distinct rows from a table while preserving the table alias for later reference?

--- AGENT ---

The `DISTINCT` pipe operator removes duplicate rows while preserving table aliases. This is different from `SELECT DISTINCT *`, which removes aliases.

```googlesql
(
  SELECT 'apples' AS item, 2 AS sales
  UNION ALL
  SELECT 'bananas' AS item, 5 AS sales
  UNION ALL
  SELECT 'bananas' AS item, 5 AS sales
  UNION ALL
  SELECT 'carrots' AS item, 8 AS sales
)
|> DISTINCT
|> WHERE sales >= 3;
```
