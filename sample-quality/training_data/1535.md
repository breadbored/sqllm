What happens if I parse a timestamp string where the format elements are in the correct order?

--- AGENT ---

The `PARSE_TIMESTAMP` function converts a formatted string into a timestamp by matching format elements in order. The string must have elements in the same positions as the format string for successful parsing.

```googlesql
SELECT PARSE_TIMESTAMP("%a %b %e %I:%M:%S %Y", "Thu Dec 25 07:30:00 2008");
```
