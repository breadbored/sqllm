## Question
Retrieve user course lesson progress records with their associated client information, excluding extended idle sessions that occurred after January 2024.

--- AGENT ---

This query retrieves lesson progress records enriched with client IDs, while filtering out certain types of idle sessions.

The logic breaks down as follows:

1. **User Clients CTE**: Builds a mapping of users to an array of client IDs. For each user with a non-null `external_client_id`, it creates an array `[1, external_client_id]`.

2. **Main SELECT**: Retrieves all columns from the `user_course_lesson_progress` table and left joins it with the user clients mapping to attach client IDs. If a user has no client mapping, it defaults the array to `[1]` using COALESCE.

3. **Filtering**: Excludes records using a NOT IN subquery that identifies and removes records meeting specific conditions:
   - Records created after 2024-01-06
   - With `interval_seconds > 0` (indicating some session duration)
   - Where a QUALIFY condition ranks rows within partitions of (user_id, course_id, lesson_id, progress_seconds) and filters based on whether the row number exceeds `10 * 60 / interval_seconds` — effectively targeting sessions that exceed approximately 10 minutes of cumulative interval time.

The result is a deduplicated set of progress records with associated client arrays and extended idle sessions filtered out.

```sql
WITH
user_clients AS (
  SELECT
    u.id as user_id,
    [1, u.external_client_id] as client_ids
  FROM
    ${ref("users")} as u
  WHERE
    u.external_client_id IS NOT NULL
)
SELECT
  _utb.id,
  _utb.progress_seconds,
  _utb.user_id,
  _utb.course_id,
  _utb.lesson_id,
  _utb.created,
  _utb.updated,
  _utb.interval_seconds,
  COALESCE(ucl.client_ids, [1]) as client_ids
FROM
  ${ref("user_course_lesson_progress")} as _utb
LEFT JOIN
  user_clients as ucl
ON
  ucl.user_id = _utb.user_id
WHERE
  -- Cutoff any >10 minute idles after Jan 6th, 2025
  _utb.id NOT IN (
    SELECT
      _uclp.id as id
    FROM
      ${ref("user_course_lesson_progress")} as _uclp
    WHERE
      _uclp.interval_seconds > 0
    AND
      _uclp.created > '2024-01-06'
    QUALIFY
      ROW_NUMBER() OVER (PARTITION BY _uclp.user_id, _uclp.course_id, _uclp.lesson_id, _uclp.progress_seconds ORDER BY _uclp.user_id, _uclp.course_id, _uclp.lesson_id, _uclp.progress_seconds) >= 10 * 60 / _uclp.interval_seconds -- 10 minutes, div by interval
  )
```
