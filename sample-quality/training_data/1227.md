How do I replace a repeated field in a protocol buffer with an array of new values?

--- AGENT ---

This query replaces the repeated reviews field with an array of two new review strings, demonstrating that repeated proto fields can be replaced with array values.

```googlesql
SELECT REPLACE_FIELDS(
  NEW Book("The Hummingbird" AS title,
    NEW BookDetails(10 AS chapters) AS details),
  ["A good read!", "Highly recommended."] AS reviews)
AS proto;
```
