How do I introduce a temporary name for a table or column in a query?

--- AGENT ---

An explicit alias assigns a temporary name to a table or expression using the `[AS] alias` syntax. The AS keyword is optional. Aliases are useful for brevity and clarity, especially in joins and nested queries.

```googlesql
SELECT s.FirstName, s2.SongName
FROM Singers AS s, (SELECT * FROM Songs) AS s2;
```
