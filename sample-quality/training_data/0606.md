How do I access a property value from a matched node in a property graph?

--- AGENT ---

This query matches all nodes labeled `Person` in the `FinGraph` property graph and returns the value of their `name` property. Graph element properties are accessed using dot notation, similar to accessing struct fields in GoogleSQL.

```googlesql
GRAPH FinGraph
MATCH (n:Person)
RETURN n.name
```
