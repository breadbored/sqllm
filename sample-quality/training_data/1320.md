How does population covariance handle a single NULL value among multiple pairs?

--- AGENT ---

The `COVAR_POP` function ignores any pair containing a NULL value. In this example, the pair (NULL, 3.0) is skipped, and covariance is computed from the remaining four valid pairs.

```googlesql
SELECT COVAR_POP(y, x) AS results
FROM
  UNNEST(
    [
      STRUCT(1.0 AS y, 1.0 AS x),
      (2.0, 6.0),
      (9.0, 3.0),
      (2.0, 6.0),
      (NULL, 3.0)])
```
