--- USER ---

I need to retrieve lesson container records enriched with the list of clients that have access to resources within each lesson.

--- AGENT ---

This query retrieves lesson container records and associates them with the set of distinct client IDs that have access to lesson resources.

The CTE `stg_lesson_container` performs a multi-table join traversing the relationship chain from clients_course_containers through courses, course_lessons, and lessons to lesson_resources. It groups by lesson_id and collects all unique (non-null) client IDs into an array using `ARRAY_AGG(DISTINCT ... IGNORE NULLS)`.

The main query selects from the lesson_container table and left joins against this aggregated view on lesson_id. It returns the lesson container's metadata (id, creator_admin_id, created, updated) along with the array of associated client IDs. For lesson containers with no matching clients, `COALESCE` returns a default array of [1].

```sql
WITH
stg_lesson_container as (
    SELECT
        l.id as lesson_id,
        ARRAY_AGG(DISTINCT ccc.client_id IGNORE NULLS) as client_ids
    FROM
        ${ref("clients_course_containers")} as ccc
    JOIN
        ${ref("courses")} as c
    ON
        c.container_id = ccc.course_container_id
    JOIN
        ${ref("course_lessons")} as cl
    ON
        cl.course_id = c.id
    JOIN
        ${ref("lessons")} as l
    ON
        l.id = cl.lesson_id
    JOIN
        ${ref("lesson_resources")} as lr
    ON
        l.id = lr.lesson_id
    GROUP BY
        l.id
)


SELECT
    lc.id,
    lc.creator_admin_id,
    lc.created,
    lc.updated,
    COALESCE(slc.client_ids, [1]) AS client_ids
FROM
    ${ref("lesson_container")} as lc
LEFT JOIN
    stg_lesson_container as slc
ON
    slc.lesson_id = lc.id
```
