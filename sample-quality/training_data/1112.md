How do I perform recursive queries on employee hierarchies using the pipe RECURSIVE UNION operator?

--- AGENT ---

The `RECURSIVE UNION` pipe operator iterates a subquery over new rows produced by the previous iteration. The recursive input table is referenced by an alias and must appear exactly once in a `FROM` or `JOIN` clause within the subquery.

```googlesql
FROM Employees
|> WHERE employee_id = 123456
|> RECURSIVE UNION ALL
(
  FROM AllReportees r  -- Recursive input table
  |> JOIN Employees e ON e.manager_id = r.employee_id
  |> SELECT e.*
) AS AllReportees
|> WHERE State = 'AK'
|> AGGREGATE COUNT(*) AS num_employees;
```
