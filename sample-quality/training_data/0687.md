How do I combine a path mode with another path mode inside a subpath pattern?

--- AGENT ---

Path modes can be nested with outer and inner modes; the inner mode is applied first. When both an outer TRAIL and inner WALK mode are specified, the inner WALK has no effect on the outer TRAIL constraint.

```googlesql
GRAPH FinGraph
MATCH TRAIL (WALK (a1:Account)-[t1:Transfers]->{4}(a5:Account))
RETURN COUNT(1) as num_paths
```
