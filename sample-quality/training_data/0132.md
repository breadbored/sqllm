How do I reverse the order of elements in an array?

--- AGENT ---

Use `ARRAY_REVERSE` to flip an array so that the last element becomes first, the second-to-last becomes second, and so on. The function preserves the array's element type in the reversed result.

```googlesql
SELECT ARRAY_REVERSE([1, 2, 3]) AS reverse_arr
```
