## Question
Create a staging table that cleans event data by filtering test users, enriching events with device classification, and preparing the data for downstream analytics.

--- AGENT ---

This query creates a staging table of events with the following logic:

**Core Structure:** It selects from an events table (`events_named`) and performs a LEFT JOIN with a users table (`stg_users`) to filter out test users.

**Device Classification:** A complex CASE statement assigns each event to a `device_category` (mobile, desktop, xr, gaming, or other) by evaluating the device operating system, device model, and browser name in a priority order—checking for known mobile and gaming patterns before falling back to browser classification.

**App Field Logic:** The `app` column is conditionally rewritten: events marked as 'devalliance' or 'app' events with client='devalliance' and created before 2026-02-13 are tagged as 'devalliance'; all others preserve their original app value.

**Window Functions:** Two LAG window functions retrieve the previous `derived_course_id` and `derived_doc_id` within each subsession, ordered by creation time, to track state transitions.

**JSON Extraction:** Four fields are extracted from the `action_meta` JSON object: `bucket`, `full_bucket_name`, `success`, and `success_condition`.

**Filtering:** The WHERE clause excludes test users (or events without a session user match), limits to either devalliance or production environments, and removes all sessions that contain any event from a test user.

  CASE 
    -- primary: mobile operating systems (highest priority)
    WHEN LOWER(COALESCE(os_name, '')) IN ('android', 'ios', 'harmonyos', 'tizen', 'windows mobile', 'webos') THEN 'mobile'
    
    -- vr/ar/gaming devices (check before desktop os since quest runs linux)
    WHEN LOWER(COALESCE(device_model, '')) LIKE '%quest%'
        OR LOWER(COALESCE(browser_name, '')) = 'oculus browser'
        THEN 'xr'
    WHEN LOWER(COALESCE(os_name, '')) = 'nintendo'

        OR LOWER(COALESCE(e.device_vendor, '')) LIKE '%xbox%'
        OR LOWER(COALESCE(e.os_name, '')) LIKE '%playstation%'
        OR LOWER(COALESCE(e.os_name, '')) LIKE '%xbox%'
        THEN 'gaming'
    
    -- primary: desktop operating systems (after vr check)
    WHEN LOWER(COALESCE(os_name, '')) IN ('windows', 'mac os', 'linux', 'ubuntu', 'chromium os') THEN 'desktop'
    
    -- secondary: mobile device models (known mobile device patterns)
    WHEN LOWER(COALESCE(device_model, '')) IN ('iphone', 'ipad', 'k')
      OR LOWER(COALESCE(device_model, '')) LIKE 'sm-%'      -- samsung models
      OR LOWER(COALESCE(device_model, '')) LIKE 'cph%'      -- oppo models  
      OR LOWER(COALESCE(device_model, '')) LIKE 'moto %'    -- motorola
      OR LOWER(COALESCE(device_model, '')) LIKE '%galaxy%'  -- samsung galaxy
      OR LOWER(COALESCE(device_model, '')) LIKE '%pixel%'   -- google pixel
      OR LOWER(COALESCE(device_model, '')) LIKE '%redmi%'   -- xiaomi redmi
      OR LOWER(COALESCE(device_model, '')) LIKE '%mi %'     -- xiaomi mi
      OR LOWER(COALESCE(device_model, '')) LIKE 'nexus %'   -- google nexus
      THEN 'mobile'
    
    -- secondary: desktop device models
    WHEN LOWER(COALESCE(device_model, '')) = 'macintosh' THEN 'desktop'
    
    -- tertiary: mobile device vendors (for cases with missing os/model)
    WHEN LOWER(COALESCE(device_vendor, '')) IN (
      'samsung', 'oppo', 'xiaomi', 'lg', 'huawei', 'motorola', 
      'realme', 'vivo', 'tecno', 'infinix', 'itel', 'zte', 'oneplus', 
      'nokia', 'sony', 'google', 'nothing', 'meizu'

    -- quaternary: mobile browsers (clearly mobile-only browsers)
    WHEN LOWER(COALESCE(browser_name, '')) IN (
      'mobile safari', 'samsung internet', 'instagram', 'facebook', 
      'chrome webview', 'miui browser', 'gsa', 'heytap', 'vivo browser', 
      'android browser', 'huawei browser', 'opera touch', 'ucbrowser', 
      'twitter', 'linkedin', 'tiktok', 'snapchat', 'line', 'kakaotalk', 'wechat'
    ) THEN 'mobile'
    
    -- quaternary: desktop browsers (for cases with null device info)
    WHEN LOWER(COALESCE(browser_name, '')) IN (
      'chrome', 'firefox', 'edge', 'safari', 'opera', 'chrome headless', 
      'yandex', 'duckduckgo', 'brave', 'whale', 'vivaldi', 'opera gx', 
      'chromium', 'seamonkey', 'ie', 'electron', 'avast secure browser',

FROM
    ${ref("events_named")} AS e
LEFT JOIN 
    ${ref("stg_users")} AS u
ON
    SAFE_CAST(e.backfilled_by_session_user_id AS INT64) = u.id
WHERE
    (
        u.test_user = 0
        OR e.backfilled_by_session_user_id IS NULL
    )
AND
    (
        -- TODO: Remove this when DA launches
        (
            e.app = 'devalliance'
        OR
            (
                e.app = 'app'
            AND
                e.client = 'devalliance'
            AND
                DATE(e.created) <= '2026-02-13'
            )
        ) OR e.environment = 'production'
    )
AND
    e.session_id NOT IN (
        SELECT
            a.session_id
        FROM
            (
                SELECT
                    e.session_id,
                    e.backfilled_by_session_user_id
                FROM
                    ${ref("events_named")} AS e
                JOIN
                    ${ref("stg_users")} AS u 
                ON 
                    SAFE_CAST(e.backfilled_by_session_user_id AS INT64) = u.id
                WHERE
                    u.test_user = 1
                GROUP BY
                    e.session_id,
                    e.backfilled_by_session_user_id
            ) as a
        GROUP BY
            a.session_id
    )
