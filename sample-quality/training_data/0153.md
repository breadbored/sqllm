How do I zip arrays with custom field names in the resulting structs?

--- AGENT ---

Use `ARRAY_ZIP` with `AS alias` clauses to name the struct fields in the output. The alias becomes the field name in each struct, making the result more readable and labeled appropriately.

```googlesql
WITH T AS (
  SELECT ['a', 'b'] AS alias_inferred
)
SELECT ARRAY_ZIP([1, 2] AS A1, alias_inferred) AS results
FROM T
```
