How can I create a chain of named windows where each one builds on the previous one's specifications?

--- AGENT ---

This query defines multiple named windows where each window builds upon a previously defined window by name. Window `b` extends window `a`, window `c` extends window `b`, and `ItemWindow` extends window `c`. The final window `ItemWindow` inherits all accumulated specifications.

```googlesql
SELECT item, purchases, category, LAST_VALUE(item)
  OVER (ItemWindow) AS most_popular
FROM Produce
WINDOW
  a AS (PARTITION BY category),
  b AS (a ORDER BY purchases),
  c AS (b ROWS BETWEEN 2 PRECEDING AND 2 FOLLOWING),
  ItemWindow AS (c)
```
