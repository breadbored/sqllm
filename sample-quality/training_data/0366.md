--- USER ---

Merge rows from a source table into a target table, updating quantities for matching products and inserting new products.

--- AGENT ---

This query performs a MERGE operation that updates inventory quantities for products that already exist and inserts new products from the NewArrivals table in a single atomic operation.

```googlesql
MERGE dataset.Inventory T
USING dataset.NewArrivals S
ON T.product = S.product
WHEN MATCHED THEN
  UPDATE SET quantity = T.quantity + S.quantity
WHEN NOT MATCHED THEN
  INSERT (product, quantity) VALUES(product, quantity)
```
