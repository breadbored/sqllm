How do I convert an entire uppercase string to lowercase?

--- AGENT ---

The LOWER function returns a string with all alphabetic characters converted to lowercase according to Unicode mappings.

```googlesql
SELECT
  LOWER('FOO BAR BAZ') AS example
FROM items;
```
