How do I format a time value to display the hour in 24-hour format?

--- AGENT ---

The CAST function with the FORMAT clause converts a time to a string. The HH24 format element outputs the hour using the 24-hour clock (0-23) as a 2-digit number.

```googlesql
SELECT CAST(TIME '21:30:00' AS STRING FORMAT 'HH24') AS date_time_to_string;
```
