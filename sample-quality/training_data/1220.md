How can I filter a protocol buffer to include only optional fields while allowing required fields to be omitted with their default values?

--- AGENT ---

This query uses the RESET_CLEARED_REQUIRED_FIELDS named argument set to TRUE, which allows omitting the required year field from the FILTER_FIELDS specification; the result automatically includes year with its default value of 0.

```googlesql
SELECT FILTER_FIELDS(
  award_col,
  +month,
  RESET_CLEARED_REQUIRED_FIELDS => TRUE) AS filtered_fields
FROM MusicAwards;
```
