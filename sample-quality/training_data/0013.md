How do I compute a differentially private percentile across items with per-row contribution bounds?

--- AGENT ---

This query uses `PERCENTILE_CONT` with `DIFFERENTIAL_PRIVACY` to compute the 50th percentile (median) of quantities per item. Contribution bounds per row clamp values before aggregating across privacy units, and noise is added for privacy.

```googlesql
SELECT
  WITH DIFFERENTIAL_PRIVACY
    OPTIONS(epsilon=10, delta=.01, max_groups_contributed=1, privacy_unit_column=id)
    item,
    PERCENTILE_CONT(quantity, 0.5, contribution_bounds_per_row => (0,100)) percentile_requested
FROM professors
GROUP BY item;
```
