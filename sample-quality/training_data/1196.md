How do I filter protocol buffer rows based on an extension field value?

--- AGENT ---

This query retrieves the extension field value and filters rows in the WHERE clause by checking if that extension equals a specific value. The same extension syntax `foo_field.(some.package.bar)` can be used in both the SELECT and WHERE clauses.

```googlesql
SELECT
  foo_field.(some.package.bar)
FROM
  Test
WHERE
  foo_field.(some.package.bar) = 5;
```
