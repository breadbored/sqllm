How do I unpack all fields from a STRUCT column using SELECT with field access?

--- AGENT ---

`SELECT expression.*` expands a STRUCT into separate columns for each field. The expression must be a table alias or evaluate to a STRUCT type with named fields.

```googlesql
WITH locations AS
  (SELECT STRUCT("Seattle" AS city, "Washington" AS state) AS location
  UNION ALL
  SELECT STRUCT("Phoenix" AS city, "Arizona" AS state) AS location)
SELECT l.location.*
FROM locations l;
```
