How do I format a date to show the abbreviated month name, day, and 4-digit year?

--- AGENT ---

The FORMAT_DATE function formats a DATE value using a format string with percent-encoded format elements. The `%b` element represents the abbreviated month name, `%d` represents the day, and `%Y` represents the 4-digit year. The hyphens in the format string are literal characters that appear in the output.

```googlesql
SELECT FORMAT_DATE("%b-%d-%Y", DATE "2008-12-25") AS formatted;
```
