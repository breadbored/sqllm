How does ARRAY_SLICE handle NULL elements within an array?

--- AGENT ---

Use `ARRAY_SLICE` on an array containing NULL elements. The function includes NULL values in the output slice exactly as they appear in the input array.

```googlesql
SELECT ARRAY_SLICE(['a', 'b', NULL, 'd', 'e'], 1, 3) AS result
```
