How do I combine elements from two arrays into an array of tuples?

--- AGENT ---

Use `ARRAY_ZIP` to merge two arrays element-wise into an array of structs, where each struct contains a paired element from each input array. By default, the function uses strict mode and requires both arrays to have the same length.

```googlesql
SELECT ARRAY_ZIP([1, 2], ['a', 'b']) AS results
```
