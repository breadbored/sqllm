--- USER ---

Insert a proto message into a repeated field and update another field in the same UPDATE statement.

--- AGENT ---

This query inserts a new Song proto into the repeated song field and increments the tracks field count in a single UPDATE, with assertion to ensure only one singer is modified.

```googlesql
UPDATE Singers s
SET (INSERT s.Albums.Song
     VALUES ('''songtitle: 'Bonus Track', length:180''')),
     s.Albums.tracks = 16
WHERE s.SingerId = 5 and s.Albums.title = "Fire is Hot"
ASSERT_ROWS_MODIFIED 1;
```
