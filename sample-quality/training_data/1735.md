--- USER ---

Create a query that produces session-level behavioral and content tags from VaaS events, capturing the earliest timestamp each tag appears within a user session.

--- AGENT ---

This query generates two parallel streams of session-level tags from VaaS event data:

**First branch (behavior tags):** Groups events by `sub_session_id` and `url_path`, creating a tag from the quality setting alone (`vaas_quality:<quality_value>`). It marks these as class='behavior' and captures the earliest timestamp (`MIN(created)`) when each quality-session combination occurs.

**Second branch (content tags):** Performs the same grouping but enriches the tag by joining to a courses table. The tag now combines quality settings with the course name (`vaas_quality:<quality_value>:<course_name>`), marking it as class='content'. Again, it captures the earliest timestamp per group.

Both branches filter to non-NULL quality settings and UNION the results, producing a unified table of behavioral and content markers with their first-occurrence timestamps within each session/URL combination.

```sql
SELECT
  e.sub_session_id as sub_session_id,
  CONCAT('vaas_quality:', e.vaas_settings_quality) as tag,
  'behavior' as class,
  'vaas' as app,
  e.vaas_url_url_path as url_path,
  MIN(e.created) as dt
FROM
  ${ref("base_buckets_events_vaas_speedy")} as e
WHERE
  e.vaas_settings_quality IS NOT NULL
GROUP BY
  sub_session_id,
  tag,
  url_path

UNION ALL

SELECT
  e.sub_session_id as sub_session_id,
  CONCAT('vaas_quality:', e.vaas_settings_quality, ':', c.name) as tag,
  'content' as class,
  'vaas' as app,
  e.vaas_url_url_path as url_path,
  MIN(e.created) as dt
FROM
  ${ref("base_buckets_events_vaas_speedy")} as e
JOIN
  ${ref("courses")} as c
ON
  c.vaas_id = e.vaas_url_video_id
WHERE
  e.vaas_settings_quality IS NOT NULL
GROUP BY
  sub_session_id,
  tag,
  url_path
```
