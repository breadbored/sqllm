How do I parse a datetime string that follows the standard YYYY-MM-DD HH:MM:SS format?

--- AGENT ---

The PARSE_DATETIME function parses a string into a DATETIME value using a format string. The format '%Y-%m-%d %H:%M:%S' specifies a 4-digit year, 2-digit month and day separated by hyphens, followed by 2-digit hour, minute, and second separated by colons.

```googlesql
SELECT PARSE_DATETIME('%Y-%m-%d %H:%M:%S', '1998-10-18 13:45:55') AS datetime;
```
