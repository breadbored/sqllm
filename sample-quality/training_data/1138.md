How do I describe a value table to see its schema?

--- AGENT ---

The `DESCRIBE` pipe operator indicates when the input table is a value table and shows its STRUCT type. It displays that the result is a value table alongside column and type information.

```googlesql
FROM (SELECT AS STRUCT 1 AS x, 'abc' AS y) AS value_table
|> DESCRIBE;
```
