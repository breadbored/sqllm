How do I return NULL only when two values are exactly equal, otherwise return the first value?

--- AGENT ---

NULLIF compares its two arguments for equality and returns NULL if they match; otherwise it returns the first argument. This is useful for suppressing specific unwanted values by converting them to NULL.

```googlesql
SELECT NULLIF(0, 0) as result
```
