How do I simplify a small polygon using different tolerance levels to see how it collapses to lower dimensions?

--- AGENT ---

`ST_SIMPLIFY` reduces a geometry to progressively simpler forms as tolerance increases, potentially collapsing polygons to linestrings or points when simplification is extreme enough.

```googlesql
WITH example AS
 (SELECT
    ST_GEOGFROMTEXT('POLYGON((0 0, 0.1 0, 0.1 0.1, 0 0))') AS polygon,
    t AS tolerance
  FROM UNNEST([1000, 10000, 100000]) AS t)
SELECT
  polygon AS original_triangle,
  tolerance AS tolerance_meters,
  ST_SIMPLIFY(polygon, tolerance) AS simplified_result
FROM example
```
