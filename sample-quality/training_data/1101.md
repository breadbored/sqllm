How do I sort results by a column in descending order using the pipe syntax?

--- AGENT ---

The `ORDER BY` pipe operator sorts the input table by the specified column. You can use `ASC` for ascending or `DESC` for descending order.

```googlesql
(
  SELECT 1 AS x
  UNION ALL
  SELECT 3 AS x
  UNION ALL
  SELECT 2 AS x
)
|> ORDER BY x DESC;
```
