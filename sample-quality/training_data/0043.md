How do I count rows and find the minimum and maximum values in a column, including null rows in the total?

--- AGENT ---

Basic aggregate functions summarize multiple rows into single values. `COUNT(*)` counts all rows including nulls, `COUNT(column)` counts non-null values, and `MIN`/`MAX` find the smallest and largest values respectively.

```googlesql
SELECT
  COUNT(*) AS total_count,
  COUNT(fruit) AS non_null_count,
  MIN(fruit) AS min,
  MAX(fruit) AS max
FROM
  (
    SELECT NULL AS fruit
    UNION ALL
    SELECT 'apple' AS fruit
    UNION ALL
    SELECT 'pear' AS fruit
    UNION ALL
    SELECT 'orange' AS fruit
  )
```
