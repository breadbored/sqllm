How do I catch and handle errors in a procedural block?

--- AGENT ---

Use `BEGIN...EXCEPTION WHEN ERROR THEN` to define error handling. If any statement in the `BEGIN` section encounters an error, execution skips remaining statements in that section and runs the statements in the `EXCEPTION` clause instead.

```googlesql
CREATE OR REPLACE PROCEDURE schema1.proc1() BEGIN
  SELECT 1/0;
END;

CREATE OR REPLACE PROCEDURE schema1.proc2() BEGIN
  CALL schema1.proc1();
END;

BEGIN
  CALL schema1.proc2();
EXCEPTION WHEN ERROR THEN
  SELECT "An error occurred when calling schema1.proc2";
END;
```
