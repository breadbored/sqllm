How do I normalize different space characters in a string using compatibility form?

--- AGENT ---

The NORMALIZE function with NFKC mode converts compatibility variants (like different Unicode space characters) to their canonical equivalents.

```googlesql
SELECT NORMALIZE('Raha\u2004Mahan', NFKC) AS normalized_name
```
