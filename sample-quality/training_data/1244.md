How do I sample rows from a large table for analysis?

--- AGENT ---

`TABLESAMPLE` selects a random sample of rows using a specified algorithm (BERNOULLI, SYSTEM, or RESERVOIR). This is useful for working with large tables where a representative sample is sufficient.

```googlesql
SELECT MessageId
FROM Messages TABLESAMPLE RESERVOIR (100 ROWS);
```
