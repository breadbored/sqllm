--- USER ---

Insert or update an inventory item and return the modified values along with a computed column showing the action taken.

--- AGENT ---

This query inserts or updates an 'oven' record and returns the product name, a computed new_quantity column, and the ACTION column indicating whether an INSERT or UPDATE occurred.

```googlesql
INSERT OR UPDATE Inventory (product, quantity)
VALUES ('oven', 100)
THEN RETURN WITH ACTION product, quantity * 10 AS new_quantity;
```
