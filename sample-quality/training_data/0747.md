How do I aggregate data in a WITH statement without an explicit GROUP BY clause?

--- AGENT ---

When a `WITH` statement includes aggregate expressions without `GROUP BY`, all non-aggregate items are implicitly used as grouping keys (equivalent to `GROUP BY ALL`). This example groups by source and destination IDs while counting transfers.

```googlesql
GRAPH FinGraph
MATCH (src:Account)-[transfer:Transfers]->(dst:Account)
WITH COUNT(*) AS transfer_total, src.id AS source_id, dst.id AS destination_id
RETURN transfer_total, destination_id, source_id
```
