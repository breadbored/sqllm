What value does GoogleSQL return when a nested protocol buffer message itself is not set?

--- AGENT ---

This query casts an empty protocol buffer where the nested message is not set at all. When the containing message is null or not set, GoogleSQL returns NULL for fields accessed through that message, regardless of any default value annotations.

```googlesql
SELECT
  proto_field.nested.value
FROM
  (SELECT
     CAST("" AS examples.package.OuterMessage) AS proto_field);
```
