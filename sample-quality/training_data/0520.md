How do I convert a sequence of bytes to a readable ASCII string?

--- AGENT ---

The CAST function with the FORMAT clause converts bytes to a string using the specified encoding. The ASCII format element interprets the byte sequence as ASCII character codes and returns the corresponding text string.

```googlesql
SELECT CAST(b'\x48\x65\x6c\x6c\x6f' AS STRING FORMAT 'ASCII') AS bytes_to_string;
```
