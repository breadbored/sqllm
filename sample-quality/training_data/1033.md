How do I access nested data within a JSON object, including both fields and array elements?

--- AGENT ---

The JSON subscript operator retrieves values from JSON using dot notation for field access and bracket notation for array indices. Field names are accessed as strings in brackets, and numeric indices access array elements by position.

```googlesql
SELECT json_value.class.students[0]['name'] AS first_student
FROM
  UNNEST(
    [
      JSON '{"class" : {"students" : [{"name" : "Jane"}]}}',
      JSON '{"class" : {"students" : []}}',
      JSON '{"class" : {"students" : [{"name" : "John"}, {"name": "Jamie"}]}}'])
    AS json_value
```
