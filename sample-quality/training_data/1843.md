--- USER ---

Show me a BigQuery query that joins user badge completion data with client badge information to enrich completion records with client metadata.

--- AGENT ---

This query enriches user badge completion records with client badge metadata. It joins the `agg_user_badge_completion` table (containing user-level completion data) with the `agg_client_badges` table (containing client badge definitions) on the condition that the badge ID in the client table matches the learning path ID in the completion table.

The result includes user completion attributes (user_id, learning_path_id, badge_name, completed_date, did_complete) and client-level attributes (client_id, client_name, legacy_badge). The join produces one row per matching badge completion-to-client relationship; if multiple rows share the same learning_path_id or badge_id, the result will contain multiple rows per user.

```sql
SELECT
  user_id,
  learning_path_id,
  badge_name,
  completed_date AS date,
  did_complete,
  cb.client_id,
  cb.client_name,
  legacy_badge
FROM
  ${ref("agg_user_badge_completion")} ubc
JOIN
  ${ref("agg_client_badges")} cb
ON
  cb.badge_id = ubc.learning_path_id
```
