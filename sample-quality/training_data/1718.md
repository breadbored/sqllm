Show me a BigQuery query that extracts `download_pdf` events grouped by sub_session, capturing the earliest event details in each session along with behavioral classification tags.

--- AGENT ---

This query filters the events table for `download_pdf` events and groups them by `sub_session_id`. For each session, it captures the earliest event (ordered by the `created` timestamp) and extracts:

- The `app` value from that earliest event using `MIN_BY(es.app, es.created)` — the app associated with the minimum timestamp
- The `url_path` value from that earliest event using `MIN_BY(es.url_path, es.created)` — the URL path associated with the minimum timestamp  
- The minimum `created` timestamp itself for that session
- Two literal classification columns: `tag` set to `'download_pdf'` and `class` set to `'behavior'`

The WHERE clause filters to events where `name` equals `'download_pdf'`. The output is one row per `sub_session_id`, with each row representing the earliest download_pdf event in that session, tagged with behavioral metadata.

```sql
SELECT
    es.sub_session_id as sub_session_id,
    'download_pdf' as tag,
    'behavior' as class,
    MIN_BY(es.app, es.created) as app,
    MIN_BY(es.url_path, es.created) as url_path,
    MIN(es.created) as dt
FROM
    ${ref("base_buckets_events_docs_speedy")} as es
WHERE
    es.name != 'page_view'
AND
    es.name != 'open_link'
AND
    es.name = 'download_pdf'

GROUP BY
    sub_session_id,
    tag
```
