How do I create a check constraint that validates a relationship between two columns?

--- AGENT ---

A check constraint restricts what data can be inserted by enforcing a boolean expression. The constraint is evaluated whenever rows are inserted or updated, and only rows satisfying the expression are accepted.

```googlesql
CREATE TABLE page_count_average (
  words_per_chapter INT64,
  words_per_book INT64,
  CHECK (words_per_chapter < words_per_book)
);
```
