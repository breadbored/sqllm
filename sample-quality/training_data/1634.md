## Question
How can I count the number of distinct days each user had transactions in each month?

--- AGENT ---

This query measures user activity distribution across calendar months by counting distinct days of activity per user. It joins transaction records to the user table (filtering to only users who exist), then groups by user and month. For each user-month combination, it counts how many distinct days had at least one transaction — providing a sense of activity spread within the month. A user with 10 transactions all on one day would show 1, while a user with transactions spread across 5 different days would show 5.

```sql
SELECT
  dt.user_id,
  datetime_trunc(dt.created, MONTH)               AS date,
  count(DISTINCT datetime_trunc(dt.created, DAY)) AS sessions
FROM
  ${ref("agg_transactions")} AS dt
INNER JOIN
  ${ref("stg_users")} AS u
  ON
    dt.user_id = u.id
GROUP BY
  dt.user_id,
  datetime_trunc(dt.created, MONTH)
```
