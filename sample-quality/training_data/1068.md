How do I ensure a volatile function like RAND() is evaluated only once in an expression?

--- AGENT ---

Use the WITH expression to assign the volatile function to a variable, which guarantees it is evaluated exactly once. Multiple references to the variable will use the same cached value.

```googlesql
SELECT WITH(a AS RAND(), a - a)
```
