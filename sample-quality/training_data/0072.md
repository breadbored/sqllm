How do I count non-NULL values differently from total rows using a window?

--- AGENT ---

`COUNT(*)` counts all rows including those with NULL values, while `COUNT(x)` counts only non-NULL values in column `x`. This window example partitions by modulo 3 and contrasts the two counts:

```googlesql
SELECT
  x,
  COUNT(*) OVER (PARTITION BY MOD(x, 3)) AS count_star,
  COUNT(x) OVER (PARTITION BY MOD(x, 3)) AS count_x
FROM UNNEST([1, 4, NULL, 4, 5]) AS x;
```
