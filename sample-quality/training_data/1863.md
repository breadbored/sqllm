--- USER ---

Provide me a query that returns all courses with their associated client IDs and normalized text fields.

--- AGENT ---

This query retrieves a flattened view of all courses with their associated client identifiers and text normalization applied.

The query operates in two steps:

1. **CTE (`course_clients`):** Pre-aggregates the many-to-many relationship between clients and course containers by collecting distinct client IDs into an array for each container. NULL client IDs are excluded.

2. **Main query:** Selects all columns from the courses table and left-joins the pre-aggregated client arrays by `container_id`. Several text columns (description, headline, pricing, level, mailgun_template_id, mailgun_subject, mailgun_data, course_type, seo) are normalized to lowercase. The `COALESCE` expression provides a fallback array `[1]` when a course container has no associated clients in the CTE.

The result is a single row per course with all its original attributes enriched by an array of client IDs and text normalization applied.

```sql
WITH
course_clients AS (
    SELECT
        ccc.course_container_id as container_id,
        ARRAY_AGG(DISTINCT ccc.client_id IGNORE NULLS) as client_ids
    FROM
        ${ref("clients_course_containers")} as ccc
    GROUP BY
        ccc.course_container_id
)
SELECT
  c.id,
  c.creator_admin_id,
  c.updated_admin_id,
  c.container_id,
  c.name,
  LOWER(c.description) AS description,
  LOWER(c.headline) AS headline,
  c.version,
  c.published,
  c.created,
  c.updated,
  c.expiration_days,
  c.thumbnail,
  c.background_image,
  LOWER(c.pricing) AS pricing,
  c.has_been_published,
  c.vaas_id,
  c.archived,
  LOWER(c.level) AS level,
  c.publish_date,
  c.assessment_id,
  c.lab_id,
  c.is_public,
  c.promoted,
  LOWER(c.mailgun_template_id) AS mailgun_template_id,
  LOWER(c.mailgun_subject) AS mailgun_subject,
  c.discover_page,
  c.facebook_sso_required,
  c.cover_image,
  LOWER(c.mailgun_data) AS mailgun_data,
  c.course_cover_mp4,
  c.course_cover_webm,
  c.disable_final_assessment,
  c.hide_assessment_score,
  c.meta_quest_sso_required,
  c.google_sso_required,
  LOWER(c.course_type) AS course_type,
  c.show_latest_version,
  LOWER(c.seo) AS seo,
  COALESCE(cc.client_ids, [1]) AS client_ids
FROM
  ${ref("courses")} as c
LEFT JOIN
    course_clients as cc
ON
    cc.container_id = c.container_id
```
