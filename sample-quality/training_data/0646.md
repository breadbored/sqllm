How do you find nodes that have a specific property defined on them?

--- AGENT ---

The PROPERTY_EXISTS predicate tests whether a graph element has a specific property. This query finds Person or Account nodes that have a name property and returns the name value for those nodes.

```googlesql
GRAPH FinGraph
MATCH (n:Person|Account WHERE PROPERTY_EXISTS(n, name))
RETURN n.name
```
