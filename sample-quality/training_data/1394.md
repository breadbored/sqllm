How do I left-pad a bytes value with a bytes pattern?

--- AGENT ---

The LPAD function works on BYTES types when both the original value and pattern are BYTES.

```googlesql
SELECT FORMAT('%T', LPAD(b'abc', 10, b'ghd')) AS results
```
