How do I replace all occurrences of a substring in multiple strings?

--- AGENT ---

The REPLACE function finds and replaces all occurrences of a pattern in a string, working across multiple rows in a table.

```googlesql
WITH desserts AS
  (SELECT 'apple pie' as dessert
  UNION ALL
  SELECT 'blackberry pie' as dessert
  UNION ALL
  SELECT 'cherry pie' as dessert)

SELECT
  REPLACE (dessert, 'pie', 'cobbler') as example
FROM desserts;
```
