--- USER ---

What does sample standard deviation return when only one non-NULL value is provided?

--- AGENT ---

When `STDDEV_SAMP` has fewer than two non-NULL values, it returns NULL because sample standard deviation requires at least two observations to compute.

```googlesql
SELECT STDDEV_SAMP(x) AS results FROM UNNEST([10, NULL]) AS x
```
