How do I convert a JSON number to a UINT32 with flexible type conversion?

--- AGENT ---

The `LAX_UINT32` function converts JSON numbers, strings representing numbers, and booleans to SQL `UINT32`, returning `NULL` for negative values or out-of-range numbers.

```googlesql
SELECT LAX_UINT32(JSON '10') AS result;

SELECT LAX_UINT32(JSON '10.0') AS result;
```
