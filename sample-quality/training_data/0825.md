How do I extract a JSON value from a specific path in a JSON-formatted string?

--- AGENT ---

The `JSON_EXTRACT` function extracts a JSON value at the specified JSONPath from a JSON-formatted string and returns it as a JSON-formatted string or JSON value.

```googlesql
SELECT
  JSON_EXTRACT(JSON '{"class": {"students": [{"id": 5}, {"id": 12}]}}', '$.class')
  AS json_data;
```
