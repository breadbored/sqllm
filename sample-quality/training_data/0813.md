How do I append multiple values to nested JSON arrays?

--- AGENT ---

The `JSON_ARRAY_APPEND` function can append values to nested arrays by specifying multiple path-value pairs, which are evaluated left to right.

```googlesql
SELECT JSON_ARRAY_APPEND(
  JSON '["a", ["b"], "c"]',
  '$[1]', [1, 2],
  '$[1][1]', [3, 4],
  append_each_element=>FALSE) AS json_data
```
