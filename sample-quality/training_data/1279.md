How do I split a date range into equal-sized daily subranges?

--- AGENT ---

GENERATE_RANGE_ARRAY splits a range into an array of subranges, each with a maximum size determined by the step interval. This query splits the date range from January 1 to January 6, 2020 into 1-day subranges.

```googlesql
SELECT GENERATE_RANGE_ARRAY(
  RANGE(DATE '2020-01-01', DATE '2020-01-06'),
  INTERVAL 1 DAY) AS results;
```
