Can you provide an example BigQuery SQL query that extracts the first occurrence of a specific event type within user sub-sessions, capturing the app and URL path from that event?

--- AGENT ---

This query identifies the earliest occurrence of 'index_filters' events within each sub-session and extracts the app and URL path from that first event, along with its timestamp.

The query filters the events table for records where the event name is 'index_filters', then groups by sub_session_id. For each group, it uses `MIN_BY()` to retrieve the `app` and `url_path` values from the row with the earliest `created` timestamp, and also captures that minimum timestamp. The result is tagged as a 'behavior' class event labeled 'index_filters', producing one row per sub-session showing where this event first occurred.

```sql
SELECT
    es.sub_session_id as sub_session_id,
    'index_filters' as tag,
    'behavior' as class,
    MIN_BY(es.app, es.created) as app,
    MIN_BY(es.url_path, es.created) as url_path,
    MIN(es.created) as dt

FROM
    ${ref('base_buckets_events_docs_speedy')} as es 

WHERE
    es.name != 'page_view'
AND
    es.name != 'open_link'
AND
    es.name = 'index_filters'

GROUP BY
    sub_session_id,
    tag
```
