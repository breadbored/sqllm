How do I join two tables on multiple columns?

--- AGENT ---

The `USING` clause in a join specifies equality on one or more column names that appear in both tables. The result deduplicates the join columns into a single output column.

```googlesql
WITH A AS (
    SELECT 1 as x, 15 as y UNION ALL
    SELECT 2, 10 UNION ALL
    SELECT 9, 16 UNION ALL
    SELECT NULL, 12),
  B AS (
    SELECT 2 as x, 10 as y UNION ALL
    SELECT 9, 17 UNION ALL
    SELECT 9, 16 UNION ALL
    SELECT 5, 15)
SELECT * FROM A INNER JOIN B USING (x, y);
```
