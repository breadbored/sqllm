How do I explicitly include partial subranges when splitting a date range?

--- AGENT ---

GENERATE_RANGE_ARRAY includes the last partial subrange by default when its third parameter is TRUE. This example explicitly passes TRUE to confirm that partial subranges are included in the output.

```googlesql
SELECT GENERATE_RANGE_ARRAY(
  RANGE(DATE '2020-01-01', DATE '2020-01-06'),
  INTERVAL 2 DAY,
  TRUE) AS results;
```
