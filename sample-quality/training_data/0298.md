How do I create a table with hidden columns that are computed from other columns?

--- AGENT ---

The `HIDDEN` attribute prevents a column from appearing in `SELECT *` expansions. A generated column with `HIDDEN` on its source columns can combine those columns without exposing the individual inputs.

```googlesql
CREATE TABLE Authors (
  first_name STRING HIDDEN,
  last_name STRING HIDDEN,
  full_name STRING GENERATED ALWAYS AS (CONCAT(first_name, ' ', last_name));
);
```
