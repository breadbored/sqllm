How do I perform a case-insensitive pattern match?

--- AGENT ---

The LIKE operator combined with the COLLATE function using 'und:ci' collation enables case-insensitive pattern matching.

```googlesql
COLLATE('Foo', 'und:ci') LIKE COLLATE('%foo%', 'und:ci')
```
