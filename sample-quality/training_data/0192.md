How do I flatten an array and access a specific indexed element within each nested array?

--- AGENT ---

Use `SAFE_OFFSET()` to access a specific element index within nested arrays before flattening with `UNNEST()`. This returns the selected element from each array in the nested structure, with `NULL` for out-of-range indexes. All three syntaxes are equivalent: explicit `FLATTEN()`, implicit `FLATTEN()`, or direct field access with indexing in the `FROM` clause.

```googlesql
-- In UNNEST (FLATTEN used explicitly):
SELECT second_prices
FROM ItemsTable
INNER JOIN UNNEST(FLATTEN(items.sales.prices[SAFE_OFFSET(1)])) AS second_prices;

-- In UNNEST (FLATTEN used implicitly):
SELECT second_prices
FROM ItemsTable
INNER JOIN UNNEST(items.sales.prices[SAFE_OFFSET(1)]) AS second_prices;

-- In the FROM clause (UNNEST used implicitly):
SELECT second_prices
FROM ItemsTable
INNER JOIN ItemsTable.items.sales.prices[SAFE_OFFSET(1)] AS second_prices;

/*---------------+
 | second_prices |
 +---------------+
 | 50            |
 | NULL          |
 | NULL          |
 | NULL          |
 +---------------*/
```
