How do I parse a JSON-formatted string with large numbers using rounding?

--- AGENT ---

Set `wide_number_mode=>'round'` in `PARSE_JSON` to round large numbers that cannot be stored without loss of precision.

```googlesql
SELECT PARSE_JSON('{"id": 922337203685477580701}', wide_number_mode=>'round') AS json_data;
```
