How can I get the unique identifier of a node that an edge points to in a graph query?

--- AGENT ---

The `DESTINATION_NODE_ID` function returns a string identifier for the destination node of a graph edge. This identifier is unique within the scope of the current query and allows you to reference specific nodes by their system-assigned IDs rather than by properties.

```googlesql
GRAPH FinGraph
MATCH (:Person)-[o:Owns]->(a:Account)
RETURN a.id AS account_id, DESTINATION_NODE_ID(o) AS destination_node_id
```
