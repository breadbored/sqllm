--- USER ---

Extract UTM attribution tags from event data by session, capturing the earliest app and URL path encountered for each unique UTM parameter value.

--- AGENT ---

This query extracts UTM attribution attributes from events, pivoting across five UTM parameters (source, campaign, medium, term, content). 

**Structure:** Five identical UNION ALL branches, each operating on the same `base_buckets_events_speedy` table but filtering on a different UTM parameter.

**Per-branch logic:**
Each branch filters to non-null, non-empty values of its UTM parameter, then:
- Cleans the UTM value by stripping query string (`?`), bracket content (`[`), and angle-bracket content (`<`) — keeping only the leftmost segment
- Creates a `tag` by concatenating the UTM parameter name (e.g., `utm_source:`) with the cleaned value
- Groups by `sub_session_id` and `tag`
- Uses `MIN_BY` to capture the earliest-by-`created` app and url_path for each session/tag pair
- Records the earliest timestamp (`MIN(created)`) as `dt`

All branches output the same schema: `sub_session_id`, `tag`, `class` (hardcoded as `'attribute'`), `app`, `url_path`, and `dt`.

**Final query:**

```sql
SELECT
  eas.sub_session_id as sub_session_id,
  CONCAT('utm_source:', SPLIT(SPLIT(SPLIT(eas.utm_source, '?')[0], '[')[0], '<')[0]) as tag,
  'attribute' as class,
  MIN_BY(eas.app, eas.created) as app,
  MIN_BY(eas.url_path, eas.created) as url_path,
  MIN(eas.created) as dt

FROM
  ${ref("base_buckets_events_speedy")} as eas

WHERE
  eas.utm_source IS NOT NULL
AND
  eas.utm_source != ''

GROUP BY 
  sub_session_id,
  tag

UNION ALL
```
