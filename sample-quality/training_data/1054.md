How do I check if two values are equal using the IS DISTINCT FROM operator?

--- AGENT ---

Use IS NOT DISTINCT FROM to return TRUE when two values are equal or both NULL. This operator never returns NULL, making it useful when you want strict equality without NULL propagation.

```googlesql
SELECT 1 IS NOT DISTINCT FROM 1
```
