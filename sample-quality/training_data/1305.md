--- USER ---

How do I create a basic recursive CTE that starts with an initial value and incrementally builds up results through multiple iterations?

--- AGENT ---

A recursive CTE uses a base term to initialize the first result set, then a recursive term references the CTE itself to generate new rows in subsequent iterations. The union operator combines results from each iteration until the termination condition is met. This query generates iteration values 1, 1, 2, 2, 3, 3 by seeding with two rows of 1, then recursively adding 1 to each previous result until reaching 3:

```googlesql
WITH RECURSIVE
  CTE_1 AS (
    (SELECT 1 AS iteration UNION ALL SELECT 1 AS iteration)
    UNION ALL
    SELECT iteration + 1 AS iteration FROM CTE_1 WHERE iteration < 3
  )
SELECT iteration FROM CTE_1
ORDER BY 1 ASC
```
