How can I output a graph element directly from GRAPH_TABLE when it cannot be selected as a column?

--- AGENT ---

Graph elements cannot be directly output as table columns; you must convert them to JSON representation using the SAFE_TO_JSON function. This function transforms the graph element into a JSON object that can be safely returned in the result set.

```googlesql
SELECT SAFE_TO_JSON(n) as json_node
FROM GRAPH_TABLE(
  FinGraph
  MATCH (n:Person)
  RETURN n
);
```
