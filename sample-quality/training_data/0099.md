How do I find approximate quantile boundaries that divide my data into equally-sized groups?

--- AGENT ---

APPROX_QUANTILES returns an array of approximate boundaries that partition the data into the specified number of quantiles, sorted in ascending order from minimum to maximum. With 2 quantiles, it returns 3 elements representing the min, median, and max values.

```googlesql
SELECT APPROX_QUANTILES(x, 2) AS approx_quantiles
FROM UNNEST([1, 1, 1, 4, 5, 6, 7, 8, 9, 10]) AS x;
```
