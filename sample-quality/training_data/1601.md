--- USER ---

How can I calculate the total purchases across all items while showing each individual item's purchases?

--- AGENT ---

This query uses an empty OVER clause to create a window that includes all rows. Every row receives the same grand total calculation (SUM of all purchases), which appears alongside each item's individual purchase count.

```googlesql
SELECT item, purchases, category, SUM(purchases)
  OVER () AS total_purchases
FROM Produce
```
