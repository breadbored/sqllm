How do I set up a foreign key that automatically cascades updates and deletes?

--- AGENT ---

The `ON UPDATE CASCADE` and `ON DELETE CASCADE` clauses in a foreign key definition ensure that when a referenced row is updated or deleted, all matching rows in the referencing table are also updated or deleted automatically.

```googlesql
CREATE TABLE top_books (
  book_name STRING,
  CONSTRAINT fk_top_books_name
    FOREIGN KEY (book_name)
    REFERENCES books (title)
    ON UPDATE CASCADE
    ON DELETE CASCADE
);
```
