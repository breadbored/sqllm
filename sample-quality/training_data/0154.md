How do I combine arrays by applying a custom expression to paired elements?

--- AGENT ---

Use `ARRAY_ZIP` with a `transformation` argument containing a lambda expression. Instead of creating structs, the lambda function receives paired elements from each input array and returns the result of the custom transformation.

```googlesql
SELECT ARRAY_ZIP([1, 2], [3, 4], transformation => (e1, e2) -> (e1 + e2))
```
