--- USER ---

How do I handle SUM when the input set is empty?

--- AGENT ---

`SUM` returns NULL when the input produces no rows or all values are NULL. This example demonstrates the NULL result from an empty array:

```googlesql
SELECT SUM(x) AS sum
FROM UNNEST([]) AS x;
```
