How do I calculate the population standard deviation of a set of values?

--- AGENT ---

The `STDDEV_POP` function computes the population (biased) standard deviation, measuring how spread out values are around their mean. NULL values are ignored, a single non-NULL value returns 0, and the function returns NULL only when all inputs are NULL.

```googlesql
SELECT STDDEV_POP(x) AS results FROM UNNEST([10, 14, 18]) AS x
```
