How do I extract a scalar value from a JSON array field?

--- AGENT ---

The `JSON_EXTRACT_SCALAR` function returns `NULL` when the path points to a non-scalar value like an array or object.

```googlesql
SELECT JSON_EXTRACT('{"fruits": ["apple", "banana"]}', '$.fruits') AS json_extract,
  JSON_EXTRACT_SCALAR('{"fruits": ["apple", "banana"]}', '$.fruits') AS json_extract_scalar;
```
