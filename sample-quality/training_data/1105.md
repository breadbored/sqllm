--- USER ---

How do I combine multiple queries using the pipe UNION operator without listing UNION multiple times?

--- AGENT ---

The `UNION` pipe operator lets you specify multiple queries separated by commas without repeating the `UNION` keyword. Queries following the operator are enclosed in parentheses.

```googlesql
-- Pipe syntax
SELECT * FROM ...
|> UNION ALL
    (SELECT 1),
    (SELECT 2);
```
