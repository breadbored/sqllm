How do I count rows per item using a differentially private view with explicit bounds?

--- AGENT ---

This query applies `COUNT(*)` with `DIFFERENTIAL_PRIVACY` to a view, adding noise based on the epsilon parameter. The contribution bounds per group clamp contributions before aggregation across the privacy unit column.

```googlesql
SELECT
  WITH DIFFERENTIAL_PRIVACY
    OPTIONS(epsilon=10, delta=.01, max_groups_contributed=1)
    item,
    COUNT(*, contribution_bounds_per_group=>(0, 100)) times_requested
FROM {{USERNAME}}.view_on_professors
GROUP BY item;
```
