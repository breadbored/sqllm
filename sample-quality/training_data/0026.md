How do I count non-null values without noise using the deprecated ANON_COUNT?

--- AGENT ---

This query shows the un-noised version of counting non-NULL values with the deprecated `ANON_COUNT` function using explicit bounds.

```googlesql
SELECT
  WITH ANONYMIZATION
    OPTIONS(epsilon=1e20, delta=.01, max_groups_contributed=1)
    item,
    ANON_COUNT(item CLAMPED BETWEEN 0 AND 100) times_requested
FROM {{USERNAME}}.view_on_professors
GROUP BY item;
```
