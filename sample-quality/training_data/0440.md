How do I extract the week number using different week start days from the same date?

--- AGENT ---

`EXTRACT()` with WEEK(SUNDAY) and WEEK(MONDAY) allows you to extract the week number using different week definitions, showing how the same date can belong to different week numbers depending on the week start day.

```googlesql
WITH table AS (SELECT DATE('2017-11-05') AS date)
SELECT
  date,
  EXTRACT(WEEK(SUNDAY) FROM date) AS week_sunday,
  EXTRACT(WEEK(MONDAY) FROM date) AS week_monday FROM table;
```
