--- USER ---

How do I compute a differentially private average without noise for demonstration purposes?

--- AGENT ---

This query uses an extremely high epsilon value (1e20) to effectively remove noise while maintaining the differential privacy query structure, useful for validating logic before adding realistic privacy parameters. The `AVG` function computes the average per privacy unit and then averages those averages.

```googlesql
SELECT
  WITH DIFFERENTIAL_PRIVACY
    OPTIONS(epsilon=1e20, delta=.01, max_groups_contributed=1, privacy_unit_column=id)
    item,
    AVG(quantity) average_quantity
FROM professors
GROUP BY item;
```
