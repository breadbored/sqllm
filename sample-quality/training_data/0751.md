--- USER ---

How do I define multiple edge types that connect different node types, with source and destination references?

--- AGENT ---

The EDGE TABLES clause defines edge types with SOURCE and DESTINATION keys that reference node tables, establishing the connections in the graph. Each edge can have a label and specify whether it has properties.

```googlesql
EDGE TABLES (
  Own
    SOURCE KEY (cid) REFERENCES Customer (cid)
    DESTINATION KEY (aid) REFERENCES Account
    NO PROPERTIES,
  Transfer
    SOURCE KEY (from_id) REFERENCES Account (aid)
    DESTINATION KEY (to_id) REFERENCES Account (aid)
    LABEL Transfer NO PROPERTIES
)
```
