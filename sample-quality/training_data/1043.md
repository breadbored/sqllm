How do I filter rows where a column matches a specific value?

--- AGENT ---

The comparison operator = in a WHERE clause filters rows where the specified column equals the given value, returning only rows where the condition evaluates to TRUE.

```googlesql
SELECT 'a' FROM entry_table WHERE entry = 'a'
```
