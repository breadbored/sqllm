--- USER ---

How can I retrieve each player's username along with the mascot associated with their team?

--- AGENT ---

A correlated scalar subquery joins each row from the outer query with a single matching value from another table by referencing a column from the outer context. This query retrieves the mascot for each player's team by correlating the Players table with the Mascots table on the team column.

```googlesql
SELECT
  username,
  (SELECT mascot FROM Mascots WHERE Players.team = Mascots.team) AS player_mascot
FROM
  Players;
```
