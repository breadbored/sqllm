How can I return any single path matching a pattern instead of all possible paths?

--- AGENT ---

Use the ANY prefix before the path pattern to return a single arbitrary path from each pair of endpoints instead of all possible paths.

```googlesql
GRAPH FinGraph
MATCH ANY (a:Account {is_blocked: true})->(mid:Account)->(b:Account)
RETURN a.id AS a_id, mid.id AS mid_id, b.id AS b_id
```
