How do I check if a JSON object contains another object with specific keys?

--- AGENT ---

The `JSON_CONTAINS` function checks object containment by verifying that all key-value pairs in the second object exist in the first object.

```googlesql
SELECT
    JSON_CONTAINS(JSON '{"a": {"b": 1}, "c": 2}', JSON '{"b": 1}') AS result1,
    JSON_CONTAINS(JSON '{"a": {"b": 1}, "c": 2}', JSON '{"a": {"b": 1}}') AS result2,
    JSON_CONTAINS(JSON '{"a": {"b": 1, "d": 3}, "c": 2}', JSON '{"a": {"b": 1}}') AS result3;
```
