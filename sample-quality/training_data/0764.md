How do I check for the existence of a pattern match using just the graph pattern without a `MATCH` keyword?

--- AGENT ---

This query uses an `EXISTS` subquery with only a graph pattern expression, bypassing the `MATCH` statement entirely. The subquery returns `TRUE` if the pattern finds at least one matching path in the graph.

```googlesql
GRAPH FinGraph
RETURN EXISTS {
  (p:Person {Name: "Lee"})-[o:Owns]->(a:Account)
} AS results;
```
