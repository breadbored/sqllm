How do I parse a BIGNUMERIC from a string that starts with a decimal point?

--- AGENT ---

PARSE_BIGNUMERIC interprets a leading decimal point as zero before the decimal, so ".1234" is parsed as 0.1234.

```googlesql
SELECT PARSE_BIGNUMERIC(".1234  ") as parsed;
```
