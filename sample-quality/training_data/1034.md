How do I access a value in a protocol buffer map field using a specific key?

--- AGENT ---

The protocol buffer map subscript operator retrieves values by key using the KEY keyword (which errors if the key is missing) or SAFE_KEY (which returns NULL if the key is missing).

```googlesql
SELECT
  m.purchased[KEY('A')] AS map_value
FROM
  (SELECT AS VALUE CAST("purchased { key: 'A' value: 2 }" AS Item)) AS m
```
