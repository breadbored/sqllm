How do I extract a specific vertex from a linestring by position, including from the end?

--- AGENT ---

`ST_POINTN` returns the point at a 1-based index in a linestring, supporting negative indices where -1 is the last point and -2 is the second-to-last.

```googlesql
WITH linestring AS (
    SELECT ST_GEOGFROMTEXT('LINESTRING(1 1, 2 1, 3 2, 3 3)') g
)
SELECT ST_POINTN(g, 1) AS first, ST_POINTN(g, -1) AS last,
    ST_POINTN(g, 2) AS second, ST_POINTN(g, -2) AS second_to_last
FROM linestring;
```
