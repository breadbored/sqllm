How do I add a window function with a named window in pipe syntax?

--- AGENT ---

The `SELECT` pipe operator supports window functions and named windows using the `WINDOW` keyword. The named window is defined within the `SELECT` operator and can be referenced by the window function with the `OVER` clause.

```googlesql
FROM Produce
|> SELECT item, sales, category, SUM(sales) OVER item_window AS category_total
   WINDOW item_window AS (PARTITION BY category);
```
