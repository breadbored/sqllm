How do I check if an array element access will succeed?

--- AGENT ---

ISERROR evaluates the array access and returns FALSE when the index exists and no error occurs.

```googlesql
SELECT ISERROR((SELECT [1,2,3][OFFSET(0)])) AS is_error
```
