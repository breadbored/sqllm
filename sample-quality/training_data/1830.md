## Question
Give me a BigQuery SQL query that enriches user feedback records with course progress metrics and client information, including progress snapshots at the time feedback was submitted.

--- AGENT ---

This query joins feedback records with multiple tables to create a fact table of user feedback enriched with contextual progress data. 

**Core structure:**
- Starts from feedback records (`stg_feedback`)
- Joins to a pre-calculated subquery (`lesson_progress`) that computes the user's total lesson progress in that course at the time the feedback was created (by filtering progress records to only those created before `f.created`)
- Joins to course duration data (aggregating video lengths across all lessons in each course)
- Joins to current course progress aggregates (`agg_user_course_progress`)
- Joins to client-course container mappings and client names

**Key calculations:**
- `course_progress_percentage` — calculated as the ratio of user's lesson progress to total course duration, capped at 100% using `LEAST(..., 1)`
- `time_on_course_minutes` — converted from seconds to minutes
- Client aggregation — collects all distinct client IDs and names associated with each course using `ARRAY_AGG(DISTINCT ...)`

**Output:** One row per feedback record, with columns for the feedback itself, the user's progress snapshot at feedback time, current progress status, course metadata, and an array of associated clients.

```sql
SELECT
  f.user_id,
  f.course_id,
  upc.container_id,
  upc.course_name,
  f.response_sentiment,
  f.response_text,
  f.lesson_id,
  f.status,
  ANY_VALUE(ulp.user_total_lesson_progress) AS course_progress_minutes,
  ANY_VALUE(cd.course_duration_minutes) AS course_length_minutes,
  LEAST(ROUND(((ANY_VALUE(ulp.user_total_lesson_progress) / ANY_VALUE(cd.course_duration_minutes))), 2), 1) AS course_progress_percentage,
  ANY_VALUE(ROUND((ulp.time_on_course / 60), 3)) AS time_on_course_minutes,
  f.created AS feedback_date,
  upc.course_started,
  upc.course_updated,
  upc.course_status AS current_course_status,
  upc.course_progress_minutes AS current_course_progress_minutes,
  upc.course_progress_percentage AS current_course_progress_percentage,
  upc.course_length_minutes AS current_course_length_minutes,
  upc.time_on_course_minutes AS current_time_on_course_minutes,
  ARRAY_AGG(DISTINCT ccc.client_id) AS client_id,
  ARRAY_AGG(DISTINCT c.name) AS client_name
FROM
  ${ref("stg_feedback")} f
  -- Get user progress at time of feedback
JOIN (
  SELECT
    lesson_progress.user_id,
    lesson_progress.course_id,
    ROUND((SUM(lesson_progress.user_total_lesson_progress) / 60), 3) AS user_total_lesson_progress,
    SUM(time_on_lesson) AS time_on_course
  FROM (
    SELECT
      uclp.user_id,
      uclp.course_id,
      MAX(progress_seconds) AS user_total_lesson_progress,
      SUM(interval_seconds) AS time_on_lesson,
    FROM
      ${ref("stg_user_course_lesson_progress")} uclp
    JOIN
      ${ref("stg_feedback")} f
    ON
      f.user_id = uclp.user_id
      AND uclp.course_id = f.course_id
    WHERE
      uclp.user_id IS NOT NULL
      AND uclp.created <= f.created
    GROUP BY
      uclp.user_id,
      uclp.course_id,
      uclp.lesson_id) AS lesson_progress
  GROUP BY
    lesson_progress.user_id,
    lesson_progress.course_id) AS ulp
ON
  ulp.course_id = f.course_id
  AND ulp.user_id = f.user_id
  --get course duration
JOIN (
  SELECT
    cl.course_id,
    ROUND((SUM(v.duration) / 60), 3) AS course_duration_minutes
  FROM
    ${ref("stg_lessons")} l
  LEFT JOIN
    ${ref("stg_videos")} v
  ON
    l.video_id = v.id
  LEFT JOIN
    ${ref("stg_course_lessons")} cl
  ON
    cl.lesson_id = l.id
  GROUP BY
    course_id
  ORDER BY
    course_id) AS cd
ON
  cd.course_id = f.course_id
JOIN
  ${ref("agg_user_course_progress")} upc
ON
  f.course_id = upc.course_id
  AND f.user_id = upc.user_id
JOIN
  ${ref("stg_clients_course_containers")} ccc
ON
  ccc.course_container_id = upc.container_id
LEFT JOIN
  ${ref("stg_clients")} c
ON
  c.id = ccc.client_id
GROUP BY
  f.user_id,
  f.course_id,
  upc.container_id,
  upc.course_name,
  f.response_sentiment,
  f.response_text,
  f.status,
  f.lesson_id,
  f.created,
  upc.course_started,
  upc.course_updated,
  upc.course_status,
  upc.course_progress_minutes,
  upc.course_progress_percentage,
  upc.course_length_minutes,
  upc.time_on_course_minutes
```
