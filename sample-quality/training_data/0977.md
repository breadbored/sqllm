--- USER ---

How do I compute hyperbolic cotangent values, with a safe version for edge cases?

--- AGENT ---

COTH computes the hyperbolic cotangent of an angle in radians, and SAFE.COTH returns NULL when the computation would error. This query demonstrates both the normal computation and the safe variant:

```googlesql
SELECT COTH(1) AS a, SAFE.COTH(0) AS b;
```
