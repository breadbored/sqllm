How do I calculate the population covariance between two numeric columns?

--- AGENT ---

The `COVAR_POP` function computes the population (biased) covariance of pairs of values. Pairs containing NULL are ignored, and if no valid pairs exist, the function returns NULL. With exactly one valid pair, it returns 0.

```googlesql
SELECT COVAR_POP(y, x) AS results
FROM
  UNNEST(
    [
      STRUCT(1.0 AS y, 1.0 AS x),
      (2.0, 6.0),
      (9.0, 3.0),
      (2.0, 6.0),
      (9.0, 3.0)])
```
