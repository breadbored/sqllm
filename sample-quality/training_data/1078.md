How do I add a window function with a named window using EXTEND?

--- AGENT ---

The `EXTEND` pipe operator supports named windows defined with the `WINDOW` keyword. The named window partitions data and can be referenced by window functions in the `EXTEND` expression.

```googlesql
FROM Produce
|> EXTEND SUM(sales) OVER item_window AS category_total
   WINDOW item_window AS (PARTITION BY category);
```
