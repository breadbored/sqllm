How do I retrieve a value from a protocol buffer map field using a key lookup?

--- AGENT ---

This query uses the protocol buffer subscript operator with `KEY()` to retrieve the value associated with a specific key in a map field. The syntax `m.purchased[KEY('A')]` returns the value `2` when the key 'A' is present in the map.

```googlesql
SELECT
  m.purchased[KEY('A')] AS map_value
FROM
  (SELECT AS VALUE CAST("purchased { key: 'A' value: 2 }" AS Item)) AS m;
```
