How do I access JSON fields using the field access operator?

--- AGENT ---

JSON values support field access notation; you can pass extracted JSON into functions like `JSON_REMOVE` using the `.` operator.

```googlesql
WITH T AS (SELECT JSON '{"a": {"b": 10, "c": 20}}' AS data)
SELECT JSON_REMOVE(data.a, '$.b') AS json_data FROM T
```
