How do I merge multiple arrays into a single array?

--- AGENT ---

Use `ARRAY_CONCAT` to combine two or more arrays of the same element type into one array. The function preserves the order of elements from each input array in sequence.

```googlesql
SELECT ARRAY_CONCAT([1, 2], [3, 4], [5, 6]) as count_to_six;
```
