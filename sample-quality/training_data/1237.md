How do I create a STRUCT from multiple columns in a SELECT query?

--- AGENT ---

`SELECT AS STRUCT` produces a value table with a single STRUCT column where each field in the STRUCT corresponds to a column in the SELECT list. This is useful for subqueries that need to return multiple values as a single column.

```googlesql
SELECT ARRAY(SELECT AS STRUCT 1 a, 2 b)
```
