## Question
Show me how to enrich top banners with the list of clients associated with their learning cues through the course hierarchy.

--- AGENT ---

This query enriches the `top_banners` table by associating each banner with a list of distinct client IDs that relate to its learning cue through the course hierarchy.

The CTE `learning_cue_clients` builds a lookup table by joining from `clients_course_containers` through `courses`, `course_lessons`, and `lessons` to reach `learning_cues`. For each learning cue ID, it aggregates all distinct client IDs into an array using `ARRAY_AGG(DISTINCT ... IGNORE NULLS)`.

The main query then performs a LEFT JOIN of `top_banners` with this CTE on `learning_cue_id`. The `COALESCE` function provides a default array `[1]` for any banner whose learning_cue_id has no matching row in the CTE—either because the banner has no learning cue or the learning cue has no associated clients.

The output includes all columns from `top_banners` plus the enriched `client_ids` column.

```sql
WITH
learning_cue_clients as (
    SELECT
        lc.id as learning_cue_id,
        ARRAY_AGG(DISTINCT ccc.client_id IGNORE NULLS) as client_ids
    FROM
        ${ref("clients_course_containers")} as ccc
    JOIN
        ${ref("courses")} as c
    ON
        c.container_id = ccc.course_container_id
    JOIN
        ${ref("course_lessons")} as cl
    ON
        cl.course_id = c.id
    JOIN
        ${ref("lessons")} as l
    ON 
        l.id = cl.lesson_id
    JOIN
        ${ref("learning_cues")} as lc
    ON
        lc.lesson_id = l.id
    GROUP BY
        lc.id
)
SELECT
  tb.id,
  tb.learning_cue_id,
  tb.course_container_id,
  tb.display_seconds,
  tb.created,
  tb.updated,
  tb.lesson_resource_id,
  COALESCE(slr.client_ids, [1]) AS client_ids
FROM
  ${ref("top_banners")} as tb
LEFT JOIN
    learning_cue_clients as slr
ON
    tb.learning_cue_id = slr.learning_cue_id
```
