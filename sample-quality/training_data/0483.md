--- USER ---

I need to confirm that a specific value exists in my dataset before continuing with processing.

--- AGENT ---

The ASSERT statement uses EXISTS to verify whether a particular value appears in the dataset. If the value is found, the assertion passes without output; if not found, an error is raised with the provided description. This example demonstrates a successful assertion—the value 7907 exists in the dataset, so no error is produced.

```googlesql
ASSERT
  EXISTS(
    (SELECT X FROM UNNEST([7877, 7879, 7883, 7901, 7907]) AS X WHERE X = 7907))
AS 'Column X must contain the value 7907.';
```
