How do I truncate a TIME value to a specific granularity, such as to the nearest hour?

--- AGENT ---

TIME_TRUNC() rounds a TIME value down to the specified granularity, returning the result rounded to the beginning of that granularity. This example truncates 15:30:00 to the hour, returning 15:00:00.

```googlesql
SELECT
  TIME "15:30:00" as original,
  TIME_TRUNC(TIME "15:30:00", HOUR) as truncated;
```
