How do I apply a path mode to a quantified subpath pattern?

--- AGENT ---

Place the path mode inside the outer parentheses before the quantified subpath. The mode applies separately to each repetition of the quantified portion.

```googlesql
GRAPH FinGraph
MATCH (TRAIL -[t1:Transfers]->()-[t2:Transfers]->()-[t3:Transfers]->){2}
RETURN COUNT(1) as num_paths
```
