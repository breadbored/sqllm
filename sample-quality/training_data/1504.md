How do I create a TVF that filters rows from any input table based on a numeric column without specifying the full table schema upfront?

--- AGENT ---

This creates a temporary table-valued function using ANY TABLE to accept input of any table structure, filtering rows where a specific numeric column (creation_time) falls within a given range. It demonstrates how polymorphic TVFs can operate on tables of unknown schema as long as the required filtering column is present.

```googlesql
CREATE TEMP TABLE FUNCTION CustomerCreationTimeRange(
    min_creation_time INT64,
    max_creation_time INT64,
    SelectedCustomers ANY TABLE)
  AS
    SELECT *
    FROM SelectedCustomers
    WHERE creation_time >= min_creation_time
    AND creation_time <= max_creation_time;
```
