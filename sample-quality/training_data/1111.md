--- USER ---

How do I write a recursive query using the standard WITH RECURSIVE clause?

--- AGENT ---

The `WITH RECURSIVE` clause combines a base query with a recursive query using `UNION ALL`. The recursive query references the CTE by name and iterates until no new rows are produced.

```googlesql
-- Standard syntax
WITH RECURSIVE name AS (
  base_query
  UNION ALL
  recursive_query
)
final_query
```
