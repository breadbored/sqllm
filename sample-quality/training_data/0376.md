How do I combine protocol buffer values from a named column table with those from a value table using set operations?

--- AGENT ---

The `UNION ALL` operator can combine a table's protocol buffer column (cast to a value with `SELECT AS VALUE`) with a value table, provided both sides have matching types. The result is always a value table. This query merges AlbumReview values from two sources.

```googlesql
SELECT AS VALUE sa.AlbumReview FROM SingersAndAlbums AS sa
UNION ALL
SELECT a FROM AlbumReviewData AS a
```
